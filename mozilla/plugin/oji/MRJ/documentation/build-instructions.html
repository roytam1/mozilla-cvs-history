MRJ Plugin Build Instructions

In 4 Easy Steps

  1. Download the latest MRJSDK (MRJSDK 2.1 EA3 or later). Run the installer, which will create a folder at the root of your system volume, called "MRJ SDK 2.1." Copy the CIncludes and Libraries folders to the "MRJSDK" folder in mozilla/plugin/oji/MRJ.
  2. Build the project mozilla/plugin/oji/MRJ/plugin/MRJInABox.mcp.
  3. Build the project mozilla/js/macbuild/LiveConnectClasses.mcp, and make an alias to the file liveconnect.jar in mozilla/plugin/oji/MRJ/plugin.
  4. Build the project mozilla/plugin/oji/MRJ/plugin/MRJConsole.mcp. Build the project, then close it.

Well that's it. You should now have a working build of the MRJ plugin.
