Présentation d'AOL Instant Messenger pour Netscape

AOL Instant Messenger pour Netscape est une version du logiciel AOL Instant Messenger entièrement intégrée dans l'environnement du navigateur Netscape. Parfaitement compatible avec les autres versions d'AIM, elle constitue un moyen aisé de communiquer instantanément avec vos amis en ligne. Vous pouvez :

  • Envoyer des messages instantanés à un ami en ligne. Ces messages sont différents du courrier électronique, dans la mesure où, si votre ami a ouvert Instant Messenger sur sa machine, votre message apparaît immédiatement sur son écran.
  • Créer un répertoire de contacts AIM en ligne auxquels adresser des messages instantanés. Instant Messenger permet de localiser vos amis pour les ajouter à la liste.
  • Discuter avec des groupes d'amis et de la famille dans votre salon de discussion privé.
  • Contrôler qui communique avec vous à l'aide de la fonction de confidentialité.
  • Rechercher des contacts par adresse de courrier électronique.
  • Avertir les utilisateurs qui vous envoient des messages choquants ou ennuyeux.

Configuration

Netscape 7.0 permet d'utiliser un seul pseudonyme pour de nombreuses propriétés Netscape, dont AIM. Si vous avez exécuté la procédure d'enregistrement au démarrage de Netscape, vous disposez déjà d'un pseudonyme et d'un mot de passe. Aucune opération supplémentaire n'est donc nécessaire. Il vous suffit de sélectionner Instant Messenger dans le menu Fenêtre ou de cliquer sur l'onglet "Mes Contacts AIM" dans Ma barre latérale. Votre pseudonyme figure déjà dans le champ. Pour utiliser le produit, il vous reste à entrer le mot de passe utilisé lors de l'enregistrement.

Si vous n'avez pas effectué la procédure d'enregistrement, procédez comme suit :

  1. Ouvrez le menu Fenêtre et cliquez sur Instant Messenger.
  2. Cliquez sur Configurer.
  3. Cliquez sur "Je n'ai pas de pseudonyme. Je dois en enregistrer un nouveau.", puis cliquez sur Suivant.
  4. Dans l'écran qui s'affiche alors, entrez votre nom, qui est en fait le nom du profil utilisé pour ouvrir une session Netscape.
  5. Entrez le pseudonyme et le mot de passe de votre choix, puis cliquez sur Suivant.
  6. Dans l'écran qui s'affiche alors, entrez votre adresse électronique, votre date de naissance, votre sexe et votre code postal, puis cliquez sur Suivant.
  7. Lisez attentivement la politique de confidentialité de Netscape, cliquez sur "J'accepte" ou "Je refuse", puis cliquez sur Suivant.
  8. L'écran qui s'affiche vous indique si votre pseudonyme est disponible. S'il a déjà été choisi par une autre personne, le logiciel vous offre la possibilité de faire une nouvelle tentative ou d'en accepter un dans une liste de propositions apparentées à votre choix initial. Cliquez sur Terminer si votre pseudonyme a été accepté. Dans le cas contraire, entrez un nouveau pseudonyme ou faites votre choix dans la liste des suggestions et cliquez enfin sur Terminer.

Remarque : Si vous cliquez sur le bouton "Je refuse", vous mettez fin automatiquement à la procédure d'enregistrement.

Peu de temps après la fin de la procédure d'enregistrement, vous recevrez une confirmation par courrier électronique. Dans ce cas, il vous suffit simplement de cliquer sur Répondre et de taper OK dans la zone de composition du message. Vous recevrez alors un second courrier électronique détaillant toutes les utilisations possibles de votre pseudonyme.

Si vous souhaitez utiliser plusieurs pseudonymes, consultez la rubrique Enregistrement de nouveaux pseudonymes.

Enregistrement de nouveaux pseudonymes

Une fois enregistrés, vos pseudonyme et mot de passe ne peuvent plus être modifiés. Vous pouvez néanmoins enregistrer plusieurs pseudonymes à utiliser avec le service Instant Messenger.

Pour enregistrer et choisir un nouveau pseudonyme et un nouveau mot de passe :

  1. Si vous êtes connecté à AIM, déconnectez-vous.
  2. Cliquez sur Enregistrer un nouveau pseudonyme.
  3. Appliquez la même procédure que celle décrite dans la rubrique Configuration, en commençant au point 3.

Connexion à Instant Messenger

Avant d'utiliser Instant Messenger pour envoyer un message ou discuter, vous devez démarrer le programme et vous connecter.

Pour démarrer et vous connecter à Instant Messenger :

  1. Ouvrez le menu Fenêtre et sélectionnez Instant Messenger.
  2. Entrez votre pseudonyme dans le champ correspondant. Si vous possédez plusieurs pseudonymes, entrez celui de votre choix.
  3. Entrez votre mot de passe dans le champ correspondant. Pour enregistrer le mot de passe pour la prochaine session, sélectionnez "Enregistrer le mot de passe".
  4. Cliquez sur le bouton de connexion ou appuyez sur Entrée.

Astuce : pour que Instant Messenger démarre en même temps que Netscape, sélectionnez "S'identifier au lancement" dans l'écran de connexion.

Remarque : Si vous avez déjà utilisé AOL Instant Messenger seul ou à partir du client AOL et installé Netscape sur votre machine, il se peut que vous deviez redéfinir vos préférences. Certaines préférences, c'est le cas notamment des répertoires de contacts AIM, ne sont pas transférées automatiquement vers la version Netscape. Voir Définition des préférences d'Instant Messenger pour plus d'informations sur la redéfinition des préférences.

Lancement d'AOL Instant Messenger au démarrage du système

AOL Instant Messenger pour Netscape est configuré pour s'exécuter au démarrage du système. Si cela ne vous satisfait pas, vous pouvez désactiver cette option en procédant comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Cliquez sur Instant Messenger.
  3. Cliquez sur "Lancer automatiquement au démarrage du système" pour désactiver cette option.
  4. Cliquez sur OK.

Cette option est contrôlée par la préférence principale d'Instant Messenger. Cependant, elle reste active, que vous utilisiez AIM pour Netscape ou ICQ pour Netscape. Elle ouvre la fenêtre AIM (ou ICQ), mais n'ouvre pas de session automatiquement, sauf si vous avez sélectionné l'option "S'identifier au lancement" dans AIM ou "Connexion au démarrage " dans ICQ.

Remarque : La disponibilité de cette option dépend de la plate-forme utilisée. Elle n'est pas disponible dans les cas suivants : une autre version d'AOL Instant Messenger ou le client ICQ autonome est configuré pour s'exécuter au démarrage du système.

Pour plus d'informations sur les préférences d'Instant Messenger, consultez la rubrique Préférences d'Instant Messenger.

Changement de mot de passe

Vous pouvez changer de mot de passe lorsque vous êtes connecté à AOL Instant Messenger. Il doit comporter entre 4 et 16 caractères.

Pour changer de mot de passe, ouvrez la fenêtre de répertoire de contacts d'Instant Messenger :

  1. Ouvrez le menu Edition, choisissez Modifier les options, puis Changer le mot de passe. La boîte de dialogue Changer le mot de passe s'affiche.
  2. Entrez le mot de passe en cours dans le champ approprié et entrez deux fois le nouveau mot de passe dans les champs correspondants.
  3. Cliquez sur OK.

Remarque : Les personnes connaissant votre mot de passe ont accès à votre compte. Vous ne devez donc jamais révéler votre mot de passe à des utilisateurs en ligne.

Mise à jour de votre adresse électronique

Si vous modifiez votre adresse électronique, vous pouvez la mettre à jour à chaque fois que vous êtes connecté à AOL Instant Messenger.

Pour mettre à jour votre adresse électronique, ouvrez la fenêtre de répertoire de contacts d'Instant Messenger :

  1. Ouvrez le menu Edition, choisissez Modifier les options, puis Modifier une adresse e-mail.
  2. Entrez votre nouvelle adresse électronique.
  3. Cliquez sur OK.

Vous recevrez, très rapidement, une confirmation électronique à laquelle vous serez tenu de répondre. Aussitôt la réponse reçue, votre adresse électronique sera modifiée.

A propos du contacts AIM

Lorsque vous vous connectez à Instant Messenger, votre répertoire de contacts s'affiche. Des dossiers pour trois groupes sont automatiquement créés dans la fenêtre de répertoire de contacts : Contacts, Famille et Collègues. Ajoutez des contacts à l'un de ces groupes ou créez des groupes.

Remarque : Le répertoire de contacts s'affiche dans la fenêtre Instant Messenger proprement dite, ainsi que dans l'onglet "Mes Contacts AIM" de Ma barre latérale. Vous pouvez accéder à la plupart des fonctions d'Instant Messenger et les utiliser en cliquant sur l'onglet Buddy List de Ma barre latérale. Pour utiliser ses menus internes, vous devez cependant sélectionner Instant Messenger dans le menu Fenêtre.

Le Répertoire comprend deux onglets<:hs>: Répertoire et En ligne.

Dans l'onglet En ligne :

  • L'onglet En ligne vous indique quand vos contacts sont disponibles. Lorsque vos contacts se connectent, ils apparaissent dans votre répertoire de contacts AIM et vous pouvez leur envoyer immédiatement des messages.
  • Les groupes de la fenêtre de répertoire de contacts AIM sont initialement affichés ouverts, pour présenter tous les contacts qu'ils contiennent. Vous pouvez basculer entre les modes d'affichage ouvert et fermé en double-cliquant sur chaque nom de groupe ou en sélectionnant le nom du groupe et en appuyant sur Entrée.
  • Le nom de chaque groupe est suivi d'un nombre entre parenthèses indiquant combien de personnes du groupe sont actuellement en ligne. Par exemple, (3/9) signifie que trois des neuf contacts du groupe sont en ligne.
  • Les noms de contacts en gris et italique signalent ceux qui viennent de se déconnecter ou sont inactifs.

Dans l'onglet "Mes Contacts AIM" :

  • Chaque groupe apparaît sous la forme d'un dossier.
  • Les contacts de chaque groupe apparaissent sous la forme de branches du dossier. Vous devez développer le groupe pour voir les contacts qu'il contient. Pour développer le groupe, double-cliquez sur son nom.
  • Vous ne pouvez pas voir lequel de vos contacts est en ligne lorsque vous affichez l'onglet "Mes Contacts AIM". Cliquez sur En ligne pour voir quels contacts sont en ligne.
  • L'onglet "Mes Contacts AIM" permet également d'ajouter des contacts à la liste et de créer des groupes de contacts. Par exemple, vous pouvez créer des groupes pour des intérêts spécifiques tels que Amis golfeurs, Camarades d'école, Voisins, etc., et y ajouter vos contacts. L'onglet "Mes Contacts AIM" permet également de supprimer des contacts et des groupes.

Astuce : Les répertoires de contacts AIM sont stockés sur un serveur et non sur votre ordinateur. Vous pouvez ainsi y accéder où que vous soyez, chaque fois que vous vous connectez au service Instant Messenger.

Ajout et recherche de contacts

Si vous connaissez le pseudonyme d'un ami, vous pouvez l'ajouter immédiatement à votre répertoire de contacts AIM.

Affichez la fenêtre de répertoire de contacts AIM.

  1. Cliquez sur l'onglet "Mes Contacts AIM".
  2. Cliquez sur le bouton Nouveau contact (le bouton affiché peut être différent si vous utilisez un thème autre que Modern). La boîte de dialogue Nouveau contact s'affiche.
  3. Entrez le pseudonyme du contact à ajouter. Pour rappel, entrez uniquement son pseudonyme, pas son adresse électronique. Par exemple, jeandupont et non jeandupont@aol.com.
  4. Cliquez sur le groupe auquel ajouter le pseudonyme.
  5. Cliquez sur OK.

Si vous ne connaissez pas le pseudonyme d'un ami, AIM peut vous aider à le localiser à l'aide de son adresse électronique.

Affichez la fenêtre de répertoire de contacts AIM dans la version autonome d'AIM (et non la version de Ma barre latérale).

  1. Ouvrez le menu Contacts et sélectionnez Rechercher un contact.
  2. Entrez l'adresse électronique du contact dans la zone "Adresse électronique de l'ami", puis cliquez sur Suivant. Instant Messenger affiche tous les pseudonymes correspondant à l'adresse spécifiée.
  3. Sélectionnez le pseudonyme à ajouter à la liste et cliquez sur Nouveau contact.
  4. Dans la boîte de dialogue Nouveau contact, sélectionnez le groupe auquel ajouter le contact et cliquez sur OK.
  5. Cliquez sur Terminé lorsque vous avez fini.

Création de groupes de contacts

Vous pouvez regrouper vos contacts dans des dossiers pour les organiser. Votre contacts AIM comprend trois dossiers de groupes prédéfinis : Contacts, Famille et Collègues.

Pour créer un groupe, ouvrez la fenêtre de répertoire de contacts :

  1. Cliquez sur l'onglet "Mes Contacts AIM".
  2. Cliquez sur le bouton Nouveau groupe (le bouton affiché peut être différent si vous utilisez un thème autre que Modern). La boîte de dialogue Nouveau groupe s'affiche.
  3. Entrez le nom du groupe et cliquez sur OK.

Un dossier s'affiche pour le nouveau groupe. Pour ajouter des contacts au dossier, suivez les étapes de la section Ajout et recherche de contacts.

Pour ouvrir ou fermer le dossier d'un groupe, sélectionnez le nom du groupe afin de le mettre en surbrillance et appuyez ensuite sur Entrée.

Vérification de l'état des contacts

Pour savoir lequel de vos contacts est en ligne, cliquez sur l'onglet En ligne dans le répertoire de contacts AIM.

Vous ne pouvez voir lequel de vos contacts est en ligne lorsque l'onglet "Mes Contacts AIM" est affiché. Cliquez sur l'onglet En ligne.

Dans l'onglet En ligne :

  • Pour développer ou réduire un groupe, double-cliquez sur son nom.
  • Pour voir combien de membres d'un groupe sont en ligne, vérifiez le nombre entre parenthèses à côté du nom du groupe. Par exemple, (4/7) signifie que quatre des sept contacts du groupe sont actuellement en ligne.
  • Si le contact est en ligne, son nom s'affiche dans le répertoire de contacts AIM accompagné d'une icône de présence en ligne .
  • Les noms affichés en gris sont inactifs (la personne n'a pas utilise son ordinateur depuis 15 minutes mais est toujours en ligne) et une petite icône en forme d'horloge les accompagne. Cette icône s'affiche uniquement si votre contact a activé la fonction de signalisation d'inactivité. Pour plus d'informations, consultez la rubrique Désactivation de la fonction de signalisation d'inactivité.
  • Si le contact a activé un message d'absence, l'icône d'absence s'affiche à côté de son nom.

Vous pouvez également rechercher les contacts en ligne dans Netscape Messagerie & Forums. Si un contact est en ligne et que vous êtes connecté au service Instant Messenger, une icône de présence s'affiche à côté de son nom dans le champ A, De ou CC. L'icône apparaît également en regard du nom des non-contacts. Il s'agit d'utilisateurs d'Instant Messenger qui ne figurent pas dans votre répertoire de contacts AIM. S'agissant des contacts et des non-contacts, l'icône de présence ne s'affiche que pour les utilisateurs enregistrés et confirmés.

L'icône de présence à côté du nom d'un contact apparaît et disparaît dès que celui-ci se déconnecte.

Obtention d'informations sur les membres

Si vous êtes connecté à l'application AIM (et non au contacts AIM de Ma barre latérale), vous pouvez obtenir des informations sur n'importe quel contact. Pour ce faire<:hs>:

  1. Sélectionnez le contact sur lequel vous souhaitez obtenir des informations.
  2. Ouvrez le menu Contacts et sélectionnez Obtenir les infos du membre.

La fenêtre Infos sur le contact s'ouvre pour ce contact. Elle affiche la durée de connexion ou d'inactivité du contact, ainsi que le niveau de blâme qui lui est associé (si tant est qu'il y en ait un). Si le contact est absent et a choisi un message d'absence, celui-ci s'affiche. Si le contact n'est pas connecté, aucune information n'est disponible. De même, si le contact s'est connecté au moyen d'une autre version d'AOL Instant Messenger, il se peut que des informations supplémentaires soient affichées à son sujet.

Définition d'alertes de contact

Les alertes de contact vous indiquent lorsque l'un de vos contacts est en ligne, ou quitte l'état Absent ou Inactif. Pour définir des alertes de contact :

  1. Dans le répertoire de contacts de la version autonome d'AIM, sélectionnez le contact pour lequel vous souhaitez définir une alerte.
  2. Ouvrez le menu Contacts, puis sélectionnez Signaler la connexion de nom du contact. La boîte de dialogue Signaler la connexion d'un contact s'affiche.
  3. Cliquez sur l'une ou plusieurs des options suivantes : Me signaler lorsque le contact : se connecte, est de nouveau actif, est de nouveau présent.
  4. Cliquez sur Options : Afficher la fenêtre de signal de connexion ou Signal sonore.

Pour plus d'informations sur la notification sonore, consultez la rubrique Préférences Instant Messenger — Notification.

Importation et exportation de répertoires de contacts AIM

Si vous avez utilisé une version précédente d'AOL Instant Messenger, vous pouvez transférer votre répertoire de contacts du AIM.

  1. Lancez AOL Instant Messenger et connectez-vous.
  2. Ouvrez le menu Mon AIM et sélectionnez Enregistrer le contact AIM. Le fichier est enregistré avec l'extension .blt.
  3. Déconnectez-vous et quittez AOL Instant Messenger.
  4. Lancez Instant Messenger dans Netscape et connectez-vous.
  5. Ouvrez le menu Fichier et choisissez Importer un liste de contacts.
  6. Sélectionnez le fichier que vous venez d'enregistrer pour importer le liste de contacts.

Pour exporter le liste de contacts, ouvrez la fenêtre de contacts :

  1. Ouvrez le menu Fichier et choisissez Exporter un liste de contacts AIM.
  2. Entrez un nom pour la liste.
  3. Choisissez un emplacement d'enregistrement pratique et cliquez sur Enregistrer.

Désactivation de la fonction de signalisation d'inactivité

Si vous ne souhaitez pas que les autres utilisateurs sachent que vous avez été inactif, vous pouvez désactiver la fonction de signalisation d'inactivité en procédant comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Sous la catégorie Instant Messenger, cliquez sur Confidentialité. (Si aucune sous-catégorie n'est visible, double-cliquez sur Instant Messenger pour développer la catégorie.)
  3. Assurez-vous que l'option "Permettre aux utilisateurs de voir combien de temps j'ai été inactif" n'est pas sélectionnée.

Pour plus d'informations, consultez la rubrique Préférences d'Instant Messenger — Confidentialité.

Utilisation des icônes de contact

En tant qu'expéditeur IM, vous pouvez agrémenter vos messages instantanés (ou des messages envoyés par des contacts sélectionnés) d'icônes de contact ; c'est-à-dire de graphiques qui vous représentent vous, ou votre correspondant. En tant que destinataire, vous avez la possibilité d'accepter ou de refuser les icônes de contact. Il ne faut cependant pas confondre ce type d'opération avec le blocage d'un message. En effet, le message vous parviendra, mais l'icône de contact ne sera pas visible. Pour choisir votre icône de contact et définir les préférences qui y sont associées, vous devez accéder au panneau des préférences Icône de contact. Pour plus d'informations, consultez la rubrique Préférences d'Instant Messenger — Icônes de contact.

Envoi d'un message instantané

Procédez comme suit pour créer et envoyer un message instantané :

  1. Sélectionnez Instant Messenger dans le menu Fenêtre ou cliquez sur l'onglet "Mes Contacts AIM" dans Ma barre latérale. La fenêtre de contacts AIM s'affiche. Vous devez être connecté à AIM pour afficher cette fenêtre.
    • Pour envoyer un message instantané à un contact de votre répertoire dont le nom figure dans l'onglet En ligne, double-cliquez sur son nom. Une fenêtre s'ouvre, vous pouvez y taper votre message. Le champ A est automatiquement rempli.
    • Pour envoyer un message instantané à quelqu'un qui n'est pas en ligne ou ne figure pas dans votre liste de contacts, cliquez sur le bouton d'envoi (l'apparence du bouton peut être différente si vous utilisez un thème autre que Modern) dans l'onglet En ligne. Entrez le nom de l'utilisateur dans le champ A de la fenêtre Instant Message.
  2. Tapez votre message dans la zone de composition de la fenêtre Instant Message.
  3. Cliquez sur le bouton d'envoi (le bouton affiché peut être différent si vous utilisez un thème autre que Modern) ou appuyez sur Entrée pour envoyer le message.

Astuce : Sous Windows et Linux, vous pouvez cliquer avec le bouton droit sur un nom et choisir de lui envoyer un message instantané ou une invitation à discuter. Sous Mac, contrôle-cliquez.

Votre message s'affiche dans le volet supérieur de la fenêtre de message. La réponse de votre contact y apparaîtra également. Vous pouvez continuer à taper et envoyer des messages aussi longtemps que vous et votre contact le souhaitez.

Dans la section "Qui peut me contacter" de ce panneau, cliquez sur la case en regard des options proposées afin de définir vos préférences pour les options suivantes :

  • Autoriser uniquement les membres de mon liste de contacts : Sélectionnez cette option pour ne recevoir de messages que des membres de votre liste de contacts AIM.

  • Empêcher tous les utilisateurs : Sélectionnez cette option pour ne recevoir de messages de personne.
  • Bloquer ces utilisateurs : Sélectionnez cette option pour ne pas recevoir de messages de certaines personnes. Cliquez sur Ajouter et entrez le pseudonyme des personnes choisies.
  • Bloquer la réception des invitations : Cochez cette case pour empêcher l'affichage des messages à l'écran lorsque quelqu'un veut vous inviter à discuter.
  • Communiquer aux autres utilisateurs mon délai d'inactivité : Cochez cette case si vous souhaitez que les autres utilisateurs sachent quand vous vous absentez de votre poste de travail (inactif), ainsi que la durée d'inactivité.

  • Savoir que je suis enregistré à AIM : Choisissez cette option pour permettre aux utilisateurs qui connaissent votre adresse électronique de savoir uniquement que vous possédez un compte à l'aide de la fonction Rechercher un contact.

Préférences d'Instant Messenger — Icônes des contacts

Cette section décrit l'utilisation du panneau de préférences Icônes des contacts d'Instant Messenger. Si ce panneau n'est pas affiché, procédez comme suit :

  1. Connectez-vous à AIM. Vous devez être connecté pour être en mesure de modifier les préférences des icônes de contact.
  2. Dans le menu Edition, choisissez Préférences.
  3. Sous la catégorie Instant Messenger, cliquez sur Icônes des contacts. Si aucune sous-catégorie n'est visible, double-cliquez sur Instant Messenger pour développer la liste.

Le panneau de préférences Icônes des contacts vous permet de sélectionner une icône qui sera envoyée avec tous les messages instantanés. Il vous offre également la possibilité de choisir une icône pour un ou plusieurs amis de votre liste de contacts AIM. Enfin, il vous permet de configurer vos préférences en tant que destinataire.

Pour configurer vos préférences en matière d'icônes de contact en tant qu'expéditeur, procédez comme suit :

  1. Choisissez une icône de contact dans l'une des catégories affichées dans le panneau des préférences.
  2. Vous pouvez également parcourir votre disque dur à la recherche d'une image à utiliser.

Plusieurs formats d'image sont acceptés : .gif, .jpg, .bmp, .ico et .xbm. La taille maximale de l'image est limitée à 7 168 octets. Que vous choisissiez l'une des images prédéfinies ou optiez pour une image personnelle, le système procédera à son redimensionnement afin de la faire tenir dans la section supérieure gauche du panneau. Lorsque vous envoyez un message instantané, l'icône s'affiche dans la partie inférieure de la fenêtre de message.

Pour définir une icône de contact pour un membre de votre liste de contacts AIM, procédez comme suit :

  1. Sélectionnez le nom du contact pour lequel vous souhaitez définir une icône.
  2. Choisissez une icône de contact dans l'une des catégories affichées dans le panneau des préférences.
  3. Vous pouvez également parcourir votre disque dur à la recherche d'une image à utiliser.

Plusieurs formats d'image sont acceptés : .gif, .jpg, .bmp, .ico et .xbm. La taille maximale de l'image est limitée à 7 168 octets. Que vous choisissiez l'une des images prédéfinies ou optiez pour une image personnelle, le système procédera à son redimensionnement afin de la faire tenir dans la section supérieure gauche du panneau. Lorsque vous envoyez un message instantané, l'icône s'affiche dans la partie inférieure de la fenêtre de message.

Remarque : L'icône que vous avez attribuée au contact remplace toute icône que ce dernier aurait pu définir. Lorsque vous recevrez des images de ce contact, vous verrez s'afficher l'icône que vous lui avez affectée. Si le contact a défini une icône, celle-ci ne sera pas visible.

Pour configurer vos préférences en matière d'icônes de contact en tant que destinataire, procédez comme suit :

  • Dans la section Afficher les icônes des contacts, les options suivantes sont disponibles : Accepter des utilisateurs de la liste de contacts, Ne jamais accepter les icônes des contacts et Ne jamais afficher les icônes des contacts.
  • Vous pouvez également cocher la case Afficher une boîte de dialogue.

Si vous cochez cette case, une boîte de dialogue s'affichera à chaque fois qu'un contact vous enverra un message accompagné d'une icône. Dans ce cas, vous aurez la possibilité d'accepter ou de refuser l'icône au cas par cas.

Remarque : Le refus des icônes de contact et le blocage des utilisateurs sont deux notions bien distinctes. Ainsi, vous continuerez à recevoir les messages instantanés envoyés par les contacts non bloqués même si vous refusez toutes les icônes de contact. Les messages s'afficheront simplement sans icônes.

Préférences d'Instant Messenger – Connexion

Cette section décrit l'utilisation du panneau Connexion AIM. Si ce panneau n'est pas affiché, procédez comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Sous la catégorie Instant Messenger, cliquez sur Connexion. Si aucune sous-catégorie n'est visible, cliquez sur ICQ pour développer la liste.

Si vous accédez à Internet par le biais d'un fournisseur d'accès à Internet, vous ne devrez probablement pas modifier la boîte de dialogue de préférences de connexion. Si vous accédez à Internet au travers du pare-feu d'un réseau local (par exemple, sur votre lieu de travail), votre administrateur réseau peut devoir vous communiquer des informations afin que vous puissiez configurer la partie Serveur Proxy de la section Connexion.

Hôte

Ce champ contient l'adresse Internet du serveur de messagerie instantanée intégré de Netscape, lequel prend en charge AOL Instant Messenger pour Netscape et ICQ pour Netscape. Normalement, vous ne devez pas modifier cette zone. Si votre serveur proxy doit résoudre le nom d'hôte, il peut être nécessaire d'entrer l'adresse IP plutôt que le nom du serveur dans cette zone. Vous pouvez obtenir cette adresse IP auprès de votre administrateur réseau.

Port

Cette zone contient le numéro du port par défaut à utiliser sur le serveur de messagerie instantanée intégré de Netscape. Normalement, vous ne devez pas modifier cette zone.

Se connecter par proxy

Si vous cochez cette case, le logiciel Instant Messenger active les commandes du serveur proxy. Entrez les informations ci-dessous après les avoir demandées à votre administrateur réseau.

Hôte (Serveur Proxy)

Entrez le nom du serveur proxy défini par votre administrateur réseau.

Port (Serveur Proxy)

Entrez le numéro de port défini par votre administrateur réseau.

Nom d'utilisateur (Authentification)

Entrez le nom de l'utilisateur authentifié défini par votre administrateur réseau.

Mot de passe (Authentification)

Entrez le mot de passe d'authentification défini par votre administrateur réseau.

Protocole

Indiquez le protocole spécifié par votre administrateur réseau : SOCKS 4, SOCKS 5, HTTPS ou HTTP.
