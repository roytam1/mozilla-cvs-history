Création de pages Web avec Netscape Composer

Netscape Composer vous permet de créer vos propres pages Web et de les publier sur Internet. Pas besoin de connaître le langage HTML pour utiliser Composer : il est aussi facile à utiliser qu'un traitement de texte.

Les boutons des barres d'outils permettent d'ajouter des listes, tableaux, images, liens vers d'autres pages, couleurs et styles de police. Vous pouvez voir de quoi votre document aura l'air sur le Web à mesure que vous le créez et pouvez facilement le partager avec d'autres utilisateurs, quel que soit le type de navigateur ou le programme de messagerie compatible HTML qu'ils utilisent.

Pour commencer à utiliser Netscape Composer :

  • Cliquez sur l'icône Composer dans l'angle inférieur gauche d'une fenêtre Netscape.

Création d'une page

Netscape Composer est un éditeur HTML (Hypertext Markup Language) permettant de créer et de modifier des pages Web. Editeur WYSIWYG (What You See Is What You Get), Composer permet de voir exactement la façon dont la page sera affichée à mesure de sa création. Il n'est pas nécessaire de connaître le langage HTML, dans la mesure où la plupart des fonctions HTML sont disponibles sous la forme de commandes des barres d'outils et des menus.

Composer permet également de modifier la source HTML si vous le souhaitez. Pour afficher ou modifier le code source HTML, ouvrez le menu Afficher et sélectionnez Source HTML, ou cliquez sur l'onglet <HTML> source dans la barre d'outils de modification au bas de la fenêtre Composer.

Pour créer une page Web, procédez comme suit. Une fois que vous avez entamé une page, vous pouvez ajouter et modifier du texte tout comme dans un traitement de texte.

Pour créer une page à partir de Navigator :

  • Ouvrez le menu Fichier et choisissez Nouveau, puis Page de Composer. Une fenêtre Composer contenant une page vierge s'ouvre.

Pour modifier une page que vous consultez dans Navigator :

  • Dans la fenêtre Navigator de la page sélectionnée, ouvrez le menu Fichier et sélectionnez Modifier la page. Une fenêtre Composer contenant la page sélectionnée s'affiche.

Pour créer une page dans Composer :

  • Cliquez sur le bouton Nouveau dans la barre d'outils de Composer.

Pour démarrer à partir d'un fichier HTML stocké sur le lecteur local :

  1. Ouvrez le menu Fenêtre et sélectionnez Composer. La fenêtre Composer s'affiche.
  2. Ouvrez le menu Fichier et choisissez Ouvrir un fichier. La boîte de dialogue Ouvrir un fichier HTML s'affiche.
  3. Sur le lecteur local, trouvez le fichier à modifier.
  4. Cliquez sur Ouvrir pour afficher le fichier spécifié dans une fenêtre Composer.

Pour modifier une page Web :

  1. Ouvrez le menu Fenêtre et choisissez Navigator.
  2. Accédez à une page Web en tapant son URL (par exemple, http://home.netscape.com) dans la barre d'adresse et en appuyant sur Entrée (Retour sur Macintosh).
  3. Ouvrez le menu Fichier et sélectionnez Modifier la page.

Astuce : Dans la fenêtre Composer, vous pouvez ouvrir rapidement les derniers fichiers sur lesquels vous avez travaillé en ouvrant le menu Fichier, puis en sélectionnant Pages récentes, et en choisissant le fichier désiré dans la liste.

Enregistrement et navigation dans la nouvelle page

Vous pouvez enregistrer des documents Composer au format HTML ou texte uniquement. L'enregistrement d'un document au format HTML conserve son formatage, notamment les styles de texte (par exemple, gras ou italique), les tableaux, les liens et les images. L'enregistrement au format texte uniquement supprime toutes les étiquettes HTML mais conserve le texte.

Pour enregistrer un document au format HTML :

  • Ouvrez le menu Fichier et sélectionnez Enregistrer ou cliquez sur le bouton Enregistrer dans la barre d'outils Composition.

    Si vous n'avez pas encore attribué de titre à la page, Composer vous invite à le faire. Composer affiche le titre de la page dans la barre de titre de la fenêtre du navigateur lorsque vous l'y affichez. Le titre de la page apparaît également dans votre liste de signets, le cas échéant.

    Composer vous invite alors à entrer un nom de fichier et à spécifier son emplacement d'enregistrement. Veillez à conserver l'extension .html du nom de fichier.

Pour changer le nom de fichier ou l'emplacement d'un fichier HTML existant :

  • Choisissez Enregistrer sous et sélectionnez un autre nom de fichier ou emplacement.

Lorsque vous enregistrez une page dans Composer, tous les éléments de la page (le code HTML, les images et les autres fichiers, tels que les fichiers son et les feuilles de style) sont enregistrés localement sur votre disque dur. Si vous souhaitez uniquement enregistrer la partie HTML de la page, vous devez modifier la préférence de Composer relative à l'enregistrement des pages. Voir Préférences de Composer - Composer pour plus d'informations sur la modification du réglage de Composer relatif à l'enregistrement des pages.

Si une adresse d'image est absolue (commence par "http://") et que vous êtes connecté à Internet, vous verrez toujours l'image du document dans Composer et Navigator. Cependant, si l'emplacement de l'image est relatif à l'emplacement de la page (commence par "file:///"), vous ne la verrez pas dans la version locale du document.

Pour enregistrer un document au format texte uniquement :

  1. Ouvrez le menu Fichier et choisissez Exporter vers texte.
  2. Entrez le nom du fichier et son emplacement d'enregistrement.

Remarque : Les images ne s'affichent pas dans les documents enregistrés au format texte uniquement.

Astuce : Vous pouvez aussi choisir Revenir à la version enregistrée dans le menu Fichier pour récupérer la dernière version enregistrée du document sur lequel vous travaillez. N'oubliez pas que les modifications en cours seront perdues.

Pour afficher la page dans une fenêtre du navigateur afin de tester les liens :

  • Ouvrez le menu Fichier et sélectionnez Parcourir une page (ou cliquez sur Parcourir dans la barre d'outils Composition). Si vous n'avez pas encore enregistré le document, Composer vous invite à entrer un titre de page, un nom de fichier et un emplacement. La fenêtre Composer reste ouverte en arrière-plan de la nouvelle fenêtre Navigator.

Formatage de paragraphes, en-têtes et listes

Pour appliquer un format à un paragraphe, commencez dans la fenêtre Composer :

  1. Cliquez pour placer le curseur à l'endroit où vous souhaitez que le format commence ou sélectionnez le texte à formater.
  2. Choisissez un format de paragraphe à l'aide de la liste déroulante de la barre d'outils de formatage :
    • Texte du corps : Utilise la police et le style par défaut de l'application pour le texte normal, sans influencer l'espacement avant ou après.
    • Paragraphe : Insère une étiquette de paragraphe (à utiliser pour commencer un nouveau paragraphe). Le paragraphe comprend des marges supérieure et inférieure.
    • En-tête 1 - En-tête 6 : Formate le paragraphe comme un en-tête. En-tête 1 est l'en-tête de plus haut niveau ; En-tête 6 est l'en-tête de plus bas niveau.
    • Adresse : Ce format peut être utilisé pour une "signature" de page Web qui indique l'auteur de la page et la personne à contacter pour plus d'informations, comme dans l'exemple suivant : utilisateur@exemple.com

      Vous pouvez y inclure la date et une note de copyright. Ce format s'affiche généralement au bas de la page Web sous une ligne horizontale. Navigator affiche le format d'adresse en italique.

    • Préformaté : Ce format est utile pour les éléments tels que les exemples de code, les données en colonnes et les messages électroniques à afficher dans une police à chasse fixe. En texte normal, la plupart des navigateurs suppriment les espaces, tabulations et retours chariot superflus. Toutefois, le texte utilisant le style Préformaté s'affiche avec l'espacement intact, ce qui préserve la mise en page du texte original.

Pour formater du texte comme un en-tête :

  1. Cliquez pour placer le curseur à un endroit quelconque du texte à formater.
  2. A l'aide de la liste déroulante de la barre d'outils de formatage, sélectionnez le niveau d'en-tête de votre choix, de 1 (le plus grand) à 6 (le plus petit). Choisissez "En-tête 1" pour l'en-tête principal, "En-tête 2" pour l'en-tête suivant, etc.

Pour appliquer un format de liste :

  1. Cliquez pour placer le curseur dans la ligne de texte à formater.
  2. Ouvrez le menu Format et choisissez Liste.
  3. Choisissez le style de liste :
    • Puces : Chaque élément de la liste est accompagné d'une puce (point) (comme dans cette liste).
    • Numérotée : Les éléments sont numérotés.
    • Terme et Définition : Ces deux styles sont associés et créent une apparence de type glossaire. Utilisez l'étiquette Terme pour le mot à définir et le l'étiquette Définition pour la définition. Le texte Terme est aligné à gauche et le texte Définition s'affiche en retrait.

Astuce : Vous pouvez appliquer rapidement un style de liste à un bloc de texte en le sélectionnant et en cliquant sur le bouton Liste numérotée ou Liste à puces de la barre d'outils de formatage.

Pour modifier le style des puces ou numéros :

  1. Cliquez pour placer le curseur dans le texte de la liste à modifier ou sélectionnez un ou plusieurs éléments de la liste pour appliquer un nouveau style à l'ensemble de la liste.
  2. Ouvrez le menu Format et choisissez Propriétés de la liste.
  3. Sélectionnez un style de puce ou de numéro dans la liste déroulante. Pour les listes numérotées, vous pouvez également spécifier un numéro de début. Pour les listes à puces, vous pouvez modifier le style de puce.

Astuce : Vous pouvez également double-cliquer sur une puce ou un numéro dans une liste pour afficher la boîte de dialogue Propriétés de la liste.

Pour aligner un paragraphe ou un texte dans la page, par exemple le centrer ou l'aligner à gauche ou à droite :

  1. Cliquez pour placer le curseur dans le paragraphe ou la ligne de texte à aligner.
  2. Ouvrez le menu Format et choisissez Aligner, puis une option d'alignement.

Remarque : Vous pouvez également utiliser la barre d'outils de formatage pour aligner du texte.

Manipulation de listes

Pour mettre fin à une liste et continuer à taper le texte du corps :

  • Cliquez pour placer le curseur à la fin du dernier élément de la liste et appuyez deux fois sur Entrée (Retour sous Macintosh) pour terminer la liste.

Pour convertir un ou plusieurs éléments de liste en texte de corps :

  1. Cliquez pour placer le curseur dans l'élément de liste ou sélectionnez les éléments voulus.
  2. Cliquez sur le bouton de liste numérotée (pour une liste numérotée) ou sur le bouton de liste à puces (pour une liste à puces) dans la barre d'outils de formatage.

Pour placer du texte en retrait sous un élément de liste :

  1. Cliquez pour placer le curseur dans l'élément de liste.
  2. Appuyez sur Maj-Entrée pour créer le retrait en sommaire.
  3. Entrez le texte à mettre en retrait.
  4. Appuyez sur Maj-Entrée pour créer un autre paragraphe en retrait ou sur Entrée pour créer l'élément de liste suivant.

Astuce : Vous pouvez augmenter ou diminuer le retrait des éléments de liste en cliquant dans un de ces éléments puis sur le bouton Mettre le texte en retrait négatif (à gauche) ou Mettre le texte en retrait (à droite) dans la barre d'outils de formatage. Ou bien, cliquez dans un élément de liste et appuyez sur Tab pour augmenter le retrait d'un niveau. Appuyez sur Maj+Tab pour réduire le retrait d'un niveau.

Pour fusionner deux listes adjacentes :

  1. Sélectionnez les deux listes à fusionner. Veillez à sélectionner tous les éléments des deux listes. Notez qu'un texte placé éventuellement entre les deux listes sera intégré dans la liste fusionnée.
  2. Cliquez sur le bouton de liste à puces ou de liste numérotée dans la barre d'outils de formatage pour fusionner les listes.

Modification de la couleur, du style et de la police du texte

Pour modifier le style, la couleur ou la police du texte sélectionné :

  1. Sélectionnez le texte à formater.
  2. Ouvrez le menu Format et choisissez l'une des options suivantes :
    • Police : Permet de choisir une police. Pour utiliser les polices spécifiées par le navigateur de l'utilisateur, sélectionnez Vectorielle ou Chasse fixe.

      Remarque : Toutes les polices installées sur votre ordinateur ne sont pas affichées. Plutôt que de spécifier une police qui n'est peut-être pas disponible à toutes les personnes visitant votre page Web, il est généralement préférable de sélectionner l'une des polices du menu dans la mesure où elles sont prises en charge par tous les ordinateurs. Par exemple, les polices Helvetica, Arial, Times et Courier ont généralement une apparence identique sur tous les ordinateurs. Si vous en choisissez une autre, elle ne s'affichera peut-être pas de la même manière selon les systèmes.

    • Taille : Permet de sélectionner une taille de police relative ou de réduire ou diminuer la taille du texte (par rapport au texte avoisinant).
    • Style de texte : Permet de sélectionner un style, comme italique, gras ou souligné, ou d'appliquer un style structuré tel que Code.
    • Couleur de texte : Permet de sélectionner une couleur dans la palette. Si vous connaissez les codes de couleur hexadécimaux HTML, vous pouvez les entrer ou simplement saisir un nom de couleur en anglais (par exemple, "blue"). Découvrez un convertisseur de codes de couleur pratique en cliquant ici.

Pour changer la couleur de fond de la page :

  1. Cliquez à un endroit quelconque de la page.
  2. Cliquez sur le bloc de couleur de fond dans la barre d'outils de formatage.
  3. Choisissez une couleur de fond dans la boîte de dialogue Couleur d'arrière-plan de bloc.
  4. Cliquez sur OK.

Astuce : Pour adopter rapidement la dernière couleur de texte utilisée, sélectionnez le texte, puis appuyez sur Maj et cliquez sur le bloc de couleur de texte dans la barre d'outils de formatage. Cela peut être utile pour utiliser une couleur pour différentes lignes de texte.

Vous pouvez également utiliser une image comme fond. Voir Définition des couleurs et du fond de la page.

Suppression ou interruption de styles de texte

Pour supprimer tous les styles (gras, italique, etc.) du texte sélectionné :

  1. Sélectionnez le texte.
  2. Ouvrez le menu Format et choisissez Supprimer tous les styles.
  3. Continuez à taper.

Pour continuer à taper du texte avec tous les styles supprimés :

  1. Placez le curseur là où vous souhaitez supprimer les styles.
  2. Ouvrez le menu Format et choisissez Annuler tous les styles de texte.
  3. Continuez à taper.

Recherche et remplacement de texte

Pour rechercher du texte dans la page en cours :

  1. Cliquez pour placer le curseur à l'endroit où vous souhaitez commencer la recherche.
  2. Ouvrez le menu Edition et choisissez Rechercher et remplacer. La boîte de dialogue Recherche et remplacement s'affiche.
  3. Tapez le texte à rechercher dans le champ "Rechercher le texte". Pour limiter la recherche, sélectionnez une ou plusieurs des options suivantes :


    • Respecter la casse : Permet de spécifier si la recherche doit faire la distinction entre les minuscules et les majuscules.
    • Retour ligne auto : Utilisez cette recherche jusqu'à la fin de la page et recommencez depuis le début ou la fin, selon que vous recherchiez vers l'avant ou l'arrière.
    • En arrière : Permet d'effectuer une recherche depuis la position du curseur jusqu'au début de la page.
  4. Cliquez sur Rechercher suivant pour commencer. Lorsque Composer trouve la première occurrence du texte, sélectionnez Rechercher suivant pour rechercher l'occurrence suivante.
  5. Cliquez sur Fermer lorsque vous avez terminé.

Pour rechercher et remplacer du texte dans la page en cours :

  1. Cliquez pour placer le curseur à l'endroit où vous souhaitez commencer la recherche.
  2. Ouvrez le menu Edition et choisissez Rechercher et remplacer. La boîte de dialogue Recherche et remplacement s'affiche.
  3. Tapez le texte à rechercher, puis celui par lequel il doit être remplacé.
  4. Pour limiter la recherche, sélectionnez une ou plusieurs des options suivantes :
    • Respecter la casse : Permet de spécifier si la recherche doit faire la distinction entre les minuscules et les majuscules. Si vous ne sélectionnez pas cette option, la recherche trouvera tout texte correspondant, qu'il soit en majuscules ou en minuscules.
    • Retour ligne auto : Cette option permet de rechercher jusqu'à la fin de la page, puis de recommencer à partir du début.
    • En arrière : Cette option permet d'effectuer une recherche de la fin de la page jusqu'à son début.
  5. Cliquez sur Rechercher suivant pour rechercher l'occurrence suivante. Composer trouve l'occurrence suivante du texte.
  6. Cliquez sur Remplacer pour remplacer le texte sélectionné par le nouveau texte. Cliquez sur Remplacer et rechercher pour remplacer le texte sélectionné et trouver l'occurrence suivante. Cliquez sur Remplacer tout pour remplacer toutes les occurrences du texte dans le document.
  7. Cliquez sur Fermer lorsque vous avez terminé.

Insertion de lignes horizontales

Les lignes horizontales permettent généralement d'établir une séparation visuelle entre différentes sections d'un document. Pour insérer une ligne horizontale (également appelée filet) dans votre page, à partir de la fenêtre Composer :

  1. Placez le curseur à l'endroit où insérer la ligne.
  2. Dans le menu Insérer, sélectionnez Ligne horizontale.

 

Définition des propriétés de la ligne horizontale

Vous pouvez personnaliser la hauteur, la longueur, la largeur, l'alignement et l'ombre de la ligne.

  1. Double-cliquez sur la ligne pour afficher la boîte de dialogue Propriétés de la ligne horizontale.
  2. Vous pouvez modifier les propriétés suivantes :
    • Largeur : Entrez la largeur, puis choisissez "% de fenêtre" ou "pixels". Si vous spécifiez la largeur sous la forme d'un pourcentage, la largeur de la ligne change en fonction de celle de la fenêtre Composer ou du navigateur.
    • Hauteur : Entrez la hauteur de la ligne (en pixels).
    • Ombrage 3 D : Cette option ajoute de la profondeur à la ligne grâce à un ombrage en relief.
    • Alignement : Spécifiez où placer la ligne (gauche, centre ou droite).
  3. Cliquez sur Utiliser comme défaut pour utiliser ces paramètres par défaut lors de la prochaine insertion d'une ligne horizontale.
  4. Pour modifier manuellement les propriétés d'une ligne horizontale, cliquez sur Edition avancée. Voir la section Editeur de propriétés avancées pour plus de détails.

Astuce :Vous pouvez sélectionner "Afficher toutes les étiquettes" dans le menu Afficher pour afficher tous les éléments HTML dans des cadres jaunes. Cliquez sur un cadre jaune pour sélectionner tout le contenu de l'étiquette ou de l'élément HTML. Double-cliquez sur un cadre jaune pour afficher la boîte de dialogue Editeur de propriétés avancées pour l'étiquette ou l'élément HTML correspondant.

Insertion de caractères spéciaux

Pour insérer des caractères spéciaux tels que des caractères accentués, de copyright ou des symboles de devise :

  1. Placez le curseur à l'endroit où insérer le caractère spécial.
  2. Ouvrez le menu Insérer et choisissez Caractères et symboles. La boîte de dialogue Insérer des caractères s'affiche.
  3. Choisissez une catégorie de caractères.
    • Si vous choisissez Majuscules accentuées ou Minuscules accentuées, ouvrez la liste déroulante Lettre et sélectionnez la lettre à accentuer. Remarque : toutes les lettres n'ont pas de forme accentuée. Sélectionnez Symboles courants pour insérer des caractères spéciaux tels que le symbole de copyright ou les fractions.
  4. Dans la liste déroulante Caractères, sélectionnez le caractère à insérer.
  5. Cliquez sur Insérer.

    Vous pouvez continuer à rédiger votre document (ou à travailler dans une fenêtre de composition de message) en gardant cette boîte de dialogue ouverte, au cas où vous en auriez encore besoin.

  6. Cliquez sur Fermer lorsque vous avez fini d'insérer des caractères spéciaux.

Insertion d'éléments et d'attributs HTML

Si vous êtes familiarisé avec le code source HTML, vous pouvez insérer d'autres étiquettes, attributs de style et codes JavaScript dans votre page. Dans le cas contraire, il est préférable de ne pas les modifier. Pour travailler avec du code HTML, utilisez l'une des deux méthodes suivantes :

  • Placez le curseur là où vous souhaitez insérer le code HTML ou sélectionnez le texte à modifier, puis ouvrez le menu Insérer et choisissez HTML. Dans la boîte de dialogue Insérer HTML, entrez des étiquettes HTML et du texte, puis cliquez sur Insérer.
  • Sélectionnez un élément tel qu'un tableau, un marqueur nommé, une image, un lien ou une ligne horizontale. Double-cliquez sur l'élément pour ouvrir la boîte de dialogue Propriétés correspondante. Cliquez sur Edition avancée pour ouvrir l'Editeur de propriétés avancées. Vous pouvez utiliser l'Editeur de propriétés avancées pour ajouter des attributs HTML, des codes JavaScript et des styles CSS à des objets.
  • Ouvrez le menu Afficher, et sélectionnez Source HTML, ou cliquez sur l'onglet <HTML> source dans la barre d'outils de modification au bas de la fenêtre Composer. Si cette barre d'outils n'est pas affichée, ouvrez le menu Afficher et choisissez Afficher/Masquer, puis vérifiez que la barre d'outils de modification est sélectionnée.

 

Utilisation de l'Editeur de propriétés avancées

Pour ajouter des attributs HTML et des événements JavaScript à des objets tels que des tableaux, images et lignes horizontales, vous pouvez utiliser l'Editeur de propriétés avancées :

Remarque : A moins de comprendre parfaitement comment ajouter, supprimer ou modifier des attributs HTML et les valeurs associées, il est préférable de s'en abstenir.

Si la boîte de dialogue Editeur des propriétés avancées n'est pas affichée, procédez comme suit :

  1. Dans le menu Afficher (ou la barre d'outils de modification), sélectionnez Afficher toutes les étiquettes.
  2. Double-cliquez sur l'objet à modifier pour ouvrir la boîte de dialogue Propriétés correspondante.
  3. Cliquez sur Edition avancée pour ouvrir l'Editeur de propriétés avancées de l'objet. L'Editeur de propriétés avancées comprend trois onglets, affichant chacun les propriétés de l'objet sélectionné :
    • Attributs HTML : Cliquez sur cet onglet pour afficher ou entrer des attributs HTML.
    • Style incorporé : Cliquez sur cet onglet pour afficher ou entrer des propriétés CSS (cascading style sheet) à l'aide de l'attribut <style>. Pour plus d'informations sur l'utilisation des styles CSS dans Composer, voir Préférences de Composer - Composer.
    • Evénements JavaScript: Cliquez sur cet onglet pour afficher ou entrer des événements JavaScript.
  4. Pour modifier une propriété ou un attribut dans l'une des trois listes, sélectionnez-le. Vous pouvez alors modifier le nom ou la valeur de l'attribut à l'aide des champs modifiables Nom et Valeur au bas de la boîte de dialogue. Pour ajouter un attribut, tapez-le dans le champ Attribut au bas de la boîte de dialogue. Le nouvel attribut est automatiquement ajouté lorsque vous cliquez dans le champ Valeur. Pour supprimer un attribut, sélectionnez-le dans la liste et cliquez sur Supprimer l'attribut.

    Remarque : Les attributs obligatoires sont mis en surbrillance dans la liste.

  5. Cliquez sur OK pour appliquer les modifications de la boîte de dialogue Editeur de propriétés avancées.
  6. Cliquez de nouveau sur OK pour quitter la boîte de dialogue Propriétés.

Composer place automatiquement le texte d'attribut entre guillemets.

Vérification orthographique

Composer vérifie toujours l'orthographe depuis le début du document, quel que soit l'endroit où vous avez placé le curseur.

Pour vérifier l'orthographe de la page :

  1. Cliquez sur le bouton Orthographe de la barre d'outils ou choisissez Orthographe dans le menu Edition.

Validation du code HTML

Avant de placer votre document sur un serveur Web pour permettre à d'autres utilisateurs de le consulter, il convient d'en vérifier le formatage HTML pour s'assurer qu'il est conforme aux standards Web. Les documents contenant du code HTML validé sont moins susceptibles de causer des problèmes lors de leur consultation à l'aide de différents navigateurs. La simple vérification de vos pages Web dans Navigator ne garantit pas que votre document s'affichera correctement dans d'autres navigateurs Web.

Composer offre un moyen pratique de vérifier que votre document est conforme aux standards HTML W3C (World Wide Web Consortium). Composer utilise le service de validation HTML W3C, qui vérifie que la syntaxe HTML de votre document est conforme aux standards HTML 4.01. Ce service fournit également des informations sur la façon de corriger les erreurs.

Remarque : Il est indispensable d'être connecté à Internet pour utiliser cette fonction.

Pour valider la syntaxe HTML de votre document :

  1. Dans le menu Outils, choisissez Valider HTML. Si des modifications n'ont pas été enregistrées, Composer affiche un message vous invitant à les enregistrer avant de continuer.
  2. Lorsque la page du service de validation HTML W3C apparaît, cliquez sur Choisir et localisez le fichier que vous souhaitez valider sur votre disque dur.
  3. Cliquez sur "Valider ce document".

Il n'est généralement pas nécessaire de changer le mode d'édition par défaut (Normal). Toutefois, si vous souhaitez travailler avec le code source HTML du document, vous devrez changer de mode d'édition.

Composer permet d'alterner rapidement entre quatre modes d'édition ou vues. Chaque mode d'édition permet de continuer à travailler sur le document, en affichant différents niveaux d'étiquettes HTML (et d'icônes d'étiquettes).

Avant de sélectionner un mode d'édition :

  • Ouvrez le menu Afficher, choisissez Afficher/Masquer et vérifiez que l'option Barre d'outils de modification est sélectionnée.

La barre d'outils de modification comprend quatre onglets :

  • <HTML> source : Choisissez ce mode pour afficher et modifier le document sous forme de code source HTML non formaté. Lorsque vous enregistrez le document, le mode Normal réapparaît.

Remarque : Les fonctions JavaScript, cadres, liens et fichiers gif animés ne sont activés dans aucun des modes d'édition. Pour afficher ces éléments à l'état activé, cliquez sur le bouton Parcourir dans la barre d'outils de rédaction pour charger la page dans une fenêtre de navigateur.

Ajout et suppression de lignes, colonnes et cellules

Composer permet d'ajouter ou de supprimer rapidement une ou plusieurs cellules, colonnes ou lignes dans le tableau. En outre, vous pouvez définir des options permettant de conserver la structure rectangulaire originale ou la présentation du tableau pendant la modification.

Pour ajouter une cellule, une ligne ou une colonne au tableau :

  1. Cliquez dans le tableau à l'endroit où insérer la ou les cellules.
  2. Ouvrez le menu Tableau, puis choisissez Insérer.
  3. Sélectionnez l'un des groupes de cellules. Vous pouvez également insérer un nouveau tableau dans une cellule.

Pour supprimer une cellule, une ligne ou une colonne :

  1. Cliquez sur une ligne, une colonne ou une cellule pour placer le curseur. Vous pouvez également sélectionner des cellules voisines pour en supprimer plusieurs à la fois. Pour sélectionner des cellules voisines, faites glisser le curseur sur les cellules à sélectionner. Pour sélectionner des cellules individuelles, maintenez la touche Ctrl (Windows et Linux) ou Commande (Macintosh) enfoncée tout en cliquant sur les cellules à sélectionner.
  2. Ouvrez le menu Tableau et choisissez Supprimer.
  3. Cliquez sur l'événement à supprimer.

Pour fusionner une cellule avec celle située à sa droite :

  • Cliquez dans la cellule de gauche, ouvrez le menu Tableau et choisissez Fusionner à droite.

Pour fusionner des cellules adjacentes :

  • Sélectionnez des cellules adjacentes en faisant glisser le curseur dessus.
  • Ouvrez le menu Tableau et choisissez Fusionner les cellules sélectionnées.

Pour scinder une cellule fusionnée :

  • Cliquez dans la cellule fusionnée, ouvrez le menu Tableau, puis choisissez Scinder la cellule. Composer place tout le contenu de la cellule fusionnée dans la première des deux cellules.

Voir Sélection d'éléments de tableau pour plus d'informations sur la sélection de cellules, lignes et colonnes non adjacentes.

 

Modification du comportement de modification de tableaux par défaut

Par défaut, lorsque vous supprimez une ou plusieurs cellules, Composer préserve la structure du tableau en ajoutant des cellules en fin de ligne, au besoin. Vous pouvez ainsi supprimer une ou plusieurs cellules tout en conservant la structure rectangulaire originale ou la présentation du tableau. Dans le cas contraire, la suppression de cellules peut entraîner la formation d'espaces dans le tableau ou de contours irréguliers en raison d'un nombre irrégulier de cellules.

Pour modifier le comportement de modification de tableaux par défaut, à partir de la fenêtre Composer :

  1. Dans le menu Edition, choisissez Préférences, puis Composer.
  2. Sous Modification de tableaux, réglez la préférence suivante :
    • Veillez à cocher "Conserver la disposition du tableau lors de l'insertion ou de la suppression de cellules" pour que le tableau ne devienne pas irrégulier.
  3. Cliquez sur OK.

Voir également Définition des préférences générales de Composer.

      Remarque : Les images d'arrière-plan sont placées sous forme de mosaïque et remplacent les options de couleur d'arrière-plan.

    • L'URL est relative à l'emplacement de la page : Si cette option est sélectionnée, Composer convertit l'adresse URL de sorte qu'elle soit relative à l'emplacement de la page. Cela s'avère particulièrement utile si vous comptez publier vos pages sur un serveur Web pour que d'autres personnes puissent les consulter.

      Si des modifications n'ont pas été enregistrées, vous devez enregistrer la page avant de pouvoir cocher cette case.

Création de liens internes à la page

Pour créer un lien interne à la page, utilisable, par exemple, pour passer d'une section à une autre, vous devez créer un marqueur (emplacement de destination), puis un lien qui pointe vers lui. Ils sont également appelés marqueurs nommés.

    • Si vous créez un lien vers un en-tête de niveau (par exemple, En-tête 1 - En-tête 6), sélectionnez-le dans la liste des en-têtes de la page.
  3. Cliquez sur OK.

Remarque : Pour tester le lien créé, ouvrez le menu Fichier et choisissez Aperçu dans le navigateur, puis cliquez sur le lien.

Création d'un site de publication

Si vous prévoyez de publier des documents à plusieurs adresses distantes, vous pouvez configurer Composer pour qu'il enregistre les informations de publication correspondant à chaque site distant que vous utilisez, de manière à ne pas devoir les entrer chaque fois que vous souhaitez publier vos documents.

Pour créer un site de publication, à partir d'une fenêtre Composer :

  1. Dans le menu Edition, choisissez Paramètres du site de publication. Composer affiche la boîte de dialogue Paramètres de publication.
  2. Cliquez sur Nouveau site.
  3. Dans la zone "Nom du site", entrez le pseudonyme par lequel vous souhaitez désigner ce site de publication.

    Par exemple, si le nouveau site doit servir à publier des documents relatifs au projet "Météore", vous pouvez appeler le site "Météore". Le nom d'un site doit rappeler les types de documents que vous y publiez.

  4. Dans la zone "Adresse de publication", entrez l'adresse URL complète fournie par votre FAI, administrateur système ou service d'hébergement Web. Cette adresse doit commencer par ftp:// ou http://.

    L'adresse de publication spécifie l'emplacement où sont publiés les documents sur ce site. Si vous ne savez pas quelle adresse indiquer, contactez votre FAI ou votre administrateur système.

  5. Sous "Adresse HTTP de votre page d'accueil", entrez l'adresse URL complète à introduire dans Navigator pour visualiser les pages de ce site. Cette adresse ne doit pas inclure de nom de fichier ni de sous-répertoire.

    Elle doit toujours commencer par http://. Dans certains cas, cette adresse est la même que l'adresse de publication. Si vous ne savez pas quelle adresse indiquer, contactez votre FAI ou votre administrateur système ou laissez le champ vide.

  6. Comme nom d'utilisateur, entrez le nom d'utilisateur que vous employez pour vous connecter à votre FAI ou service d'hébergement Web.
  7. Comme mot de passe, entrez le mot de passe correspondant à votre nom d'utilisateur.
  8. Sélectionnez "Enregistrer le mot de passe" pour enregistrer votre mot de passe de manière sécurisée avec le Gestionnaire de mots de passe et vous éviter de devoir le retaper chaque fois que vous publiez des pages sur ce site.
  9. Cliquez sur OK.

Vérification de vos paramètres de publication

Pour vérifier vos paramètres de publication :

  1. Le cas échéant, fermez la boîte de dialogue Etat de publication.
  2. Dans le menu Edition, choisissez Paramètres du site de publication.
  3. Dans la boîte de dialogue Paramètres de publication, vérifiez que les paramètres du site sur lequel vous tentez de publier sont corrects. En cas de doute, consultez votre FAI ou service d'hébergement Web.
    • Vérifiez que vous avez correctement introduit les paramètres de publication. Vous avez peut-être mal tapé un des paramètres.
    • Vérifiez que vous avez introduit l'adresse de publication correcte. Les services d'hébergement Web ou FAI peuvent désigner l'adresse de publication par les termes "nom de serveur", "nom d'hôte" ou "serveur/hôte". Souvent, ils spécifient l'adresse de publication sous la forme ftp.monfai.com/utilisateur, où utilisateur est votre nom d'utilisateur.

      Pour que l'adresse de publication soit correcte, vous devez la faire précéder de ftp:// ou http://. Par exemple, l'adresse de publication correcte pour le site susmentionné serait ftp://ftp.monfai.com/utilisateur.


Vérification de vos noms de fichiers

Examinez le nom des fichiers dont la publication a échoué. Vérifiez les points suivants :

  • Les noms de fichiers ne peuvent utiliser que des chiffres ou des lettres minuscules ou majuscules. S'il est admis de créer des noms de fichiers utilisant des majuscules, vous éviterez des erreurs potentielles dans la localisation du fichier publié en n'utilisant que des minuscules dans les noms de fichiers.

    Lorsque vous publiez des fichiers sur un serveur Web, une distinction est faite entre les majuscules et les minuscules dans les noms de fichiers y résidant. Il peut être plus difficile de se rappeler les noms qui utilisent des majuscules uniquement ou en combinaison avec des minuscules.

    Par exemple, lorsque vous tentez de localiser un fichier publié en tapant son adresse Web dans le champ d'adresse de Navigator, vous devez entrer le nom de fichier exactement tel que vous l'avez créé, avec la même combinaison de majuscules et de minuscules.

  • N'utilisez pas de caractères de ponctuation ni d'espaces. Les traits de soulignement ( _ ) et tirets ( - ) sont admis.
  • Le nom de fichier doit se terminer par .html ou .htm (pour Composer).
  • Un nom de fichier est limité à 32 caractères.

Publier la page - Publier

L'onglet Publier la page - Publier permet de spécifier où vous souhaitez publier un document. Ces paramètres s'appliquent au document en cours.

Si l'onglet Publier la page - Publier n'est pas affiché, procédez comme suit :

  1. Dans le menu Fichier, choisissez Publier sous. La boîte de dialogue Publier la page s'affiche.
  2. Cliquez sur l'onglet Publier.
  • Nom du site : Enumère tous les sites de publication que vous avez créés, pour vous permettre de choisir le site où vous souhaitez publier. Pour créer un site, cliquez sur Nouveau site.
  • Titre de la page : Spécifie le titre de page du document tel qu'il apparaît dans la barre de titre de la fenêtre du navigateur lorsque vous l'y affichez. Le titre de la page apparaît également dans votre liste de signets, le cas échéant.
  • Nom de fichier : Spécifie le nom de fichier du document. Veillez à inclure l'extension .html ou .htm dans le nom du fichier.

  • Avertissement : Si le site distant où vous publiez contient un fichier de même nom que celui que vous téléchargez, ce dernier remplacera le fichier existant. Cet écrasement se fait sans demande de confirmation.

  • Sous-répertoire du site pour cette page : Si vous laissez ce champ vide, Composer publie la page dans le répertoire de publication principal (root) du site. Si vous souhaitez publier la page dans un sous-répertoire distant du répertoire de publication principal du site, entrez le nom du sous-répertoire ou choisissez-le dans la liste. Composer mémorise les emplacements que vous tapez ici, pour vous permettre de sélectionner un site parmi une liste des emplacements distants que vous avez utilisés précédemment. N'oubliez pas que les noms de sous-répertoires sont sensibles aux minuscules et majuscules.

  • Remarque : Le sous-répertoire que vous choisissez doit exister sur le serveur distant.

  • Inclure les images et les autres fichiers: Si cette option est cochée, Composer publie les images et autres fichiers référencés sur cette page. Vous pouvez choisir de publier ces fichiers à la même adresse que la page ou dans un sous-répertoire distant du répertoire de publication principal.

Astuce : Pour créer des sous-répertoires distants ou supprimer des pages ou images publiées, vous devez utiliser un programme FTP (File Transfer Protocol). Demandez à votre fournisseur d'accès s'il recommande un programme FTP en particulier. Vous trouverez généralement des informations sur les programmes FTP dans les sections d'aide ou d'assistance du site Web de votre FAI. Des programmes FTP sont également disponibles sur des sites de sharewares tels que ZDNet Downloads.

L'onglet Publier la page - Paramètres permet de spécifier les informations de connexion au site de publication distant, ainsi que les paramètres de publication de ce site. Ces paramètres s'appliquent au document courant ainsi qu'à tout autre fichier que vous publiez à cet emplacement.

Préférences de Composer - Composer

Les préférences de Composer permettent de spécifier des paramètres pour l'enregistrement de fichiers et la modification de tableaux. Ces paramètres s'appliquent à tous les documents créés.

Si les préférences de Composer ne sont pas encore affichées, procédez comme suit :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Cliquez sur la catégorie Composer.
  • Nombre maximum de pages listées : Spécifie le nombre maximal de pages à afficher sous l'option Pages récentes dans le menu Fichier.
  • Conserver le formatage source original : Cette option préserve les sauts de ligne et le formatage original de la page pour le code source HTML. Cochez cette case pour que Composer ne supprime pas les espaces (lignes supplémentaires, tabulations, etc.) qui rendent le code HTML plus lisible. Cette préférence n'influence pas l'affichage des pages dans une fenêtre de navigateur.
  • Reformater la source HTML : Cette option reformate le code source HTML pour le rendre plus lisible, en insérant des sauts de ligne et des retraits. Cette préférence n'influence pas l'affichage des pages dans une fenêtre de navigateur.
  • Enregistrer les images et les autres fichiers associés lors de l'enregistrement des pages : Si cette option est sélectionnée, toutes les images, les scripts JavaScript (JS), les styles CSS et autres fichiers associés sont enregistrés au même endroit que le document lorsque celui-ci est enregistré la première fois ou lorsqu'il est enregistré dans un nouvel emplacement. Sinon, seul le fichier HTML est enregistré.

  • Par exemple, lorsque vous modifiez une page distante, ce paramètre veille à ce que tous les fichiers associés à la page distante soient enregistrés localement au moment où vous enregistrez la page sur votre disque dur.

  • Toujours afficher la boîte de dialogue de publication lors de la publication de pages : Si cette option est sélectionnée, Composer affiche toujours la boîte de dialogue Publier la page lorsque vous cliquez sur le bouton Publier ou que vous choisissez Publier dans le menu Fichier. Sinon, Composer n'affiche la boîte de dialogue Publier la page que s'il a besoin de davantage d'informations pour publier la page.
  • Conserver la disposition du tableau lors de l'insertion ou de la suppression de cellules : Cochez cette case pour que Composer conserve toujours la disposition du tableau (c'est-à-dire une forme rectangulaire) en ajoutant des cellules au besoin. Si vous la désélectionnez, lorsque vous supprimez une ou plusieurs cellules, Composer en supprime également la bordure, ce qui peut entraîner l'affichage d'espaces vides ou un contour irrégulier en raison du nombre variable de cellules.
  • Utiliser des styles CSS plutôt que des éléments et des attributs HTML : Permet l'utilisation de styles CSS dans vos documents Composer. Lorsque cette préférence est sélectionnée, Composer génère un formatage HTML 4.01 avec des styles CSS intégrés.

  • Sinon, Composer génère un formatage HTML 4.01 mais n'utilise pas de styles CSS.

    Comparé à HTML, le code HTML avec un formatage CSS est plus portable, plus facile à mettre à jour et plus compatible avec différents navigateurs. Si vous activez cette préférence et modifiez un document créé sans CSS, Composer remplace les éléments modifiés par des styles CSS.

    Si vous activez les styles CSS, vous pouvez choisir une couleur de mise en surbrillance du texte sélectionné avec le bouton ad hoc de la barre d'outils de formatage. Vous pouvez également choisir un arrière-plan de couleur pour tout élément de la page. Ces fonctions ne sont pas disponibles si cette préférence n'est pas activée.
