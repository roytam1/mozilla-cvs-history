Préférences d'aspect - Aspect

Cette section décrit l'utilisation des panneaux de préférences Aspect. Si aucun de ces panneaux n'est affiché, procédez comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Cliquez sur la catégorie Aspect.


Le panneau de préférences Aspect permet de définir les composants à utiliser au démarrage de Netscape :

  • Au démarrage de Netscape 6, lancer : Sélectionnez les composants à utiliser au démarrage de Netscape.
  • Afficher les info-bulles : Cochez cette case pour que les info-bulles s'affichent lorsque vous placez le curseur sur certaines parties de l'interface utilisateur de Netscape.
  • Afficher les icônes de site Web : Cochez cette case si vous souhaitez voir les icônes spécifiques au site, s'il y en a, au lieu de l'icône de signet . Les icônes de sites Web sont utilisées à gauche de la barre d’adresse et dans la liste de signets.

 

Préférences d'aspect - Polices

Cette section décrit l'utilisation du panneau de préférences Polices. Si ce panneau n'est pas affiché, procédez comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Polices. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.


Le panneau de préférences Polices permet de définir le type et la taille de police de la page, ainsi que la résolution d'écran.

Remarque : Certains styles de polices peuvent ne pas être sélectionnables si la langue sélectionnée ne possède pas de polices pour ce style.

  • Sélectionner : Choisissez un jeu de caractères. Par exemple, pour définir les polices par défaut pour le jeu de caractères occidental (roman), sélectionnez "Occidental".
    • Proportionnelle : Sélectionnez si le texte proportionnel doit être de type serif (comme Times Roman) ou sans serif (comme Arial). Vous pouvez également attribuer une taille au texte proportionnel. La largeur du texte proportionnel est variable, de même que celle des caractères et des lettres
    • Serif : Sélectionnez une police serif à utiliser pour les pages Web.
    • Sans serif : Sélectionnez une police sans serif à utiliser pour les pages Web.
    • Cursive : Sélectionnez une police cursive à utiliser pour les pages Web.
    • Fantaisie : Sélectionnez une police de fantaisie à utiliser pour les pages Web.
    • Fixe : Sélectionnez une police fixe (comme Courier) et une taille à utiliser pour les pages Web. Le texte fixe a une largeur fixe, de sorte que chaque caractère ou lettre occupe le même espace.
    • Taille de police minimum : Sélectionnez la plus petite taille de police admissible sur les pages Web.
  • Autoriser les documents à utiliser d'autres polices : Si possible, cochez cette case pour conserver les paramètres de taille et de police d'une page Web au lieu de vos préférences.
  • Résolution d'affichage : Sélectionnez dans la liste déroulante la résolution d'affichage des pages Web en points par pouce (dpi). Sélectionnez "Autres" pour ouvrir la boîte de dialogue Etalonner la résolution, qui permet d'étalonner la résolution en mesurant la longueur d'une ligne à l'écran. Une résolution accrue améliorera la lisibilité du texte sur certains écrans.

 

Préférences d'aspect - Couleurs

Cette section décrit l'utilisation du panneau de préférences Couleurs. Si ce panneau n’est pas affiché, procédez comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Couleurs. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.


Le panneau de préférences Couleurs permet de définir les couleurs d’arrière-plan et de texte sur les pages Web :

  • Texte et arrière-plan : Cliquez sur les blocs de couleur pour sélectionner une couleur de texte et d'arrière-plan pour les pages Web.
    • Utiliser les couleurs système : Cochez cette case pour utiliser les paramètres de couleur du système.
  • Couleurs des liens : Cliquez sur les blocs de couleur d'affichage des liens visités et non visités sur les pages Web.
    • Liens soulignés : Cochez cette case pour afficher les liens soulignés sur les pages Web.
  • Lorsqu'une page Web fournit ses propres couleurs et arrière-plan :
    • Toujours utiliser les couleurs et l'arrière-plan de la page web : Permet à la page Web de choisir les couleurs et arrière-plans affichés.
    • Utiliser mes propres couleurs, ignorer les couleurs et l'image d'arrière-plan spécifiées : Permet de choisir les couleurs affichées, en ignorant les couleurs et l'image d'arrière-plan définies par la page Web.
  • Activer la mise en surbrillance de la syntaxe : Sélectionnez cette option pour afficher la syntaxe HTML de la source d'une page Web dans des couleurs spécifiques.

    Astuce : Pour afficher la source d'une page Web, ouvrez le menu Afficher de Navigator et choisissez Source de la page.

 

Préférences d'aspect - Thèmes

Cette section décrit l'utilisation du panneau de préférences Thèmes. Si ce panneau n’est pas affiché, procédez comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Thèmes. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.


Le panneau de préférences Thèmes permet de changer l'apparence et le style de Netscape à l'aide de différents thèmes.

  • Thème à utiliser : Choisissez un des thèmes installés pour changer l'apparence de Netscape.
    • Désinstaller [Nom du thème] : Cliquez pour supprimer un thème sélectionné.
  • Obtenir de nouveaux thèmes : Cliquez sur ce lien pour accéder à une page Web d'où vous pouvez télécharger d'autres thèmes.

    Remarques :

    • Pour voir un nouveau thème, vous devez quitter et redémarrer Netscape après son application.
    • Vous ne pourrez plus rétablir un thème désinstallé.

 

Préférences d'aspect - Langues/Contenu

Cette section décrit l'utilisation du panneau de préférences Modules de langue et de contenu. Si ce panneau n’est pas affiché, procédez comme suit :

  1. Dans le menu Edition, choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Langues/Contenu. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.


Le panneau de préférences Modules de langue et de contenu permet de sélectionner un module de contenu et/ou de langue à utiliser dans Netscape.

  • Eléments de contenu installés : Sélectionnez un élément de contenu à utiliser dans Netscape.
  • Modules de langue installés : Sélectionnez un module de langue à utiliser dans Netscape.
  • Télécharger plus : Permet de télécharger davantage de modules de contenu ou de langue.

    Remarque : Vous devez redémarrer Netscape pour qu'un nouveau module de contenu ou de langue entre en vigueur.
