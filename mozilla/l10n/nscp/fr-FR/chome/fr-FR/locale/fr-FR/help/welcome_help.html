Pour afficher des informations concernant Netscape 7.0 dans cette fenêtre,
cliquez sur les onglets ou sur les rubriques du volet gauche :

  • Sommaire présente les principales rubriques et sous-rubriques.
  • Rechercher permet d'effectuer une recherche dans l'aide intégrée de Netscape 7.0.
  • Index affiche la liste alphabétique des rubriques.
  • Glossaire définit des termes clés.

Pour plus d'informations sur l'utilisation de l'aide, voir Utilisation de la fenêtre d'aide.

Pour le système requis et les procédures d'installation, voir le fichier Lisez-moi de Netscape 7.0.
