Préférences de Navigator - Navigator

Cette section explique comment utiliser le panneau de préférences Navigator principal. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Cliquez sur la catégorie Navigator.


Le panneau de préférences Navigator permet de personnaliser trois aspects du navigateur.

  • Navigator démarre avec : Choisissez un des boutons suivant pour déterminer ce que Netscape affiche au démarrage :
    • Page vierge : Fait démarrer Navigator sans charger automatiquement une page Web.
    • Page d'accueil : Charge votre page d'accueil (spécifiée plus bas) à chaque démarrage.
    • Dernière page consultée : Charge la page qui était affichée lorsque vous avez quitté Netscape la dernière fois.
  • Navigateur par défaut : Permet de définir Netscape comme navigateur par défaut ou indique que c'est le cas.
    • Définir comme navigateur par défaut : Si elle est disponible, cliquez sur cette option pour faire de Netscape votre navigateur par défaut.
  • Page d'accueil : Entrez l'adresse de la page Web à choisir comme page d'accueil ou cliquez sur "Utiliser la page courante" pour choisir la page Web en cours dans Navigator. Cliquez sur "Choisir le fichier" pour rechercher sur le disque un fichier à charger comme page d'accueil.
  • Sélectionnez les boutons que vous souhaitez voir dans les barres d'outils : Cochez les cases disponibles de votre choix pour les afficher dans les barres d'outils.

    Les boutons Atteindre, Rechercher et Imprimer apparaissent dans la barre de navigation, près de l'angle supérieur droit de la fenêtre Navigator. Tous les autres boutons apparaissent dans la barre d'outils personnelle. Pour plus d'informations sur l'ajout de signets à cette barre d'outils, reportez-vous à Barre d'outils personnelle.

 

Préférences de Navigator - Historique

Cette section décrit l'utilisation du panneau de préférences Historique. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, cliquez sur Historique. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Le panneau de préférences Historique permet de configurer trois paramètres d'historique dans Navigator.

  • Historique de Navigation :
    • Mémoriser les pages visitées pour les derniers __ jours : Entrez le nombre de jours pendant lequel Netscape doit garder la trace des pages Web visitées. Par exemple, si vous réglez cette valeur sur 10 jours, les pages visitées les dix derniers jours seront conservées dans l'historique.
    • Effacer l'historique : Permet de supprimer la liste des sites visités.
  • Historique de la barre d'outils d'adresse :
    • Effacer la barre d'outils d'adresse : Permet d'effacer la liste des sites dans le menu de la barre d'adresses.
  • Historique de session :
    • Nombre de pages dans l'historique de session : Entrez le nombre de pages Web dont Netscape garde la trace via les boutons Précédent et Suivant.

Pour plus d'informations sur l'historique dans Netscape, reportez-vous à Retour en arrière.

 

Préférences de Navigator - Langues

Cette section décrit l'utilisation du panneau de préférences Langues. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, cliquez sur Langues. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Le panneau de préférences Langues permet de choisir les langues et le codage de caractères pour l'affichage des pages Web :

  • Langues pour les pages Web :
    • Monter / Descendre : Cliquez sur un de ces boutons pour déplacer une langue sélectionnée vers le haut ou vers le bas, afin de définir l'ordre de préférence des langues indiquées.
    • Ajouter : Permet d'ajouter d'autres langues pour l'affichage de pages Web.
    • Supprimer : Cliquez pour supprimer une langue sélectionnée.
  • Codage des caractères :
    • Codage de caractères par défaut : Utilisez la liste déroulante pour sélectionner le codage des caractères pour l'affichage de pages Web.

 

Préférences de Navigator - Applications auxiliaires

Cette section décrit l'utilisation du panneau de préférences Applications auxiliaires. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, cliquez sur Applications auxiliaires. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Le panneau de préférences Applications auxiliaires permet de configurer la façon dont différents types de fichiers sont ouverts par d'autres applications dans Netscape :

  • Spécifiez quelle application doit traiter chaque type de fichier :
    • Types de fichiers : Affiche les types de fichiers utilisés par Netscape. Sélectionnez un type pour afficher les informations suivantes :
      • Extension : Affiche l'extension de fichier correspondant au type de fichier sélectionné.
      • Type MIME : Affiche le type MIME du type de fichier sélectionné.
      • Géré par : Affiche le programme à utiliser pour ouvrir chaque fichier du type sélectionné.
    • Nouveau type : Permet d'ajouter un nouveau type de fichier. Tapez ou choisissez la description, l'extension de fichier, le type MIME et l'application.
    • Modifier : Permet de modifier les informations de traitement de fichiers pour un type de fichier sélectionné.
    • Supprimer : Permet de supprimer un type de fichier sélectionné.
  • Plug-in Finder Service : Sélectionnez cette option pour utiliser Plugin Finder Service (PFS) chaque fois que Netscape a besoin d'un nouveau module externe. Si cette option n'est pas sélectionnée, PFS n'est utilisé que lorsque l'emplacement d'un module externe n'est pas indiqué par la page Web qui exige ce module externe.
  • Ouverture de fichiers :
    • Réinitialiser : Cette option permet d'effacer toutes les préférences en matière d'ouverture de fichiers que vous avez éventuellement définies en désélectionnant la case à cocher "Toujours demander avant d'ouvrir ce type de fichier" dans la boîte de dialogue de téléchargement.

Pour plus d'informations sur le traitement de différents types de fichiers dans Netscape, reportez-vous à Gestion de différents types de fichiers.

 

Préférences de Navigator - Assistants de navigation

Cette section décrit l'utilisation du panneau de préférences Assistants de navigation. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, choisissez Assistants de navigation. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Le panneau de préférences Assistants de navigation permet de mieux naviguer dans les pages Web avec Internet Keywords et le remplissage automatique de la barre d'adresses.

  • Mots-clés Internet :
    • Activer Internet Keywords : Permet un accès rapide aux services tels que les cotations boursières, la recherche et d'autres informations de la barre d'adresses.
    • Plus d'informations : Permet d'obtenir davantage d'informations sur l'utilisation des mots-clés Internet.
  • Remplissage automatique de la barre d'adresses :
    • Remplir automatiquement le texte tapé dans la barre d'adresses : Permet de compléter automatiquement le texte précédemment entré dans la barre d'adresses.
    • Avancé : Cliquez sur cette option, puis sélectionnez une ou plusieurs des options suivantes :
      • Remplissage auto meilleure correspondance en cours de saisie : Lors de la saisie dans la barre d'adresses, Netscape complète automatiquement l'adresse Web en utilisant l'adresse d'un site Web visité dont la correspondance est la plus proche.
      • Afficher une liste de correspondances : Lors de la saisie dans la barre d'adresses, Netscape affiche une liste déroulante d'adresses de sites Web visités correspondantes.
      • Afficher le moteur de recherche Internet : Affiche une option de liste déroulante permettant de rechercher, à l'aide du moteur de recherche par défaut, des mots que vous saisissez .
      • Correspondance uniquement avec les sites Web que vous avez tapés précédemment : N'affiche que les sites Web dont l'adresse a été entrée dans la barre d'adresses, et non les sites ouverts d'une autre manière (p. ex., en cliquant sur un lien dans une page Web).

 

Préférences de Navigator - Recherche sur Internet

Cette section décrit l'utilisation du panneau de préférences Recherche sur Internet. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, cliquez sur Recherche sur Internet. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Le panneau de préférences Recherche sur Internet permet de définir la recherche à l'aide de Netscape :

  • Moteur de recherche par défaut :
    • Rechercher avec : Utilisez la liste déroulante pour sélectionner le moteur de recherche Web à utiliser.
  • Résultats de la recherche :
    • Ouvrir l'onglet Recherche de la barre latérale lorsque des résultats de recherche sont disponibles : Netscape ouvre la barre latérale et affiche les résultats de la recherche.
  • Préférence de l'onglet Recherche de Ma barre latérale :
    • De base : Utilise un seul moteur de recherche pour les recherches dans Netscape.
    • Avancé : Permet de choisir un ou plusieurs moteurs de recherche dans une liste lors d'une recherche dans Netscape.

 

Préférences de Navigator - Navigation par onglets

Cette section décrit l'utilisation du panneau de préférences Navigation par onglets. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, choisissez Navigation par onglets. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Ce panneau de préférences permet de configurer la navigation par onglets :

Remarque : Sur Mac OS, lorsque vous êtes invité à appuyer sur Control (Ctrl), appuyez sur Commande.

  • Affichage des onglets :
    • Masque la barre d'onglets lorsqu'un seul onglet est ouvert : Sélectionnez cette option pour n'afficher la barre Navigation par onglets que lorsque plusieurs onglets Navigator sont ouverts.
    • Charger les liens en arrière-plan : Sélectionnez cette option pour empêcher Netscape de basculer vers un nouvel onglet lorsque la fonction "Ouvrir dans un nouvel onglet" est utilisée pour ouvrir un lien.
  • Ouvrir des onglets plutôt que des fenêtres pour :
    • Clic du bouton central ou Ctrl-clic des liens dans une page Web : Sélectionnez cette option pour ouvrir des liens vers des pages Web dans un nouvel onglet lorsque vous cliquez sur un lien avec le bouton central de la souris ou appuyez sur Ctrl.
    • Ctrl+Entrée dans la barre d'adresse : Sélectionnez cette option pour ouvrir un lien de page Web dans un nouvel onglet lorsque vous tapez une adresse URL dans la barre d'adresses et appuyez sur Ctrl+Entrée.

     

Préférences de Navigator - Téléchargements

Cette section décrit l'utilisation du panneau de préférences Téléchargements. Si ce panneau n’est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Navigator, cliquez sur Télécharger. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.


Le panneau de préférences Téléchargements permet de configurer la façon dont Netscape traite les fichiers téléchargés à partir de pages Web :

  • Au démarrage d'un téléchargement :
    • Ouvrir le Gestionnaire de téléchargement : Ouvre le Gestionnaire de téléchargement, qui affiche l'état du téléchargement en cours et des téléchargements précédents. L'état de tous les téléchargements est donné dans une fenêtre unique.
    • Ouvrir une boîte de dialogue de progression : Affiche une boîte de dialogue de progression, indiquant l'état du téléchargement en cours. L'état de chaque téléchargement est donné dans une fenêtre séparée.
    • Ne rien ouvrir : Permet de télécharger les fichiers de manière invisible. Aucun état n'est donné pour tous les téléchargements.

Pour plus d'informations sur le téléchargement de fichiers à partir de pages Web, reportez-vous à Gestion de différents types de fichiers.
