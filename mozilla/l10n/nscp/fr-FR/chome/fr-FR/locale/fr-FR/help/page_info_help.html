Ce document est fourni par Nestcape pour information uniquement. Il peut vous aider à prendre certaines mesures destinées à protéger la confidentialité et la sécurité de vos informations personnelles sur Internet. Ce document ne couvre cependant pas tous les aspects de la confidentialité et de la sécurité en ligne, et ne constitue pas une recommandation de la part de Netscape concernant ce qui constitue une protection adéquate de la confidentialité et de la sécurité sur Internet.

Consultation d'informations sur la page

La boîte de dialogue Infos page se compose de plusieurs onglets qui affichent différents types d'informations sur la page en cours de consultation. Cette section vous donne un bref aperçu des informations disponibles dans chaque onglet.

Pour consulter les informations relatives à la page affichée par le navigateur, sélectionnez Infos page dans le menu Afficher. Vous pouvez ensuite cliquer sur l'onglet correspondant au type d'information que vous souhaitez visualiser.

La majorité des informations affichées dans les onglets Générales, Formulaires, Liens et Média de la boîte de dialogue Infos page sont extraites de la source HTML de la page. En règle générale, ces informations n'intéressent que les développeurs Web et d'autres spécialistes Internet.

Pour plus d'informations sur le langage HTML, y compris les balises affichées par la boîte de dialogue Infos page, consultez la Spécification HTML 4.01.

Les onglets Confidentialité et Sécurité contiennent des informations sur les politiques P3P (Platform for Privacy Preferences) du site Web dont vous consultez la page, ainsi que l'état de sécurité de cette page.

Générales (onglet Infos page)

Lorsque vous sélectionnez Infos page dans le menu Afficher, l'onglet Générales affiche des informations de base sur la page affichée dans le navigateur.

Dans la partie supérieure de la boîte de dialogue, sont affichés le nom de la page (si tant est qu'il y en ait un) et les informations suivantes :

  • URL :  Adresse URL (Uniform Resource Locator) de la page, c'est-à-dire l'adresse normalisée qui apparaît dans la barre d'adresses située en haut de la fenêtre du navigateur.
  • Type :  Type de média affiché par la page, tel que text/HTML, text/XML, audio ou application.
  • Mode de rendu :  Indique si le navigateur utilise le mode de compatibilité ou le mode de conformité aux standards pour le rendu de la page. Le mode de compatibilité tient compte du comportement non standard que peuvent présenter certaines pages Web plus anciennes, conçues pour des versions antérieures de navigateurs Web qui ne sont pas totalement conformes aux standards. Le mode de conformité aux standards adhère scrupuleusement aux spécifications standard. Votre navigateur choisit automatiquement le mode de rendu conformément aux informations contenues dans la page Web.
  • Source :  Indique si le code source de la page a été placé en mémoire cache.
  • Encodage :  Jeu de caractères utilisé pour ce document HTML.
  • Taille :  Taille du fichier, si cette information est disponible.
  • Modifié :  Date de la dernière modification de la page, si cette information est disponible.
  • Date d'expiration :  Date d'expiration des informations affichées par la page.

La partie inférieure de la boîte de dialogue affiche les métatags spécifiés par la page. Ces métatags fournissent des informations sur le type de contenu affiché par une page : description générale de la page, mots-clés pour les moteurs de recherche, informations sur les droits d'auteur, etc.

 

Formulaires (onglet Infos page)

Lorsque vous sélectionnez Infos page dans le menu Afficher, puis cliquez sur l'onglet Formulaires, vous accédez à des informations sur tous les formulaires affichés par la page affichée actuellement par le navigateur.

La partie supérieure de la boîte de dialogue affiche des informations de base sur la manière dont chaque formulaire est spécifié dans la source HTML:

  • Nom :  Nom du formulaire, le cas échéant.
  • Méthode :  Méthode HTML utilisée pour renvoyer au serveur Web les informations capturées par le formulaire. La commande GET ajoute les valeurs que vous avez renseignées à l'adresse du site Web auquel elle envoie le formulaire. La commande POST envoie les valeurs au site Web sous la forme de paramètres lisibles par un programme.
  • Action :  URL du programme à appeler lors de l'envoi du formulaire.

Lorsque vous sélectionnez un formulaire répertorié dans la partie supérieure de l'onglet Formulaires, la partie inférieure affiche des informations détaillées sur la manière dont chaque élément du formulaire est spécifié dans la source HTML :

  • Libellé : Libellé de l'élément (s'il est balisé comme tel dans le code HTML).
  • Nom du champ : Nom de l'élément.
  • Type : Type de saisie de l'élément, comme TEXT (pour l'envoi de texte), RADIO (pour une case d'option) ou encore HIDDEN (pour le stockage d'informations envoyées, mais non visibles à l'écran).
  • Valeur actuelle : Valeur actuelle de l'élément. Par exemple, dans le cas d'un élément de texte, la valeur actuelle est le texte qu'il contient actuellement.

 

Liens (onglet Infos page)

Lorsque vous sélectionnez Infos page dans le menu Afficher, puis cliquez sur l'onglet Liens, la boîte de dialogue affiche la liste de tous les liens disponibles sur cette page. Les informations suivantes sont affichées pour chaque lien :

  • Nom : Texte affiché dans le navigateur sous la forme d'un lien.
  • Adresse : URL de la page vers laquelle pointe le lien.
  • Type :  Type de lien, tel que Marqueur (pour un lien pointant vers un endroit précis dans un document HTML) ou Soumission de formulaire.

 

Média (onglet Infos page)

Lorsque vous sélectionnez Infos page dans le menu Afficher, puis cliquez sur l'onglet Média, la boîte de dialogue répertorie l'ensemble des images et des éléments multimédia affichés par la page.

La partie supérieure de la boîte de dialogue affiche des informations de base sur chaque élément, y compris son adresse (URL) et son type.

Lorsque vous sélectionnez un élément multimédia dans la partie supérieure de l'onglet Média, la partie inférieure affiche les informations disponibles à son sujet, telles que sa description, sa taille ou ses dimensions.

L'élément sélectionné s'affiche également au bas de cette boîte de dialogue. Pour visualiser des images plus grandes, vous devez cliquer sur l'angle inférieur droit de la boîte de dialogue Infos page et faire glisser le pointeur.

Pour enregistrer un élément multimédia en tant que fichier distinct :

  • Enregistrer sous : Sélectionnez l'élément de votre choix, cliquez sur Enregistrer sous, puis accédez à l'emplacement où vous souhaitez enregistrer l'élément.

 

Sécurité (onglet Infos page)

Lorsque vous choisissez Infos page dans le menu Afficher, puis cliquez sur l'onglet Sécurité, des informations s'affichent concernant l'authentification et du chiffrement de la page Web en cours de visualisation. La partie supérieure de la boîte de dialogue indique si l'identité du site Web a été vérifiée, tandis que la partie inférieure signale si la page a été chiffrée lorsqu'elle a été reçue par le navigateur.

Il est également possible d'accéder directement à l'onglet Sécurité en cliquant sur l'icône représentant un cadenas située dans le coin inférieur droit de toute fenêtre du navigateur.

La partie supérieure de l'onglet Sécurité peut afficher l'un des messages suivants :

  • Identité du site Web non vérifiée : Le site Web affiché n'a pas présenté de certificat pour s'authentifier. Le Gestionnaire de certificats ne peut donc vérifier son identité. Il est possible, bien qu'improbable, que le site Web ne soit pas ce qu'il prétend être.
  • Identité du site Web vérifiée : Le certificat utilisé par le Gestionnaire de certificats pour vérifier l'identité de ce site Web a été émis par une autorité de certification marquée comme fiable. Vous pouvez être assez certain que le site Web est bien ce qu'il prétend être.

La partie inférieure de l'onglet Sécurité peut afficher l'un des messages suivants :

  • Connexion non chiffrée : Il est possible, bien qu'improbable, que d'autres personnes visualisent les informations envoyées par votre ordinateur au site Web, ou inversement.
  • Connexion chiffrée : En règle générale, le niveau d'une connexion chiffrée dépend de la longueur des clés utilisées pour le chiffrement, mesurée en bits. Plus la clé est longue, plus le chiffrement est fort, c'est-à-dire plus il est difficile pour une personne non autorisée de décrypter les informations.

    La fenêtre Infos page distingue trois niveaux de chiffrement :

    • Chiffrement de qualité haute : Chiffrement le plus élevé disponible, utilisant au moins une clé de 128 bits.
    • Chiffrement de qualité moyenne : Un peu plus fort que le chiffrement de faible niveau, avec des clés de 56 ou 64 bits.
    • Chiffrement de qualité basse : Chiffrement le plus faible disponible, avec des clés de 40 bits.

    La plupart des sites Web prennent en charge le chiffrement de haut niveau. Si vous consultez un site Web plus ancien prenant en charge un niveau de chiffrement plus faible, il est possible que d'autres personnes puissent voir les informations échangées entre votre ordinateur et le site Web. Cependant, il est improbable que cela soit le cas.

L'absence de tout chiffrement, ou d'un haut niveau de chiffrement, ne doit vous inquiéter que si vous envoyez ou affichez des informations confidentielles, comme votre numéro de carte de crédit.

Vous pouvez connaître, en un clin d'œil, l'état de chiffrement d'une page Web grâce à l'icône représentant un cadenas située dans l'angle inférieur droit de la fenêtre du navigateur. Pour plus d'informations à ce sujet, consultez la rubrique Vérification de la sécurité d'une page Web.

 

Confidentialité (onglet Infos page)

Lorsque vous sélectionnez Infos page dans le menu Afficher et cliquez sur l'onglet Confidentialité, la boîte de dialogue affiche la liste des différents types de liens que contient la page, tels que les liens pointant vers d'autres pages, images ou scripts.

Vous pouvez utiliser cet onglet pour visualiser les politiques P3P (Platform for Privacy Preferences) du site Web dans son ensemble, ou d'autres sites vers lesquels pointent des éléments de la page. Ces politiques décrivent le type d'informations collectées par le site, les personnes auxquelles il les communique, ainsi que la façon dont elles sont utilisées.

Les sites Web publient les politiques de confidentialité sous une forme lisible par l'homme et sous la forme d'un fichier XML interprétable par le navigateur. Pour plus d'informations sur les politiques de confidentialité, consultez la rubrique Consultation des politiques de confidentialité.

Pour obtenir des informations de confidentialité sur n'importe quel site Web spécifié par un lien de la page, procédez comme suit :

  1. Double-cliquez sur la catégorie à laquelle appartiennent les liens sur lesquels vous souhaitez obtenir des informations de confidentialité.
  2. Cliquez sur le lien sur lequel vous souhaitez afficher des informations.
  3. Cliquez sur l'un des boutons ci-dessous pour afficher des informations sur la politique de confidentialité du lien sélectionné :
    • Politique : Cliquez sur ce bouton pour afficher la politique de confidentialité du site sous une forme lisible.
    • Résumé : Cliquez sur ce bouton pour afficher une interprétation lisible du fichier de politique XML du site, lequel définit avec force détails les politiques du site.
    • Options : Cliquez sur ce bouton pour afficher une page Web, fournie par le site, où vous pouvez choisir de donner ou non votre consentement à la collecte de certains types d'informations.

Pour afficher des informations de confidentialité sur le site Web dans son ensemble, cliquez sur l'adresse URL située en haut de la liste, puis cliquez sur l'un des boutons.
