Raccourcis clavier de Netscape

Pour bon nombre de commandes, Netscape vous permet d'utiliser le clavier plutôt que la souris. L'utilisation du clavier pour exécuter une action permet souvent de gagner un temps précieux. De plus, ces raccourcis peuvent également s'avérer très utiles pour les personnes présentant des troubles physiques, visuels ou auditifs.

Utilisation des raccourcis

Ce document utilise le format suivant pour dresser la liste des raccourcis :

Exemple :

Commande
Windows
Mac
Linux
Copier
Ctrl+C
Cmd+C
Ctrl+C

Pour exécuter une commande, appuyez simultanément sur les touches spécifiées. Par exemple, vous devez appuyer simultanément sur les touches Ctrl et C pour effectuer une copie dans Windows.

Astuce : Le signe plus (+) indique que vous devez appuyer simultanément sur les deux touches qui l'entourent. S'il y a plusieurs signes plus (exemple : Ctrl+Maj+C), cela signifie que vous devez appuyer simultanément sur les trois touches.

Abréviations :

  • Ctrl = Touche Contrôle
  • Cmd = Touche Commande sur un clavier Macintosh

Remarque : Certains raccourcis clavier permettent d'exécuter des fonctions différentes suivant l'emplacement du curseur (focus). Ainsi, si vous appuyez sur la touche Origine alors que vous visualisez une page Web, Netscape vous ramène au début de la page. En revanche, si vous appuyez sur cette même touche alors que vous êtes dans un champ texte, le curseur se place au début du champ.

Astuce : Sous Windows et Linux, vous pouvez utiliser le clavier pour accéder aux options de menu. Pour ouvrir un menu de niveau supérieur, il vous suffit d'appuyer simultanément sur la touche Alt et sur la lettre soulignée dans le nom du menu. Lorsqu'un menu est ouvert, vous pouvez ouvrir les sous-menus ou commandes sans appuyer sur la touche Alt. Par exemple, il vous suffit d'appuyer sur les touches Alt+F pour ouvrir le menu Fichier.
