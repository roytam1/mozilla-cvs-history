Navigation sur le Web

Bienvenue dans Netscape ! Netscape est l'un des outils les plus populaires pour naviguer sur le Web. Navigator, le composant Netscape qui vous permet de consulter des pages Web, vous offre une foule de fonctionnalités pour parcourir le Web et effectuer des recherches sur la Toile.

Cette section vous présente Navigator et vous explique comment l'utiliser pour consulter des pages Web, faire des recherches sur Internet et enregistrer des pages.

Affichage de la page d'accueil

Au démarrage de Netscape, Navigator, votre navigateur, s'affiche. Une page "Nouveautés" s'affiche automatiquement dans la fenêtre du navigateur lors du premier démarrage de Netscape.

Après le premier lancement, votre page d'accueil s'affiche normalement au démarrage de Netscape. A moins de sélectionner une page d'accueil vous-même, votre page d'accueil est choisie par votre administrateur réseau ou fournisseur d'accès à Internet, à moins que la page d'accueil de Netscape ne soit affichée.

Pour sélectionner votre propre page d'accueil, consultez la rubrique Spécification du démarrage de Netscape.

Astuces :

  • Pour optimiser l'interface de Netscape, vous pouvez utiliser le mode Plein écran afin d'afficher les pages sur pratiquement toute la surface de votre écran. Pour ce faire, ouvrez le menu Afficher de Navigator et choisissez Plein écran, ou appuyez sur la touche F11.
  • Pour accéder rapidement à votre page d'accueil, appuyez sur Alt+Origine (Cmd+Origine sous Mac OS).

Déplacement dans une autre page

Vous vous déplacez dans une autre page en entrant son URL (son adresse sur le Web). Les URL commencent généralement par "http://", suivie d'un ou plusieurs noms identifiant l'adresse. Par exemple, "http://netscape.com".

  1. Cliquez sur la barre d'adresses pour sélectionner l'URL qui s'y trouve.
  2. Entrez l'URL de la page à visiter. L'adresse URL entrée remplace le texte existant dans la barre d'adresses.
  3. Appuyez sur Entrée.

L'icône représentant un cadenas située dans le coin inférieur droit de la fenêtre vous permet de vérifier, à tout moment, l'état de sécurité d'une page Web. Pour plus d'informations à ce sujet, consultez la rubrique Vérification de la sécurité d'une page Web.

Astuce : Pour sélectionner rapidement l'URL dans la barre d'adresses, appuyez simultanément sur les touches Ctrl et L.

 

Barre d'adresses

 

Si vous ne connaissez pas l'URL… Vous pouvez entrer une partie d'une URL, comme "cnn" (for www.cnn.com) ou un terme générique, tel que "cadeaux" ou "fleurs". Navigator devine la page que vous souhaitez visiter ou affiche une page avec une sélection de liens relatifs au terme entré.

Si vous êtes un débutant sur Internet, consultez la rubrique Didacticiel de découverte d'Internet.

Activation d'un lien

La plupart des pages Web contiennent des liens sur lesquels vous pouvez cliquer pour accéder à d'autres pages.

  1. Déplacez le pointeur jusqu'à ce qu'il se transforme en main. Cela signifie que le pointeur est placé sur un lien. La plupart des liens sont du texte souligné, mais ils peuvent également se présenter sous la forme de boutons ou d'images.
  2. Cliquez une fois sur le lien. Pendant que le réseau recherche la page désignée par le lien, des messages d'état s'affichent au bas de la fenêtre.

Retour en arrière

Il existe plusieurs manières de revisiter des pages :

  • Pour passer à la page suivante ou revenir à la précédente, cliquez sur la flèche vers l'avant ou l'arrière.
  • Pour vous déplacer de plusieurs pages, cliquez sur les petits triangles des boutons fléchés. Vous verrez ainsi une liste des pages visitées ; pour revenir à une page, sélectionnez-la dans la liste.

 

Arrière   Avant

 

  • Pour afficher une liste des URL entrées dans la barre d'adresses, cliquez sur la flèche à l'extrême droite de cette dernière. Pour afficher une page, sélectionnez-la dans la liste.

 

Barre d'adresses

 

  • Pour sélectionner les pages visitées pendant la session en cours, ouvrez le menu Aller à et utilisez la liste de la section inférieure du menu.
  • Pour sélectionner les pages visitées au cours des sessions précédentes, ouvrez le menu Aller à, puis choisissez Historique. L'historique s'affiche. Cette fenêtre contient une liste de dossiers. Double-cliquer sur les dossiers affiche des sous-dossiers ou des signets de pages Web. Vous pouvez double-cliquer sur l'URL en regard de l'icône Signet pour afficher cette page.

Astuce : L'onglet Historique de Ma barre latérale permet également de choisir des pages que vous avez visitées au cours des quelques dernières sessions. Pour plus d'informations à ce sujet, consultez la rubrique Ajout d'onglets à Ma barre latérale.

 

A propos des historiques

L'historique contient des liens vers les pages visitées récemment. La liste de la barre d'adresses contient les adresses de pages que vous avez saisies dans la barre d'adresses, puis visitées.

Pour accéder à l'historique à partir de Navigator, sélectionnez le menu Aller à, puis choisissez Historique. Pour accéder à la liste de la barre d'adresses, cliquez sur la flèche située à l'extrême droite de cette barre.

Astuce : Pour accéder rapidement à l'historique, appuyez sur les touches Ctrl+H.

Si vous ne souhaitez pas que la barre d'adresses ou l'historique affiche les pages que vous avez visitées, vous pouvez effacer les historiques entièrement ou de manière sélective.

Pour supprimer toutes les pages de la barre d'adresses ou de l'historique à partir de la fenêtre Navigator :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Navigator, cliquez sur Historique. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.
  3. Cliquez sur Effacer l'historique et sur Effacer la barre d'outils d'adresse pour supprimer toutes les pages Web précédemment visitées des listes.
  4. Pour limiter le nombre de pages conservées dans l'historique, cliquez sur le champ en regard de "Nombre de pages dans l'historique de session" et entrez un nombre de pages.

Pour supprimer des pages de manière sélective de l'historique, effectuez l'une des opérations suivantes :

  • Pour supprimer toutes les pages d'un domaine, sélectionnez une page de ce domaine (dossier) dans l'historique, ouvrez le menu Edition et sélectionnez "Supprimer tout le domaine [nom de domaine]". Par exemple, utilisez cette commande pour supprimer toutes les pages qui se terminent par "netscape.com".
  • Pour supprimer des pages d'un sous-domaine, sélectionnez une page de ce sous-domaine dans l'historique, ouvrez le menu Edition et sélectionnez "Supprimer tout de [sous-domaine]". Par exemple, utilisez cette commande pour supprimer toutes les pages de qui se "health.netscape.com", mais pas "netscape.com".
  • Pour ne supprimer qu'une seule page, sélectionnez-la dans l'historique et appuyez sur Supprimer.

Astuce : Pour trier l'historique, cliquez sur l'une des catégories (Titre, Adresse ou Dernière consultation). Cliquez de nouveau sur le titre pour inverser l'ordre de tri.

Arrêt et rechargement

Si une page met trop de temps à s'afficher ou si vous avez changé d'avis et ne souhaitez plus la consulter, cliquez sur le bouton Arrêter.

Pour actualiser la page en cours ou en obtenir la dernière version, cliquez sur le bouton Recharger ou appuyez sur Ctrl+Maj+R (Cmd+Maj+R sous Mac OS).

Visite des pages référencées

Les adresses, ou URL, des pages Web peuvent être assez longues et difficiles à mémoriser. Il n'est heureusement pas nécessaire de mémoriser les URL pour naviguer sur le Web. Votre navigateur comprend une liste de signets, qui pointent vers des pages Web intéressantes.

Pour visiter une page référencée, ouvrez la fenêtre Navigator :

  1. Ouvrez le menu Signets. Il contient des signets représentés par une icône correspondante et des dossiers contenant d'autres signets.
  2. Pour visiter une page référencée, choisissez un signet dans le menu ou ouvrez un dossier et choisissez un signet.

Astuce : Pour revenir en arrière, cliquez sur la flèche Précédent.

Vous pouvez enregistrer des signets pointant vers vos pages préférées ou d'autres sites Web intéressants. Pour plus d'informations, consultez la rubrique Création de signets.

Navigation par onglets

Si vous consultez plusieurs pages Web à la fois, vous pouvez utiliser la fonction de navigation par onglets afin de naviguer de manière plus rapide et plus facile.

Cette option vous permet d'ouvrir plusieurs onglets, chacun contenant une page Web, dans une seule fenêtre Navigator. Il n'est donc pas nécessaire d'ouvrir plusieurs fenêtres pour visiter des pages Web différentes. Cela a pour effet de libérer de l'espace sur votre bureau. Vous pouvez ainsi ouvrir, fermer et recharger des pages Web en toute simplicité, sans devoir basculer vers une autre fenêtre.

Vous pouvez facilement gérer vos onglets de navigation et contrôler leur ouverture automatique. Pour plus d'informations sur la définition des préférences de la fonction Navigation par onglets, consultez la rubrique Préférences de Navigator – Navigation par onglets.

Pour en savoir davantage sur l'utilisation de la fonction Navigation par onglets, consultez la rubrique Navigation par onglets.

Utilisation de Ma barre latérale

Outre la navigation sur le Web avec Navigator, vous pouvez laisser le Web venir à vous avec Ma barre latérale.

Ma barre latérale est un cadre personnalisable de votre navigateur, permettant de conserver les éléments dont vous avez besoin en permanence (l'actualité et la météo, votre carnet d'adresses ou liste d'amis, les cours de la bourse, un calendrier) et comportant de nombreuses autres options disponibles. Elle vous présente ces éléments dans des onglets continuellement mis à jour.

Netscape est fourni avec quelques onglets prédéfinis dans Ma barre latérale, mais vous pouvez la personnaliser en ajoutant, supprimant et réorganisant des onglets. Pour plus d'informations, consultez la rubrique Ma barre latérale.

Pour afficher un élément dans Ma barre latérale, cliquez sur son onglet.

Poignée de Ma barre latérale

Si ce n'est déjà fait, ouvrez Ma barre latérale en cliquant sur sa poignée. Si la poignée n'apparaît pas, ouvrez le menu Afficher de Navigator, choisissez Afficher/Masquer, puis cliquez sur Ma barre latérale dans le sous-menu.

Recherches rapides

Il existe trois méthodes de recherche rapide : à partir de la barre d'adresses, de Ma barre latérale et de la page de recherche de Netscape.

 

Recherche à partir de la barre d'adresses

Rechercher des pages Web sur un sujet particulier est aussi facile que taper une question, ou seulement un mot ou deux, dans la barre d'adresses, comme illustré ci-dessous.

Par exemple, si vous recherchez des informations sur les poupées :

  1. Double-cliquez dans la barre d'adresses pour sélectionner le texte en cours.
  2. Tapez le mot "poupées". Il remplace le texte en cours.
  3. Effectuez l'une des opérations suivantes :
  • Cliquez sur le bouton Rechercher.
  • Cliquez sur "Recherche - Netscape Recherche de "poupée" au bas de la liste déroulante dans la barre d'adresses. Votre moteur de recherche par défaut peut être différent.
    Le moteur de recherche par défaut que vous choisissez dans Préférences est utilisé. Les résultats de la recherche sur "poupées" s'affichent dans la fenêtre du navigateur et dans Ma barre latérale. Cliquez sur les liens pour visiter les pages Web sur les poupées.

Après une recherche, les résultats sont enregistrés dans l'onglet Recherche de Ma barre latérale jusqu'à la prochaine recherche. Vous ne devez pas cliquer sur le bouton Précédent pour récupérer les résultats de la recherche.

Remarque : Vous pouvez également utiliser des Mots clés Internet pour effectuer une recherche à partir de la barre d'adresses.

 

Recherche à partir de Ma barre latérale

L'onglet Recherche de Ma barre latérale permet d'effectuer des recherches rapides et d'en enregistrer les résultats dans des signets. Par exemple, si vous recherchez des informations sur les voitures miniatures :

  1. Si ce n'est déjà fait, ouvrez Ma barre latérale en cliquant sur sa poignée.
  2. Cliquez sur l'onglet Recherche.
  3. Ouvrez la liste déroulante "dans" et choisissez un moteur de recherche.
  4. Entrez "voiture miniature" dans le champ de recherche.
  5. Cliquez sur Rechercher. Les résultats de la recherche sur "voiture miniature" s'affichent dans la fenêtre du navigateur et dans Ma barre latérale. Cliquez sur les liens pour visiter des pages Web sur les voitures miniatures.

Remarque : Si l'onglet Recherche de Ma barre latérale contient une liste de moteurs de recherche, cela signifie qu'il est configuré en mode recherche avancée. Vous devrez donc choisir un ou plusieurs moteurs de recherche avant de cliquer sur le bouton Rechercher. Vous pouvez conserver ce mode ou passer à un réglage de base dans les Préférences.

Astuce : Pour afficher rapidement la page précédente ou suivante des résultats de la recherche, cliquez sur le bouton Précédent ou Suivant au bas de l'onglet Recherche de Ma barre latérale.

 

Recherche à partir de la page de recherche de Netscape

La page de recherche de Netscape permet de rechercher des mots ou d'explorer des rubriques (telles que Arts et culture, Finance, etc.) susceptibles de vous intéresser. Pour visitez la page Net Search, procédez comme suit :

  • Cliquez sur le bouton Recherche de la barre d'outils personnelle.
  • Ouvrez le menu Outils et choisissez Rechercher sur Internet.

 

Recherche de mots dans une page Web

Netscape vous permet d'effectuer une recherche sur des mots sélectionnés dans une page Web :

  1. Sélectionnez des mots dans une page Web.
  2. Cliquez sur le bouton droit de la souris, puis choisissez "Rechercher [mots sélectionnés] sur le Web" dans le menu contextuel. Pour simuler un clic droit sous Mac OS, appuyez sur Ctrl et cliquez sur le bouton de la souris.

Netscape ouvre une nouvelle fenêtre et utilise votre moteur de recherche par défaut pour rechercher les mots que vous avez sélectionnés. Pour savoir comment changer le moteur utilisé pour rechercher les mots que vous avez sélectionnés, consultez la rubrique Préférences de Navigator – Recherche sur Internet.

[ Retour au début de la section ]

 

Mode de recherche avancée

Vous pouvez régler l'onglet Recherche de Ma barre latérale en mode avancé, ce qui permet une recherche plus ciblée ou l'utilisation d'un ou plusieurs moteurs de recherche.

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Navigator, cliquez sur Recherche sur Internet. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.
  3. Dans la section Préférences de l'onglet Recherche de la barre latérale, cliquez sur Avancé, puis sur OK.
  4. Si ce n'est déjà fait, ouvrez Ma barre latérale en cliquant sur sa poignée.
  5. Cliquez sur l'onglet Recherche de Ma barre latérale.
  6. Ouvrez la liste déroulante "dans" et choisissez une catégorie.
  7. Sélectionnez un ou plusieurs moteurs de recherche pour la catégorie sélectionnée (si cette option est disponible).
  8. Dans la zone de recherche, entrez le mot à rechercher.
  9. Cliquez sur Rechercher.

 

 

Personnalisation des catégories de recherche

Vous pouvez spécifier les moteurs de recherche à utiliser selon les catégories. Par exemple, vous pouvez sélectionner un ensemble de moteurs recherche pour rechercher des voyages et un autre pour rechercher des logiciels.

  1. Configurez le mode de recherche Avancé de Ma barre latérale. Voir Mode de recherche avancé pour plus de détails.
  2. Si ce n'est déjà fait, ouvrez Ma barre latérale en cliquant sur sa poignée.
  3. Cliquez sur l'onglet Recherche de Ma barre latérale.
  4. Ouvrez la liste déroulante "dans", cliquez sur "Modifier les catégories". La boîte de dialogue Modifier les catégories s'affiche.
  5. Dans la liste déroulante Catégories, sélectionnez la catégorie à personnaliser.
    Ou
    Pour créer une catégorie, cliquez sur Nouveau et entrez un nom dans la boîte de dialogue. La liste déroulante affiche à présent le nom de la catégorie sélectionnée.
  6. Effectuez l'une des opérations suivantes :
    • Pour ajouter un moteur de recherche pour la catégorie sélectionnée, cliquez sur le moteur de recherche dans la liste de gauche, puis sur Ajouter.
    • Pour supprimer un moteur de recherche pour la catégorie sélectionnée, sélectionnez-le dans la liste de droite, puis cliquez sur Supprimer.
    • Pour renommer la catégorie sélectionnée, cliquez sur Renommer et entrez un autre nom.
    • Pour supprimer la catégorie sélectionnée, cliquez sur Supprimer.
  7. Cliquez sur OK une fois que vous avez terminé la personnalisation.

Définition des préférences de recherche

Lorsque vous cliquez sur Rechercher, c'est le moteur de recherche Netscape qui est utilisé par défaut. Vous pouvez choisir un moteur de recherche par défaut différent. Vous pouvez également définir l'affichage des résultats de recherche dans Ma barre latérale.

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Navigator, cliquez sur Recherche sur Internet. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.
  3. Dans Moteur de recherche par défaut, choisissez un moteur pour les recherches sur le Web.
  4. Dans Résultats de la recherche, sélectionnez "Ouvrir l'onglet Ma barre latérale de recherche Internet lorsque des résultats de recherche sont disponibles" si vous souhaitez que l'onglet de recherche de Ma barre latérale s'ouvre et présente vos résultats de recherche.
  5. Dans Préférences de l'onglet Search de Ma barre latérale, sélectionnez De base si vous ne souhaitez rechercher qu'avec un seul moteur ou sélectionnez Avancé pour utiliser plusieurs moteurs de recherche.

Utilisation de mots-clés Internet

Les mots-clés Internet peuvent être utilisés avec la barre d'adresses pour vous aider à trouver rapidement des types d'information spécifiques. Vous pouvez utiliser les mots-clés Internet de diverses manières :

  • Entrez un produit, une marque ou un nom de société spécifique et appuyez sur Entrée (Retour sous Mac OS), pour vous connecter directement à son site Web.
  • Entrez certains verbes, comme "acheter" ou "offrir", suivis du mot à rechercher, pour obtenir des informations spécifiques. Par exemple, "acheter un animal domestique" permet d'accéder à une page comportant des liens vers des magasins d'animaux en ligne.
  • Entrez le nom d'une ville suivi de certains mots tels que "cinémas" ou "restaurants" pour des informations locales. Par exemple, entrez "San Francisco restaurants".

Conseil : Les mots-clés Internet les plus courants figurent dans la liste déroulante à gauche de la barre d'adresses. Cliquez sur le triangle et choisissez dans la liste. Pour afficher une liste plus complète de mots-clés, sélectionnez Liste de mots clés.

Important : Pour utiliser les mots-clés Internet, vous devez appuyer sur Entrée sur le clavier (Retour sous Mac OS) au lieu de cliquer sur le bouton Rechercher. La touche Entrée lance la fonction mots clés Internet ; cliquer sur Rechercher lance une recherche.

Pour voir la différence, procédez comme suit :

  1. Entrez "apple imac" dans la barre d'adresses et appuyez sur Entrée (Retour). La page Web de l'iMac d'Apple Computer s'affiche.
  2. Entrez maintenant "apple imac" dans la barre d'adresses et cliquez sur Rechercher. Une liste de liens liés aux modèles iMac d'Apple Computer.

Si l'entrée de mots-clés Internet dans la barre d'adresses ne fonctionne pas bien, vérifiez que la fonction Mots-clés Internet est activée dans vos préférences Assistants de navigation.

 

Cotation rapide en bourse

Pour voir la cotation en bourse récente d'une société, entrez "quote", suivi du symbole boursier de la société, et appuyez sur Entrée.

Si vous ignorez le symbole boursier d'une société, entrez son nom. La page Quote (Bourse) affiche toutes les sociétés au nom similaire, parmi lesquelles vous pouvez choisir celle que vous désirez.

Recherche dans une page

Pour rechercher du texte dans la page en cours de consultation dans Navigator :

  1. Ouvrez le menu Edition, choisissez "Rechercher sur la page". Si la page en cours contient des cadres, il peut être nécessaire de cliquer dans l'un d'eux pour entamer la recherche. La boîte de dialogue "Rechercher sur cette page" s'affiche.
  2. Entrez le texte à rechercher.
  3. Choisissez parmi les options suivantes :
    • Respecter la casse : Cochez cette case pour rechercher la correspondance exacte d'un mot (en respectant les majuscules et les minuscules).
    • Retour ligne auto : Cochez cette case pour permettre au navigateur de rechercher à partir du point d'insertion du curseur au bas de la page et de recommencer ensuite la recherche à partir du haut de la page. Remarque : Normalement, la recherche s'effectue à partir de l'endroit où vous avez cliqué en dernier jusqu'au bas de la page.
    • En arrière : Choisissez cette option pour effectuer une recherche de la position du curseur en arrière jusqu'au début de la page.
  4. Cliquez sur Rechercher pour commencer.

Pour rechercher de nouveau le même mot ou la même phrase :

  • Ouvrez le menu Edition et choisissez Rechercher.

Recherche dans les signets ou l'historique

Pour effectuer une recherche dans la liste des signets, ouvrez la fenêtre Navigator :

  1. Ouvrez le menu Signets et choisissez Gérer les signets. La fenêtre des signets s'affiche.
  2. Dans la fenêtre Signets, ouvrez le menu Outils et choisissez Rechercher des signets. La boîte de dialogue Rechercher les signets s'affiche.
  3. Utilisez les listes déroulantes pour sélectionner des options pour définir votre recherche.
  4. Entrez un mot à rechercher.
  5. Cliquez sur Rechercher. Les signets qui correspondent à votre critère de recherche s'affichent dans la fenêtre Résultats de la recherche – Signets.

Astuce : Pour ouvrir rapidement le menu Signets, appuyez sur les touches Ctrl+B.

Pour rechercher dans l'historique :

  1. Sélectionnez Historique dans le menu Aller à. L'historique s'affiche.
  2. Dans l'historique, ouvrez le menu Outils et sélectionnez Rechercher dans l'historique. La boîte de dialogue Rechercher dans l'historique s'affiche.
  3. Utilisez les listes déroulantes pour sélectionner des options pour définir votre recherche.
  4. Choisissez parmi les options suivantes :
  • Sélectionnez "titre" ou "emplacement" pour choisir l'endroit où doit porter la recherche. (La fenêtre Signets vous permet d'effectuer une recherche sur le "nom", la "description" et le "mot clé".)
  • Sélectionnez "contient", "commence par" ou "finit par" si vous ne connaissez qu'une partie du mot ou de la phrase recherchée.
  • Sélectionnez "est" si vous savez exactement ce que vous recherchez.
  • Sélectionnez "n'est pas" ou "ne contient pas" pour exclure des pages.
  • Cliquez dans le champ et entrez l'URL (adresse Web) ou un mot, partiel ou complet, des signets ou entrées d'historique à rechercher ou exclure.
  1. Cliquez sur Rechercher. Les entrées de l'historique qui correspondent à votre critère de recherche s'affichent dans la fenêtre Résultats de la recherche.

Pour utiliser les résultats de recherche :

  • Double-cliquez sur un signet de la fenêtre Résultats de la recherche pour accéder à cette page.

Astuces :

  • Pour accéder rapidement à l'historique, appuyez sur les touches Ctrl+H.
  • Si la liste est difficile à lire, essayez d'agrandir la fenêtre des résultats de recherche.

  3. Sélectionnez Copier l'adresse du lien ou Copier l'adresse de l'image. Si une image sert également de lien, les deux options sont disponibles.

Vous pouvez coller le lien dans d'autres programmes ou dans la barre d'adresses de Navigator.

Enregistrement intégral ou partiel d'une page

Pour enregistrer une page entière, ouvrez la fenêtre de Navigator :

  1. Ouvrez le menu Fichier et choisissez Enregistrer la page sous. La boîte de dialogue Enregistrer sous s'affiche.
  2. Sélectionnez un dossier dans lequel enregistrer ce fichier.
  3. Sélectionnez le format de la page que vous souhaitez enregistrer :
    • Page Web, complète : Enregistre tout le contenu de la page Web, y compris les images. Cette option vous permet de visualiser la page telle qu'elle apparaissait initialement avec des images, sans toutefois conserver la structure des liens HTML de la page d'origine. Netscape crée un répertoire (dans lequel la page est enregistrée) pour enregistrer les images et les autres fichiers nécessaires à l'affichage de la page Web complète.
    • Page Web, HTML uniquement : Enregistre la page d'origine, sans les images. Cette option préserve la structure des liens HTML d'origine dans un seul fichier.
    • Fichier texte : Enregistre la page d'origine sous la forme d'un fichier texte. Cette option ne préserve pas la structure des liens HTML d'origine, mais vous permet de visualiser une version texte de la page Web dans n'importe quel éditeur de texte.
  4. Entrez un nom de fichier pour la page et cliquez sur Enregistrer.

Lorsque vous affichez une page contenant des cadres dont l'un est sélectionné, l'option de la liste déroulante Enregistrer le cadre sous est disponible en plus de l'option Enregistrer la page sous. Vous pouvez donc enregistrer uniquement le contenu du cadre sélectionné.

L'enregistrement d'un fichier sur le disque dur permet d'afficher la page (ou son code HTML) sans être connecté à Internet.

Pour enregistrer une image d'une page :

  1. Positionnez le pointeur de la souris sur l'image.
  2. Cliquez avec le bouton droit (sous Mac OS, appuyez sur la touche Contrôle et cliquez) sur l'image pour afficher un menu contextuel.
  3. Cliquez sur Enregistrer l'image sous. La boîte de dialogue Enregistrer le fichier s'affiche.
  4. Sélectionnez un dossier dans lequel enregistrer cette image.
  5. Entrez un nom de fichier pour l'image et cliquez sur Enregistrer.

Pour enregistrer une page sans l'afficher (ce qui est utile pour récupérer une page non formatée, comme un fichier de données, qui n'est pas prévu pour l'affichage) :

  1. Positionnez le pointeur de la souris sur le lien vers la page.
  2. Cliquez avec le bouton droit (sous Mac OS, appuyez sur la touche Contrôle et cliquez) sur l'hyperlien de la page pour afficher un menu contextuel.
  3. Cliquez sur Enregistrer la cible du lien sous. La boîte de dialogue Enregistrer le fichier s'affiche.
  4. Sélectionnez un dossier dans lequel enregistrer ce fichier.
  5. Entrez un nom de fichier pour la page et cliquez sur Enregistrer.

Important : Certains liens téléchargent et enregistrent automatiquement les fichiers sur le disque dur lorsque vous cliquez dessus. L'URL de ces liens commence souvent par "ftp" ou se termine par une extension de type fichier, telle que "au" ou "mpeg". Ces liens peuvent transférer des logiciels ou des fichiers son ou vidéo et lancer des applications compatibles avec les fichiers.

Astuce : Pour enregistrer une image comme papier peint du bureau de Windows, cliquez sur celle-ci avec le bouton droit de la souris, puis sélectionnez Définir comme papier peint dans le menu contextuel.

Utilisation de la fonction Aperçu avant impression

Pour avoir un aperçu de la page telle qu'elle sera imprimée, vous pouvez utiliser la fonction Aperçu avant impression. Ouvrez la fenêtre de Navigator :

  • Dans Navigator, ouvrez le menu Fichier et choisissez Aperçu avant impression.

Le mode Aperçu avant impression vous permet d'effectuer les opérations suivantes avec les pages que vous souhaitez imprimer :

  • Visualiser un aperçu de chaque page : Cliquez sur le bouton Suivant , Précédent , Premier ou Dernier pour vous déplacer entre les différentes pages.
  • Changer l'échelle (taille) des pages : Dans le champ Echelle, entrez un pourcentage de la taille initiale et appuyez ensuite sur la touche Entrée (Retour sous Mac OS). Vous pouvez, par exemple entrer "50" dans ce champ et appuyer ensuite sur Entrée pour diviser par deux la taille de chacune des pages. Cliquez sur le bouton Haut ou Bas pour redimensionner l'image par incrément d'une unité.
  • Changer l'orientation de la page : Sélectionnez Portrait pour imprimer la page verticalement. Sélectionnez Paysage pour imprimer la page horizontalement.
  • Accéder à la mise en page : Cliquez sur Mise en page pour personnaliser davantage les pages que vous souhaitez imprimer.
  • Procéder à l'impression : Cliquez sur Imprimer pour imprimer les pages.

Remarque : Certaines fonctions de l'aperçu avant impression sont différentes, voire indisponibles, sous Mac OS et Linux.

 

Utilisation de la mise en page

Remarque : Certaines fonctions de l'aperçu avant impression sont différentes, voire indisponibles, sous Mac OS et Linux.

Pour personnaliser le mode d'impression des pages dans Netscape, vous pouvez utiliser la fonction de mise en page :

  • Dans Navigator, ouvrez le menu Fichier et choisissez Mise en page.

La boîte de dialogue Mise en page vous permet de modifier les paramètres suivants pour les pages que vous souhaitez imprimer :

  • Format et options : Choisissez l'orientation, l'échelle et d'autres options :
    • Orientation :
      • Portrait : Sélectionnez la case d'option Portrait pour positionner la page dans le sens vertical.
      • Paysage : Sélectionnez la case d'option Paysage pour positionner la page dans le sens horizontal.
    • Echelle : Entrez un pourcentage de la taille initiale. Vous pouvez, par exemple, entrer "50" dans ce champ pour diviser par deux la taille de chacune des pages.
      • Réduire pour ajuster à la largeur de la page : Sélectionnez cette option pour redimensionner automatiquement la page en fonction de la largeur du papier.
    • Options :
      • Imprimer l'arrière-plan (couleurs et images) : Sélectionnez cette option pour imprimer les couleurs et images d'arrière-plan. Lorsque cette option est désélectionnée, seules les images et la couleur d'avant-plan sont imprimées.
  • Marges et en-tête/pied de page : Cliquez sur cet onglet pour configurer les marges, en-têtes et pieds de page :
    • Marges :
      • Haut, Bas, Gauche, Droite : Entrez une valeur, exprimée en pouces, pour les marges supérieure, inférieure, gauche et droite.
    • En-têtes/pieds de page : Chaque liste déroulante représente une zone d'en-tête ou de pied de page. La rangée supérieure des listes déroulantes est destinée aux zones gauche, centrale et droite de l'en-tête. La rangée inférieure concerne les zones gauche, centrale et droite du pied de page. Chaque liste déroulante vous propose les options suivantes :
      • --vierge-- : Rien n'est affiché dans cette zone.
      • Titre : Affichage du titre de la page Web.
      • URL : Affichage de l'URL de la page Web (les URL commencent généralement par "http://").
      • Date/heure : Affichage de la date et de l'heure d'impression de la page Web.
      • Page # : Affichage du numéro de page sur chaque page. Par exemple,
      • Page # sur # : Affichage du numéro de page par rapport au nombre total de pages. Par exemple, si vous imprimez une page Web composée de cinq feuilles, la mention "3 sur 5" apparaîtra sur la troisième page.
      • Personnaliser : Tapez votre propre texte. Vous pouvez ajouter l'un des codes suivants afin d'imprimer des informations spécifiques :
        • &PT : Numéro de page avec le total (Exemple : "3 sur 5")
        • &P : Numéro de page
        • &D : Date
        • &U : URL
        • &T : Titre de la page

Astuce : Pour avoir un aperçu des modifications effectuées dans la boîte de dialogue Mise en page, utilisez la fonction Aperçu avant impression.

Sélection de polices et de codages des caractères

Si vous surfez ou créez, envoyez et recevez du courrier électronique en plusieurs langues, vous devez sélectionner les polices et codages des caractères appropriés.

Une méthode de codage des caractères est la façon dont un document ou un message a été converti en données utilisables par l'ordinateur. Tous les documents Web et messages utilisent une méthode de codage des caractères (également appelée encodage ou jeu de caractères).

La méthode de codage des caractères d'un document dépend de sa langue. Certaines langues, telles que la plupart des langues occidentales, partagent la même méthode de codage. D'autres, telles que le chinois, le japonais et le russe, utilisent d'autres méthodes.

Votre version de Netscape est réglée sur un codage des caractères par défaut approprié pour votre région. Cependant, si vous utilisez plusieurs langues, il peut s'avérer nécessaire de sélectionner des méthodes de codage des caractères appropriées et de désigner les polices correspondantes.

Pour sélectionner les codages des caractères, ouvrez la fenêtre de Navigator :

  1. Ouvrez le menu Afficher, sélectionnez Codage des caractères et Autres.
  2. Sélectionnez ensuite une région dans la section supérieure du sous-menu.
  3. Sélectionnez une méthode de codage des caractères dans le sous-menu de la région. Répétez les étapes 1 à 3 pour chaque méthode de codage des caractères désirée.

Les méthodes de codage des caractères sélectionnées sont ajoutées au menu Codage des caractères. Si plusieurs méthodes de codage sont sélectionnées, la méthode active est accompagnée d'une puce (point).

Netscape peut détecter le codage des caractères utilisé par un document et l'afficher correctement. Pour tirer parti de cette fonctionnalité, ouvrez la fenêtre de Navigator :

  1. Ouvrez le menu Afficher, choisissez Codage des caractères et Détection auto.
  2. Sélectionnez l'une des options de détection automatique ou cliquez sur Pas de détection auto. dans le sous-menu.

Pour modifier la liste de jeux de caractères actifs :

  1. Ouvrez le menu Afficher, choisissez Codage des caractères et Personnaliser. La boîte de dialogue Personnaliser le codage des caractères s'affiche.
  2. Choisissez l'une des procédures suivantes :
    • Sélectionnez un jeu de caractères à activer dans la liste de gauche et cliquez sur Ajouter.
    • Sélectionnez le jeu de caractères à supprimer dans la liste de droite et cliquez sur Supprimer.
    • Pour modifier l'ordre de vérification des jeux de caractères par le navigateur, sélectionnez les jeux de caractères dans la liste de droite et utilisez les boutons fléchés pour les déplacer vers le haut ou le bas.

Pour changer les polices par défaut dans un groupe de langues :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Polices. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.
  3. Dans la liste déroulante "Sélectionner", choisissez une méthode de codage des caractères. Par exemple, pour définir les polices par défaut du jeu de caractères occidental, sélectionnez "Occidental".
  4. Sélectionnez si le texte proportionnel doit être de type serif (comme Times Roman) ou sans serif (comme Arial). Vous pouvez également attribuer une taille au texte proportionnel. Le texte proportionnel varie en largeur.
  5. (Le cas échéant) Sélectionnez une police pour Serif, Sans serif, Cursive et Fantaisie.
  6. Sélectionnez la police fixe (comme Courier) à utiliser pour les pages Web. Le texte fixe a une largeur fixe, de sorte que chaque caractère ou lettre occupe le même espace.

De nombreux auteurs de pages Web sélectionnent leurs propres polices et tailles. Vous pouvez utiliser les paramètres de police des auteurs en sélectionnant "Autoriser les documents à utiliser d'autres polices".

Pour régler la lisibilité des polices, sélectionnez dans la liste déroulante la résolution d'affichage des pages Web en points par pouce (dpi). Sélectionnez "Autres" pour ouvrir la boîte de dialogue Etalonner la résolution, qui permet d'étalonner la résolution en mesurant la longueur d'une ligne à l'écran.

Définition des préférences linguistiques

La langue utilisée pour Netscape affecte le texte des boutons, boîtes de dialogue, menus, outils et autres éléments. La fonctionnalité de ces éléments reste inchangée.

Pour définir votre langue de prédilection :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Langues/Contenu. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.
  3. Effectuez une sélection dans "Modules de langue installés".
  4. Cliquez sur OK.

    Remarque : Vous devez redémarrer Netscape après avoir changé votre langue de travail.

L'élément de contenu utilisé affecte la page d'accueil, les signets, le contenu de la barre d'outils, Ma barre latérale et d'autres éléments.

Pour utiliser un élément de contenu :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Aspect, cliquez sur Langues/Contenu. Si aucune sous-catégorie n'est visible, double-cliquez sur Aspect pour développer la liste.
  3. Effectuez une sélection dans "Modules de contenu installés".
  4. Cliquez sur OK.

    Remarque : Il est nécessaire de redémarrer Netscape après un changement d'élément de contenu.

Astuce : Pour télécharger d'autres modules de contenu ou de langue, cliquez sur Télécharger plus.

Les pages Web sont parfois disponibles en plusieurs langues. Si elle est disponible, Netscape présente les pages dans la langue de votre choix. Vous pouvez sélectionner les langues d'affichage par ordre de préférence.

Pour définir les préférences linguistiques :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Navigator, cliquez sur Langues. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.
  3. Cliquez sur Ajouter. La boîte de dialogue Ajouter des langues s'affiche.
  4. Sélectionnez une langue et son code. Pour ajouter un code ne figurant pas dans la liste, entrez la langue et son code dans la zone Autres (voir remarque ci-dessous). Pour réorganiser les langues dans la liste, sélectionnez une langue et cliquez sur les boutons fléchés.
  5. Cliquez sur OK.

Remarque : Les codes de deux lettres entre crochets apparaissent à côté de chaque langue de la liste. Il s'agit des abréviations standard des langues. Pour obtenir la liste complète des codes, consultez le document en ligne intitulé Codes for the Representation of Names of Languages.

Gestion de différents types de fichiers

Navigator peut prendre en charge de nombreux types de fichiers. Cependant, pour traiter certains fichiers, tels que la musique ou les films, Navigator a besoin de modules externes ou applications auxiliaires. Même si ces modules ou applications ne sont pas installés, Netscape est en mesure d'enregistrer le fichier sur votre disque dur. Lorsque vous enregistrez des fichiers, vous pouvez en opérer le suivi grâce au Gestionnaire de téléchargement.

 

Modules externes

Les modules externes sont des applications auxiliaires qui étendent les fonctionnalités de Navigator et s'exécutent dans Netscape. L'utilisation de modules externes tels que Sun Java, Macromedia Flash et RealNetworks RealPlayer permet à Netscape d'afficher des fichiers multimédia et d'exécuter de petites applications, comme des films, des animations et des jeux.

Lors de l'installation de Netscape, le système vous a offert la possibilité d'installer quelques-uns de ces modules externes. Vous n'êtes pas obligé de les installer séparément. Sachez que vous pouvez également ajouter des modules après avoir installé Nestcape. Pour consulter la liste des modules externes déjà installés, consultez la page Web en ligne du Netscape Plug-in Manager.

Pour consulter la liste de tous les modules externes qu'il est possible d'installer dans Netscape, rendez-vous à la page Browser Plug-ins sur le site Web de Netscape.

 

Applications auxiliaires

S'il s'avère impossible d'utiliser des fichiers dans Netscape, vous avez la possibilité de lancer des applications auxiliaires qui s'exécutent en dehors de Netscape. Ainsi, pour lire des fichiers MP3, vous pouvez ouvrir des programmes externes tels que Winamp.

Les paramètres de la section Applications auxiliaires des préférences indiquent à Navigator les applications auxiliaires à utiliser sur l'ordinateur. En règle générale, ces préférences ne doivent pas être modifiées car elles sont prédéfinies. En outre, si Navigator ignore quelles applications auxiliaires utiliser, il vous permet d'en trouver une dans la boîte de dialogue de téléchargement.

La boîte de dialogue de téléchargement propose les options suivantes :

  • Ouvrir en utilisant : Netscape ouvre le fichier en utilisant le programme par défaut de vos paramètres système.
  • Choisir : Netscape vous permet d'ouvrir le fichier en utilisant l'application de votre choix.
  • Enregistrer ce fichier sur disque : Netscape permet de sélectionner un emplacement pour l'enregistrement du fichier.
  • Toujours demander avant d'ouvrir ce type de fichier : Lorsque cette option est sélectionnée, Netscape ouvre la boîte de dialogue de téléchargement lors du téléchargement d'un type de fichier particulier. Vous pouvez réinitialiser ces choix par la suite dans les préférences des applications auxiliaires.
  • Avancé : Netscape permet d'optimiser la gestion des fichiers. Voir "Pour ajouter une application auxiliaire à la liste" pour plus de détails.

Astuce : Vous pouvez effacer les options enregistrées pour "Toujours demander avant d'ouvrir ce type de fichier" dans les préférences. Pour plus d'informations à ce sujet, consultez la rubrique Préférences d'Instant Messenger – Applications auxiliaires.

Pour optimiser les préférences des applications auxiliaires (pour les utilisateurs expérimentés) :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Navigator, cliquez sur Applications auxiliaires. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.
  3. Exécutez l'une des procédures suivantes :

    Pour ajouter une application auxiliaire à la liste :

    1. Cliquez sur Nouveau type. La boîte de dialogue Nouveau type s'affiche.
    2. Entrez une description du type de fichier avec lequel fonctionne l'application ; par exemple, "fichiers vidéo" ou "fichiers son".
    3. Entrez l'extension de fichier (suffixe) utilisée par ce type de fichier. Si ce type de fichier utilise plusieurs extensions, entrez-les toutes en les séparant par des espaces.
    4. Entrez le type MIME. Les types MIME sont constitués de deux mots séparés par une barre oblique. Par exemple, le type MIME d'un fichier image TIFF est "image/tiff".
      Remarque : Lors du téléchargement d'un fichier, Netscape indiquera le type MIME dans la boîte de dialogue de téléchargement.
    5. Cliquez sur Choisir, puis sélectionnez une application prenant en charge les fichiers de ce type.
    6. Cliquez sur OK.

    Pour spécifier la gestion par Navigator des fichiers téléchargés :

    1. Sélectionnez un type de fichier dans la liste et cliquez sur Edition.
    2. Entrez le type MIME qui sera utilisé par ce fichier.
    3. Sélectionnez une option "Géré par" pour ouvrir les fichiers téléchargés enregistrés sur le disque dur ou ouverts dans une application spécifiée. Cliquez sur Choisir pour spécifier l'application.
    4. Cliquez sur OK.

    Pour supprimer un élément de la liste :

    1. Sélectionnez un type de fichier dans la liste et cliquez sur Supprimer.
    2. Cliquez sur OK.

Pour plus d'informations sur les préférences des applications auxiliaires, consultez la rubrique Préférences de Navigator – Applications auxiliaires.

 

Gestionnaire de téléchargement

    Remarque : L'option Afficher dans Explorer se nomme Afficher dans le navigateur sous Linux et Afficher dans le Finder sous Mac OS.

Sélection de Netscape comme navigateur par défaut

Remarque : Cette section porte sur des paramètres et fonctions disponibles uniquement dans Windows.

Netscape est un outil très prisé pour l'affichage de pages Web, que ce soit sur Internet ou sur votre ordinateur. Pour ouvrir aisément des pages Web, vous pouvez faire en sorte que Netscape devienne votre navigateur par défaut.

 

Protocoles et fichiers Internet courants

En faisant de Netscape votre navigateur par défaut, vous pourrez ouvrir automatiquement des protocoles et formats de fichier utilisés couramment sur Internet. Voici un aperçu de ces fichiers et protocoles Internet :

  • Fichiers image :

    • JPEG, GIF, PNG et MNG

  • Fichiers de langage et de document Internet :

    • HTML, XML, XHTML et XUL

  • Protocoles Internet :

    • HTTP, HTTPS, FTP, Chrome, Gopher

 

Modification automatique des paramètres par défaut du navigateur

Une fois l'installation terminée, Netscape vérifie s'il est le navigateur par défaut pour les protocoles ou formats de fichiers Internet courants. S'il ne l'est pas, le système vous pose la question suivante : "Netscape n'est pas actuellement défini comme votre navigateur par défaut. Souhaitez-vous en faire votre navigateur par défaut ?"

Cliquez sur Oui pour faire de Netscape votre navigateur par défaut. Si vous cliquez sur Non, cette question vous sera posée à chaque fois que vous lancerez Netscape, à moins que vous n'ayez désactivé la case à cocher "Vérifier au prochain démarrage aussi".

Si vous désactivez la case à cocher "Vérifier au prochain démarrage aussi", vous aurez toujours la possibilité de faire de Netscape votre navigateur par défaut en modifiant vos paramètres dans les Préférences. Pour savoir comment faire de Netscape votre navigateur par défaut en utilisant le panneau des préférences, consultez la rubrique Préférences de Navigator – Navigator.

Modification des paramètres de la mémoire cache

Votre ordinateur stocke des copies des pages fréquemment visitées dans la mémoire cache. Cela évite à l'ordinateur de devoir récupérer la page du réseau à chaque affichage.

Pour définir la taille de la mémoire cache ou la vider (Windows et Unix uniquement) :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Sélectionnez la catégorie Avancé, cliquez sur Cache. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.
  3. Entrez un nombre dans la zone Cache en mémoire pour en spécifier la taille. Il est conseillé de sélectionner une taille entre 1 024 Ko et 2 000 Ko. Pour vider immédiatement la mémoire cache, cliquez sur Vider le cache en mémoire.

Important : Une mémoire cache plus volumineuse permet de récupérer rapidement davantage de données. Cependant, à moins que la mémoire de votre ordinateur soit très importante, 1 024 Ko est suffisant.

Pour définir la taille de la cache sur disque ou la vider :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Sélectionnez la catégorie Avancé, cliquez sur Cache. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.
  3. Entrez un nombre dans la zone Cache sur disque pour en spécifier la taille. 8 000 Ko est une taille suffisante sous Windows (5000 Ko sous Unix). Pour vider immédiatement la mémoire cache, cliquez sur Vider le cache sur disque.

Important : Une mémoire cache sur disque plus volumineuse permet de récupérer rapidement davantage de pages, mais occupe davantage d'espace sur le disque dur.

Lorsque vous quittez Netscape, il assure la maintenance de la mémoire cache. Si vous estimez qu'elle prend trop de temps, essayez de réduire la taille de la mémoire cache sur disque.

Pour spécifier la fréquence de vérification par Navigator des révisions de page sur le réseau (pour ne pas conserver trop longtemps des pages obsolètes dans la mémoire cache) :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Sélectionnez la catégorie Avancé, cliquez sur Cache. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.
  3. Sélectionnez parmi les options suivantes :
  • Chaque fois que je visualise la page : Pour que Netscape compare une page Web au cache à chaque visualisation.
  • Lorsque la page est périmée : Indique à Netscape de comparer une page Web au cache lorsque la page est déterminée comme ayant expiré par le serveur.
  • Une fois par session : Indique à Netscape de comparer une page Web au cache à chaque démarrage de Netscape.
  • Jamais : Indique à Netscape de ne jamais comparer les informations du cache au réseau.

Si des pages devant se trouver dans la mémoire cache prennent un temps excessif pour s'afficher, vérifiez que la préférence n'est pas réglée sur "Chaque fois que je visualise la page" dans la mesure où la vérification nécessite une connexion réseau.

Pour rafraîchir une page à tout moment :

  • Cliquez sur le bouton Recharger dans la barre d'outils de navigation du navigateur. L'ordinateur consulte le réseau pour vérifier que vous disposez de la dernière version de la page.

Utilisation de Quick Launch

Remarque : Quick Launch n'est disponible que sur les versions Windows de Netscape.

Quick Launch réduit le temps nécessaire au démarrage de Netscape. Lorsque Netscape utilise Quick Launch, l'icône du logo de Netscape est présente dans la barre des tâches (près de l'heure).
Icône de Netscape Quick Launch

Astuce : Pour démarrer rapidement des composants (tels que Composer, Messagerie & Forums et la fenêtre Carnet d'adresses), cliquez avec le bouton droit de la souris sur l'icône du logo Netscape et sélectionnez le composant dans le menu contextuel.

Lors de l'installation de Netscape, vous avez eu la possibilité de choisir entre l'activation ou la désactivation de Quick Launch. Si cette option est activée, Quick Launch charge une partie de Netscape en mémoire lors du démarrage initial de Windows (et à chaque fois que Netscape est exécuté). Quick Launch permet à Netscape de rester en mémoire après la fermeture de toutes les fenêtres Nestcape. Cela se traduit par un démarrage rapide de Netscape lorsque vous en avez besoin, sans qu'il faille charger tous les composants de Netscape. Si la mémoire de votre ordinateur est saturée, vous pouvez désactiver Quick Launch afin d'économiser la mémoire.


Pour activer ou désactiver Quick Launch dans Netscape :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Cliquez sur la catégorie Avancé.
  3. Sélectionnez "Activer Quick Launch" pour activer Quick Launch, ou désélectionnez cette option pour désactiver Quick Launch.
  4. Cliquez sur OK.

    Astuce : Pour désactiver aisément Quick Launch, cliquez avec le bouton droit sur l'icône du logo de Netscape dans la barre des tâches (près de l'heure) et sélectionnez "Désactiver Quick Launch" dans le menu contextuel.

Définition des valeurs du serveur proxy

Cette section vous explique comment utiliser des proxies.

Définition des valeurs du serveur proxy

De nombreuses sociétés bloquent l'accès depuis Internet à leurs réseaux. Cela évite que de tierces parties accèdent à des informations confidentielles. Cette protection est appelée un pare-feu.

Si votre société possède un pare-feu, le navigateur doit probablement passer par un serveur proxy pour vous connecter à Internet. Le serveur proxy empêche les intrus de pénétrer dans le réseau privé de la société.

Avant de commencer :

  • Si votre société possède un fichier de configuration de proxy, demandez son URL à l'administrateur système.
  • Dans le cas contraire, demandez à l'administrateur système les noms et numéros de port des serveurs exécutant le logiciel proxy pour chaque service réseau.

Pour configurer le navigateur pour le fonctionnement avec le serveur proxy :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Avancé, cliquez sur Proxies. Si aucune sous-catégorie n'est visible, double-cliquez sur Navigator pour développer la liste.
  3. Sélectionnez l'une des options suivantes :
    • Connexion directe à Internet : Sélectionnez cette option si vous ne souhaitez pas utiliser de proxy.
    • Configuration manuelle du Proxy : Sélectionnez cette option si vous ne possédez pas d'adresse de proxy (URL).
      • Proxy HTTP, Proxy SSL, Proxy FTP, Proxy Gopher: Entrez le nom ou l'adresse IP numérique du serveur proxy. Entrez le port dans le champ Port.
      • Hôte SOCKS : Entrez le nom ou l'adresse IP numérique du serveur proxy. Entrez le port dans le champ Port.
        • SOCKS v4, SOCKS v5 : Lorsque vous entrez un hôte SOCK, sélectionnez "SOCKS v4" ou "SOCKS v5", selon la version de SOCKS utilisée pour le proxy.
      • Pas de proxy pour : Entrez les domaines pour lesquels vous ne souhaitez pas utiliser de proxy. Séparez les domaines par des virgules. Exemple : .votreentreprise.com, .votreentreprise.fr.
    • Adresse de configuration automatique des proxies : Choisissez cette option si vous avez un fichier de configuration ou une URL de proxy, puis entrez l'URL de configuration.

Les noms de domaine sont la partie d'un URL contenant le nom d'une organisation, d'une société ou d'une faculté, comme netscape.com ou washington.org. Si vous utilisez des noms d'hôtes locaux sans nom de domaine, énumérez-les de la même manière. Utilisez des virgules pour séparer les noms d'hôtes. Le caractère générique [*] ne peut pas être utilisé.

Pour plus d'informations sur le panneau des préférences Proxies, consultez la rubrique Préférences avancées – Proxies.
