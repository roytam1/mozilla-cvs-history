Préférences avancées - Avancé

Cette section explique comment utiliser le panneau de préférences Avancé principal. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Cliquez sur la catégorie Avancé.

Le panneau de préférences Avancé principal permet de personnaliser Java, les mots de passe FTP et Quick Launch :

  • Activer Java :Autorise Netscape à interpréter les pages contenant du code Java. Java est utilisé sur certaines page Web pour incorporer du contenu interactif et multimédia.
  • Activer XSLT : Permet à Netscape d'utiliser XSLT. XSLT (eXtensible Stylesheet Language/Transform) est un langage utilisé pour transformer un document XML en d'autres documents XML. Pour plus d'informations concernant XSLT, voir le document en ligne Transforming XML: Netscape and XSLT.
  • Envoyer cette adresse électronique comme mot de passe FTP anonyme : Entrez une adresse électronique à utiliser comme mot de passe lors de la connexion anonyme à un site FTP.
  • Activer Quick Launch : Permet de réduire le temps nécessaire pour démarrer Netscape. Voir "Utilisation de Quick Launch" pour plus d'informations sur l'utilisation de Quick Launch.

    Remarque : Quick Launch n'apparaît que sur les plates-formes Windows.

 

Préférences avancées - Scripts et plugins

Cette section décrit l'utilisation du panneau de préférences Scripts et plugins. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Avancé, cliquez sur Scripts et plugins. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.


Le panneau de préférences Scripts et plugins permet de contrôler la façon dont JavaScript et les plugins sont utilisés dans Netcape :

  • Activer JavaScript pour : Sélectionnez les cases à cocher suivantes pour activer JavaScript :
    • Navigator : Active ou désactive JavaScript pour les pages Web ouvertes dans Navigator.
    • Messagerie & forums : Active ou désactive JavaScript pour les pages Web ouvertes dans Messagerie & forums.
  • Autoriser les pages Web à : Sélectionnez les cases à cocher suivantes pour contrôler la façon dont JavaScript peut être utilisé :
    • Ouvrir un lien dans une nouvelle fenêtre : Permet d'ouvrir dans une nouvelle fenêtre les liens sur lesquels vous cliquez. Certains liens sont configurés pour s'ouvrir dans une nouvelle fenêtre plutôt que dans la même fenêtre. Une fois modifié, ce contrôle exige le redémarrage de Netscape pour être pris en compte.
    • Déplacer ou redimensionner des fenêtres existantes : Permet de redimensionner ou déplacer des fenêtres ouvertes.
    • Monter ou descendre des fenêtres: Permet de placer des fenêtres devant ou derrière les autres.
    • Masquer la barre d'état : Permet de masquer la barre d'état.
    • Modifier le texte de la barre d'état : Permet de modifier le texte de la barre d'état, par exemple pour y faire défiler du texte.
    • Modifier des images : Permet de modifier ou d'animer des images, par exemple pour faire changer l'image lorsque le curseur de la souris passe dessus.
    • Créer ou modifier des cookies : Permet de modifier ou créer des cookies avec JavaScript.
    • Lire des cookies : Permet la lecture de cookies avec JavaScript.
  • Activer les plugins pour: Cochez cette case pour contrôler la façon dont les plugins sont utilisés :
    • Messagerie & forums : Permet l'utilisation de plugins dans Messagerie et forums.

Pour plus d'informations sur les plugins, voir Gestion de différents types de fichiers.

 

Préférences avancées - Cache

Cette section décrit l'utilisation du panneau de préférences Cache. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Sélectionnez la catégorie Avancé, cliquez sur Cache. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.


Le panneau de préférences Cache permet d'ajuster le cache de Netscape en mémoire et sur disque :

  • Cache en mémoire : Entrez la quantité de cache en mémoire à attribuer à Netscape. Le cache est effacé lorsque vous redémarrez l'ordinateur.
  • Cache sur disque : Entrez la quantité de cache sur disque à attribuer à Netscape. Le cache sur disque est enregistré sur disque dur et peut être réutilisé même si vous avez éteint l'ordinateur entre-temps.
  • Dossier de cache sur disque : Affiche l'emplacement actuel du dossier de cache sur disque
    • Choisir un dossier : Permet de choisir un emplacement pour le dossier de cache sur disque.
  • Vider le cache en mémoire : Permet de vider le cache en mémoire.
  • Vider le cache sur disque : Permet de vider le cache sur disque.
  • Le document du cache est comparé à celui sur le réseau :
    • Chaque fois que je visualise la page : Pour que Netscape compare une page Web au cache à chaque visualisation.
    • Lorsque la page est périmée : Indique à Netscape de comparer une page Web au cache lorsque la page est déterminée comme ayant expiré par le serveur.
    • Une fois par session : Indique à Netscape de comparer une page Web au cache à chaque démarrage de Netscape.
    • Jamais : Indique à Netscape de ne jamais comparer les informations du cache au réseau.

 

Préférences avancées - Proxies

Cette section décrit l'utilisation du panneau de préférences Proxies. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Avancé, cliquez sur Proxies. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.


Le panneau de préférences Proxies permet de configurer Netscape pour l'utilisation d'un proxy :

Avant de commencer : Demandez à l'administrateur réseau si vous avez un fichier de configuration de proxy ou le nom et le numéro de port du proxy.

  • Connexion directe à Internet : Sélectionnez cette option si vous ne souhaitez pas utiliser de proxy.
  • Configuration manuelle du Proxy : Sélectionnez cette option si vous ne possédez pas d'adresse de proxy (URL).
    • Proxy HTTP, Proxy SSL, Proxy FTP, Proxy Gopher: Entrez le nom ou l'adresse IP numérique du serveur proxy. Entrez le port dans le champ Port.
    • Hôte SOCKS : Entrez le nom ou l'adresse IP numérique du serveur proxy. Entrez le port dans le champ Port.
      • SOCKS v4, SOCKS v5 : Lorsque vous entrez un hôte SOCK, sélectionnez "SOCKS v4" ou "SOCKS v5", selon la version de SOCKS utilisée pour le proxy.
    • Pas de proxy pour : Entrez les domaines pour lesquels vous ne souhaitez pas utiliser de proxy. Séparez les domaines par des virgules. Exemple : .votreentreprise.com, .votreentreprise.fr.
  • Adresse de configuration automatique des proxies : Choisissez cette option si vous avez un fichier de configuration ou une URL de proxy, puis entrez l'URL de configuration.

 

Préférences avancées - Réseau HTTP

Cette section décrit l'utilisation du panneau de préférences Réseau HTTP. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Avancé, cliquez sur Réseau HTTP. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.


Le panneau de préférences Réseau HTTP permet de configurer la mise en réseau basée sur HTTP :

  • Options de connexion directe, Options de connexion par proxy : Choisissez la version HTTP et les options de connexion directe et par proxy.
    • Utiliser HTTP 1.0 : Choisissez cette option pour utiliser la version originale de HTTP, standardisée en 1996.
    • Utiliser HTTP 1.1 : Choisissez cette option pour utiliser la nouvelle version de HTTP, qui apporte des améliorations de performances, dont une utilisation plus efficace des connexions HTTP, un meilleur support de la mise en cache côté client, la prise en charge de requêtes HTTP multiples (pipelines) et un contrôle affiné des politiques d'expiration et de remplacement des caches.
    • Activer les diagnostics: Sélectionnez cette option pour maintenir une connexion ouverte en vue d'autres requêtes HTTP, ce qui permet un gain de rapidité.
    • Activer les pipelines : Sélectionnez cette option pour activer les pipelines, qui permettent d'envoyer au serveur plusieurs requêtes HTTP à la fois, réduisant ainsi le délai de chargement des pages Web.
      Remarque : Les pipelines ne sont disponibles qu'avec HTTP 1.1.

 

Préférences avancées - Installation de logiciels

Cette section décrit l'utilisation du panneau de préférences Installation du logiciel. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Avancé, cliquez sur Installation du logiciel. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.


Le panneau de préférences Installation du logiciel permet d'activer l'installation de logiciels et la notification des mises à jour :

  • Activer l'installation de logiciels : Sélectionnez cette option pour que Netscape affiche une invite lors de l'installation et de l'utilisation de nouveaux logiciels avec Netscape.
  • Recherche de mises à jour : Sélectionnez cette option pour recevoir une notification lorsqu'une nouvelle version de Netscape est disponible. Votre données personnelles ne sont pas transmises à Netscape lors de la vérification de votre version de Netscape.
    • hebdomadaire : Choisissez cette option pour que Netscape vérifie une fois par semaine si une nouvelle version de Netscape est disponible.
    • mensuelle : Choisissez cette option pour que Netscape vérifie une fois par mois si une nouvelle version de Netscape est disponible.

 

Préférences avancées - Roulette de la souris

Cette section décrit l'utilisation du panneau de préférences Roulette de la souris. Si ce panneau n'est pas affiché, procédez comme suit :

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Avancé, cliquez sur Roulette de la souris. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.


Le panneau de préférences Roulette de la souris permet de contrôler l'utilisation de la roulette de la souris (entre les boutons de votre souris) dans Netscape :

  • Touche de modification : Sélectionnez dans la liste déroulante une touche à utiliser en même temps que la roulette de la souris pour effectuer une des opérations suivantes.
    • Faire défiler le document : Fait défiler le document du nombre de lignes spécifié dans le champ.
      • Utiliser la valeur par défaut du système : Ignore le nombre de lignes introduit et utilise à la place la valeur par défaut du système.
    • Faire défiler une page vers le haut ou vers le bas : Fait défiler le document d'une page à la fois vers le haut ou vers le bas. Ce paramètre permet un défilement plus rapide, mais moins précis, de la page avec la roulette de la souris.
    • Historique : Effectue un déplacement vers l'arrière ou vers l'avant dans les pages visitées précédemment.
    • Agrandir ou réduire le texte : Augmente ou réduit la taille du texte sur une page Web. Ce paramètre permet de faciliter la lecture d'une page ou de faire tenir le texte sur l'écran.

    Remarque : Une fonction différente peut être affectée à chaque touche de modification.

1. Dans le menu Edition, choisissez Préférences.
2. Dans la catégorie Avancé, cliquez sur Système. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Avancé pour développer la liste.

Remarque : Le panneau de préférences Système n'apparaît que sur les plates-formes Windows.


Ce panneau permet de spécifier les fichiers et protocoles ouverts avec Netscape :

  • Windows doit utiliser Netscape pour ouvrir ces types de fichiers : Sélectionnez les types de fichiers à ouvrir par défaut avec Netscape.
  • Windows doit utiliser Netscape pour gérer ces protocoles : Sélectionnez les protocoles à ouvrir par défaut avec Netscape.
  • Avertissez-moi si d'autres applications modifient ces paramètres : Sélectionnez cette option pour que Netscape vous prévienne si d'autres applications ont modifié vos paramètres de fichier et de protocole Netscape par défaut.

Astuce : Pour faire facilement de Netscape le navigateur par défaut, voir Préférences de Navigator - Navigator. Sélectionnez également "Avertissez-moi si d'autres applications modifient ces paramètres" afin de savoir quand d'autres programmes deviennent les programmes par défaut au lieu de Netscape.
