Gestion des profils

Si vous utilisez Internet chez vous et au travail, vous souhaiterez peut-être accéder à différents jeux de signets, préférences, carnets d'adresses, comptes de messagerie, configuration de barre latérale, etc. De même, les différents membres de votre famille peuvent vouloir partager une copie du même navigateur mais conserver leurs identités Internet séparées.

Le Gestionnaire de profils permet de créer différents profils, chacun avec ses propres signets, préférences, paramètres de messagerie, etc. Un profil par défaut est créé automatiquement lors de la première installation de votre logiciel de navigation. Après avoir crée un ou plusieurs profils supplémentaires, vous serez invité à préciser celui que vous souhaitez utiliser à chaque démarrage du navigateur.

Création d'un profil

Pour créer un profil :

  1. Quittez le navigateur. Il est impossible de créer un profil tant que Navigator, Mail ou un autre composant de votre logiciel de navigation est actif.
  2. Ouvrez le Gestionnaire de profils :
    • Sous Windows, ouvrez le menu Démarrer et choisissez Programmes, Netscape 7,0, puis Gestionnaire de profils.
    • Sous Mac OS, ouvrez le dossier de Netscape, puis double-cliquez sur le Gestionnaire de profils.
    • Sous Linux, tapez ce qui suit sur la ligne de commande : ./netscape -profilemanager
  3. Cliquez sur Créer profil, lisez la description du profil, puis cliquez sur Suivant.
  4. Entrez un nom de profil. Vous pouvez entrer ce que vous souhaitez : votre véritable nom ou un nom associé à l'usage du profil, comme "Ecole".
  5. Acceptez le répertoire par défaut pour le nouveau profil ou cliquez sur le bouton Choisir un dossier et sélectionnez l'emplacement de votre choix.
  6. Si des modules linguistiques ou de contenu régional sont installés, cliquez sur Sélection de la région et choisissez la langue et/ou le contenu désiré.
  7. Cliquez sur Terminé.

L'écran d'ouverture du Gestionnaire de profil réapparaît, avec le nouveau profil en surbrillance dans la liste.

Pour configurer votre navigateur pour le nouveau profil, cliquez sur Démarrer Netscape 7,0. Le système vous invite à enregistrer un pseudonyme (si vous en avez déjà un) que vous utiliserez avec Netscape Webmail, Netscape Instant Messenger, ainsi que pour personnaliser votre page My Netscape. Si vous ne disposez pas d'un pseudonyme, cliquez sur Enregistrer maintenant pour en obtenir un.

Si vous ne souhaitez pas enregistrer de pseudonyme pour l'instant, cliquez sur Annuler et le programme s'ouvrira. Pour enregistrer un pseudonyme alors qu'une session Netscape est ouverte, ouvrez le menu Aide et sélectionnez Enregistrement Netscape.

Lorsque vous commencerez à utiliser le nouveau profil, vous serez invité à fournir des informations supplémentaires à mesure que vous exécuterez des tâches qui les exigent, par exemple vérifier vos messages.

 

Suppression ou changement de nom d'un profil

Vous souhaiterez peut-être supprimer les profils que vous n'utilisez normalement pas. Pour supprimer ou renommer un profil existant :

  1. Quittez le navigateur. Il est impossible de créer, supprimer ou renommer un profil tant que Navigator, Mail ou un autre composant de votre logiciel de navigation est actif.
  2. Ouvrez le Gestionnaire de profils :
    • Sous Windows, ouvrez le menu Démarrer et choisissez Programmes, Netscape 7,0, puis Gestionnaire de profils.
    • Sous Mac OS, ouvrez le dossier de Netscape, puis double-cliquez sur le Gestionnaire de profils.
    • Sous Linux, tapez ce qui suit sur la ligne de commande : ./netscape -profile
  3. Pour supprimer un profil, sélectionnez son nom et cliquez sur Supprimer un profil.
  4. Pour renommer un profil, sélectionnez son nom, cliquez sur Renommer le profil et suivez les instructions.
