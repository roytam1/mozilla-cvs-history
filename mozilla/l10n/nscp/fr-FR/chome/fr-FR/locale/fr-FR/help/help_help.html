Utilisation de la fenêtre d'aide

Cette section s'attache à décrire l'utilisation de la fenêtre d'aide, c'est-à-dire l'intégralité de la fenêtre dans laquelle s'affichent ce texte et les onglets de la partie gauche.

Pour plus d'informations à ce sujet, cliquez sur les liens affichés dans la partie inférieure de la section Aide et centre d'assistance.

Recherche de la rubrique souhaitée

Lorsque la fenêtre d'aide est affichée, vous pouvez cliquer sur les onglets situés dans la partie gauche pour afficher le contenu de l'Aide de diverses manières :

  • Sommaire affiche la liste des principales rubriques.

    Cliquez sur une rubrique pour accéder aux informations qu'elle contient. Double-cliquez pour afficher les sous-rubriques.

    Si vous ne trouvez pas la rubrique que vous recherchez, cliquez sur l'onglet Rechercher ou Index.

  • Rechercher vous permet d'effectuer une recherche dans l'Aide intégrée de Netscape 7.0.

    Entrez un mot ou une expression et appuyez sur Entrée pour afficher les rubriques connexes. Cliquez ensuite sur la rubrique que vous souhaitez consulter.

    Si votre recherche ne retourne aucune rubrique, faites une nouvelle tentative en saisissant moins de mots ou une combinaison de mots différente.

  • Index affiche la liste alphabétique des rubriques.

    Cliquez sur une entrée d'index pour afficher les informations relatives à cette rubrique.

  • Glossaire affiche la liste alphabétique des définitions.

    Cliquez sur un terme du glossaire pour afficher sa définition.

Pour consulter les informations relatives à la rubrique d'aide affichée, cliquez sur les liens de la fenêtre d'aide, tout comme vous le feriez dans une page Web normale.

Ce lien vous permet, par exemple, d'accéder à une définition du glossaire : cookie. (Cliquez sur le bouton Précédent pour revenir à cette page.)

 

Retour en arrière

Pour revenir en arrière dans l'Aide, cliquez sur les boutons situés dans le coin supérieur droit de la fenêtre d'aide :

         Aide et centre d'assistance
Arrière   Avant

  • Cliquez sur le bouton Précédent pour parcourir les pages que vous avez visitées précédemment (de la plus récente à la plus ancienne). Le fonctionnement est identique à celui du bouton Précédent de votre navigateur.
  • Cliquez sur le bouton Suivant pour parcourir les pages que vous avez visitées précédemment (de la plus ancienne à la plus récente). Le fonctionnement est identique à celui du bouton Suivant de votre navigateur.
  • Cliquez sur le bouton Page d'accueil pour afficher la page Aide et centre d'assistance, ainsi que les liens vers les options d'assistance et les ressources basées sur le Web.

Si vous éprouvez des difficultés pour conserver la fenêtre d'aide au premier plan, tandis que vous suivez les instructions, cliquez sur Aide Netscape 7.0 dans la barre des tâches de Windows pour la faire réapparaître.

Vous pouvez également choisir d'imprimer les instructions que vous devez suivre :

  • Pour imprimer la totalité de la page qui est affichée dans la fenêtre d'aide, cliquez sur le bouton Imprimer situé dans le coin supérieur droit de la fenêtre :


      Imprimer

  • Pour n'imprimer qu'une partie de la page, sélectionnez la partie de votre choix, puis cliquez sur le bouton Imprimer.

 

Utilisation des Raccourcis de l'aide

Pour naviguer dans la fenêtre d'aide, vous pouvez utiliser les raccourcis clavier. Pour plus d'informations à ce sujet, consultez la rubrique Raccourcis clavier de l'Aide.

 

Utilisation des boutons d'aide

Bon nombre de boîtes de dialogue et de fenêtres spécialisées de Netscape sont pourvues d'un bouton "Aide".

Ce bouton vous permet d'afficher des informations détaillées sur la fenêtre dans laquelle il se trouve.
