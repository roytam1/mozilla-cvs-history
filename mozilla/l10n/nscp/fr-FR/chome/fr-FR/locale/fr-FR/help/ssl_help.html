Versions du protocole SSL

Le protocole SSL (Secure Sockets Layer) définit les règles régissant l'authentification mutuelle entre un site Web et le navigateur et le chiffrement des informations qu'ils échangent. Le protocole TLS (Transport Layer Security) est un standard IETF fondé sur SSL. TLS 1.0 peut être considéré comme SSL 3.1.

Vous devriez normalement laisser ces trois cases cochées pour garantir la compatibilité du navigateur avec les anciens et les nouveaux serveurs Web :

  • Activer SSL version 2 : Permet aux anciens serveurs Web de fonctionner avec le navigateur.
  • Activer SSL version 3 : Permet aux nouveaux serveurs Web de fonctionner avec le navigateur.
  • Activer TLS : Permet aux serveurs Web compatibles TLS d'exploiter ce protocole.

Pour activer ou désactiver des codes SSL individuels, cliquez sur Modifier les chiffres :

  • Modifier les chiffres : N'essayez pas de modifier les codes SSL individuellement à moins de savoir exactement ce que vous faites et d'en avoir discuté avec votre administrateur système. Pour plus d'informations, voir Modification des codes SSL.

Remarque importante concernant TLS : Certains serveurs qui ne mettent pas correctement en œuvre SSL ne peuvent pas négocier la prise de contact SSL avec le logiciel client (tel que le navigateur) qui prend en charge TLS. Ces serveurs sont dits "TLS intolérants".

Lorsque l'option Activer TLS du panneau de préférences SSL est sélectionnée, le navigateur tente d'utiliser le protocole TLS lorsqu'il établit des connexions protégées avec un serveur. Si cette connexion échoue parce que le serveur est "TLS intolérant", le navigateur se replie sur l'utilisation de SSL 3.0.

 

Avertissements SSL

Il est facile de savoir si le site Web affiché utilise une connexion chiffrée. Dans ce cas, l'icône représentant un cadenas dans l'angle inférieur droit de la fenêtre de Navigator est verrouillée. Dans le cas contraire, elle est déverrouillée.

Si vous souhaitez des avertissements supplémentaires, vous pouvez cocher une ou plusieurs des cases d'avertissement dans le panneau de préférences SSL. Certains trouvent ces avertissements gênants.

Pour activer des avertissements de Navigator, cochez les cases correspondantes :

  • Chargement d'une page qui prend en charge le chiffrement : Sélectionnez cet avertissement si vous souhaitez obtenir un rappel chaque fois que vous chargez une page qui prend en charge le chiffrement.
  • Chargement d'une page qui utilise un chiffrement de bas niveau : Sélectionnez cet avertissement si vous souhaitez obtenir un rappel chaque fois que vous chargez une page qui prend en charge un chiffrement de bas niveau. Le chiffrement de bas niveau est le plus faible utilisant une clé de 40 bits.
  • Sortie d'une page qui prend en charge le chiffrement : Sélectionnez cet avertissement si vous souhaitez obtenir un rappel chaque fois que vous quittez une page qui prend en charge le chiffrement pour passer à une autre qui ne le prend pas en charge.
  • Envoi des données de formulaire d'une page non chiffrée à une page non chiffrée : Sélectionnez cet avertissement si vous souhaitez obtenir un rappel lorsque vous envoyez des données sur une connexion non chiffrée. Si vous envoyez des informations non chiffrées sur Internet, elles peuvent aisément être interceptées par d'autres personnes.
  • Afficher une page semi-chiffrée : Sélectionnez cet avertissement si vous souhaitez être alerté chaque fois que vous affichez une page qui comprend des informations qui ne sont pas chiffrées.

Pour des définitions succinctes, cliquez sur authentification, chiffrement ou certificat.

Modification des codes SSL

Cette section décrit l'utilisation de la boîte de dialogue Modifier les chiffres. Si cette boîte de dialogue n'est pas affichée, procédez comme suit :

  1. Ouvrez le menu Edition et choisissez Préférences.
  2. Dans la catégorie Confidentialité et sécurité, cliquez sur SSL. Si aucune sous-catégorie n'est visible, double-cliquez sur Confidentialité et sécurité pour développer la liste.
  3. Cliquez sur le bouton Modifier les chiffres.

La boîte de dialogue Modifier les chiffres permet de sélectionner ou déselectionner des codes ou algorithmes cryptographiques. Consultez votre administrateur système avant toute modification de ces paramètres.

Chaque case correspond à une suite de codage (un ensemble d'algorithmes cryptographiques).

Pour plus d'informations sur les codes et le chiffrement, consultez les documents en ligne suivants :

  • Introduction to Public-Key Cryptography
  • Introduction to SSL
  • Encryption Technologies Available in NSS 3.4.
