A propos des signatures numériques et du chiffrement

Lorsque vous composez un message, vous avez la possibilité d'y associer votre signature numérique. Une signature numérique permet aux destinataires du message de vérifier sa provenance et de s'assurer qu'il n'a subi aucune falsification depuis son envoi.

Il vous est également possible de chiffrer les messages que vous rédigez. La technique de chiffrement met votre message à l'abri des regards indiscrets sur Internet.

Les techniques de signature et de chiffrement ne sont pas disponibles pour les messages de forum.

Avant de pouvoir signer ou chiffrer un message, il convient de prendre quelques mesures :

  1. Vous devez obtenir un ou plusieurs certificats (équivalents numériques des cartes d'identité). Pour plus d'informations, consultez la rubrique Obtention de votre propre certificat.
  2. Vous devez configurer les paramètres de sécurité de votre compte de messagerie. Pour plus d'informations, consultez la rubrique Configuration de vos paramètres de sécurité .

Une fois ces opérations effectuées, vous pouvez poursuivre la procédure décrite dans la rubrique Signature et chiffrement d'un nouveau message.

Les sections suivantes vous donnent un bref aperçu du fonctionnement des signatures numériques et du chiffrement. Pour obtenir davantage d'informations techniques à ce sujet, consultez le document en ligne intitulé Introduction to Public-Key Cryptography (Présentation de la cryptographie à clé publique).

 

Fonctionnement des signatures numériques

Une signature numérique est un code spécial, propre à chaque message, créé au moyen de la technique de cryptographie à clé publique.

Une signature numérique n'a rien à voir avec une signature manuscrite, bien qu'elle puisse parfois être utilisée à des fins juridiques similaires, par exemple pour la signature d'un contrat.

Deux éléments sont nécessaires à la création d'une signature numérique pour un message envoyé :

  • Un certificat de signature qui vous identifie à cette fin. Chaque fois que vous signez un message, votre certificat de signature y est inclus. Ce certificat comprend une clé publique. La présence du certificat dans le message permet au destinataire de vérifier votre signature numérique.

    • Votre certificat est comparable aux coordonnées (nom et numéro de téléphone) figurant dans l'annuaire téléphonique ; il s'agit d'informations publiques permettant à d'autres personnes de communiquer avec vous.

  • Une clé privée, créée et stockée dans votre ordinateur dès que vous obtenez un certificat.

    • Votre clé privée destinée à un certificat de signature est protégée par votre mot de passe principal. Le programme Netscape ne la divulgue à personne. Le logiciel Messagerie & forums se sert de votre clé privée pour créer une signature numérique unique et vérifiable pour tout message auquel vous attribuez une signature.

 

Mécanisme de chiffrement

Pour chiffrer un message, vous devez disposer d'un certificat de chiffrement pour chacun des destinataires. La clé publique de chaque certificat sert à chiffrer le message pour ce destinataire.

Si vous ne disposez de certificats pour aucun des destinataires, il s'avérera impossible de chiffrer le message.

Pour déchiffrer le message, le logiciel du destinataire utilise la clé privée de ce dernier, qui reste sur cet ordinateur.

Obtention des certificats d'autres personnes

Chaque fois que vous envoyez un message signé numériquement, votre certificat de chiffrement y est inclus. Dès lors, l'une des manières les plus simples d'obtenir le certificat d'une autre personne consiste à lui demander de vous envoyer un message signé numériquement.

Lorsque vous recevez un message de ce type, le certificat de la personne est stocké automatiquement par le Gestionnaire de certificats, qui fait partie intégrante du navigateur qui assure le suivi des certificats. Cela se révèle particulièrement utile dans la mesure où vous devez disposer d'un certificat pour chaque destinataire d'un message que vous souhaitez transmettre sous forme chiffrée.

Pour obtenir des certificats, une autre solution consiste à les rechercher dans un annuaire public, tel que les "annuaires téléphoniques" gérés par bon nombre d'entreprises.

Il est également possible d'opter pour une recherche automatique des certificats. Pour configurer cette fonction, vous devez accéder aux Préférences de Messagerie & Forums – Adressage ou aux Paramètres de compte de messagerie/forum – Adressage . Vous pouvez y définir la collecte automatique d'adresses électroniques dans un annuaire.

Lorsque vous utilisez un compte configuré pour rechercher des adresses dans un annuaire, c'est également dans cet annuaire que seront recherchés les certificats correspondants lorsque vous tenterez d'envoyer un message chiffré à un ou plusieurs destinataires pour lesquels vous ne disposez pas de certificat.

Le logiciel y recherchera également les certificats manquants lorsque vous ouvrirez le menu déroulant situé sous l'icône de sécurité dans la fenêtre de composition et choisirez l'option Sécurité des messages.

Configuration des paramètres de sécurité

Après avoir obtenu un ou plusieurs certificats de messagerie, vous devez spécifier ceux que vous souhaitez utiliser pour la signature et le chiffrement des messages.

Pour plus d'informations sur l'obtention de certificats de messagerie, consultez la rubrique Obtention de votre propre certificat.

Pour spécifier les certificats de signature et de chiffrement à utiliser pour un compte donné, ouvrez la fenêtre de messagerie :

  1. Ouvrez le menu Edition et sélectionnez Paramètres de compte de messagerie/forum.
  2. Cliquez sur Sécurité sous le nom du compte de messagerie dont vous souhaitez configurer les paramètres de sécurité.
  3. Cliquez sur Sélectionner dans la section Signature numérique. Il se peut que le logiciel vous invite à saisir un mot de passe principal avant de poursuivre la procédure.

    4. La boîte de dialogue qui s'affiche alors vous permet de faire votre choix parmi les certificats de signature disponibles.

  4. Sélectionnez le certificat de signature à utiliser, puis cliquez sur OK.
  5. Effectuez la même procédure dans la section Chiffrement, en l'occurrence : cliquez sur le bouton Sélectionner, sélectionnez le certificat de chiffrement à utiliser, puis cliquez sur OK.

    6. Dans certains cas, il peut être possible de spécifier le même certificat dans les deux sections ; pour savoir ce qu'il en est exactement dans votre cas, consultez votre administrateur système.

Vous pouvez éventuellement indiquer que vous souhaitez signer ou chiffrer tous les messages envoyés à partir d'un compte spécifique. Ces paramètres propres à un compte sont donnés uniquement dans un souci de commodité. Vous pouvez remplacer les paramètres par défaut définis pour chaque message.

Pour configurer les paramètres de signature et de chiffrement par défaut, ouvrez le panneau de sécurité du compte (voir procédure ci-dessus) et sélectionnez vos paramètres :

  • Sous Signature numérique :
    • Signer numériquement les messages : Lorsque cette case est cochée, tous les messages envoyés à partir de ce compte sont signés numériquement, sauf indication contraire avant l'envoi. Pour désactiver cette option par défaut, désélectionnez la case à cocher.
  • Sous Chiffrement (sélectionner une option) :
    • Jamais : Lorsque cette option est sélectionnée, les messages envoyés à partir de ce compte ne sont pas chiffrés, sauf indication contraire avant l'envoi.
    • Obligatoire : Lorsque cette option est sélectionnée, tous les messages envoyés à partir de ce compte sont chiffrés, à la condition expresse que vous disposiez de certificats valides pour chaque destinataire du message. Si vous ne disposez pas de tous les certificats nécessaires, il s'avérera impossible d'envoyer le message, à moins que vous ne désactiviez le chiffrement.

Dès que vous avez configuré vos paramètres de sécurité de messagerie, cliquez sur OK pour les confirmer.

Signature et chiffrement d'un nouveau message

Avant de pouvoir signer ou chiffrer un message, vous devez obtenir au moins un certificat de messagerie et configurer correctement vos paramètres de sécurité de messagerie. Pour obtenir des informations générales sur ces tâches, consultez la rubrique A propos des signatures numériques et du chiffrement.

Les paramètres définis dans le panneau Paramètres de compte de messagerie/forum – Sécurité déterminent la configuration par défaut appliquée à chaque fenêtre de composition ouverte en vue de rédiger un courrier électronique.

Pour ouvrir une fenêtre de composition, ouvrez la fenêtre de messagerie et cliquez sur le bouton Composer. La présence ou l'absence des icônes suivantes dans le coin inférieur droit de la fenêtre vous permet d'identifier immédiatement les paramètres de sécurité par défaut :

 Le message sera signé numériquement (pour autant que vous disposiez d'un certificat de messagerie valide qui vous identifie).
 Le message sera chiffré (pour autant que vous disposiez de certificats valides pour tous les destinataires).

Pour activer ou désactiver ces paramètres, cliquez sur la flèche située sous l'icône de sécurité dans la barre d'outils de messagerie (partie supérieure de la fenêtre). Sélectionnez ensuite l'élément de votre choix dans la liste déroulante :

  • Aucun chiffrement : Cette option vous permet de désactiver le chiffrement pour ce message. Dès lors, le message ne sera pas chiffré lorsqu'il sera envoyé sur Internet.
  • Chiffrement requis : Cette option vous permet d'activer le chiffrement pour ce message. Dès lors, le message sera envoyé sous forme chiffrée. Cependant, l'envoi du message implique que vous disposiez de certificats valides pour tous les destinataires.
  • Signer numériquement : Cette option vous permet d'activer ou de désactiver la signature numérique de ce message. Une marque de sélection indique que le message sera signé.
  • Sécurité des messages : Cliquez sur cette option pour afficher des informations détaillées sur l'état de sécurité du message ; cela vous permettra, par exemple, de déterminer s'il y a lieu d'obtenir un certificat pour l'un des destinataires.

Pour afficher des informations détaillées sur l'état de sécurité du message, vous pouvez également cliquer sur l'icône représentant une clé ou un cadenas, ainsi qu'il est décrit dans la rubrique Sécurité des messages – Fenêtre de composition.

Lecture de messages signés et chiffrés

Lorsque vous consultez un message chiffré ou signé dans la fenêtre de messagerie, les icônes suivantes, situées dans le coin supérieur droit de l'en-tête de message, indiquent l'état de sécurité du message :

 Le message est signé numériquement et a été validé. En cas de problème avec la signature, le crayon est cassé.
 Le message est signé, mais il possède une pièce jointe volumineuse qui n'a pas encore été téléchargée à partir du serveur IMAP. En conséquence, la signature ne peut pas être validée. Cliquez sur l'icône pour télécharger la pièce jointe et valider la signature.
 Le message est chiffré. En cas de problème avec le chiffrement, la clé est cassée.

Pour plus d'informations sur la validation des certificats, consultez la rubrique Contrôle de la validation.

Pour obtenir des informations plus détaillées sur la sécurité du message, cliquez sur l'icône représentant une clé ou un cadenas, ou suivez les instructions de la rubrique Sécurité des messages – Message reçu.

Sécurité des messages – Fenêtre de composition

Cette section décrit la fenêtre Sécurité des messages accessible pour tout message en cours de composition. Le cas échéant, cliquez sur l'icône de sécurité dans la barre d'outils de la fenêtre de composition.

La fenêtre Sécurité des messages décrit la manière dont votre message sera envoyé :

  • Signé numériquement : Cette ligne indique si une signature numérique sera, ou non, appliquée à votre message. Trois possibilités s'offrent à vous :
    • Oui : La fonction de signature numérique a été activée pour ce message, vous disposez d'un certificat valide qui vous identifie et le message peut être signé.
    • Non : La fonction de signature numérique a été désactivée pour ce message.
    • Impossible : La fonction de signature numérique a été activée pour ce message. Cependant, vous ne disposez d'aucun certificat valide qui vous identifie ou un autre problème rend la signature impossible.
  • Chiffré : Cette ligne indique si votre message sera chiffré ou non. Trois possibilités s'offrent à vous :
    • Oui : La fonction de chiffrement a été activée pour ce message, vous disposez de certificats valides pour tous les destinataires répertoriés et le message peut être chiffré.
    • Non : Le chiffrement a été désactivé ou ne peut pas être effectué pour ce message.
    • Impossible : Le chiffrement a été activé pour ce message. Cependant, il manque un certificat valide pour au moins un des destinataires répertoriés, aucun destinataire n'est répertorié ou un autre problème rend impossible le chiffrement.

La fenêtre Sécurité des messages dresse également la liste des certificats disponibles pour les destinataires de votre message :

  • Afficher. Pour afficher des informations détaillées sur un certificat de la liste, sélectionnez son nom, puis cliquez sur Afficher.

Pour plus d'informations sur l'obtention de certificats et la configuration des paramètres de sécurité des messages, consultez la rubrique Signature et chiffrement des messages

Pour indiquer vos préférences en matière de chiffrement ou de signature d'un message, cliquez sur la flèche située à côté du bouton Sécurité dans la fenêtre de composition, puis sélectionnez les options de votre choix.

Pour définir les paramètres de chiffrement et de signature par défaut pour tous les messages, consultez la rubrique Paramètres de compte de messagerie/forum – Sécurité

Sécurité des messages – Message reçu

Cette section décrit la fenêtre Sécurité des messages accessible pour tout message reçu. Si vous n'avez pas encore ouvert la fenêtre de sécurité d'un message reçu, procédez comme suit :

  1. Dans la fenêtre de messagerie, sélectionnez le message pour lequel vous souhaitez afficher des informations de sécurité.
  2. Ouvrez le menu Afficher et sélectionnez Infos de sécurité du message.

La fenêtre Sécurité des messages affiche les informations suivantes :

  • Signature numérique. La section supérieure de cette fenêtre indique si le message est signé numériquement et, le cas échéant, si la signature est valide.

    • Si la validation a échoué alors que le protocole OCSP était activé, vérifiez les paramètres OCSP définis dans la section Préférences de confidentialité et sécurité – Validation. Si vous n'êtes pas rompu à l'utilisation du protocole OCSP, confirmez les paramètres avec votre administrateur système. S'ils sont corrects, il se peut que le service OCSP connaisse un problème ou que le certificat utilisé pour créer la signature ne soit plus valide.

    Si la signature n'est plus valide en raison d'un problème affectant les paramètres de confiance du certificat, vous pouvez utiliser le Gestionnaire de certificats pour afficher ou modifier ces paramètres.

  • Afficher le certificat de signature. Si le message est signé, cliquez sur ce bouton pour afficher le certificat utilisé pour le signer.
  • Chiffrement. La section inférieure indique si le message est chiffré et signale les éventuels problèmes de chiffrement.
    • Si le contenu du message a été modifié pendant la transmission, demandez à l'expéditeur de le renvoyer. Les modifications peuvent avoir été causées par des problèmes réseau.
    • Si aucune copie de votre propre certificat (utilisé par l'expéditeur pour chiffrer le message) n'est disponible sur votre ordinateur, il est impossible de récupérer la clé privée nécessaire au déchiffrement du message. Dans ce cas, la seule solution consiste à importer une copie de sauvegarde de votre certificat et de sa clé privée (pour plus d'informations à ce sujet, consultez la rubrique Vos certificats). Si vous ne pouvez accéder à une copie de sauvegarde du certificat, vous ne pourrez déchiffrer le message.
