Digitale Signaturen und Verschlüsselung

Wenn Sie eine eMail-Nachricht erstellen, können Sie eine digitale Signatur anfügen. Anhand einer digitalen Signatur können Empfänger der Nachricht feststellen, dass die Nachricht wirklich von Ihnen stammt und seit dem Senden nicht manipuliert wurde.

Beim Erstellen einer eMail-Nachricht haben Sie auch die Möglichkeit, sie zu verschlüsseln. Bei einer Verschlüsselung ist es für jeden anderen als den beabsichtigen Empfänger äußerst schwierig, während der Übertragung im Internet die Nachricht zu lesen.

Für Nachrichten von Diskussionsforen sind Signaturen und Verschlüsselung nicht verfügbar.

Damit Sie eine Nachricht signieren oder verschlüsseln können, müssen Sie zuerst folgende Schritte ausführen:

  1. Einholen eines oder mehrerer Zertifikate (gewissermaßen ein digitaler Ausweis). Weitere Informationen finden Sie unter Beantragen eines eigenen Zertifikats.
  2. Konfigurieren der Sicherheitseinstellungen für Ihr eMail-Konto. Weitere Informationen finden Sie unter Konfigurieren der Sicherheitseinstellungen.

Wenn Sie diese Schritte ausgeführt haben, können Sie den Anweisungen unter Signieren und Verschlüsseln einer neuen Nachricht folgen.

Die folgenden Abschnitte liefern eine kurze Übersicht zur Funktionsweise von digitalen Signaturen und der Verschlüsselung. Weitere Informationen zu diesem Thema finden Sie im Onlinedokument Introduction to Public-Key Cryptography.

 

Funktionsweise digitaler Signaturen

Eine digitale Signatur ist ein spezieller Code, der für jede Nachricht eindeutig ist und mithilfe der Public-Key-Kryptografie erstellt wird.

Eine digitale Signatur hat nichts mit einer handschriftlichen Unterschrift zu tun, auch wenn sie gelegentlich für dieselben rechtlichen Zwecke eingesetzt wird.

Zum Erstellen einer digitalen Signatur für eine eMail-Nachricht sind zwei Voraussetzungen erforderlich:

  • Ein Signaturzertifikat, das Sie für diesen Zweck identifiziert. Bei jedem Signieren einer Nachricht wird Ihr Signaturzertifikat in die Nachricht aufgenommen. Das Zertifikat enthält einen öffentlichen Schlüssel. Anhand des Zertifikats in der Nachricht kann der Empfänger Ihre digitale Signatur überprüfen.

    • Ihr Zertifikat ist in etwa so etwas wie Ihr Namen und Ihre Telefonnummer in einem Telefonbuch. Es handelt sich dabei um öffentliche Informationen, die anderen die Kommunikation mit Ihnen erleichtern.

  • Ein privater Schlüssel, der beim ersten Beantragen eines Zertifikats auf Ihrem Computer erstellt und gespeichert wird.

    • Ihr privater Schlüssel zum Signieren eines Zertifikats wird durch Ihr Master-Passwort geschützt, das Netscape für niemanden zugänglich macht. Die eMail & Diskussionsforen-Software erstellt anhand Ihres privaten Schlüssels eine eindeutige, überprüfbare digitale Signatur für jede Nachricht, die Sie signieren.

 

Funktionsweise der Verschlüsselung

Zum Verschlüsseln einer eMail-Nachricht benötigen Sie für jeden Empfänger der Nachricht ein Verschlüsselungszertifikat. Der öffentliche Schlüssel jedes Zertifikats wird zum Verschlüsseln der Nachricht für diesen Empfänger verwendet.

Wenn ein Zertifikat für mindestens einen der Empfänger nicht vorhanden ist, kann die Nachricht nicht verschlüsselt werden.

Die Software des Empfängers verwendet dessen privaten Schlüssel, der auf dem Computer dieser Person verbleibt, zum Entschlüsseln der Nachricht.

Einholen fremder Zertifikate

Bei jedem Senden einer digital signierten Nachricht wird Ihr Verschlüsselungszertifikat automatisch in die Nachricht aufgenommen. Die einfachste Methode, das Zertifikat einer anderen Person zu erhalten, besteht folglich darin, dass Ihnen die Person eine digital signierte Nachricht sendet.

Wenn Sie diese Nachricht erhalten, wird das Zertifikat der Person automatisch vom Zertifikats-Manager gespeichert, einer Browser-Komponente zur Verwaltung von Zertifikaten. Das ist nützlich, weil Sie ein Zertifikat für jeden Empfänger einer eMail-Nachricht benötigen, die Sie verschlüsselt senden möchten.

Eine weitere Methode zum Einholen von Zertifikaten besteht darin, sie in einem öffentlichen Verzeichnis zu suchen, wie etwa in den "Telefonbuch"-Verzeichnissen, die von vielen Unternehmen geführt werden.

Zertifikate können auch automatisch gesucht werden. Diese Funktion wird von Einstellungen für eMail & Diskussionsforen – Adressieren oder eMail & Diskussionsforen-Kontoeinstellungen – Adressieren gesteuert. Hier kann konfiguriert werden, dass in einem Verzeichnis nach den eMail-Adressen der Empfänger gesucht wird.

Wenn Sie ein Konto verwenden, das für die Suche von Adressen in einem Verzeichnis konfiguriert wurde, wird dieses Verzeichnis auch nach passenden Zertifikaten durchsucht, sobald Sie versuchen, eine verschlüsselte Nachricht an einen oder mehrere Empfänger zu senden, für die noch keine Zertifikate gespeichert sind.

Das Verzeichnis wird auch nach fehlenden Zertifikaten durchsucht, wenn Sie im Fenster in dem Dropdown-Menü unter dem Sicherheitssymbol auf Sicherheitsinformationen anzeigen klicken.

Konfigurieren der Sicherheitseinstellungen

Wenn Sie eMail-Zertifikate eingeholt haben, müssen Sie die Zertifikate angeben, die Sie für das Signieren und Verschlüsseln von Nachrichten verwenden möchten.

Weitere Informationen zum Einholen von eMail-Zertifikaten finden Sie unter Beantragen eines eigenen Zertifikats.

So geben Sie die zu verwendenden Signatur- und Verschlüsselungszertifikate für ein bestimmtes Konto an (ausgehend vom Mail-Fenster):

  1. Klicken Sie im Menü Bearbeiten auf eMail und Diskussionsforen-Kontoeinstellungen.
  2. Klicken Sie unter dem Namen des eMail-Kontos, dessen Sicherheitseinstellungen Sie konfigurieren möchten, auf Sicherheit.
  3. Klicken Sie unter Digitale Unterschrift auf Wählen. (Möglicherweise werden Sie nach Ihrem Master-Passwort gefragt, bevor Sie fortfahren können.)

    4. In einem Dialogfeld können Sie eines der verfügbaren Signaturzertifikate auswählen.

  4. Wählen Sie das Signaturzertifikat aus, das Sie verwenden möchten, und klicken Sie auf OK.
  5. Führen Sie dieselben Schritte für die Verschlüsselung aus: Klicken Sie auf die Schaltfläche Wählen, wählen Sie das Verschlüsselungszertifikat aus und klicken Sie auf OK.

    6. In machen Fällen können Sie bei der Verschlüsselung dasselbe Zertifikat auswählen wie bei digitalen Signaturen. Wenden Sie sich an Ihren Systemadministrator, um sicherzugehen.

Sie können optional auch angeben, dass Sie in der Regel alle Nachrichten signieren oder verschlüsseln möchten, die Sie mit einem bestimmten Konto senden. Diese kontenspezifischen Einstellungen dienen lediglich der Vereinfachung. Sie können die Standardeinstellungen für einzelne Nachrichten überschreiben.

Beginnen Sie im Fenster Sicherheit für das Konto (wie oben beschrieben) und wählen Sie die Einstellungen für Ihre Standardsignatur und -verschlüsselung folgendermaßen aus:

  • Für digitale Signaturen:
    • Nachrichten digital unterzeichnen: Wenn dieses Kontrollkästchen aktiviert ist, werden alle mit diesem Konto gesendeten Nachrichten digital signiert, falls Sie vor dem Senden nichts anderes einstellen. Um diese Standardeinstellung auszuschalten, deaktivieren Sie das Kontrollkästchen.
  • Für die Verschlüsselung:
    • Nie: Wenn diese Option aktiviert ist, werden alle mit diesem Konto gesendeten Nachrichten verschlüsselt, falls Sie vor dem Senden der Nachricht nichts anderes einstellen.
    • Erforderlich: Wenn diese Option aktiviert ist, werden alle mit diesem Konto gesendeten Nachrichten verschlüsselt – jedoch nur, wenn Sie für jeden Empfänger der Nachricht ein gültiges Zertifikat besitzen. Falls Sie nicht alle erforderlichen Zertifikate besitzen, kann die Nachricht erst gesendet werden, wenn Sie die Verschlüsselung für diese Nachricht deaktivieren.

Wenn Sie mit dem Konfigurieren der Sicherheitseinstellungen für eMail fertig sind, klicken Sie zur Bestätigung auf OK.

Signieren und Verschlüsseln einer neuen Nachricht

Damit Sie eine Nachricht digital signieren oder verschlüsseln können, müssen Sie mindestens ein eMail-Zertifikat einholen und die Sicherheitseinstellungen der eMail richtig konfigurieren. Weitere Informationen zu diesen Aufgaben finden Sie unter Digitale Signaturen und Verschlüsselung.

Die in eMail & Diskussionsforen-Kontoeinstellungen – Adressieren festgelegten Einstellungen bestimmen die Standardeinstellungen für jedes neue Fenster, das Sie zum Verfassen einer eMail öffnen.

Um ein Fenster zu öffnen, klicken Sie im Mail-Fenster auf Verfassen. Sie erkennen die Standardsicherheitseinstellungen ganz einfach am Vorhandensein oder Fehlen dieser Symbole in der unteren rechten Ecke des Fensters.

 Die Nachricht wird digital signiert (vorausgesetzt Sie verfügen über ein gültiges eMail-Zertifikat, das Sie identifiziert).
 Die Nachricht wird verschlüsselt (vorausgesetzt Sie verfügen über gültige Zertifikate für alle Empfänger).

Um diese Einstellungen zu aktivieren oder zu deaktivieren, klicken Sie auf den Pfeil unter dem Sicherheitssymbol in der Mail-Symbolleiste oben im Fenster. Wählen Sie dann den gewünschten Eintrag in der Dropdown-Liste aus:

  • Diese Nachricht nicht verschlüsseln: Wählen Sie diese Option aus, um die Verschlüsselung für diese Nachricht zu deaktivieren. Die Nachricht wird dann beim Versenden über das Internet nicht verschlüsselt.
  • Diese Nachricht verschlüsseln: Wählen Sie diese Option aus, um die Verschlüsselung für diese Nachricht zu aktivieren. Die Nachricht wird verschlüsselt gesendet. Sie kann jedoch erst gesendet werden, wenn Sie gültige Zertifikate für alle Empfänger besitzen.
  • Diese Nachricht digital unterzeichnen : Wählen Sie diese Option aus, um digitale Signaturen für diese Nachricht zu aktivieren oder zu deaktivieren. Ein Häkchen gibt an, dass die Nachricht signiert wird.
  • Sicherheitsinformationen anzeigen: Wählen Sie diese Option aus, um genaue Informationen zum Sicherheitsstatus der Nachricht anzuzeigen, beispielsweise, um festzustellen, ob für einen der Empfänger ein Zertifikat eingeholt werden muss.

Um genaue Informationen zum Sicherheitsstatus der Nachricht anzuzeigen, können Sie auch wie unter Nachrichtensicherheit – Neue Nachricht verfassen beschrieben auf das Schlüssel- oder Schlosssymbol klicken.

Lesen von unterzeichneten und verschlüsselten Nachrichten

Beim Anzeigen einer signierten oder verschlüsselten Nachricht im Mail-Fenster geben diese Symbole in der oberen rechten Ecke der Nachrichtenkopfzeile den Sicherheitsstatus der Nachricht an:

 Die Nachricht ist digital signiert und wurde ausgewertet. Falls es ein Problem mit der Signatur gibt, ist der Stift zerbrochen.
 Die Nachricht ist signiert, hat jedoch eine große Anlage, die noch nicht vom IMAP-Server gedownloadet wurde. Deshalb kann die Signatur nicht ausgewertet werden. Klicken Sie auf das Symbol, um die Anlage zu downloaden und die Signatur auszuwerten.
 Die Nachricht ist verschlüsselt. Falls es ein Problem mit der Verschlüsselung gibt, ist der Schlüssel zerbrochen.

Weitere Informationen zur Zertifikatsauswertung finden Sie unter Steuern der Auswertung.

Um genaue Informationen zum Sicherheitsstatus der Nachricht anzuzeigen, klicken Sie auf das Schlüssel- oder Schlosssymbol oder folgen Sie den Anweisungen unter Nachrichtensicherheit – Empfangene Nachricht.

Nachrichtensicherheit – Neue Nachricht verfassen

In diesem Abschnitt wird das Fenster Nachrichtensicherheit beschrieben, das Sie für jede Nachricht öffnen können, die Sie verfassen. Wenn Sie das Fenster Nachrichtensicherheit noch nicht geöffnet haben, klicken Sie auf das Sicherheitssymbol auf der Symbolleiste des Fensters.

Im Fenster Nachrichtensicherheit wird beschrieben, wie Ihre Nachricht gesendet wird:

  • Digital unterschrieben: In dieser Zeile steht, ob Ihre Nachricht signiert wird. Es gibt drei Möglichkeiten:
    • Ja: Digitale Signaturen wurden für diese Nachricht aktiviert, Sie verfügen über ein gültiges Zertifikat, das Sie identifiziert und die Nachricht kann signiert werden.
    • Nein: Digitale Signaturen wurden für diese Nachricht deaktiviert.
    • Nicht möglich: Digitale Signaturen wurden für diese Nachricht aktiviert. Es ist jedoch kein gültiges Zertifikat verfügbar, das Sie für diesen Zweck identifiziert, oder ein anderes Problem macht das Signieren unmöglich.
  • Verschlüsselt: In dieser Zeile steht, ob Ihre Nachricht verschlüsselt wird. Es gibt drei Möglichkeiten:
    • Ja: Die Verschlüsselung wurde für diese Nachricht aktiviert, gültige Zertifikate sind für alle aufgeführten Empfänger verfügbar und die Nachricht kann verschlüsselt gesendet werden.
    • Nein: Die Verschlüsselung wurde deaktiviert oder ist bei dieser Nachricht nicht möglich.
    • Nicht möglich: Die Verschlüsselung wurde für diese Nachricht aktiviert. Es ist jedoch nicht einmal für einen der aufgeführten Empfänger ein gültiges Zertifikat verfügbar oder es sind keine Empfänger aufgeführt oder ein anderes Problem macht das Verschlüsseln unmöglich.

Das Fenster Nachrichtensicherheit führt ebenfalls die verfügbaren Zertifikate für die Empfänger der Nachricht auf:

  • Anzeigen: Um die Details für ein Zertifikat in der Liste anzuzeigen, wählen Sie seinen Namen aus und klicken Sie auf Anzeigen.

Weitere Informationen zum Einholen von Zertifikaten und zum Konfigurieren der Nachrichtensicherheit finden Sie unter Signieren und Verschlüsseln von Nachrichten.

Um die Einstellungen für die Signatur und Verschlüsselung für einzelne Nachrichten festzulegen, klicken Sie im Fenster auf den Pfeil neben der Schaltfläche Sicherheit, und wählen Sie die gewünschten Optionen aus.

Informationen zum Angeben der Standardsignatur- und -verschlüsselungseinstellungen finden Sie unter eMail und Diskussionsforen-Kontoeinstellungen – Sicherheit

Nachrichtensicherheit – Empfangene Nachricht

In diesem Abschnitt wird das Fenster Nachrichtensicherheit beschrieben, das Sie für jede Nachricht öffnen können, die Sie empfangen haben. Wenn Sie das Fenster Nachrichtensicherheit noch nicht für eine empfangene Nachricht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Wählen Sie die Nachricht, für die Sie die Sicherheitsinformationen anzeigen möchten, im Mail-Fenster aus.
  2. Klicken Sie im Menü Anzeigen auf Info zu Nachrichtensicherheit.

Das Fenster Info zu Nachrichtensicherheit zeigt folgende Informationen an:

  • Digitale Signatur. Im oberen Bereich wird beschrieben, ob die Nachricht digital signiert ist und, wenn ja, ob die Signatur gültig ist.

    • Falls die Auswertung fehlschlug, während OCSP aktiviert war, überprüfen Sie die OCSP-Einstellungen unter Einstellungen für Privatsphäre und Sicherheit – Auswertung. Falls Sie mit OCSP nicht vertraut sind, wenden Sie sich an Ihren Systemadministrator. Wenn die Einstellungen stimmen, besteht womöglich ein Problem mit dem OCSP-Dienst oder das Zertifikat ist nicht länger gültig, mit dem die Signatur erstellt wurde.

    Falls die Signatur wegen eines Problems mit den Vertrauenseinstellungen des Zertifikats ungültig ist, können Sie diese Einstellungen mithilfe des Zertifikats-Managers anzeigen oder bearbeiten.

  • Unterschriftenzertifikat anzeigen: Wenn die Nachricht signiert ist, klicken Sie auf diese Schaltfläche, um das Zertifikat anzuzeigen, mit dem sie signiert wurde.
  • Verschlüsselung: Im unteren Bereich wird angegeben, ob die Nachricht verschlüsselt ist und eventuell Verschlüsselungsprobleme auftraten.
    • Falls der Nachrichteninhalt bei der Übermittlung geändert wurde, sollten Sie den Absender bitten, sie noch einmal zu senden. Die Änderungen könnten durch Netzwerkprobleme verursacht worden sein.
    • Falls eine Kopie Ihres eigenen Zertifikats (das vom Absender zum Verschlüsseln der Nachricht verwendet wird) auf Ihrem Computer nicht verfügbar ist, kann der zum Entschlüsseln der Nachricht benötigte private Schlüssel nicht abgerufen werden. Die einzige Lösung ist, eine Sicherheitskopie Ihres Zertifikats und des zugehörigen privaten Schlüssels zu importieren (siehe Ihre Zertifikate ). Falls Sie auf keine Sicherheitskopie des Zertifikats zugreifen können, ist die Entschlüsselung der Nachricht nicht möglich.
