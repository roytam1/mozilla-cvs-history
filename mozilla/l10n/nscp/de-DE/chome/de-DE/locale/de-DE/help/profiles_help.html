Verwalten von Profilen

Wenn Sie das Internet sowohl zu Hause als auch am Arbeitsplatz nutzen, verwenden Sie wahrscheinlich unterschiedliche Lesezeichen, Einstellungen, Adressbücher, eMail-Konten, Seitenleisten usw. Ebenso möchten möglicherweise Familienmitglieder, die eine Kopie derselben Browsersoftware verwenden, separate Internet-Identitäten pflegen.

Mithilfe des Profil-Managers können Sie verschiedene Profile erstellen, die jeweils eigene Lesezeichen, eMail-Einstellungen usw. haben. Bei der Installation der Browsersoftware wird automatisch ein Standardprofil angelegt. Wenn Sie weitere Profile erstellen, werden Sie bei jedem Start des Browsers gefragt, welches Profil Sie verwenden möchten.

In diesem Abschnitt werden folgende Themen behandelt:

Erstellen eines neuen Profils

Löschen oder Umbenennen eines Profils

 

Erstellen eines neuen Profils

So erstellen Sie ein Profil:

  1. Beenden Sie den Browser. Ein neues Profil kann nicht angelegt werden, solange Navigator, Mail oder eine andere Komponente der Browsersoftware aktiv ist.
  2. Öffnen Sie den Profil-Manager:
    • In Windows klicken Sie im Startmenü auf Programme, Netscape 7.0 und dann auf Profil-Manager.
    • In Mac OS öffnen Sie den Ordner Netscape und doppelklicken auf den Profil-Manager.
    • In Linux geben Sie die folgende Befehlszeile ein: ./netscape -profilemanager
  3. Klicken Sie auf Profil erstellen, lesen Sie die Profilbeschreibung und klicken Sie dann auf Weiter.
  4. Geben Sie einen Profilnamen ein. Dies kann der Name des Profilbenutzers sein oder auch ein Name, der auf den Verwendungszweck des Profils hinweist, beispielsweise "Schule".
  5. Übernehmen Sie den Standardordner für das neue Profil, oder klicken Sie auf die Schaltfläche Ordner auswählen, um zum gewünschten Ordner zu wechseln.
  6. Wenn Sie Sprachpakete oder regionalen Inhaltspakete installiert haben, klicken Sie auf Sprache wählen, und wählen Sie die gewünschte Sprache/das gewünschte Inhaltspaket aus.
  7. Klicken Sie auf Fertig stellen.

Dadurch kehren Sie zum Anfangsbildschirm des Profil-Managers zurück, in dem das neue Profil aufgelistet und markiert ist.

Um Ihren Browser für das neue Profil einzurichten, klicken Sie auf Netscape 7.0 starten. Sie werden gebeten, einen Benutzernamen (falls Sie bereits einen besitzen) für die Verwendung mit Netscape Webmail und Netscape Instant Messenger  zu registrieren. Falls Sie noch keinen Benutzernamen besitzen, klicken Sie auf Jetzt registrieren, um einen Namen zu erhalten.

Wenn Sie noch keinen Benutzernamen registrieren möchten, klicken Sie auf Abbrechen. Das Programm wird geöffnet. Um einen Benutzernamen zu registrieren, während Netscape ausgeführt wird, klicken Sie im Menü Hilfe auf Netscape registrieren.

Wenn Sie das neue Profil zum ersten Mal verwenden, werden Sie bei der Ausführung von Aufgaben, die das Profil benötigen (z. B. beim Abruf von eMail-Nachrichten), nach weiteren Informationen gefragt.

 

Löschen oder Umbenennen eines Profils

Profile, die Sie nicht mehr benötigen, sollten Sie löschen. So löschen Sie ein vorhandenes Profil oder benennen es um:

  1. Beenden Sie den Browser. Profile können nicht angelegt, gelöscht oder umbenannt werden, solange Navigator, Mail oder eine andere Komponente der Browsersoftware aktiv ist.
  2. Öffnen Sie den Profil-Manager:
    • In Windows klicken Sie im Startmenü auf Programme, Netscape 7.0 und dann auf Profil-Manager.
    • In Mac OS öffnen Sie den Ordner Netscape und doppelklicken auf den Profil-Manager.
    • In Linux geben Sie die folgende Befehlszeile ein: ./netscape -profile
  3. Um ein Profil zu löschen, wählen Sie dessen Namen aus und klicken auf Löschen.
  4. Um ein Profil umzubenennen, wählen Sie dessen Namen aus, klicken Sie auf Profil umbenennen, und folgen Sie den Anleitungen.
