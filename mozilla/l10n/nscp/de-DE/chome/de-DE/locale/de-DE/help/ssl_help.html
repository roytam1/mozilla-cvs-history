SSL-Protokollversionen

Das Protokoll Secure Sockets Layer (SSL) definiert Regeln für die wechselseitige Authentifizierung zwischen einer Website und einer Browsersoftware sowie für die Verschlüsselung der übertragenen Daten. Das Protokoll Transport Layer Security (TLS) ist ein IETF-Standard, der auf SSL aufbaut. TLS 1.0 kann als SSL 3.1 betrachtet werden.

In der Regel sollten Sie diese drei Kontrollkästchen aktiviert lassen, damit der Browser auf ältere und neuere Webserver zugreifen kann.

  • SSL-Version 2 aktivieren: Ermöglicht den Zugriff des Browsers auf ältere Webserver.
  • SSL-Version 3 aktivieren: Ermöglicht den Zugriff des Browsers auf neuere Webserver.
  • TLS aktivieren: Ermöglicht Webservern TLS zu nutzen, wenn sie es unterstützen.

Um einzelne Schlüssel zu aktivieren oder zu deaktivieren, klicken Sie auf die Schaltfläche Schlüssel bearbeiten.

  • Schlüssel bearbeiten Bearbeiten Sie einzelne Schlüssel nur, wenn Sie genau wissen, was Sie damit bezwecken möchten und die Änderungen mit Ihrem Systemadministrator besprochen haben. Weitere Informationen finden Sie unter Bearbeiten von Schlüsseln.

Wichtiger Hinweis in Bezug auf TLS: Einige Server implementieren SSL nicht korrekt, sodass der SSL-Handshake mit Clientsoftware, die TLS unterstützt (z. B. dem Browser), nicht funktioniert. Derartige Server werden als "TLS-intolerant" bezeichnet.

Wenn im Fenster SSL-Einstellungen die Option TLS aktivieren ausgewählt ist, versucht der Browser das TLS-Protokoll beim Aufbau einer sicheren Verbindung mit einem Server zu verwenden. Wenn die Verbindung fehlschlägt, da der Server TLS nicht toleriert, arbeitet der Browser stattdessen mit SSL 3.0.

 

SSL-Warnungen

Es kann schnell festgestellt werden, ob die besuchte Website, eine verschlüsselte Verbindung benutzt. Wenn die Verbindung verschlüsselt ist, wird das Schlosssymbol in der rechten unteren Ecke des Navigator-Fensters geschlossen dargestellt. Wenn die Verbindung nicht verschlüsselt ist, ist das Schloss offen.

Wenn Sie zusätzliche Warnungen wünschen, können Sie die entsprechenden Kontrollkästchen im Fenster SSL-Einstellungen aktivieren. Manche Personen empfinden diese Warnung jedoch als störend.

Um eine dieser Navigator-Warnungen zu aktivieren, wählen Sie das entsprechende Kontrollkästchen aus:

  • Laden einer Seite, die Verschlüsselung unterstützt: Aktivieren Sie diese Warnung, wenn Sie bei jedem Laden einer Webseite, die Verschlüsselung unterstützt, gewarnt werden wollen.
  • Laden einer Seite mit niedriger Verschlüsselung. Aktivieren Sie diese Warnung, wenn Sie bei jedem Laden einer Webseite, die nur einen niedrigen Verschlüsselungsgrad unterstützt, gewarnt werden wollen. (Die niedrigste Verschlüsselung stellt mit 40-Bit-Schlüsseln die schwächste Art der Verschlüsselung dar.)
  • Verlassen einer Seite, die Verschlüsselung unterstützt. Aktivieren Sie diese Warnung, wenn Sie gewarnt werden wollen, wenn Sie eine Seite verlassen, die Verschlüsselung unterstützt, um eine Seite ohne Verschlüsselung aufzurufen.
  • Senden von Formulardaten von einer unverschlüsselten Seite zur anderen. Aktivieren Sie diese Warnung, um immer dann eine Nachricht zu erhalten, wenn Sie Daten über eine unverschlüsselte Verbindung senden. Wenn Sie unverschlüsselte Daten über das Internet senden, können diese Daten leicht von Dritten abgefangen werden.
  • Anzeige einer Seite mit einer Mischung aus verschlüsselten und unverschlüsselten Informationen. Aktivieren Sie diese Warnung, wenn Sie eine Meldung beim Anzeigen von Seiten erhalten wollen, die unverschlüsselte Daten enthalten.

Klicken Sie auf Echtheitsbestätigung, Verschlüsselung oder Zertifikat, um Kurzbeschreibungen zu erhalten.

Bearbeiten von Schlüsseln

Dieser Abschnitt beschreibt die Verwendung des Dialogfelds Schlüssel bearbeiten. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Privatsphäre und Sicherheit auf SSL. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Privatsphäre und Sicherheit, um die Liste zu erweitern.)
  3. Klicken Sie auf die Schaltfläche Schlüssel bearbeiten.

Im Dialogfeld Schlüssel bearbeiten können Sie Schlüssel oder kryptografische Algorithmen aktivieren oder deaktivieren. Wenden Sie sich an Ihren Systemadministrator, bevor Sie diese Einstellungen ändern.

Jedes Kontrollkästchen entspricht einer Schlüsselpalette oder einer Reihe von kryptografischen Algorithmen.
