Einstellungen für Privatsphäre und Sicherheit – Auswertung

Dieser Abschnitt beschreibt die Verwendung des Fensters Einstellungen für Auswertung. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Privatsphäre und Sicherheit auf Auswertung. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Privatsphäre und Sicherheit, um die Liste zu erweitern.)

Weitere Informationen über die Zertifikatsauswertung finden Sie unter Funktionsweise der Zertifikatsauswertung.

 

CRL

Eine CRL (Certificate Revocation List, Zertifikatsrücknahmeliste) ist eine Liste widerrufener Zertifikate, die regelmäßig von einem Zertifikatsaussteller (ZA) erstellt und signiert wird. Es ist möglich, eine CRL in Ihren Browser herunterzuladen, damit er Zertifikate anhand diese Lister auf Gültigkeit überprüfen kann, bevor er sie zur Identifikation akzeptiert.

Klicken Sie auf CRLs verwalten, um die CRLs anzuzeigen, die dem Zertifikats-Manager zur Verfügung stehen.

Weitere Informationen zur Verwaltung von CRLs finden Sie unter Verwalten von CRLs.

 

OCSP

Das Online Certificate Status Protocol (OCSP) ermöglicht dem Zertifikats-Manager, eine Onlineprüfung der Gültigkeit eines Zertifikats durchzuführen, wenn das Zertifikat benutzt oder angezeigt wird. Bei diesem Vorgang wird das Zertifikat anhand einer Zertifikatsrücknahmeliste (Certificate Revocation List, CRL) überprüft, die sich auf einer bestimmten Website befindet. Ihr Computer muss online sein, damit OCSP funktioniert.

Um festzulegen, wie der Zertifikats-Manager OCSP benutzt, wählen Sie eine der folgenden Einstellungen im Abschnitt OCSP des Fensters Auswertungseinstellungen aus.

  • OCSP nicht für Zertifikatsauswertung verwenden.Wählen Sie diese Einstellung aus, wenn der Zertifikats-Manager keine Online-Statusprüfung bei der Zertifikatsauswertung durchführen soll. Stattdessen überprüft der Zertifikats-Manager bei der Zertifikatsauswertung nur den Gültigkeitszeitraum und verifiziert, ob das Zertifikat von einem Zertifikatsaussteller korrekt signiert wurde, dessen ZA-Zertifikat in der Liste auf der Registerkarte ZA-Zertifikate (im Hauptfenster des Zertifikats-Managers) enthalten und als vertrauenswürdig für die Ausstellung von Zertifikaten dieser Art markiert ist.
  • OCSP nur zur Auswertung von Zertifikaten verwenden, die eine OCSP-Dienst-URL angeben. Wählen Sie diese Einstellung aus, wenn der Zertifikats-Manager die Online-Statusprüfung nur für Zertifikate durchführen soll, die eine URL für eine solche Prüfung enthalten. Wenn im Zertifikat eine URL angegeben ist, überprüft der Zertifikats-Manager, ob das Zertifikat dort als gültig geführt wird. Diese Prüfung erfolgt zusätzlich zu den Standardprüfungen des Gültigkeitszeitraums und der Vertrauenseinstellungen.
  • Alle Zertifikate mit OCSP auswerten unter Verwendung der hier eingegebenen URL und des Unterzeichners. Wählen Sie diese Einstellung aus, wenn der Zertifikats-Manager bei jeder Zertifikatsauswertung eine Online-Statusprüfung durchführen soll. Wenn Sie diese Einstellung wählen, müssen Sie auch im Popupmenü Antwortunterzeichner das Zertifikat auswählen, das den Unterzeichner von OCSP-Antworten identifiziert. Bei dieser Einstellung akzeptiert der Zertifikats-Manager nur Zertifikate, die durch eine mit dem unter Antwortunterzeichner angegebenen Zertifikat (oder einem Zertifikat aus dessen Zertifikatskette) unterzeichnete OCSP-Antwort bestätigt werden.

    Wenn Sie ein Zertifikat aus dem Popupmenü Antwortunterzeichner auswählen, trägt der Zertifikats-Manager die URL des Dienstes (sofern verfügbar) für diesen Unterzeichner automatisch ein. Wenn die Dienst-URL nicht automatisch eingetragen wird, müssen Sie diese selbst angeben. Fragen Sie Ihren Systemadministrator nach weiteren Informationen.

Verwalten von CRLs

Dieser Abschnitt beschreibt die Verwendung des Dialogfelds CRLs verwalten. So öffnen Sie das Dialogfeld:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Privatsphäre und Sicherheit auf Auswertung. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Privatsphäre und Sicherheit, um die Liste zu erweitern.)
  3. Klicken Sie auf CRLs verwalten.

In diesem Dialogfeld wird eine Liste der CRLsangezeigt, die Sie für die Verwendung durch den Browser heruntergeladen haben. Normalerweise laden Sie eine CRL herunter, indem Sie auf eine URL klicken. Weitere Informationen zur Funktionsweise von CRLs finden Sie unter Verwalten von CRLs.

Um eine CRL auszuwählen, klicken Sie darauf. Sie können eine der folgenden Aktionen ausführen:

  • Löschen: Eine CRL wird unwiderruflich von der Festplatte gelöscht. Machen Sie das nur, wenn die CRL zur Auswertung von Zertifikaten mit Sicherheit nicht mehr benötigen. Wenden Sie sich an Ihren Systemadministrator, wenn Sie sich nicht sicher sind.
  • Einstellungen:Das Dialogfeld Einstellungen für automatische CRL-Aktualisierung wird geöffnet. Sie können hier automatische CRL-Aktualisierungen für die ausgewählte CRL aktivieren und die Häufigkeit dieser Aktualisierungen angeben.
  • Aktualisieren: Aktualisiert sofort die ausgewählte CRL (sofern möglich).

Das Dialogfeld CRLs verwalten enthält die folgenden Informationen zu den CRLs:

  • Organisation (O): Der Name der Organisation, die die CRL ausgestellt hat.
  • Organisationseinheit (OU): Der Name der Organisationseinheit, die die CLR ausgestellt hat (wie etwa die Stamm-ZA für eine bestimmte Art von Zertifikaten).
  • Letzte Aktualisierung: Das Datum, an dem die vom Browser gespeicherte CRL-Kopie zuletzt aktualisiert wurde.
  • Nächste Aktualisierung: Das nächste Datum, an dem eine aktualisierte Version der CRL vom CRL-Aussteller veröffentlicht wird.
  • Automatische Aktualisierung: Gibt an, ob die automatische Aktualisierung für diese CRL aktiviert wurde. Um die Einstellungen für die automatische Aktualisierung anzuzeigen, wählen Sie die CLR aus und klicken auf Einstellungen.
  • Status der automatischen Aktualisierung:
    • Dieses Feld ist leer, wenn die automatische Aktualisierung nicht aktiviert oder zwar aktiviert wurde, aber der Termin der geplanten Aktivierung noch nicht fällig ist.
    • Wenn bereits eine automatische Aktivierung stattfand, zeigt dieses Feld Fehlgeschlagen an, falls die letzte Aktualisierung fehlschlug, oder OK, falls sie erfolgreich war.

CRL-Importstatus

Dieser Abschnitt beschreibt die Verwendung des Dialogfelds Status des CRL-Imports. Es wird beim ersten Versuch, eine CRL zu importieren angezeigt oder wenn Sie die CRL erfolgreich manuell aktualisieren.

Das Dialogfeld liefert Ihnen folgende Informationen:

  • Ob der Import oder das Aktualisieren der CRL erfolgreich war.
  • Welche Organisation die CRL ausgestellt hat.
  • Wann die nächste Aktualisierung der CRL veröffentlicht wird.
  • Ob die automatische Aktualisierung für diese CRL aktiviert ist.

Wenn die automatische Aktualisierung nicht aktiviert ist, können Sie sie hier aktivieren:

  • Ja: Klicken Sie hierauf, wenn die automatische Aktualisierung der CRL aktiviert werden soll. Wenn Sie auf diese Schaltfläche klicken, wird das Dialogfeld Einstellungen für automatische CRL-Aktualisierung angezeigt. Der nächste Abschnitt beschreibt, wie Sie diese Einstellungen festlegen.
  • Nein: Klicken Sie hierauf, wenn die automatische Aktualisierung nicht aktiviert werden soll.

Einstellungen für die automatische CRL-Aktualisierung

Dieser Abschnitt beschreibt die Verwendung des Dialogfelds Einstellungen für automatische CRL-Aktualisierung. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Privatsphäre und Sicherheit auf Auswertung. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Privatsphäre und Sicherheit, um die Liste zu erweitern.)
  3. Klicken Sie auf CRLs aktualisieren und wählen Sie die CRL aus, deren Aktualisierungseinstellungen Sie anzeigen oder ändern möchten.
  4. Klicken Sie auf Einstellungen.

Dieses Dialogfeld enthält folgende Optionen und Informationen:

  • Automatische Aktualisierung für diese CRL aktivieren: Wählen Sie diese Option aus, wenn die ausgewählte CRL automatisch nach dem von Ihnen hier festgelegten Terminplan aktualisiert werden soll. (Sie können diese Option nicht auswählen, wenn die CRL kein Datum für die nächste Aktualisierung angibt.)

    Wenn Sie die automatische Aktualisierung aktivieren, müssen Sie eines der folgenden Optionsfelder auswählen:

    • Aktualisieren X Tag(e) bis zur nächsten Aktualisierung. Wählen Sie diese Option aus, wenn die Aktualisierung von der Häufigkeit abhängen soll, mit der die CRL-Aussteller eine neue Version veröffentlichen.
    • Aktualisieren alle X Tag(e). Wählen Sie diese Option aus, wenn die Aktualisierung unabhängig von der Aktualisierung der CLR erfolgen soll.
  • CRL wird importiert aus: Gibt die URL an, von dem der Browser die CLR ursprünglich importiert hat. Diese Einstellung kann nicht geändert werden. Wenn Sie eine andere Adresse angeben möchten, löschen Sie die CRL und importieren Sie sie noch einmal von der neuen Adresse.
  • Fehler bei früheren aufeinanderfolgenden Aktualisierungen: Gibt an, wie oft Aktualisierungsversuche dieser CRL nacheinander fehlschlugen, einschließlich des aktuellsten Versuchs:
    • Wenn der aktuellste Versuch erfolgreich war, wird "Keine" angegeben, auch wenn vorhergehende Versuche fehlschlugen.
    • Wenn der aktuellste Versuch fehlschlug, wird die Anzahl der aufeinander folgenden fehlgeschlagenen Versuche und eine entsprechende Fehlermeldung angezeigt.

Klicken Sie auf OK, um die ausgewählten Optionen zu bestätigen.
