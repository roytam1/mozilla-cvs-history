Erstellen von Webseiten mit Netscape Composer

Mit Netscape Composer können Sie Ihre eigenen Webseiten erstellen und im Web veröffentlichen. Sie müssen sich nicht mit HTML auskennen, um Composer verwenden zu können. Der Composer ist so einfach zu bedienen wie ein Textverarbeitungsprogramm.

Mit Symbolleisten-Schaltflächen können Sie Listen, Tabellen, Grafiken, Verknüpfungen zu anderen Seiten, Farben und Schriftschnitte hinzufügen. Sie können eine Vorschau auf das Aussehen Ihres Dokuments im Web erstellen und Ihr Dokument auf einfache Weise mit anderen Benutzern gemeinsam bearbeiten und zwar unabhängig vom verwendeten Browser oder HTML-kompatiblen eMail-Programm.

So verwenden Sie Netscape Composer:

  • Klicken Sie in der linken unteren Ecke eines beliebigen Netscape-Fensters auf das Composer-Symbol.

Anlegen einer neuen Webseite

Netscape Composer ist ein HTML-Editor (Hypertext Markup Language), mit dem Sie Webseiten erstellen und bearbeiten können. Composer unterstützt WYSIWYG (What You See Is What You Get). Dadurch können Sie schon beim Erstellen sehen, wie Ihre Webseite für den Betrachter aussieht. Sie müssen dazu HTML nicht beherrschen, da die meisten HTML-Basisfunktionen als Befehle in Menüs und Symbolleisten verfügbar sind.

Composer ermöglicht Ihnen auf Wunsch auch die Bearbeitung des HTML-Quellcodes. Um den HTML-Quellcode anzuzeigen oder zu bearbeiten, klicken Sie im Menü Anzeigen auf HTML-Quelle oder klicken Sie auf der Bearbeitungs-Symbolleiste am unteren Rand des Composer-Fensters auf HTML-Quelle.

Verwenden Sie zum Erstellen einer Webseite eine der unten beschriebenen Methoden. Sobald Sie die Seite aufgerufen haben, können Sie Text wie mit einem Textverarbeitungsprogramm hinzufügen und bearbeiten.

So erstellen Sie eine neue Webseite im Navigator-Browser:

  • Klicken Sie im Menü Datei auf Neu und dann auf Composer-Seite. Ein Composer-Fenster mit einer leeren Seite wird geöffnet.

So öffnen Sie eine Seite, die gerade in Navigator angezeigt wird:

  • Klicken Sie im Navigator-Fenster, in dem die gewünschte Seite angezeigt wird, im Menü Datei auf Seite bearbeiten. Ein Composer-Fenster mit der aktuellen Seite wird angezeigt.

So erstellen Sie eine neue Webseite in Composer:

  • Klicken Sie in Composer auf der Symbolleiste auf die Schaltfläche Neu.

So beginnen Sie mit einer HTML-Datei, die auf einem lokalen Laufwerk gespeichert ist:

  1. Klicken Sie im Menü Fenster auf Composer. Das Composer-Fenster wird geöffnet.
  2. Klicken Sie im Menü Datei auf Datei öffnen. Das Dialogfeld HTML-Datei öffnen wird angezeigt.
  3. Wählen Sie auf Ihrem lokalen Laufwerk die Datei aus, die Sie bearbeiten möchten.
  4. Klicken Sie auf Öffnen, um die Datei in einem Composer-Fenster zu öffnen.

So bearbeiten Sie eine Webseite:

  1. Klicken Sie im Menü Fenster auf Navigator.
  2. Öffnen Sie eine Webseite, indem Sie den entsprechende URL (z. B. http://home.netscape.de) in der Adressleiste eingeben und die Eingabetaste drücken .
  3. Klicken Sie im Menü Datei auf Seite bearbeiten.

Tipp: Sie können die von Ihnen zuletzt im Composer-Fenster bearbeiteten Dateien schnell öffnen, indem Sie im Menü Datei auf Zuletzt geöffnete Seiten klicken und die gewünschte Datei in der Liste auswählen.

Speichern und Anzeigen Ihrer neuen Webseite

Composer-Dokumente können im HTML- oder im Textformat gespeichert werden. Beim Speichern im HTML-Format bleiben die Formatierung des Dokuments, beispielsweise Textformate (fett, kursiv usw.), Tabellen, Verknüpfungen und Grafiken erhalten. Wenn Sie ein Dokument im Textformat speichern, werden alle HTML-Tags entfernt und nur der Text des Dokuments bleibt erhalten.

So speichern Sie ein Dokument als HTML-Datei:

  • Klicken Sie im Menü Datei auf Speichern, oder klicken Sie auf der Erstellungs-Symbolleiste auf die Schaltfläche Speichern.

    Wenn Sie Ihre Seite noch nicht mit einem Titel versehen haben, werden Sie von Composer nun dazu aufgefordert. Composer zeigt den Seitentitel in der Titelleiste des Browsers an, wenn Sie die Seite im Browser betrachten. Der Seitentitel des Dokuments wird außerdem in Ihrer Lesezeichenliste aufgeführt, wenn Sie die Seite den Lesezeichen hinzufügen.

    Composer fordert Sie anschließend zur Eingabe eines Dateinamens und eines Pfades zum Speichern der Datei auf. Achten Sie darauf, dass die Erweiterung .html erhalten bleibt.

So ändern Sie den Dateinamen oder den Speicherort einer vorhandenen HTML-Datei:

  • Klicken Sie auf Speichern unter und wählen Sie einen anderen Dateinamen oder Speicherort aus.

Beim Speichern einer Seite in Composer werden alle Bestandteile der Seite (HTML-Code, Grafiken und sonstige Dateien wie beispielsweise Klangdateien und Stylesheets) lokal auf der Festplatte gespeichert. Wenn Sie nur den HTML-Code der Seite speichern möchten, müssen Sie die Composer-Einstellung zum Speichern von Seiten ändern. Weitere Informationen zum Ändern der Composer-Einstellung zum Speichern von Webseiten finden Sie unter Composer-Einstellungen – Composer.

Wenn die Adresse einer Grafik absolut ist (mit http:// beginnt) und Sie mit dem Internet verbunden sind, wird die Grafik in Composer und Navigator angezeigt. Wenn die Grafikadresse jedoch relativ zur Seitenadresse ist (mit file:/// beginnt), wird die Grafik in der lokalen Version des Dokuments nicht angezeigt.

So speichern Sie ein Dokument als Textdatei:

  1. Klicken Sie im Menü Datei auf In Textformat exportieren.
  2. Geben Sie den Dateinamen und den Speicherort zum Speichern der Datei an.

Anmerkung: In Dokumenten, die im Textformat gespeichert wurden, werden keine Grafiken angezeigt.

Tipp: Sie können im Menü Datei auf Zuletzt gespeicherte Version wiederherstellen klicken, um die zuletzt gespeicherte Kopie des gerade bearbeiteten Dokuments abzurufen. Denken Sie aber daran, dass Änderungen verloren gehen, die nach der letzten Speicherung vorgenommen wurden.

So zeigen Sie Ihre Webseite zum Testen der Verknüpfungen in einem Browserfenster an:

  • Klicken Sie im Menü Datei auf Seite durchsuchen (oder klicken Sie auf der Erstellungs-Symbolleiste auf das entsprechende Symbol). Wenn Sie Ihr Dokument noch nicht gespeichert haben, fordert Composer Sie auf Seitentitel, Dateiname und Speicherort einzugeben. Das Composer-Fenster bleibt unter dem neuen Navigatorfenster geöffnet.

Formatieren von Absätzen, Überschriften und Listen

So weisen Sie einem Absatz ein Format zu (ausgehend vom Composer-Fenster):

  1. Klicken Sie dort, wo Sie die Formatierung beginnen wollen oder markieren Sie den Text, den Sie formatieren möchten.
  2. Wählen Sie ein Absatzformat in der Dropdown-Liste auf der Formatierungs-Symbolleiste aus.
    • Haupttext: Weist Standardschriftart und -stil der Anwendung für regulären Text zu, ohne den Abstand vor und nach dem Text zu beeinflussen.
    • Absatz: Fügt Absatz-Tags ein (hiermit beginnen Sie neue Absätze). Zum Absatz zählt auch der obere und untere Rand.
    • Überschrift 1 - Überschrift 6: Formatiert den Absatz als Überschrift. Überschrift 1 ist die oberste Überschriftenebene, Überschrift 6 die unterste.
    • Adresse: Verwenden Sie dieses Format als "Unterschrift" für Ihre Webseite. Geben Sie beispielsweise den Namen des Autors oder einer Kontaktperson für weitere Informationen an: benutzer@beispiel.de

      Sie können auch Datum und Copyright-Hinweise einfügen. Dieses Format wird normalerweise unten auf der Webseite unter einer horizontalen Linie angezeigt. Navigator zeigt die Adresseninformationen gewöhnlich kursiv an.

    • Formatiert: Verwenden Sie dies beispielsweise für Codebeispiele, Spaltendaten und Mail-Nachrichten, die in fester Schriftbreite angezeigt werden sollen. Bei normalem Text entfernen die meisten Browser zusätzliche Leerzeichen, Tab-Stopps und Absatzendemarken. Bei einem Text, der das Format Formatiert verwendet, bleibt das Originallayout des Textes einschließlich der Leerzeichen erhalten.

So formatieren Sie einen Text als Überschrift:

  1. Klicken Sie an einer beliebigen Stelle innerhalb des zu formatierenden Textes, um die Einfügemarke zu platzieren.
  2. Wählen Sie auf der Formatierungs-Symbolleiste in der Dropdown-Liste die gewünschte Überschriftenebene von 1 (oberste Ebene) bis 6 (unterste Ebene) aus. Wählen Sie Überschrift 1 als Hauptüberschrift, Überschrift 2 für die nächste Ebene usw. aus.

So weisen Sie ein Listenpunktformat zu:

  1. Klicken Sie an einer beliebigen Stelle innerhalb der zu formatierenden Textzeile, um die Einfügemarke zu platzieren.
  2. Klicken Sie im Menü Absatz auf Liste.
  3. Wählen Sie den gewünschten Listenstil aus:
    • Auflistung: Jeder Listeneintrag hat einen Punkt (wie diese Liste).
    • Aufzählung: Die Listeneinträge sind nummeriert.
    • Begriff und Definition: Diese beiden Formate ergeben gemeinsam das Erscheinungsbild eines Glossars. Verwenden Sie das Format Begriff für das zu definierende Wort und Definition für die Erläuterung. Der Begriffstext wird linksbündig und die Definition eingerückt angezeigt.

Tipp: Sie können Listenstile schnell einem Textblock zuweisen, indem Sie den Text markieren und auf der Formatierungs-Symbolleiste auf die Schaltflächen für Aufzählung () oder Punkteliste () klicken.

So ändern Sie den Stil von Aufzählungspunkten oder Nummern:

  1. Platzieren Sie die Einfügemarke in dem Text des Listeneintrags, den Sie ändern möchten, oder wählen Sie einen oder mehrere Einträge in der Liste aus, wenn Sie das neue Format für die komplette Liste verwenden möchten.
  2. Klicken Sie im Menü Absatz auf Listeneigenschaften.
  3. Wählen Sie in der Dropdown-Liste ein Aufzählungspunkte- oder Nummernformat aus. Bei Aufzählungen können Sie zusätzlich die Startnummer angeben. Bei Punktelisten können Sie das Punkteformat ändern.

Tipp: Wenn Sie in einer Liste auf einen Punkt oder eine Nummer doppelklicken, wird das Dialogfeld Listeneigenschaften geöffnet.

So richten Sie einen Absatz oder Text Ihrer Webseite aus (z. B. zentriert, rechts- oder linksbündig):

  1. Platzieren Sie die Einfügemarke in dem Absatz oder der Textzeile, die Sie ausrichten möchten.
  2. Klicken Sie im Menü Absatz auf Ausrichten und anschließend auf eine der Ausrichtungsoptionen.

Anmerkung: Sie können auch die Formatierungs-Symbolleiste zum Ausrichten von Text verwenden.

Arbeiten mit Listen

So beenden Sie eine Liste und fahren mit dem Fließtext fort:

  • Platzieren Sie die Einfügemarke am Ende des letzten Listeneintrags und drücken Sie die Eingabetaste zweimal, um die Liste zu beenden.

So ändern Sie einen oder mehrere Listeneinträge in Fließtext:

  1. Platzieren Sie die Einfügemarke in dem Listeneintrag oder wählen Sie die Listeneinträge aus.
  2. Bei einer Aufzählung klicken Sie auf der Formatierungs-Symbolleiste auf die Schaltfläche für Aufzählungen (bei einer Punkteliste auf die Schaltfläche für Punktelisten).

So positionieren Sie eingerückten Text unter einem Listeneintrag:

  1. Platzieren Sie die Einfügemarke in dem Listeneintrag.
  2. Drücken Sie gleichzeitig die Umschalt- und die Eingabetaste, um einen hängenden Einzug zu erzeugen.
  3. Geben Sie den Text ein, den Sie einrücken möchten.
  4. Drücken Sie erneut die Umschalt- und Eingabetaste, um einen weiteren eingerückten Absatz zu erzeugen, oder drücken Sie nur die Eingabetaste, um einen weiteren Listeneintrag zu erzeugen.

Tipp: Sie können die Einrückung der Listeneinträge vergrößern oder verkleinern, indem Sie an einer beliebigen Stelle eines Listeneintrags klicken und dann auf der Formatierungs-Symbolleiste auf die Schaltfläche Text einrücken oder Text herausrücken klicken. Sie können aber auch an einer beliebigen Stelle in einem Listeneintrag klicken und die TAB-Taste drücken, um den Text um eine Ebene einzurücken. Drücken Sie Umschalt+Tab, um den Text um eine Ebene herauszurücken.

So führen Sie zwei aneinander folgende Listen zusammen:

  1. Markieren Sie die beiden Listen, die Sie zusammenführen möchten. Stellen Sie sicher, dass alle Elemente beider Listen markiert sind. Text, der sich zwischen den beiden Listen befindet, wird ebenfalls Teil der zusammengeführten Liste.
  2. Klicken Sie auf der Formatierungs-Symbolleiste auf die Schaltfläche für Aufzählungen oder Punktelisten, um die Listen zusammenzuführen.

Ändern von Textfarbe, -stil und Schriftart

So ändern Sie Farbe, Stil oder Schriftart von markiertem Text:

  1. Markieren Sie den Text, den Sie formatieren möchten.
  2. Klicken Sie im Menü Absatz auf eine der folgenden Optionen:
    • Schriftart: Hier können Sie Schriftarten wählen. Wenn die Schriftart vom Browser des Betrachters verwendet werden soll, klicken Sie auf Variable Breite oder Feste Breite.

      Anmerkung: Nicht alle auf dem Computer installierten Schriftarten werden angezeigt. Statt eine Schriftart anzugeben, die möglicherweise nicht allen Besuchern der Webseite zur Verfügung steht, sollten Sie eine Schriftart aus dem Menü auswählen, weil diese Schriftarten auf jedem Computer angezeigt werden können. Die Schriftarten Helvetica, Arial, Times und Courier beispielsweise sehen im Allgemeinen auf verschiedenen Computern gleich aus. Wenn Sie eine andere Schriftart wählen, wird diese auf einem anderen Computer möglicherweise anders dargestellt.

    • Größe: Hier legen Sie eine relative Schriftgröße fest oder Sie wählen eine Option, um die Textgröße (relativ zum umgebenden Text) zu vergrößern oder zu verkleinern.
    • Textformat: Hier legen Sie den Schriftschnitt wie beispielsweise kursiv, fett oder unterstrichen fest, oder Sie können ein strukturiertes Format wie beispielsweise Code festlegen.
    • Textfarbe: Hier wählen Sie eine Farbe mit dem Farbauswahlfeld aus. Wenn Sie mit den hexadezimalen HTML-Farbcodes vertraut sind, können Sie einen bestimmten Code oder aber einfach eine Farbbezeichnung (z. B. "blue") eingeben. Hier finden Sie ein praktisches Konvertierungsprogramm für Farbcodes.

So ändern Sie die Hintergrundfarbe der Webseite:

  1. Klicken Sie an einer beliebigen Stelle auf der Seite.
  2. Klicken Sie auf der Formatierungs-Symbolleiste auf den Hintergrundfarbblock.
  3. Wählen Sie im Dialogfeld Tabellen- oder Zellenfarbe eine Hintergrundfarbe aus.
  4. Klicken Sie auf OK.

Tipp: Um die zuletzt benutzte Farbe auf einen Text anzuwenden, markieren Sie den Text, und klicken Sie dann bei gedrückter Umschalttaste auf der Formatierungs-Symbolleiste auf den Textfarbblock. Dies ist hilfreich, wenn Sie für separate Textzeilen dieselbe Farbe verwenden möchten.

Sie können auch eine Grafik als Hintergrund verwenden. Siehe Festlegen der Seitenfarben und des Hintergrunds.

Entfernen oder Unterbrechen von Textformaten

So entfernen Sie alle Textformatierungen (fett, kursiv usw.) für markierten Text:

  1. Markieren Sie den Text.
  2. Klicken Sie im Menü Absatz auf Alle Textformate entfernen.
  3. Setzen Sie die Texteingabe fort.

So setzen Sie die Texteingabe ohne jegliche Textformatierung fort:

  1. Platzieren Sie die Einfügemarke dort, wo die Textformatierungen enden sollen.
  2. Klicken Sie im Menü Absatz auf Textformate deaktivieren.
  3. Setzen Sie die Texteingabe fort.

Suchen und Ersetzen von Text

So suchen Sie auf der Seite, die Sie gerade bearbeiten, nach Text:

  1. Platzieren Sie die Einfügemarke an der Stelle, an der Sie die Suche beginnen möchten.
  2. Klicken Sie im Menü Bearbeiten auf Suchen und Ersetzen. Das Dialogfeld Suchen und Ersetzen wird geöffnet.
  3. Geben Sie den Text, den Sie suchen möchten, in das Feld Suchen nach ein. Wählen Sie eine oder mehrere der folgenden Optionen aus, um die Suche einzuschränken:


    • Groß-/Kleinschreibung beachten: Hier legen Sie fest, ob bei der Suche die Groß-/Kleinschreibung beachtet wird.
    • Am Dokumentanfang fortsetzen: Diese Option sucht bis zum Ende der Seite und setzt die Suche dann am Anfang fort (wenn Sie aufwärts suchen, wird bis zum Seitenanfang gesucht und dann am Seitenende erneut begonnen).
    • Aufwärts suchen: Diese Option sucht von der Position der Einfügemarke an rückwärts bis zum Seitenanfang.
  4. Starten Sie die Suche, indem Sie auf Weitersuchen klicken. Klicken Sie, nachdem Composer die erste Textstelle gefunden hat, erneut auf Weitersuchen, um weitere Textstellen zu finden.
  5. Klicken Sie auf Schließen, wenn Sie fertig sind.

So suchen und ersetzen Sie Text auf der Seite, die Sie gerade bearbeiten:

  1. Platzieren Sie die Einfügemarke an der Stelle, an der Sie die Suche beginnen möchten.
  2. Klicken Sie im Menü Bearbeiten auf Suchen und Ersetzen. Das Dialogfeld Suchen und Ersetzen wird geöffnet.
  3. Geben Sie den zu suchenden Text und den Ersatztext ein.
  4. Wählen Sie eine oder mehrere der folgenden Optionen aus, um die Suche einzuschränken:
    • Groß-/Kleinschreibung beachten: Hier legen Sie fest, ob bei der Suche die Groß-/Kleinschreibung beachtet wird. Ohne diese Option wird Text in Groß- und Kleinbuchstaben gefunden.
    • Am Dokumentanfang fortsetzen: Verwenden Sie diese Option, damit die Suche nach dem Seitenende wieder am Anfang fortgesetzt wird.
    • Aufwärts suchen: Suchen Sie hiermit vom Ende zum Anfang der Seite.
  5. Klicken Sie auf Weitersuchen, um nach der nächsten Textstelle zu suchen. Composer markiert das nächste Vorkommen im Text.
  6. Klicken Sie auf Ersetzen, um den markierten Text durch den Ersatztext zu ersetzen. Klicken Sie auf Suchen und Ersetzen, um den markierten Text zu ersetzen und nach der nächsten Textstelle zu suchen. Klicken Sie auf Alle ersetzen, um alle Vorkommen des Suchtextes mit dem Ersatztext zu ersetzen.
  7. Klicken Sie auf Schließen, wenn Sie fertig sind.

Einfügen horizontaler Linien

Horizontale Linien werden normalerweise verwendet, um verschiedene Abschnitte eines Dokuments optisch voneinander zu trennen. So fügen Sie Ihrer Seite eine horizontale Linie (eine so genannte Trennlinie) hinzu (ausgehend vom Composer-Fenster):

  1. Platzieren Sie die Einfügemarke an der Stelle, an der die Linie eingefügt werden soll.
  2. Klicken Sie im Menü Einfügen auf Horizontale Linie.

 

Festlegen von Eigenschaften horizontaler Linien

Sie können Höhe, Länge, Breite, Ausrichtung und Schattierung einer Linie anpassen.

  1. Doppelklicken Sie auf die Linie, um das Dialogfeld Eigenschaften horizontaler Linien zu öffnen.
  2. Sie können folgende Eigenschaften bearbeiten:
    • Breite: Geben Sie die Breite ein und klicken Sie anschließend auf % des Fensters oder Pixel. Wenn Sie die Breite als Prozentsatz eingeben, ändert sich die Breite der Linie, sobald die Größe des Composer- oder Browserfensters verändert wird.
    • Höhe: Geben Sie einen Wert für die Höhe der Linie (in Pixeln) ein.
    • 3-D-Schattierung: Aktivieren Sie dieses Kontrollkästchen, um der Linie durch eine Schattierung eine Tiefenwirkung zu verleihen.
    • Ausrichtung: Geben Sie an, wo die Linie platziert werden soll (links, zentriert oder rechts).
  3. Klicken Sie auf Als Standard verwenden, um diese Einstellungen beim nächsten Einfügen einer horizontalen Linie als Standardeinstellungen zu verwenden.
  4. Um die Eigenschaften einer horizontalen Linie manuell zu bearbeiten, klicken Sie auf Fortgeschrittene Bearbeitung. Weitere Informationen finden Sie unter Fenster "Editor – Weitere Eigenschaften".

Tipp:Wenn Sie alle HTML-Elemente in gelben Kästchen anzeigen möchten, klicken Sie im Menü Anzeigen auf Alle Tags anzeigen. Klicken Sie auf ein gelbes Kästchen, um alle Bestandteile in diesem HTML-Tag oder –Element zu markieren. Doppelklicken Sie auf ein gelbes Kästchen, um das Dialogfeld Editor – Weitere Eigenschaften für dieses HTML-Tag oder –Element zu öffnen.

Einfügen von Sonderzeichen

So fügen Sie Sonderzeichen wie Akzente, Copyright- und Währungssymbole ein:

  1. Platzieren Sie die Einfügemarke an der Stelle, an der das Sonderzeichen eingefügt werden soll.
  2. Klicken Sie im Menü Einfügen auf Zeichen und Symbole. Das Dialogfeld Zeichen einfügen wird geöffnet.
  3. Wählen Sie eine Zeichenkategorie aus.
    • Wenn Sie Akzent (groß) oder Akzent (klein) auswählen, müssen Sie den gewünschten Buchstaben im Dropdown-Menü Buchstabe auswählen. (Anmerkung: Akzente sind nicht für alle Buchstaben verfügbar.) Klicken Sie auf Häufige Symbole, um Sonderzeichen wie das Copyright-Symbol oder Bruchzahlen einzufügen.
  4. Klicken Sie in der Dropdown-Liste Zeichen auf das Zeichen, das Sie einfügen möchten.
  5. Klicken Sie auf Einfügen.

    Sie können die Texteingabe in Ihrem Dokument (oder im Fenster Erstellen in Mail) fortsetzen, während dieses Dialogfeld geöffnet ist, falls Sie es noch einmal brauchen.

  6. Klicken Sie auf Schließen, wenn Sie fertig sind.

Einfügen von HTML-Elementen und Attributen

Wenn Sie mit HTML-Quellcode vertraut sind, können Sie Ihrer Seite zusätzliche Tags, Stile, Attribute und JavaScript hinzufügen. Wenn Sie nicht gewohnt sind, mit HTML-Quellcode zu arbeiten, sollten Sie lieber keine Änderungen vornehmen. Wenn Sie mit HTML-Code arbeiten möchten, können Sie eine der folgenden Methoden verwenden:

  • Platzieren Sie die Einfügemarke an der Stelle, an der HTML-Code eingefügt werden soll, oder markieren Sie den zu bearbeitenden Text, und klicken Sie im Menü Einfügen auf HTML. Geben Sie im Dialogfeld HTML einfügen HTML-Tags und -Text ein, und klicken Sie anschließend auf Einfügen.
  • Wählen Sie ein Objekt wie beispielsweise eine Tabelle, einen benannten Anker, ein Bild, eine Verknüpfung oder eine horizontale Linie aus. Doppelklicken Sie auf das Objekt, um das entsprechende Eigenschaften-Dialogfeld zu öffnen. Klicken Sie auf Fortgeschrittene Bearbeitung, um das Fenster Editor – Weitere Eigenschaften zu öffnen. Im Fenster Editor – Weitere Eigenschaften können Sie Objekten HTML-Attribute, JavaScript und Cascading Style Sheets (CSS) hinzuzufügen.
  • Klicken Sie im Menü Anzeigen auf HTML-Quelle oder klicken Sie auf der Bearbeitungs-Symbolleiste am unteren Rand des Composer-Fensters auf HTML-Quelle. (Wenn die Bearbeitungs-Symbolleiste nicht angezeigt wird, klicken Sie im Menü Anzeigen auf Ein-/Ausblenden, und stellen Sie sicher, dass Bearbeitungs-Symbolleiste ausgewählt ist.)

 

Verwenden des Fensters "Editor – Weitere Eigenschaften"

Sie können das Fenster Editor – Weitere Eigenschaften verwenden, um Tabellen, Grafiken und horizontalen Linien HTML-Attribute und JavaScript hinzuzufügen.

Anmerkung: Bearbeiten Sie die HTML-Attribute und die ihnen zugeordneten Werte nur, wenn Sie mit dem Hinzufügen, Löschen und Ändern dieser Attribute vertraut sind.

Wenn Sie das Fenster Editor – Weitere Eigenschaften noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Anzeigen (oder auf der Bearbeitungs-Symbolleiste) auf Alle Tags anzeigen.
  2. Doppelklicken Sie auf das Objekt, das Sie ändern möchten. Dadurch wird das entsprechende Eigenschaften-Dialogfeld geöffnet.

Auswerten des HTML-Codes

Bevor Sie Ihr Dokument auf einem Webserver der Allgemeinheit zur Verfügung stellen, sollten Sie die HTML-Formatierung überprüfen, um sicherzustellen, dass sie den Webstandards entspricht. Geprüfte HTML-Dokumente laufen selten Gefahr, bei der Anzeige in unterschiedlichen Browsern Probleme zu verursachen. Eine rein visuelle Überprüfung Ihrer Webseiten in Navigator gewährleistet nicht, dass Ihr Dokument auch in anderen Webbrowsern korrekt dargestellt wird.

Composer bietet eine bequeme Möglichkeit, Ihr Dokument auf die Einhaltung der HTML-Standards des W3C (World Wide Web Consortium) zu überprüfen. Composer verwendet den W3C HTML Validation Service, der für die HTML-Syntax Ihres Dokuments die Einhaltung der HTML 4.01-Standards überprüft und außerdem Informationen zur Korrektur von Fehlern liefert.

Anmerkung: Sie müssen mit dem Internet verbunden sein, um diese Funktion nutzen zu können.

So werten Sie die HTML-Syntax Ihres Dokuments aus:

  1. Klicken Sie im Menü Extras auf HTML auswerten. Wenn Sie Änderungen noch nicht gespeichert haben, fordert Composer Sie zum Speichern auf, bevor der Vorgang fortgesetzt wird.
  2. Wenn die Seite des W3C HTML Validation Service geöffnet wird, klicken Sie auf Durchsuchen und suchen Sie auf Ihrer Festplatte nach der Datei, die Sie auswerten möchten.
  3. Klicken Sie auf Validate this document.

Auswählen des richtigen Bearbeitungsmodus

In der Regel brauchen Sie die Voreinstellung für den Bearbeitungsmodus (Standard) nicht zu ändern. Wenn Sie allerdings mit dem HTML-Quellcode des Dokuments arbeiten möchten, müssen Sie den Bearbeitungsmodus ändern.

Sie können in Composer zwischen vier Bearbeitungsmodi oder Ansichten wechseln. In allen Bearbeitungsmodi können Sie weiter Ihre Webseite bearbeiten, es werden jedoch verschiedene Ebenen von HTML-Tags (und Tag-Symbolen) angezeigt.

Führen Sie die folgenden Schritte aus, bevor Sie einen Bearbeitungsmodus auswählen:

  • Klicken Sie im Menü Anzeigen auf Ein-/ausblenden, und stellen Sie sicher, dass Bearbeitungs-Symbolleiste ausgewählt ist.

Auf der Bearbeitungs-Symbolleiste befinden sich vier Schaltflächen:

  • Standard: Wählen Sie diesen Bearbeitungsmodus, um das Dokument währen der Bearbeitung so anzuzeigen, wie es später online dargestellt wird. Wählen Sie diesen Modus, um Tabellenrahmen und benannte Ankersymbole anzuzeigen. Alle anderen Symbole für HTML-Tags sind ausgeblendet.
  • Alle Tags anzeigen: Wählen Sie diesen Modus, um alle Symbole für HTML-Tags anzuzeigen.
  • HTML-Quelle: Wählen Sie diesen Modus, um das Dokument als unformatierten HTML-Code darzustellen und zu bearbeiten. Wenn Sie das Dokument speichern, wird der Standardmodus wieder aktiviert.
  • Ansicht: Wählen Sie diesen Modus, um das Dokument genau so darzustellen, wie es im Browserfenster angezeigt wird. Verknüpfungen und JavaScript-Funktionen sind allerdings nicht aktiv.

Anmerkung: JavaScript-Funktionen, Frames, Verknüpfungen und animierte GIF-Dateien sind in keinem der Bearbeitungsmodi aktiviert. Um diese Elemente im aktiven Status anzuzeigen, klicken Sie auf der Erstellungs-Symbolleiste auf die Schaltfläche Vorschau, um die Seite in einem Browserfenster zu laden.

Einfügen einer Tabelle

Verwenden Sie Tabellen, um Text, Grafiken oder Daten in formatierten Zeilen und Spalten anzuordnen. So fügen Sie eine Tabelle ein:

  1. Platzieren Sie die Einfügemarke an der Stelle, an der die Tabelle eingefügt werden soll.
  2. Klicken Sie auf der Erstellungs-Symbolleiste auf die Schaltfläche Tabelle (). Das Dialogfeld Tabelle einfügen wird geöffnet.
  3. Geben Sie die gewünschte Zeilen- und Spaltenanzahl ein.
    • (Optional) Geben Sie die Tabellenbreite ein, und wählen Sie als Maßeinheit Pixel oder % des Fensters aus.
  4. Geben Sie einen Wert für die Randbreite (in Pixel) ein oder geben Sie Null (0) ein, falls Sie keinen Rand wünschen.

    Anmerkung: Composer verwendet eine rot gepunktete Linie, um Tabellen ohne Rand anzuzeigen. Diese Linie wird im Browser nicht angezeigt.

  5. Wenn Sie zusätzliche Tabellenattribute oder JavaScript zuweisen möchten, können Sie auf Fortgeschrittene Bearbeitung klicken, um das Fenster Editor – Weitere Eigenschaften zu öffnen.
  6. Klicken Sie auf OK, um die Einstellungen zu bestätigen und die neue Tabelle anzuzeigen.

Weitere Informationen zum Ändern zusätzlicher Eigenschaften für die neue Tabelle finden Sie unter Ändern von Tabelleneigenschaften.

Tipp: Um eine Tabelle innerhalb einer anderen Tabelle einzufügen, klicken Sie im Menü Einfügen auf Tabelle.

Ändern von Tabelleneigenschaften

Dieser Abschnitt beschreibt, wie Sie Eigenschaften sowohl der ganzen Tabelle als auch von Zeilen, Spalten oder einzelnen Zellen einer Tabelle ändern. Wenn Sie das Dialogfeld Tabelleneigenschaften noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Wählen Sie die Tabelle aus oder klicken Sie an eine beliebige Stelle der Tabelle.
  2. Klicken Sie auf der Symbolleiste auf die Schaltfläche Tabelle () oder klicken Sie im Menü Tabelle auf Tabelleneigenschaften. Das Dialogfeld Tabelleneigenschaften enthält zwei Registerkarten: Tabelle und Zellen.
  3. Mit der Registerkarte Tabelle können Sie folgende Eigenschaften bearbeiten:
    • Größe: Hier können Sie die Anzahl der Zeilen und Spalten festlegen. Geben Sie die Breite der Tabelle ein und klicken Sie anschließend auf % des Fensters oder Pixel. Wenn Sie die Breite als Prozentsatz eingeben, ändert sich die Breite der Tabelle, sobald die Größe des Composer- oder Browserfensters geändert wird.
    • Ränder und Abstände: Hier legen Sie in Pixeln die Randbreite, die Zellabstände und die Zellauffüllung (Abstand zwischen dem Inhalt und dem Rand der Zelle) fest.

      Anmerkung: Composer verwendet einen gepunkteten Umriss, um Tabellen ohne Rand anzuzeigen. Diese Linie wird nicht im Browser angezeigt.

    • Tabellenausrichtung: Hier richten Sie die Tabelle auf der Seite aus. Wählen Sie in der Dropdown-Liste eine Option aus.
    • Titel: Wählen Sie in der Dropdown-Liste die Position des Titels aus.
    • Hintergrundfarbe: Hier wählen Sie eine Farbe für den Tabellenhintergrund aus oder belassen ihn transparent.
  4. Wenn Sie zusätzliche Attribute oder JavaScript-Ereignisse zuweisen möchten, können Sie auf Fortgeschrittene Bearbeitung klicken, um das Fenster Editor – Weitere Eigenschaften zu öffnen.
  5. Klicken Sie auf Anwenden, um Ihre Änderungen anzuzeigen, ohne das Dialogfeld zu schließen oder klicken Sie auf OK, um sie zu bestätigen.

Sie können die Eigenschaften für eine oder mehrere Zellen folgendermaßen anzeigen, ändern oder hinzufügen:

  1. Wählen Sie die Zeile, Spalte oder Zelle aus, und klicken Sie im Menü Tabelle auf Tabelleneigenschaften. Das Dialogfeld Tabelleneigenschaften wird geöffnet.
  2. Mit der Registerkarte Zellen können Sie folgende Eigenschaften bearbeiten:
    • Auswahl: Wählen Sie in der Dropdown-Liste Zelle, Zeile oder Spalte aus. Klicken Sie auf Vorige oder Weiter, um zwischen Zeilen, Spalten oder Zellen zu wechseln.
    • Größe: Geben Sie eine Zahl für Höhe und Breite ein und klicken Sie anschließend auf % der Tabelle oder Pixel.
    • Inhaltsausrichtung: Wählen Sie die Art der vertikalen und horizontalen Ausrichtung für den Text oder die Daten in jeder Zelle.
    • Zellstil: Wählen Sie in der Dropdown-Liste Kopfzeile für Spalten- oder Zeilenüberschriften aus (der Text wird in Fettschrift dargestellt und innerhalb der Zelle zentriert); ansonsten wählen Sie Standard aus.
    • Textumbruch: Wählen Sie in der Dropdown-Liste Ohne Umbruch aus, damit der Text nicht in die nächste Zeile umgebrochen wird, außer Sie fügen einen Absatz ein. Wählen Sie andernfalls Umbruch aus.
    • Hintergrundfarbe: Wählen Sie eine Farbe für den Zellenhintergrund aus, oder belassen Sie ihn transparent.

    Anmerkung: Wenn Sie zusätzliche Attribute oder JavaScript-Ereignisse zuweisen möchten, können Sie auf Fortgeschrittene Bearbeitung klicken, um das Fenster Editor – Weitere Eigenschaften zu öffnen.

  3. Klicken Sie auf Anwenden, um Ihre Änderungen anzuzeigen, ohne das Dialogfeld zu schließen, oder klicken Sie auf OK, um sie zu bestätigen.

Tipp: Wenn Sie die Text- oder Hintergrundfarbe mehrerer Zellen oder der ganzen Tabelle ändern möchten, wählen Sie die Zellen aus oder klicken an einer beliebigen Stelle in der Tabelle, und klicken Sie dann auf der Formatierungs-Symbolleiste auf das Symbol für Textfarbe oder Hintergrundfarbe.

Tipp: Um die zuletzt verwendete Farbe auf eine Zelle anzuwenden, markieren Sie die Zelle und klicken Sie dann bei gedrückter Umschalttaste auf die Farbauswahl. Dies ist hilfreich, wenn Sie für mehrere einzelne Zellen dieselbe Farbe verwenden möchten.

Hinzufügen und Löschen von Zeilen, Spalten und Zellen

Sie können mit Composer Ihrer Tabelle schnell eine oder mehrere Zellen, Spalten oder Zeilen hinzufügen oder aus ihr löschen. Des Weiteren können Sie Optionen festlegen, mit denen Sie die ursprüngliche rechteckige Struktur oder das Layout der Tabelle beibehalten können, während Sie sie bearbeiten.

So fügen Sie eine Zelle, Zeile oder Spalte zur Tabelle hinzu:

  1. Klicken Sie an die Stelle der Tabelle, an der Sie eine oder mehrere Zellen hinzufügen möchten.
  2. Klicken Sie im Menü Tabelle auf Einfügen.
  3. Wählen Sie eine Zellengruppierung aus. (Sie können auch eine neue Tabelle innerhalb einer Tabellenzelle einfügen.)

So löschen Sie eine Zelle, Zeile oder Spalte:

  3. Wählen Sie das Element aus, das Sie löschen möchten.

So verbinden Sie eine Zelle mit der rechten Zelle:

  • Klicken Sie in die Zelle auf der linken Seite, klicken Sie im Menü Tabelle und klicken Sie auf Mit der Zelle rechts verbinden.

So verbinden Sie mehrere benachbarte Zellen:

  • Wählen Sie benachbarte Zellen aus, indem Sie die Maus über die gewünschten Zellen ziehen.
  • Klicken Sie im Menü Tabelle auf Ausgewählte Zellen verbinden.

So spalten Sie eine verbundene Zelle wieder in zwei oder mehr Zellen auf:

Informationen zum Auswählen mehrerer nicht benachbarter Zellen, Zeilen oder Spalten finden Sie unter Auswählen von Tabellenelementen.

 

Ändern der standardmäßigen Bearbeitung von Tabellen

Deshalb können Sie eine oder mehrere Zellen löschen, aber dabei das ursprüngliche Layout oder die Struktur der Tabelle beibehalten. Andernfalls würden durch das Löschen von Zellen leere Stellen oder ein unregelmäßiger Umriss aufgrund einer ungeraden Zellenanzahl entstehen.

So ändern Sie das Standardverhalten bei der Bearbeitung von Tabellen (ausgehend vom Composer-Fenster):

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen und dann auf Composer.
  2. Legen Sie unter Tabellenbearbeitung die folgende Einstellungen fest:
    • Stellen Sie sicher, dass das Kontrollkästchen Tabellenlayout beim Einfügen oder Löschen von Zellen bewahren aktiviert ist, damit keine unregelmäßig geformten Tabellen entstehen.
  3. Klicken Sie auf OK.

Siehe auch Festlegen allgemeiner Composer-Einstellungen.

Einfügen einer Grafik in Ihre Webseite

In Ihre Webseite können GIF-, JPEG-, BMP- und PNG (Portable Network Graphics)-Grafiken eingefügt werden. Sie können damit auch Verknüpfungen erstellen. Beim Einfügen der Grafik speichert Composer einen Verweis auf die Grafikdatei in Ihrer Seite.

Anmerkung: Wenn Sie Ihre Webseiten im Web veröffentlichen möchten, sollten Sie keine BMP-Grafiken auf Ihren Seiten verwenden.

Tipp: Es empfiehlt sich, die Webseite zuerst zu speichern oder zu veröffentlichen, bevor Sie Grafiken hinzufügen. Auf diese Weise kann Composer automatisch relative Verweise auf eingefügte Grafiken verwenden.

So fügen Sie eine Grafik ein:

  1. Platzieren Sie die Einfügemarke an der Stelle, an der die Grafik eingefügt werden soll.
  2. Klicken Sie auf der Symbolleiste auf die Schaltfläche Grafik () oder klicken Sie im Menü Einfügen auf Grafik. Das Dialogfeld Grafikeigenschaften wird geöffnet.
  3. Geben Sie den Speicherort und den Namen der Grafikdatei ein oder klicken Sie auf Datei wählen, um die Grafikdatei auf Ihrer Festplatte oder im Netzwerk zu suchen.
  4. Geben Sie eine einfache Beschreibung der Grafik als Alternativtext ein, der in Browsern, die nur Text darstellen können (sowie in anderen Browsern), anstelle der Grafik angezeigt wird, wenn die Grafik geladen wird oder wenn das Laden von Grafiken deaktiviert wurde.

    Es ist auch möglich, keinen Alternativtext hinzuzufügen.

  5. Klicken Sie bei Bedarf auf weitere Registerkarte, um die Einstellungen (z. B. Ausrichtung) im Dialogfeld Grafikeigenschaften anzupassen.

Tipp: So fügen Sie schnell eine Grafik ein: Fügen Sie die Grafik per Drag & Drop auf der Webseite ein.

Bearbeiten von Grafikeigenschaften

Wenn Sie eine Grafik auf Ihrer Webseite eingefügt haben, können Sie deren Eigenschaften bearbeiten und das Layout, beispielsweise Höhe, Breite, Abstand und Textausrichtung, anpassen. Wenn Sie das Dialogfeld Grafikeigenschaften noch nicht geöffnet haben, gehen Sie folgendermaßen vor.

So bearbeiten Sie die Eigenschaften einer ausgewählten Grafik:

  1. Doppelklicken Sie auf die Grafik, oder wählen Sie sie aus, und klicken Sie auf der Symbolleiste auf die Schaltfläche Grafik (), um das Dialogfeld Grafikeigenschaften zu öffnen.
  2. Mit der Registerkarte Adresse können Sie folgende Eigenschaften bearbeiten:
    • Grafikinformationen: Geben Sie den Dateinamen und den Speicherort der Grafikdatei ein. Klicken Sie auf Datei wählen, um nach einer Grafikdatei auf Ihrer Festplatte oder im Netzwerk zu suchen.
    • URL ist relativ zur Seitenadresse: Wenn dieses Kontrollkästchen aktiviert ist, konvertiert Composer die URL, sodass er einen relativen Pfad zur Adresse der Seite darstellt. Dies ist besonders dann hilfreich, wenn Sie Ihre Seiten auf einem Webserver für die Allgemeinheit veröffentlichen möchten. Durch die Verwendung relativer URLs haben Sie die Möglichkeit, alle verknüpften Dateien an derselben Position in Relation zueinander zu halten, unabhängig davon, wo sie auf der Festplatte oder dem Webserver gespeichert sind.

      Wenn dieses Kontrollkästchen deaktiviert wurde, konvertiert Composer die URL in einen vollständigen (absoluten) URL. Absolute URLs werden normalerweise für die Verknüpfung mit Grafiken verwendet, die auf anderen Webservern (nicht auf der lokalen Festplatte) gespeichert sind.

      Wenn Sie die Seite noch nie gespeichert oder veröffentlicht haben, müssen Sie die Seite zunächst speichern, um dieses Kontrollkästchen aktivieren zu können. (Dieses Kontrollkästchen ist nicht verfügbar, wenn Sie das Dialogfeld Grafikeigenschaften in einem Nachrichtenerstellungsfenster öffnen.)

    • Alternativtext: Geben Sie Text ein, der anstelle der ursprünglichen Grafik angezeigt wird, beispielsweise einen Titel oder eine kurze Beschreibung der Grafik. Dieser Text wird in Webbrowsern angezeigt, die nur Text darstellen können oder für die das Laden von Grafiken deaktiviert wurde.
    • Keinen alternativen Text verwenden: Wählen Sie diese Option, wenn die Grafik keinen Alternativtext erfordert oder wenn Sie keinen Alternativtext verwenden möchten.
  3. Mit der Registerkarte Abmessungen können Sie folgende Eigenschaften bearbeiten:
    • Tatsächliche Größe: Wählen Sie diese Option, um Änderungen an den Abmessungen rückgängig zu machen und die ursprüngliche Größe der Grafik wiederherzustellen.
    • Benutzerdefinierte Größe: Wählen Sie diese Option, und geben Sie die neue Höhe und Breite in Pixeln oder als Prozentsatz an. Diese Einstellung hat keinen Einfluss auf die ursprüngliche Grafikdatei, nur auf die Darstellung der Grafik auf Ihrer Seite.
    • Beschränkung: Wenn Sie die Grafikgröße ändern, sollten Sie dieses Kontrollkästchen aktivieren, um das Seitenverhältnis der Grafik beizubehalten (damit diese nicht verzerrt angezeigt wird). Wenn Sie diese Option aktivieren, brauchen Sie nur eines der Felder Höhe oder Breite zu ändern.
  4. Mit der Registerkarte Gesamtbild können Sie folgende Eigenschaften bearbeiten:
    • Abstände: Geben Sie hier den Abstand zwischen der Grafik und dem umgebenden Text an. Sie können auch einen durchgezogenen, schwarzen Rand um die Grafik einfügen und dessen Breite in Pixeln angeben. Geben Sie 0 für keinen Rand ein.
    • Text an Grafik ausrichten: Wenn Sie Ihre Grafik neben einen beliebigen Text platzieren, wählen Sie das Ausrichtungssymbol aus, um anzugeben, wie der Text in Bezug auf die Grafik positioniert werden soll.
    • Grafikzuordnung: Klicken Sie auf Entfernen, um alle Grafikzuordnungseinstellungen zu beseitigen.
  5. Mit der Registerkarte Verknüpfung können Sie folgende Eigenschaften bearbeiten:
    • Geben Sie einen Web-Seiten-Speicherort ein: Wenn Sie eine Verknüpfung für diese Grafik definieren möchten, geben Sie die URL einer entfernten oder lokalen Seite ein, oder wählen Sie in der Dropdown-Liste einen benannten Anker oder eine Überschrift aus. Klicken Sie auf Datei wählen, um nach einer Grafikdatei auf Ihrer Festplatte oder im Netzwerk zu suchen.
    • URL ist relativ zur Seitenadresse: Wenn dieses Kontrollkästchen aktiviert ist, konvertiert Composer die URL, sodass er einen relativen Pfad zur Adresse der Seite darstellt. Dies ist besonders dann hilfreich, wenn Sie Ihre Seiten auf einem Webserver für die Allgemeinheit veröffentlichen möchten. Durch die Verwendung relativer URLs haben Sie die Möglichkeit, alle verknüpften Dateien an derselben Position in Relation zueinander zu halten, unabhängig davon, wo sie auf der Festplatte oder dem Webserver gespeichert sind.

      Wenn dieses Kontrollkästchen deaktiviert wurde, konvertiert Composer die URL in einen vollständige (absoluten) URL. Absolute URLs werden normalerweise für die Verknüpfung mit Grafiken verwendet, die auf anderen Webservern (nicht auf der lokalen Festplatte) gespeichert sind.

      Wenn Sie Änderungen noch nicht gespeichert haben, müssen Sie die Seite zunächst speichern, um dieses Kontrollkästchen aktivieren zu können. (Dieses Kontrollkästchen ist nicht verfügbar, wenn Sie das Dialogfeld Grafikeigenschaften in einem Nachrichtenerstellungsfenster öffnen.)

    • Um jedes verknüpfte Bild einen Rahmen anzeigen: Wenn dieses Kontrollkästchen aktiviert ist, werden verknüpfte Grafiken farblich hervorgehoben.
  6. Wenn Sie zusätzliche Attribute oder JavaScript-Ereignisse zuweisen möchten, können Sie auf Fortgeschrittene Bearbeitung klicken, um das Fenster Editor – Weitere Eigenschaften zu öffnen.
  7. Klicken Sie auf OK, um die Änderungen zu bestätigen.

Erstellen von Verknüpfungen auf der gleichen Seite

Sie müssen einen Anker und anschließend eine Verknüpfung, die auf den Anker verweist, erstellen, um eine Verknüpfung auf derselben Seite zu realisieren, beispielsweise von einem Abschnitt zum anderen. Anker werden auch als benannte Anker bezeichnet.

  1. Platzieren Sie die Einfügemarke am Anfang einer Zeile, in der Sie einen Anker erstellen wollen, oder wählen Sie Text aus.
  2. Klicken Sie im Menü Einfügen auf Benannter Anker. Das Dialogfeld Eigenschaften benannter Anker wird geöffnet.
  3. Geben Sie in das Feld Zielname einen eindeutigen Namen für den Anker ein (bis zu 30 Zeichen). Wenn Sie Leerzeichen eingeben, werden diese durch Unterstriche (_) ersetzt. Falls Sie in Schritt 1 Text ausgewählt haben, enthält dieses Feld bereits einen Namen.
  4. Klicken Sie auf OK. Ein Ankersymbol wird in Ihrem Dokument angezeigt, um die Position des Ankers zu markieren:

So erstellen Sie eine Verknüpfung, mit der die Leser zum Anker springen können:

  1. Wählen Sie den Text oder die Grafik aus, den bzw. die Sie mit dem Anker verknüpfen möchten.
  2. Klicken Sie auf die Schaltfläche Verknüpfung oder klicken Sie im Menü Einfügen auf Verknüpfung. Das Dialogfeld Verknüpfungseigenschaften wird geöffnet.
    • Wenn Sie eine Verknüpfung zu einer HTML-Datei auf Ihrem Computer erstellen, klicken Sie auf Datei wählen, um die HTML-Datei zu suchen.
    • Wenn Sie eine Verknüpfung zu einem benannten Anker (Ziel) erstellen, wählen Sie einen Anker in der Liste der derzeit verfügbaren Anker auf dieser Seite aus.
    • Wenn Sie eine Verknüpfung zu einer Überschriftenebene erstellen (z. B. Überschrift 1 bis Überschrift 6), wählen Sie eine Überschrift in der Liste der derzeit verfügbaren Überschriften auf dieser Seite aus.
  3. Klicken Sie auf OK.

Anmerkung: Klicken Sie im Menü Datei auf Seite öffnen und anschließend auf die Verknüpfung, um die erstellte Verknüpfung zu testen.

Tipp: Wenn Sie noch keine benannten Anker erstellt haben, können Sie im Dialogfeld Verknüpfungseigenschaften Verknüpfungen zu Überschriften herstellen, die bereits auf der Seite vorhanden sind.

Erstellen einer neuen Veröffentlichungssite

Wenn Sie Dokumente an mehreren entfernten Websites veröffentlichen, können Sie Composer anweisen, die Veröffentlichungsinformationen für jede verwendete entfernte Website zu speichern. Auf diese Weise müssen Sie diese Informationen nicht jedes Mal beim Veröffentlichen eingeben.

So erstellen Sie eine neue Veröffentlichungssite (ausgehend vom Composer-Fenster):

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen für publizierende Web-Site. Composer öffnet das Dialogfeld Publizierungs-Einstellungen.
  2. Klicken Sie auf Neue Website.
  3. Geben Sie für Name der Web-Site das gewünschte Pseudonym für diese Veröffentlichungssite ein.

    Wenn Sie beispielsweise die neue Website zum Veröffentlichen von Dokumenten für das Projekt "Meteor" verwenden werden, könnten Sie "Meteor" als Websitenamen eingeben. Sitenamen sollten Sie an die Dokumente erinnern, die Sie auf der jeweiligen Website veröffentlichen.

  • Die Veröffentlichungsadresse bezeichnet den Speicherort, an dem Dokumente auf dieser Site veröffentlicht werden (dies wird als "Upload" bezeichnet). Wenn Sie nicht sicher sind, was Sie hier eingeben sollten, ziehen Sie Ihren ISP oder Systemadministrator zu Rate.

  • HTTP-Adresse Ihrer Homepage: Die vollständige HTTP-Adresse des Veröffentlichungsbasisverzeichnisses. Geben Sie keine Dateinamen oder Unterverzeichnisse als Bestandteil der URL ein.

  • Diese URL muss immer mit http:// beginnen. In manchen Fällen ist diese URL mit der Veröffentlichungsadresse identisch. Wenn Sie nicht sicher sind, was Sie hier eingeben sollten, ziehen Sie Ihren ISP oder Systemadministrator zu Rate, oder lassen Sie dieses Feld leer.

  • Benutzername: Der Benutzername, mit dem Sie sich bei Ihrem ISP oder Netzwerk anmelden.
  • Passwort: Das Passwort für Ihren Benutzernamen.
  • Passwort speichern: Wählen Sie diese Option, um das Passwort auf sichere Weise mit dem Passwort-Manager zu speichern, damit Sie das Passwort nicht jedes Mal eingeben müssen, wenn Sie Seiten auf dieser Website veröffentlichen.

Composer-Einstellungen – Composer

Mit den Composer-Einstellungen können Sie Einstellungen zum Speichern von Dateien und zum Bearbeiten von Tabellen definieren. Diese Einstellungen gelten für alle Dokumente, die Sie erstellen.

Wenn Sie das Fenster Einstellungen für Composer noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie auf die Kategorie Composer.
  • Maximale Anzahl der aufgelisteten Seiten: Geben Sie die Anzahl der Seiten an, die im Menü Datei unter Zuletzt geöffnete Seiten maximal angezeigt werden sollen.
  • Formatierung der ursprünglichen Quelle beibehalten: Diese Option behält die Zeilenumbrüche und die ursprüngliche Formatierung der Seite für den HTML-Quellcode bei. Wählen Sie diese Option, wenn Composer zusätzliche Leerschritte (zusätzliche Zeilen, Tabulatoren usw.) nicht entfernen soll, die die Lesbarkeit des HTML-Codes verbessern. Diese Voreinstellung wirkt sich nicht auf die Darstellung der Seiten im Browserfenster aus.
  • HTML-Quelle neu formatieren: Diese Option formatiert den HTML-Quellcode neu durch Einfügen von Zeilenumbrüchen und Einrückungen, um die Lesbarkeit zu erhöhen. Diese Voreinstellung wirkt sich nicht auf die Darstellung der Seiten im Browserfenster aus.
  • Beim Speichern von Seiten Bilder und andere zugehörige Dateien speichern: Wenn diese Option aktiviert ist, werden alle Grafiken, JavaScript (JS), Cascading Style Sheet (CSS) und sonstige verknüpfte Dateien im gleichen Pfad wie das Dokument gespeichert, wenn Sie das Dokument zum ersten Mal speichern oder wenn das Dokument in einem neuen Pfad gespeichert wird. Wenn diese Option deaktiviert ist, wird nur die HTML-Datei gespeichert.

  • Wenn Sie beispielsweise eine Remoteseite bearbeiten, wird damit sichergestellt, dass alle Dateien, die zu dieser Remoteseite gehören, beim Speichern lokal auf der Festplatte gespeichert werden.

  • Beim Publizieren von Seiten immer das Dialogfeld 'Publizieren' anzeigen: Wenn diese Option aktiviert ist, öffnet Composer stets das Dialogfeld Seite publizieren, wenn Sie auf die Schaltfläche Veröffentlichen klicken oder im Menü Datei auf Veröffentlichen klicken. Wenn diese Option nicht aktiviert ist, öffnet Composer das Dialogfeld Seite publizieren nur, wenn zusätzliche Informationen zum Veröffentlichen der Seite erforderlich sind.
  • Tabellenlayout beim Einfügen oder Löschen von Zellen bewahren: Aktivieren Sie dieses Kontrollkästchen, wenn in Composer immer das Layout der Tabelle (rechteckige Form) beibehalten werden soll, indem bei Bedarf zusätzliche Zellen hinzugefügt werden. Wenn Sie diese Option deaktivieren, entfernt Composer auch den Zellenrand, wenn Sie eine oder mehrere Zellen löschen, wodurch die Tabelle leere Stellen oder einen unregelmäßigen Umriss aufgrund einer ungeraden Anzahl von Zellen aufweisen kann.
  • Verwenden von CSS-Stilen anstelle von HTML-Elementen und -Attributen: Diese Option ermöglicht die Verwendung von CSS-Formatierungen in Composer-Dokumenten. Wenn diese Option aktiviert ist, generiert Composer HTML 4.01-Formatierungen mit CSS-Formaten für die Elemente.

  • Wenn diese Option nicht aktiviert ist, generiert Composer HTML 4.01-Formatierungen, jedoch ohne CSS-Formate.

    Im Vergleich zu HTML zeichnet sich HTML mit CSS-Formatierungen durch eine höhere Portierbarkeit, eine einfachere Verwaltung und eine höhere Kompatibilität beim Betrachten der Seiten in unterschiedlichen Browsern aus. Wenn Sie diese Option aktivieren und ein Dokument bearbeiten, das ohne CSS erstellt wurde, ersetzt Composer die bearbeiteten Elemente durch CSS-Formate.

    Wenn Sie CSS-Formate aktivieren, können Sie auf der Formatierungs-Symbolleiste mithilfe der entsprechenden Schaltfläche eine Farbe zum Hervorheben von markiertem Text auswählen. Darüber hinaus können Sie für jedes Seitenelement eine Hintergrundfarbe auswählen. (Diese Funktionen sind nur verfügbar, wenn diese Option aktiviert ist.)

Composer-Einstellungen – Einstellungen für Neue Seite

Mit den Einstellungen für Neue Seite definieren Sie Einstellungen für Farben und Hintergrundgrafiken, die für jedes Dokument, das Sie erstellen, gelten sollen.

Wenn Sie das Fenster Einstellungen für Neue Seite noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Doppelklicken Sie auf die Kategorie Composer, und klicken Sie auf Einstellungen für Neue Seite.
  • Autor: Geben Sie Ihren Namen ein. Damit wird Ihr Name für jede neue Seite, die Sie erstellen, zum HTML-Quellcode hinzugefügt.
  • Standardfarben des Leseprogramms: Wählen Sie diese Option, wenn Ihre Seiten immer die Farbeinstellungen für Text- und Verknüpfungselemente des Browsers der anzeigenden Person verwenden sollen.
  • Benutzerdefinierte Farben verwenden: Wählen Sie diese Option, wenn Sie die Farben der Text- und Verknüpfungselemente immer angeben möchten. Wählen Sie dann für jedes Element eine Farbe aus, indem Sie neben jedem Element auf die Farbschaltfläche klicken.
  • Hintergrundgrafik: Geben Sie den Speicherort und den Namen einer Grafikdatei ein, oder klicken Sie auf Datei wählen, um die Grafikdatei auf Ihrer Festplatte oder im Netzwerk zu suchen.

    Anmerkung: Hintergrundgrafiken sind gekachelt und überschreiben die Hintergrundfarbe.

So ändern Sie den Namen des Autors für eine einzelne Seite: Klicken Sie im Menü Absatz auf Seitentitel und Eigenschaften.

So ändern Sie die Seitenfarben und die Hintergrundgrafik für eine einzelne Seite: Klicken Sie im Menü Absatz auf Seitenfarben und –hintergrund.
