Gesamtbildeinstellungen – Gesamtbild

Dieser Abschnitt beschreibt die Verwendung der Einstellungsfenster für Gesamtbild. Wenn Sie noch keines dieser Fenster geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie auf die Kategorie Gesamtbild.


Im Fenster Einstellungen für Gesamtbild legen Sie die Komponenten fest, die beim Start von Netscape verwendet werden sollen.

  • Beim Start von Netscape öffnen: Wählen Sie hier die Komponenten aus, die beim Start von Netscape verwendet werden sollen.
  • QuickInfo anzeigen: Wählen Sie diese Option aus, um Kurzinformationen anzuzeigen, wenn Sie den Mauszeiger auf ein Oberflächenelement von Netscape setzen.
  • Web-Site-Symbole anzeigen: Wählen Sie diese Option aus, wenn Sie die speziellen Symbole der Websites, falls vorhanden, statt des Lesezeichensymbols anzeigen möchten. Websitesymbole werden links von der Adressleiste und in Ihren Lesezeichenlisten verwendet.

 

Gesamtbildeinstellungen – Schriftarten

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Schriftarten beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Gesamtbild auf Schriftarten. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Gesamtbild, um die Liste zu erweitern.)


Im Fenster Einstellungen für Schriftarten legen Sie die Schriftart- und -größe für Seiten sowie die Bildschirmauflösung fest.

Anmerkung: Einige Schriftschnitte können möglicherweise nicht ausgewählt werden, da die ausgewählte Schriftart diese Schriftschnitte nicht zur Verfügung stellt.

  • Schriftart für: Wählen Sie hier einen Zeichensatz für eine Sprache aus. Klicken Sie beispielsweise auf Western, wenn Sie Standardschriftarten für den Zeichensatz Western (Roman) festlegen möchten.
    • Proportional: Legen Sie fest, ob Proportionaltext mit Serifen (z. B. in Times Roman) oder ohne (z. B. in Arial) dargestellt werden soll. Sie können auch die Schriftgröße für Proportionaltext angeben. Proportionaltext bedeutet, dass Buchstaben und Ziffern unterschiedlich breit sind ("i" beansprucht weniger Platz als "m").
    • Serif: Wählen Sie eine Serifenschrift aus, die Sie für Webseiten verwenden möchten.
    • Sans-serif: Wählen Sie eine serifenlose Schrift aus, die Sie für Webseiten verwenden möchten.
    • Kursiv: Wählen Sie eine Kursivschrift, die Sie für Webseiten verwenden möchten.
    • Fantasy: Wählen Sie eine Schmuckschrift aus, die Sie für Webseiten verwenden möchten.
    • Nichtproportional: Wählen Sie eine Schrift mit fester Zeichenbreite (z. B. Courier) aus, die Sie für Webseiten verwenden möchten. Bei einer solchen Schrift sind alle Zeichen und Ziffern gleich breit ("i" beansprucht genauso viel Platz wie "m").
    • Mindestschriftgröße: Wählen Sie die kleinste Schriftgröße aus, die auf Webseiten dargestellt werden soll.
  • Andere Schriftarten in Dokumenten zulassen: Wenn dieses Kontrollkästchen verfügbar ist, aktivieren Sie es, um die Originalschrift einer Webseite anstelle Ihrer Einstellung zu verwenden.
  • Bildschirmauflösung: Wählen Sie in der Dropdown-Liste die Auflösung in dpi (Punkte pro Zoll) für die Darstellung von Webseiten. Klicken Sie auf Sonstige, um das Dialogfeld Auflösung kalibrieren zu öffnen, in dem Sie Ihre Auflösung durch Messen der Länge einer auf dem Bildschirm dargestellten Zeile einstellen lassen können. Eine höhere Bildschirmauflösung kann auf einigen Bildschirmen die Lesbarkeit von Text verbessern.

 

Gesamtbildeinstellungen – Farben

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Farben beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Gesamtbild auf Farben. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Gesamtbild, um die Liste zu erweitern.)


Im Fenster Einstellungen für Farben können Sie die Hintergrund- und Textfarbe für Webseiten einstellen.

  • Text und Hintergrund: Klicken Sie auf die farbigen Blöcke, um eine Farbe für Text und Hintergrund von Webseiten auszuwählen.
    • Systemfarben verwenden: Wählen Sie diese Option aus, um die Farbeinstellungen des Systems zu verwenden.
  • Verknüpfungsfarben: Klicken Sie auf die farbigen Blöcke, um eine Farbe für die Darstellung besuchter und nicht besuchter Verknüpfungen auf Webseiten auszuwählen.
    • Verknüpfungen unterstreichen: Wählen Sie diese Option aus, um Verknüpfungen auf Webseiten unterstrichen darzustellen.
  • Wenn eine Webseite eigene Farben und eigenen Hintergrund bietet:
    • Immer Farben und Hintergrund der Webseite verwenden: Überlässt der Webseite die Darstellung der Farben und des Hintergrunds.
    •  Die selbst gewählten Farben verwenden und die vorhandenen Farben und Hintergrundbilder  ignorieren: Ermöglicht Ihnen die Auswahl der dargestellten Farben unabhängig von den Farben und dem Hintergrundbild der Webseite.
  • Syntaxmarkierung aktivieren: Wählen Sie diese Option aus, um die HTML-Syntax der Quelle einer Webseite in bestimmten Farben hervorgehoben anzuzeigen.

    Tipp: Um die Quelle einer Webseite anzuzeigen, klicken Sie in Navigator im Menü Anzeigen auf Seitenquelltext.

 

Gesamtbildeinstellungen – Themes

Dieser Abschnitt beschreibt die Verwendung des Fensters Einstellungen für Themes. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie Gesamtbild auf Themes. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Gesamtbild, um die Liste zu erweitern.)


Im Fenster Einstellungen für Themes können Sie das Erscheinungsbild von Netscape mithilfe verschiedener Themes ändern.

  • Installierte Themes: Wählen Sie zum Ändern des Erscheinungsbilds von Netscape eines der installierten Themes aus.
    • Theme-Name deinstallieren: Klicken Sie auf diese Schaltfläche, um ein ausgewähltes Theme zu entfernen.
  • Neue Themes holen: Klicken Sie auf diese Verknüpfung, um zu einer Webseite zum Downloaden weiterer Themes zu wechseln.

    Anmerkungen:

    • Damit ein neues Theme wirksam wird, müssen Sie Netscape nach der Auswahl des Theme beenden und neu starten.
    • Wenn Sie ein Theme deinstalliert haben, können Sie nicht wieder darauf zurückgreifen.

 

Gesamtbildeinstellungen – Sprachen/Inhalt

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Sprachen und Inhaltspakete beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie unter der Kategorie Gesamtbild auf Sprachen/Inhalt. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Gesamtbild, um die Liste zu erweitern.)


Im Fenster Einstellungen für Sprachen und Inhaltspakete können Sie Inhalts- und Sprachpakete für die Verwendung in Netscape auswählen.

  • Installierte Inhaltspakete: Wählen Sie ein Inhaltspaket für die Benutzung mit Netscape aus.
  • Installierte Sprachpakete: Wählen Sie ein Sprachpaket für die Verwendung mit Netscape aus.

Anmerkung: Sie müssen Netscape neu starten, damit das neue Inhalts- oder Sprachpaket wirksam wird.
