Privatsphäre im Internet

Dieser Abschnitt enthält Hintergrundinformationen über den Schutz Ihrer Privatsphäre im Internet. Er beschreibt außerdem verschiedene Maßnahmen, die Sie ergreifen können, um Ihre Privatsphäre zu schützen. Er kann jedoch nicht als umfassende Beschreibung zum Thema Privatsphäre im Internet betrachtet werden.

In diesem Abschnitt werden folgende Themen behandelt:

Welche Informationen gibt der Browser an eine Website weiter?

Wenn der Browser eine Webseite anzeigt, beispielsweise beim Klicken auf eine Verknüpfung oder bei der Eingabe eines URL, oder wenn eine Webseite in einer eMail-Nachricht dargestellt wird, werden bestimmte Arten von Informationen an die Website weitergegeben. Diese Informationen können unter anderem Angaben zu Ihrer Betriebsumgebung, Ihrer Internetadresse und Ihrer Startseite beinhalten.

 

Betriebsumgebung

Der Site wird etwas über Ihre Betriebsumgebung mitgeteilt, beispielsweise über den von Ihnen verwendeten Browser und Informationen über Ihr Betriebssystem. So kann die Website die aufgerufene Seite optimal auf Ihrem Bildschirm präsentieren. Die Website könnte zum Beispiel die Information erhalten, dass Sie die französische Version des Browsers Netscape 6.2.3 auf einem Computer mit dem Betriebssystem Windows 2000 ausführen.

 

Internetadresse

Der Browser muss der Website Ihre Internetadresse (auch Internet-Protocol- bzw. IP-Adresse genannt) angeben, damit die Website "weiß", wohin sie die angeforderte Seite übertragen soll. Die Website kann die gewünschte Seite nur dann anzeigen, wenn Ihre IP-Adresse bekannt ist.

Ihre IP-Adresse kann entweder temporär oder fest (statisch) sein.

Wenn Sie die Verbindung zum Internet über ein Standardmodem aufbauen, das an Ihre Telefonleitung angeschlossen ist, weist Ihnen der Internetdienstanbieter (ISP) möglicherweise bei jedem Verbindungsaufbau eine temporäre IP-Adresse zu. Diese temporäre IP-Adresse verwenden Sie für die Dauer Ihrer Internetsitzung, beispielsweise bis Sie die DFÜ-Verbindung unterbrechen oder auf eine andere Weise die bestehend Verbindung mit dem Internet trennen. Jeder ISP verfügt über eine Vielzahl von IP-Adressen und weist diese den Benutzern nach dem Zufallsprinzip zu.

Falls Sie über eine DSL-, Kabelmodem- oder Glasfaserverbindung verfügen, besitzen Sie möglicherweise eine feste IP-Adresse, die Sie bei jedem Einwählvorgang verwenden.

Ihre IP-Adresse ist nicht mit Ihrer eMail-Adresse identisch.

 

Referenzseite

Der Website wird auch mitgeteilt, welche Seite Sie gelesen haben, als Sie auf eine Verknüpfung zu einer der Seiten der Website geklickt haben. So wird die Website darüber informiert, von welcher anderen Website Sie weitergeleitet wurden. Wenn Sie sich innerhalb einer Website bewegen, wird übermittelt, von welcher Seite Sie gerade kommen.

 

Was sind Cookies und wie funktionieren sie?

Cookies bestehen aus wenigen Informationen und werden von manchen Websites verwendet. Wenn Sie eine Website besuchen, die Cookies verwendet, fordert diese Ihren Browser möglicherweise auf, ein oder mehrere Cookies auf Ihrer Festplatte zu speichern.

Beim nächsten Besuch der Website sendet Ihr Browser die entsprechenden Cookies zurück.

Wenn Sie die Standardeinstellung für Cookies verwenden, ist dieser Vorgang für Sie nicht erkennbar. Sie wissen nicht, wann eine Website ein Cookie einrichtet oder der Browser das Cookie einer Website zurücksendet. Sie können jedoch einstellen, dass Sie vor dem Einrichten eines Cookies gefragt werden möchten. Weitere Informationen dazu finden Sie unter Festlegen der Privatsphärenebene.

 

Wie verwenden Sites die Informationen der Cookies?

Cookies ermöglichen einer Website, etwas über Ihre früheren Besuche zu erfahren. Wenn Sie beispielsweise häufig nach dem aktuellen Wetter sehen oder Bücher in einer Website kaufen, wird mit einem Cookie gespeichert, in welcher Stadt Sie wohnen oder welche Autoren Sie vorziehen. Auf diese Weise kann Ihr nächsten Besuch der Website einfacher und effizienter gestaltet werden.

Seriöse Websites verfügen über Vertraulichkeitsrichtlinien, in denen festgelegt ist, wie die gesammelten Informationen verwendet werden.

Sie können die Sicherheitsrichtlinien einer Website überprüfen und feststellen, was sie über die gesammelten Informationen, deren Verwendung und Ihren möglichen Einfluss in Bezug auf die Verwendung dieser Informationen besagen. Weitere Informationen dazu finden Sie unter Anzeigen der Privatsphärenrichtlinien.

Weitere Informationen zum Verwalten von Cookies mithilfe von veröffentlichten Sicherheitsrichtlinien finden Sie unter Festlegen der Privatsphärenebene.

 

Was sind Cookies von Drittanbietern?

Wenn der Browser das Cookie einer bestimmten Website speichert, gibt er es später nur an diese eine Website zurück. Der Browser sendet keine Cookies zu einer Website, die von einer anderen Website gespeichert wurden. Da eine Website nur ihre eigenen Cookies empfangen kann, kann sie Ihre Aktivitäten in dieser Website verfolgen, nicht aber, was Sie sonst noch tun, wenn Sie im Internet surfen.

Manchmal stellt eine Website jedoch Inhalte dar, die sich auf einer anderen Website befinden. Das kann alles Mögliche sein, von Grafiken zu Text oder Werbung. Die Website, die diese Elemente enthält, kann ebenfalls ein Cookie in Ihrem Browser hinterlassen, auch wenn Sie die Site nicht direkt besuchen.

Cookies, die von anderen Websites als der gerade besuchten gespeichert werden, sind Cookies von Drittanbietern bzw. Fremde Cookies. Bei manchen Websites werden Cookies von Drittanbietern mit transparenten GIFs verwendet. Dabei handelt es sich um spezielle Grafiken, die den Betreibern der Website dabei helfen, Besucher zu zählen, eMail-Antworten zu protokollieren, mehr über die Besucher zu erfahren oder sich auf deren Surfverhalten einzustellen. (Transparente GIFs werden auch als Web-Beacons oder Web-Bugs bezeichnet.)

Sie können die Cookie-Einstellungen so anpassen, dass herkömmliche Cookies, aber keine Cookies von Drittanbietern gespeichert werden.

 

Wie können Webseiten in eMail-Nachrichten gesteuert werden?

Sie können Cookies, Grafiken, JavaScript und Plugins für Webseiten, die in eMail-Nachrichten gesendet werden, vollständig deaktivieren.

Es kann zwar praktisch sein, einige oder alle dieser Funktionen beim Surfen im Web aktiviert zu haben, doch in einzelnen, als Anlage zu Nachrichten gesendeten Webseiten, sind sie möglicherweise überflüssig.

Wie kann sichergestellt werden, dass unbefugte Personen keine persönlichen Informationen verwenden?

Die beste Methode zum Schutz Ihrer Informationen besteht darin, die Privatsphärenrichtlinien der besuchten Websites und der verwendeten Internetdienste zu lesen und vorsichtig bei der Online-Weitergabe von persönlichen Informationen zu sein.

Das Internet ist ein öffentliches Netzwerk. Wenn Sie Namen, Telefonnummer, Adresse und andere persönlichen Informationen in diesem Netzwerk weiterleiten (per Webseite, eMail oder auf andere Weise), besteht die Möglichkeit, dass sie abgefangen werden.

Hier sind einige Fragen, mit denen Sie herausfinden können, wie vertraulich Ihre Daten von einer Website behandelt werden:

  • Welche Arten von persönlichen Daten werden von der Website gesammelt?
  • Wofür werden diese Daten verwendet?
  • Werden diese Daten weitergegeben und haben Sie Einfluss auf die Verwendung der weitergegebenen Daten?
  • Können Sie auf einige oder alle von der Website über Sie gesammelten Informationen zugreifen, um diese zu prüfen oder zu aktualisieren?
  • Wie werden diese Daten geschützt?
  • Wie kontaktieren Sie die Website, wenn Fragen oder Probleme auftreten?

Weitere Informationen zum Überprüfen der Privatsphärenrichtlinien einer Website finden Sie unter Anzeigen der Privatsphärenrichtlinien.
