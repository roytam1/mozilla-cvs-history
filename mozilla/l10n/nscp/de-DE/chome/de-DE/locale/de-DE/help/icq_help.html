Verwenden von ICQ in Netscape

Mit dem Instant Messaging-Client von Netscape können Sie AIM- und ICQ-Nachrichten senden und empfangen. Zum Senden und Empfangen von ICQ-Nachrichten müssen Sie bei ICQ angemeldet sein. AIM und ICQ können nicht gleichzeitig verwendet werden. Sie benötigen ein ICQ-Konto und müssen eine ICQ-Adressliste erstellen. Einige AIM-Buddies besitzen möglicherweise ICQ-Konten und können deshalb als ICQ-Adresse verwendet werden, das geschieht jedoch nicht automatisch.

Öffnen Ihres ersten ICQ-Kontos

So öffnen Sie ein neues ICQ-Konto in Netscape:

  1. Klicken Sie in Meine Seitenleiste auf Buddy-Liste oder klicken Sie im Menü Fenster auf Instant Messenger.
  2. Wählen Sie ICQ in der Dropdown-Liste Dienst zum Anmelden auswählen aus.
  3. Klicken Sie auf Neuen ICQ-Benutzer registrieren.

Die Seite ICQ Web Registration wird in Navigator geöffnet.

  1. Geben Sie Ihre eMail-Adresse ein und klicken Sie auf Senden.

Eine zweite Seite Web Registration wird angezeigt.

  1. Klicken Sie auf Click here to register and get a new ICQ number.
  2. Lesen Sie den Lizenzvertrag und klicken Sie auf I Decline oder I Agree.

Wenn Sie auf I Agree geklickt haben, wird die letzte Seite Web Registration mit einem Textfeld für Ihren Bestätigungscode angezeigt.

  1. Sie erhalten eine eMail mit Ihrem Bestätigungscode (in der Regel innerhalb weniger Minuten). Kopieren Sie ihn in das Textfeld für den Bestätigungscode. Klicken Sie dann auf Senden. Sie können auch auf die Verknüpfung in der eMail klicken und den Code automatisch eintragen lassen.
  2. Geben Sie in das angezeigte Formular ein Pseudonym ein, dann Ihren Vor- und Nachnamen, Ihr Geschlecht und Alter und ein vier bis acht Zeichen langes Passwort.
  3. Klicken Sie auf Senden.

Ein neues Fenster wird mit Ihrer ICQ-Nummer angezeigt. Sie sind jetzt für das Austauschen von ICQ-Nachrichten mit anderen ICQ-Mitgliedern bereit.

Öffnen eines ICQ-Kontos mithilfe einer vorhandenen Nummer und eines Passworts

So richten Sie Ihr ICQ-Konto für die Verwendung in Netscape ein, wenn Sie bereits eine ICQ-Nummer und ein Passwort besitzen:

  1. Klicken Sie in Meine Seitenleiste auf Buddy-Liste oder klicken Sie im Menü Fenster auf Instant Messenger.
  2. Wählen Sie ICQ in der Dropdown-Liste Dienst zum Anmelden auswählen aus.
  3. Klicken Sie auf Neuen ICQ-Benutzer registrieren.

Die Seite ICQ Web Registration wird in Navigator geöffnet.

  1. Geben Sie Ihre eMail-Adresse ein und klicken Sie auf Senden.
  2. Geben Sie auf der zweiten Seite der Web Registration Ihre ICQ-Nummer und das Passwort ein und klicken Sie auf Senden.
  3. Lesen Sie den Lizenzvertrag und klicken Sie auf I Decline oder I Agree.

Wenn Sie auf I Agree geklickt haben, wird die letzte Seite der Web Registration mit einem Textfeld für Ihren Bestätigungscode angezeigt.

  1. Sie erhalten eine eMail mit Ihrem Bestätigungscode (in der Regel innerhalb weniger Minuten). Kopieren Sie ihn in das Textfeld für den Bestätigungscode. Klicken Sie dann auf Senden. Sie können auch auf die Verknüpfung in der eMail klicken und den Code automatisch eintragen lassen.
  2. Geben Sie in das angezeigte Formular ein Pseudonym ein, dann Ihren Vor- und Nachnamen und optional Ihr Geschlecht und Alter. Klicken Sie dann auf Senden.

Sie sind jetzt für das Austauschen von ICQ-Nachrichten mit anderen ICQ-Benutzern bereit.

Anmeldeverfahren

Sie können sich jedes Mal, wenn Sie den Dienst verwenden möchten, manuell bei ICQ anmelden oder Sie können ICQ so einrichten, dass es beim Starten von Netscape oder des Computers ebenfalls gestartet wird.

In diesem Abschnitt werden folgende Themen behandelt:

Anmelden bei ICQ

Starten von ICQ beim Systemstart

 

Anmelden bei ICQ

So melden Sie sich bei ICQ an:

  1. Öffnen Sie ICQ, indem Sie im Menü Fenster auf Instant Messenger oder in Meine Seitenleiste auf den Eintrag Buddy-Liste klicken.
  2. Wenn ICQ noch nicht ausgewählt ist, wählen Sie es in der Dropdown-Liste Dienst zum Anmelden auswählen aus.
  3. Geben Sie Ihre ICQ-Nummer oder Ihren Anzeigenamen und Ihr ICQ-Passwort ein.

  4. Aktivieren Sie eines oder beide Kontrollkästchen Passwort speichern und Beim Start anmelden.

Wenn Sie Beim Start anmelden aktivieren, werden Sie anschließend bei jedem Starten von Netscape automatisch bei ICQ angemeldet. Dieses Mal müssen Sie sich manuell anmelden.

  1. Klicken Sie auf die Schaltfläche Mit ICQ-Server verbinden.

Anmerkung: Wenn Sie sich bei ICQ angemeldet haben, ändert sich die Schaltfläche Mit ICQ-Server verbinden in Verbindung zu ICQ-Server trennen. Klicken Sie darauf, wenn Sie sich abmelden möchten.

Das Fenster zeigt jetzt die Registerkarte Online-Modus mit einer Liste all Ihrer ICQ-Kontakte an, die momentan online sind. Wer nicht online ist, wird nicht angezeigt. Ist ein Kontakt online, dann können Sie ihm ICQ-Nachrichten senden.

Starten von ICQ beim Systemstart

Auf Windows-Computern können Sie ICQ so einrichten, dass es beim Starten des Computers automatisch gestartet wird, ohne dass auch der Netscape-Browser startet. Sie können diese Option jedoch nicht in ICQ einstellen, sondern müssen dazu bei AOL Instant Messenger für Netscape angemeldet sein. Sie sollten dazu das ICQ/AIM-Hauptfenster geöffnet haben, das Sie über das Menü Fenster und nicht über Meine Seitenleiste öffnen.

  1. Wenn Sie bei ICQ angemeldet sind, klicken Sie auf die Schaltfläche Verbindung zu ICQ-Server trennen.
  2. Wählen Sie AIM in der Dropdown-Liste Dienst zum Anmelden auswählen aus.
  3. Geben Sie Ihren AIM-Namen und Ihr Passwort ein und klicken Sie auf die Schaltfläche Anmelden bei Instant Messenger.
  4. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  5. Klicken Sie auf die Instant Messenger-Einstellung.
  6. Wählen Sie Automatisch bei Systemstart starten aus.
  7. Klicken Sie auf OK.
  8. Klicken Sie auf die Schaltfläche Abmelden vomInstantMessenger.
  9. Wählen Sie ICQ in der Dropdown-Liste Dienst zum Anmelden auswählen aus.
  10. Geben Sie Ihre ICQ-Nummer oder Ihren Anzeigenamen und Ihr Passwort ein und klicken Sie auf die Schaltfläche Mit ICQ-Server verbinden.

ICQ wird jetzt bei jedem Starten des Computers geöffnet. Obwohl diese Option das ICQ-Fenster öffnet, werden Sie jedoch erst dann automatisch angemeldet, wenn Sie auch im ICQ-Fenster die Option Beim Start anmelden aktiviert haben.

Anmerkung: Das Starten beim Systemstart ist nur auf Windows-Computern verfügbar. Die Option ist auch nicht verfügbar, wenn Sie zuvor die separate Version des AOL Instant Messenger-Clients oder den separaten ICQ-Client so konfiguriert haben, dass er zusammen mit dem Computer automatisch gestartet wird.

Hinzufügen von ICQ-Kontakten

So fügen Sie ICQ-Kontakte hinzu:

  1. Klicken Sie auf die Registerkarte Kontaktliste, um die Kontaktliste zu öffnen.
  2. Klicken Sie auf die Schaltfläche Kontakt hinzufügen. Das Dialogfeld Kontakt hinzufügen wird angezeigt.
  3. Geben Sie die ICQ-Nummer und den Anzeigenamen (ein beliebiger von Ihnen gewählter Name) des Kontakts ein, aktivieren Sie das Kontrollkästchen für die Gruppe, der Sie den Kontakt zuweisen möchten, und klicken Sie auf OK.

Ein Kontakt kann nur einer Gruppe hinzugefügt werden.

Eine Autorisierungsanforderung wird automatisch gesendet, wenn der Benutzer diese Sicherheitsoption konfiguriert hat. Bis der vorgeschlagene Kontakt auf diese Nachricht antwortet, bleibt sein Name im Ordner Autorisation ausstehend. Wenn der Kontakt zustimmt, wird er der Kontaktliste hinzugefügt. Falls er ablehnt, bleibt sein Name im Ordner Autorisation ausstehend. Hat der Kontakt keine Autorisierung konfiguriert, wird keine Anforderung gesendet. Sein Anzeigename wird sofort in Ihre Kontaktliste aufgenommen.

Entfernen von ICQ-Kontakten

So entfernen Sie einen ICQ-Kontakt:

  1. Wählen Sie den Namen des Kontakts in der Kontaktliste aus.
  2. Klicken Sie auf die Schaltfläche Kontakt entfernen.
  3. Klicken Sie im Bestätigungsdialogfeld auf OK.

Der Kontakt wird entfernt.

Hinzufügen von ICQ-Gruppen

Der ICQ-Client wird mit vier Standardgruppen installiert: Allgemein, Familie, Freunde und Kollegen. So können Sie weitere Gruppen hinzufügen:

  1. Klicken Sie auf die Schaltfläche Gruppe hinzufügen. Das Dialogfeld Gruppe hinzufügen wird angezeigt.
  2. Geben Sie einen Gruppennamen ein und klicken Sie auf OK.

Die neue Gruppe wird in der Gruppenliste angezeigt.

Senden von ICQ-Nachrichten

Sie können ICQ-Nachrichten an jedes ICQ-Mitglied senden, unabhängig davon, ob es ICQ in Netscape oder als separaten ICQ-Client verwendet.

  1. Wenn ICQ geschlossen ist, öffnen Sie in Meine Seitenleiste die Buddy-Liste oder klicken Sie im Menü Fenster auf Instant Messenger.
  2. Wählen Sie gegebenenfalls ICQ in der Dropdown-Liste Dienst zum Anmelden auswählen aus.
  3. Klicken Sie gegebenenfalls auf die Registerkarte Online-Modus, um sie im Vordergrund anzuzeigen.
  4. Doppelklicken Sie auf die Nummer oder den Anzeigenamen des ICQ-Kontakts, mit dem Sie kommunizieren möchten.
  5. Geben Sie Ihre Nachricht in das Nachrichtenfenster ein und drücken Sie die EINGABETASTE.
  6. Wenn Sie eine Antwort erhalten, geben Sie eine Erwiderung in den unteren Bereich des Nachrichtenfensters ein und drücken Sie die EINGABETASTE.
  7. Wiederholen Sie die Schritte 5 und 6 bis zum Ende Ihrer Konversation.

Falls das ICQ-Mitglied nicht in Ihrer Kontaktliste steht, folgen Sie den obigen Schritten 1 bis 3, und öffnen Sie dann ein leeres Fenster, indem Sie auf die Schaltfläche Senden der Registerkarte Online-Modus klicken. Geben Sie den ICQ-Anzeigenamen oder die ICQ-Nummer der Person ein, mit der Sie kommunizieren möchten und folgen Sie den obigen Schritten 5 bis 7.

Empfangen von ICQ-Nachrichten

Das Empfangen von ICQ-Nachrichten funktioniert genauso wie das Empfangen von AIM-Nachrichten, außer dass Sie bei dem Dienst angemeldet sein müssen. Der ICQ-Dienst kann keine Nachrichten empfangen, wenn Sie offline sind. Beim Eingang einer Nachricht wird ein Nachrichtenfenster mit dem gesendeten Inhalt geöffnet. So antworten Sie:

  1. Geben Sie eine Antwort in den unteren Bereich des Nachrichtenfensters ein und drücken Sie die EINGABETASTE.
  2. Wiederholen Sie Schritt 1, wenn Sie eine Antwort erhalten.

ICQ-Einstellungen  Allgemein

Dieser Abschnitt beschreibt die Verwendung des allgemeinen Einstellungsfensters von ICQ. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie auf ICQ.

Wählen Sie unter folgenden Optionen aus:

  • Eingabetaste fügt einen Zeilenumbruch ein: Aktivieren Sie diese Option, wenn mit der Eingabetaste ein Zeilenumbruch eingefügt werden soll. Deaktivieren Sie diese Option, wenn beim Drücken der Eingabetaste die Nachricht gesendet werden soll.
  • Tabulatortaste fügt einen Tabulator ein: Aktivieren Sie diese Option, wenn mit der Tabulatortaste ein Tabulator eingefügt werden soll. Deaktivieren Sie diese Option, wenn Sie mit der Tabulatortaste im ICQ-Fenster zwischen den Feldern wechseln möchten.
  • Zeitanzeige auf allen Nachrichten anzeigen: Aktivieren Sie diese Option, wenn Nachrichten in neuen ICQ-Sitzungen mit dem Sendedatum versehen werden sollen.

ICQ-Einstellungen  Privatsphäre & Authorisierung

In diesem Abschnitt wird die Verwendung des ICQ-Fensters Einstellungen für Privatsphäre & Authorisierung beschrieben. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Melden Sie sich bei ICQ an. (Sie müssen angemeldet sein, um die Einstellungen für die Privatsphäre ändern zu können.)
  2. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  3. Klicken Sie in der Kategorie ICQ auf Privatsphäre & Authorisierung. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf ICQ, um die Liste zu erweitern.)

Wählen Sie unter folgenden Optionen aus:

  • Wenn Sie möchten, dass Ihnen nur die Benutzer in Ihrer Kontaktliste Nachrichten senden können, klicken Sie unter Wer kann mich kontaktieren auf Nur Benutzer in meiner Kontaktliste zulassen.
  • Wenn Sie einen oder mehrere Kontakte blockieren möchten, klicken Sie auf Hinzufügen von ICQ für den ersten zu blockierenden Benutzer und geben Sie dann in dem Dialogfeld die ICQ-Nummer des Kontakts ein. Wiederholen Sie diesen Vorgang, bis alle gewünschten Benutzer blockiert sind.
  • Wenn Sie die Blockierung eines oder mehrerer Benutzer aufheben möchten, wählen Sie deren Nummer in der Liste der blockierten Benutzer aus und klicken Sie auf Entfernen.

ICQ-Einstellungen  Benachrichtigung & Klänge

In diesem Abschnitt wird die Verwendung des ICQ-Fensters Einstellungen für Benachrichtigung & Klänge beschrieben. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie ICQ auf Benachrichtigung & Klänge. (Wenn in dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf ICQ, um die Liste zu erweitern.)

Aktivieren Sie die Kontrollkästchen der zur Auswahl stehenden Einstellungen:

  • Einen Klang abspielen, wenn Wenn Sie diese Option auswählen, wird standardmäßig für alle aufgelisteten Ereignisse ein Klang abgespielt. Deaktivieren Sie die Ereignisse, für die kein Klang abgespielt werden soll. Sie können nach einer Klangdatei suchen, indem Sie auf die Schaltfläche Durchsuchen klicken und dann zu Systemsteuerung, Sounds und Multimedia wechseln. Wenn Sie einen Klang ausgewählt haben, können Sie eine Vorschau abspielen, indem Sie auf die Schaltfläche Vorschau klicken. Wenn Sie die Option zum Abspielen eines Klangs auswählen, aber keinen bestimmten Klang angeben, wird das Standardsignal verwendet.

ICQ-Einstellungen  Verbindung

In diesem Abschnitt wird die Verwendung des ICQ-Fensters Einstellungen für Verbindung beschrieben. Wenn Sie dieses Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

  1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
  2. Klicken Sie in der Kategorie ICQ auf Verbindung. (Wenn keine Unterkategorien angezeigt werden, doppelklicken Sie auf ICQ, um die Liste zu erweitern.)

Wenn Sie Internetverbindungen über einen Internetdienstanbieter herstellen, müssen Sie höchstwahrscheinlich keine Änderungen in diesem Fenster vornehmen. Wenn Sie Internetverbindungen über eine LAN-Firewall herstellen (z. B. am Arbeitsplatz), benötigen Sie möglicherweise vom Netzwerkadministrator Informationen zum Konfigurieren des Proxyabschnitts in diesem Fenster.

Host

In diesem Feld ist die Standardinternetadresse des in Netscape integrierten Instant Messaging-Servers enthalten, der AOL Instant Messenger für Netscape und ICQ für Netscape unterstützt. Normalerweise müssen Sie dieses Feld nicht ändern. Wenn der Proxyserver den Hostnamen auflösen muss, müssen Sie möglicherweise die IP-Adresse anstelle des Servernamens in dieses Feld eingeben. Die IP-Adresse erhalten Sie von Ihrem Netzwerkadministrator.

Port

In diesem Feld ist die Standard-Serverportnummer, die der in Netscape integrierte Instant Messaging-Server verwenden soll, enthalten. Normalerweise müssen Sie dieses Feld nicht ändern.

Verbinden mit Proxy

Wenn Sie dieses Kontrollkästchen aktivieren, wird durch die Instant Messenger-Software die Proxyserversteuerung aktiviert. Lassen Sie sich die entsprechenden Proxyserver-, Authentifizierungs- und Protokollinformationen vom Netzwerkadministrator geben und tragen Sie sie ein.
