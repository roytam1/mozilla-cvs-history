In Netscape können Sie für viele Befehle die Tastatur anstelle der Maus verwenden. Die Verwendung der Tastatur für einen Befehl kann oft Zeit sparen. Außerdem können Menschen mit körperlichen Einschränkungen Netscape mithilfe der Tastatur besser nutzen.

Verwenden von Tastaturbefehlen

Zur Beschreibung der Tastaturbefehle wird das folgende Format verwendet:

Beispiel:

Befehl
Windows
Mac
Linux
Kopieren
STRG+C
BEFEHL+C
STRG+C

Zum Ausführen eines Befehls drücken Sie die aufgeführten Tasten gleichzeitig. Wenn Sie beispielsweise in Windows etwas kopieren möchten, drücken Sie gleichzeitig STRG und C.

Tipp: Das Pluszeichen (+) bedeutet, dass Sie die Tasten vor und hinter dem Zeichen gleichzeitig drücken müssen. Falls mehr als ein Pluszeichen vorhanden ist (wie bei STRG+UMSCHALT+C), müssen alle aufgeführten Tasten gleichzeitig gedrückt werden.

Abkürzungen:

  • STRG = die Steuerungstaste
  • BEFEHL = die Befehltaste der Macintosh-Tastatur

Anmerkung: Bei einigen Tastaturbefehlen hängt die Funktion von der Position des Cursors ab (Fokus). Wenn Sie beispielsweise POS1 in Windows drücken, während Sie eine Webseite anzeigen, wechseln Sie an den Anfang der Webseite. Wenn Sie jedoch POS1 in Windows drücken, während der Cursor in einem Textfeld platziert ist, bewegt sich der Cursor an den Anfang des Textfeldes.

Tipp: In Windows und Linux können Sie mithilfe der Tastatur auf Menüoptionen zugreifen. Um ein Hauptmenü zu öffnen, drücken Sie ALT und den unterstrichenen Buchstaben im Menünamen. In einem geöffneten Menü können Sie die Untermenüs oder Befehle ohne Drücken von ALT öffnen. Drücken Sie beispilesweise ALT+F, um das Menü Datei zu öffnen.
