Erweiterte Einstellungen

Dieser Abschnitt beschreibt die Verwendung des Fensters Einstellungen für Erweitert. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie auf die Kategorie Erweitert.

In diesem Abschnitt werden folgende Themen behandelt:

Erweitert

Scripts & Plugins

Cache

Proxies

HTTP-Networking

Software-Installation

Mausrad

System

 

Erweiterte Einstellungen – Erweitert

Dieser Abschnitt beschreibt die Verwendung des Hauptfensters Einstellungen für Erweitert. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie auf die Kategorie Erweitert.

Im Hauptfenster Einstellungen für Erweitert können Sie Java, FTP-Passwörter und Quick Launch anpassen:

  • Java aktivierenWählen Sie diese Option aus, damit Netscape Seiten interpretieren kann, die Java enthalten. Java wird auf einigen Webseiten zur Integration interaktiver und multimedialer Inhalte verwendet.
  • XSLT aktivieren: Wählen Sie diese Option aus, damit Netscape XSLT verwenden kann. Mit der Sprache XSLT (eXtensible Stylesheet Language/Transform) werden XML-Dokumente in andere XML-Dokumente umgewandelt. Weitere Informationen zu XSLT finden Sie im Onlinedokument Transforming XML: Netscape and XSLT (Nur auf Englisch verfügbar).
  • Diese eMail-Adresse als anonymes Passwort senden: Geben Sie eine eMail-Adresse ein, die Sie als Passwort verwenden möchten, wenn Sie sich anonym bei einem FTP-Server anmelden.
  • Quick Launch aktivieren: Wählen Sie diese Option aus, um die Startzeit von Netscape zu verkürzen. Weitere Informationen zu Quick Launch finden Sie unter Verwenden von Quick Launch.

    Anmerkung: Quick Launch ist nur für Windows-Plattformen verfügbar.

 

Erweiterte Einstellungen – Scripts & Plugins

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Scripts & Plugins beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Einstellungen auf Scripts & Plugins. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)


Im Fensters Einstellungen für Scripts & Plugins können Sie die Verwendung von JavaScript und Plugins in Netscape steuern:

  • JavaScript für Navigator aktivieren: Wählen Sie diese Kontrollkästchen aus, um JavaScript zu aktivieren.
    • Navigator: Schaltet für Webseiten, die in Navigator geöffnet werden, auf JavaScript um.
    • eMail & Diskussionsforen: Schaltet für Webseiten, die in eMail & Diskussionsforen geöffnet werden, auf JavaScript um.
  • Webpages erlauben: Aktivieren Sie diese Kontrollkästchen, um die Verwendung von JavaScript zu steuern.
    • Verknüpfung in neuem Fenster öffnen: Ermöglicht, dass angeklickte Verknüpfungen in einem neuen Fenster geöffnet werden. Einige Verknüpfungen sind so konfiguriert, dass sie in einem neuen und nicht im selben Fenster geöffnet werden. Das Ändern dieses Steuerelements wird nach dem Neustarten von Netscape wirksam.
    • Bestehende Fenster zu verschieben oder ihre Größe zu ändern: Ermöglicht, dass geöffnete Fenster verschoben oder in der Größe geändert werden können.
    • Fenster zu positionieren: Ermöglicht, dass Fenster unter oder über anderen Fenstern platziert werden können.
    • Die Statusleiste auszublenden: Ermöglicht das Verbergen der Statusleiste.
    • Den Text in der Statusleiste zu ändern: Ermöglicht das Ändern von Text in der Statusleiste, wie beispielsweise Bildlauftext der Statusleiste.
    • Bilder zu ändern: Ermöglicht das Ändern oder Animieren von Bildern, wie bei Bild-Rollovers (Bilder, die sich ändern, wenn der Mauszeiger darüber bewegt wird).
    • Cookies zu erstellen bzw. zu ändern: Ermöglicht das Ändern oder Erstellen von Cookies mithilfe von JavaScript.
    • Cookies zu lesen: Ermöglicht das Lesen von Cookies mithilfe von JavaScript.
  • Aktivieren von Plugins für: Aktivieren Sie dieses Kontrollkästchen, um die Verwendung von Plugins zu steuern.
    • eMail & Diskussionsforen: Ermöglicht die Verwendung von Plugins in eMail & Diskussionsforen.

Weitere Informationen zu Plugins finden Sie unter Verwalten von verschiedenen Dateitypen.

 

Erweiterte Einstellungen – Cache

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Cache beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Erweitert auf Cache. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)


Im Fenster Einstellungen für Cache können Sie den Arbeitsspeicher für Netscape und den Festplatten-Cache einstellen.

  • Speicher-Cache: Geben Sie hier ein, wie viel Cache im Arbeitsspeicher Sie Netscape zuordnen möchten. Der Inhalt des Speicher-Cache wird gelöscht, sobald Sie den Computer ausschalten oder neu starten.
  • Festplatten-Cache: Geben Sie hier ein, wie viel Festplatten-Cache Sie Netscape zuordnen möchten. Dieser auf der Festplatte gespeicherte Cache kann später wiederverwendet werden, auch wenn Sie den Computer ausgeschaltet haben.
  • Festplatten-Chache: Zeigt den aktuellen Speicherort des Festplatten-Cache-Ordners an.
    • Ordner wählen: Klicken Sie hierauf, um einen Ordner für den Festplatten-Cache auszuwählen.
  • Speicher-Cache löschen: Klicken Sie hierauf, um den Inhalt des Speicher-Cache zu löschen.
  • Festplatten-Cache löschen: Klicken Sie hierauf, um den Inhalt des Festplatten-Cache zu löschen.
  • Die Seite im Cache mit der Seite im Netzwerk vergleichen:
    • Immer beim Öffnen der Seite: Wählen Sie diese Option aus, wenn Netscape bei jedem Aufrufen einer Webseite diese Seite mit dem Cache vergleichen soll.
    • Wenn die Seite nicht mehr aktuell ist: Wählen Sie diese Option aus, wenn Netscape eine Webseite, die vom Server als abgelaufen gekennzeichnet wird, mit dem Cache vergleichen soll.
    • Einmal pro Sitzung: Wählen Sie diese Option, wenn Netscape eine Webseite einmal in jeder Sitzung mit dem Cache vergleichen soll.
    • Nie: Wählen Sie diese Option aus, wenn Netscape die im Cache gespeicherten Informationen nicht mit dem Netzwerk vergleichen soll.

 

Erweiterte Einstellungen – Proxies

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Proxies beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Erweitert auf Proxies. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)


Im Fenster Einstellungen für Proxies richten Sie Netscape für die Verwendung eines Proxy ein:

Bevor Sie beginnen: Fragen Sie Ihren Netzwerkadministrator, ob eine Proxy-Konfigurationsdatei vorhanden ist, oder lassen Sie sich die Namen und Portnummern des Proxy geben.

  • Direkte Verbindung zum Internet: Aktivieren Sie diese Option, wenn Sie die Internetverbindung nicht über Proxy herstellen müssen.
  • Manuelle Proxy-Konfiguration: Aktivieren Sie diese Option, wenn Sie nicht über eine Proxy-Adresse (URL) verfügen.
    • HTTP-Proxy, SSL-Proxy, FTP-Proxy, Gopher-Proxy: Geben Sie den Namen oder die nummerische IP-Adresse des Proxyservers ein. Geben Sie den Port in die Port-Felder ein.
    • SOCKS-Host: Geben Sie den Namen oder die nummerische IP-Adresse des Proxyservers ein. Geben Sie den Port in die Port-Felder ein.
      • SOCKS v4, SOCKS v5: Wählen Sie bei der Eingabe eines SOCK-Hosts "SOCKS v4" oder "SOCKS v5" aus, je nachdem, welche Version von SOCKS für den Proxy verwendet wird.
    • Kein Proxy für: Geben Sie die Domains ein, für die Sie keinen Proxy verwenden möchten. Trennen Sie die einzelnen Domains durch ein Komma. (Beispiel: .ihrefirma.de, .ihrefirma.au)
  • Automatische Proxy-Konfiguration-URL: Wählen Sie diese Option, wenn Sie eine Proxy-Konfigurationsdatei oder -URL haben, und geben Sie anschließend die Konfigurations-URL ein.

 

Erweiterte Einstellungen – HTTP-Networking

Dieser Abschnitt beschreibt die Verwendung des Fensters Einstellungen für HTTP-Networking. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Erweitert auf HTTP-Networking. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)


Im Fenster Einstellungen für HTTP-Networking konfigurieren Sie ein HTTP-Netzwerk:

  • Optionen für direkte Verbindung, Optionen für Verbindung über Proxy: Wählen Sie die HTTP-Version und -optionen für direkte und Proxy-Verbindungen aus.
    • HTTP 1.0 verwenden: Wählen Sie diese Option aus, um die 1996 standardisierte Originalversion von HTTP zu verwenden.
    • HTTP 1.1 verwenden: Wählen Sie diese Option aus, um die neue HTTP-Version zu verwenden. Diese Version bietet Leistungsverbesserungen, einschließlich einer effizienteren Verwendung von HTTP-Verbindungen, einen besseren Support für Client-Caching, mehrfache HTTP-Anforderungen (Pipelining) und eine umfassendere Steuerung der Cache-Ablauf- und Cache-Austauschrichtlinien.
    • Dauerverbindung aktivieren: Wählen Sie diese Option aus, um die Verbindung für weitere HTTP-Anforderungen aufrechtzuerhalten. Die Geschwindigkeit wird dadurch erhöht.
    • Pipelining aktivieren: Wählen Sie diese Option zum Aktivieren von Pipelining aus, damit gleichzeitig mehrere HTTP-Anforderungen an den Server gesendet werden können. Dadurch werden Verzögerungen beim Laden verringert.
      Anmerkung: Pipelining ist nur bei HTTP 1.1 verfügbar.

Dieser Abschnitt beschreibt die Verwendung des Fensters Einstellungen für Software-Installation. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Erweitert auf Software-Installation. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)


Im Fenster Einstellungen für Software-Installation aktivieren Sie die Software-Installation und Update-Benachrichtigung:

  • Software-Installation aktivieren: Wählen Sie diese Option aus, wenn Sie eine Benachrichtigung zur Installation und Verwendung neuer Software mit Netscape erhalten möchten.
  • Nach Aktualisierungen suchen: Wählen Sie diese Option aus, wenn Sie über neue Versionen von Netscape informiert werden möchten. Ihre persönlichen Informationen werden bei der Versionsüberprüfung nicht an Netscape weitergegeben.
    • wöchentlich: Wählen Sie diese Option aus, wenn einmal in der Woche nach einer neuen Version von Netscape gesucht werden soll.
    • monatlich: Wählen Sie diese Option aus, wenn einmal im Monat nach einer neuen Version von Netscape gesucht werden soll.

 

Erweiterte Einstellungen – Mausrad

In diesem Abschnitt wird die Verwendung des Fensters Einstellungen für Mausrad beschrieben. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Erweitert auf Mausrad. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)


Im Fenster Einstellungen für Mausradkönnen Sie die Verwendung des Mausrads (zwischen den Maustasten) in Netscape steuern.

  • Modifizierungstaste: Wählen Sie in der Dropdown-Liste eine Taste aus, die zusammen mit dem Mausrad für eine der folgenden Funktionen verwendet wird.
    • Dokument scrollen um: Wählen Sie diese Option aus, um in dem Dokument um die Anzahl der in dem Feld eingegebenen Zeilen zu scrollen.
      • System-Einstellung verwenden: Wählen Sie diese Option aus, wenn die Anzahl der eingegebenen Zeilen überschrieben und die System-Standardeinstellung verwendet werden soll.
    • Seite aufwärts oder abwärts scrollen: Wählen Sie diese Option aus, um jeweils eine Seite nach oben oder unten zu scrollen. Diese Einstellung ermöglicht ein schnelles, aber weniger genaues Scrollen durch eine Seite mit dem Mausrad.
    • In History vorwärts und rückwärts bewegen: Wählen Sie diese Option, um mit dem Mausrad zwischen zuvor besuchten Seiten vor oder zurück zu wechseln.
    • Text vergrößern oder verkleinern: Wählen Sie diese Option, um mit dem Mausrad den Text auf einer Webseite zu vergrößern oder zu verkleinern. Mithilfe dieser Einstellung können Sie eine Seite besser lesen oder den Text dem Bildschirm anpassen.

    Anmerkung: Jede Modifizierungstaste kann einer anderen Funktion zugewiesen werden.

 

Erweiterte Einstellungen – System

Dieser Abschnitt beschreibt die Verwendung des Fensters Einstellungen für System. Wenn Sie das Fenster noch nicht geöffnet haben, gehen Sie folgendermaßen vor:

1. Klicken Sie im Menü Bearbeiten auf Einstellungen.
2. Klicken Sie unter der Kategorie Erweitert auf System. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf Erweitert, um die Liste zu erweitern.)

Anmerkung: Die Systemeinstellungen sind nur auf Windows-Plattformen verfügbar.


Im Fenster Einstellungen für System geben Sie an, welche Dateien und Protokolle mit Netscape geöffnet werden.

  • Windows soll Netscape zum Öffnen folgender Dateitypen verwenden: Wählen Sie die Dateitypen aus, die standardmäßig mit Netscape geöffnet werden sollen.
  • Windows soll Netscape zur Handhabung folgender Protokolle verwenden: Wählen Sie die Protokolle aus, die standardmäßig mit Netscape geöffnet werden sollen.
  • Benachrichtigen, wenn andere Anwendungen diese Einstellungen ändern: Wählen Sie diese Option aus, wenn Netscape Sie benachrichtigen soll, falls andere Anwendungen die Standardeinstellungen für Netscape-Dateien und -Protokolle geändert haben.

Tipp: Wie Sie Netscape ganz einfach zum Standardbrowser machen, finden Sie unter Navigator-Einstellungen – Navigator. Wählen Sie auch Benachrichtigen, wenn andere Anwendungen diese Einstellungen ändern aus, um informiert zu werden, falls andere Programme zum Standardprogramm werden.
