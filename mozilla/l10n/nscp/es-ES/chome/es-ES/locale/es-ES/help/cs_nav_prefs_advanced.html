Preferencias avanzadas: avanzadas

En esta sección se explica el uso del panel principal de Preferencias avanzadas. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. Haga clic en la categoría Avanzadas.

El panel principal de Preferencias avanzadas permite personalizar Java, las contraseñas de FTP y Quick Launch:

  • Activar Java: seleccione esta opción para que Netscape interprete páginas con Java. en algunas páginas web se usa Java para incorporar contenido interactivo y multimedia.
  • XSLT: seleccione esta opción para que Netscape pueda usar XSLT. XSLT (eXtensible Stylesheet Language/Transform) es un lenguaje que transforma documentos XML en otros documentos XML. Si desea más información sobre XSLT, consulte en Internet el documento Transforming XML: Netscape and XSLT (Transformación de XML: Netscape y XSLT).
  • Enviar esta dirección de correo electrónico como contraseña de FTP anónima: escriba la dirección de correo electrónico que desee utilizar como contraseña al iniciar una sesión anónima en un sitio FTP.
  • Activar Quick Launch: seleccione esta opción si desea reducir el tiempo que tarda Netscape en iniciar. Consulte Cómo se activa Quick Launch Si desea más información sobre el empleo de Quick Launch.

    Nota: Quick Launch sólo aparece en sistemas basados en Windows.

 

Preferencias avanzadas: secuencias de comandos y conectores

En esta sección se explica el uso del panel de preferencias de secuencias de comandos y conectores. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. En la categoría Avanzadas, haga clic en Secuencias de comandos y Windows. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)


Este panel de preferencias de las secuencias de comandos y los conectores permite controlar el empleo de JavaScript y los conectores en Netscape.

  • Activar JavaScript para: active estas casillas para activar JavaScript en:
    • Navigator: activa JavaScript para las páginas web que se abren en Navigator.
    • Correo/Grupos de noticias activa JavaScript para las páginas web que se abren en el programa de correo y grupos de noticias.
  • Permitir que las páginas web puedan: active estas casillas para controlar el modo en que se emplea JavaScript:
    • Abrir un enlace en una ventana nueva (es necesario reiniciar Netscape): permite abrir en una ventana nueva los enlaces en los que se hace clic. Algunos enlaces están configurados para que se abran en ventanas nuevas, en lugar de abrirse en la misma ventana. Si se cambia esta opción, el cambio empezará a tener efecto después de reiniciar Netscape.
    • Mover o cambiar el tamaño de las ventanas existentes: permite trasladar las ventanas abiertas y cambiarles el tamaño.
    • Subir o bajar las ventanas: permite situar las ventanas encima o debajo de otras ventanas.
    • Ocultar la barra de estado: permite ocultar la barra de estado.
    • Cambia el texto de la barra de estado: permite cambiar el texto de la barra de estado, como por ejemplo, el texto que se desplaza en la barra de estado.
    • Cambiar imágenes: permite cambiar o animar las imágenes, como las imágenes que cambian al situar sobre ellas el cursor.
    • Crear o cambiar cookies: permite crear o cambiar los cookies con JavaScript.
    • Leer cookies: permite leer cookies con JavaScript.
  • Activar conectores para: active esta casilla para controlar el uso de los conectores:
    • Correo/Grupos de noticias permite usar conectores con el correo y los grupos de noticias.

Si desea más información, sobre los conectores, consulte Cómo se gestionan tipos de archivos diferentes.

 

Preferencias avanzadas: caché

En esta sección se explica el uso del panel de preferencias de la caché. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. En la categoría Avanzadas, haga clic en Caché. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)


El panel de preferencias Caché permite ajustar caché de disco y de memoria de Netscape.

  • Caché de memoria: escriba la cantidad de caché de memoria que desea asignar a Netscape. La caché de memoria se borra al apagar o reiniciar el equipo.
  • Caché de disco: escriba la cantidad de caché de disco que desea asignar a Netscape. La caché de disco se guarda en la unidad de disco duro y puede volver a usarse aun después de apagar y volver a encender el equipo.
  • Carpeta de caché de disco: muestra la ubicación de la carpeta de caché de disco.
    • Elegir carpeta: haga clic aquí para elegir la ubicación de la carpeta de caché de disco.
  • Vaciar caché de memoria: haga clic en esta opción para vaciar la caché de memoria.
  • Vaciar caché de disco: haga clic en esta opción para vaciar la caché de disco.
  • Compara la página en caché con la página de la red:
    • Cada vez que veo la página: seleccione esta opción si desea que Netscape compare una página web con la caché cada vez que la vea.
    • Cuando caduca la página: seleccione esta opción si desea que Netscape compare una página web con la caché cuando el servidor determine que la página ha caducado.
    • Una vez por sesión: seleccione esta opción si desea que Netscape compare una página web con la caché una vez cada vez que inicie Netscape.
    • Nunca: seleccione esta opción si no desea que Netscape compare la información de la caché con la red.

 

Preferencias avanzadas: proxy

En esta sección se explica el uso del panel de preferencias avanzadas de proxy. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. Abra la categoría Avanzadas y haga clic en Proxy. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)


El panel de preferencias de proxy permite configurar Netscape para usar un servidor proxy:

Antes de empezar: pregunte al administrador de red si usted dispone de un archivo de configuración de proxy o pídale los nombres y números de puerto del servidor proxy.

  • Conexión directa con Internet: seleccione esta opción si no desea utilizar un servidor proxy.
  • Configuración manual del servidor proxy: seleccione esta opción si no dispone de una ubicación de proxy (URL).
    • Proxy HTTP, Proxy SSL, Proxy FTP, Proxy de Gopher: escriba el nombre o la dirección IP numérica del servidor proxy. Indique el puerto en el campo Puerto.
    • Sistema SOCKS: escriba el nombre o la dirección IP numérica del servidor proxy. Indique el puerto en el campo Puerto.
      • SOCKS v4, SOCKS v5: cuando indique un sistema SOCKS, seleccione SOCKS v4 o SOCKS v5, según la versión de SOCKS que se use para el proxy.
    • No hay proxy para: escriba los dominios para los que no desea usar proxy. Separe cada dominio con una coma. (Ejemplo: suempresa.com, .suempresa.es)
  • URL de configuración automática del servidor proxy: seleccione esta opción si dispone de una URL o un archivo de configuración de servidor proxy; a continuación escriba la URL de configuración.

 

Preferencias avanzadas: conexiones de red HTTP

En esta sección se explica el uso del panel de preferencias de conexiones de red HTTP. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. Abra la categoría Avanzadas y haga clic en Conexiones de red HTTP. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)


El panel de preferencias de conexiones de red HTTP sirve para configurar las conexiones de red basadas en HTTP:

  • Opciones de conexión directa, opciones de conexión por proxy: elija una versión de HTTP y las opciones para las conexiones directa y por proxy.
    • Utilizar HTTP 1.0: seleccione esta casilla para usar la versión original de HTTP, normalizada en 1996.
    • Utilizar HTTP 1.1: seleccione esta casilla para usar la versión nueva de HTTP, que ofrece mejoras de rendimiento, como un uso más eficaz de las conexiones HTTP, una mayor compatibilidad con el almacenamiento en caché del programa, varias solicitudes de HTTP (canalización) y un control más avanzado de la caducidad de la caché y las normas de sustitución.
    • Activar conexiones persistentes: seleccione esta casilla para mantener una conexión abierta para hacer más solicitudes de HTTP y aumentar así la velocidad.
    • Activar canalización: seleccione esta casilla para activar la canalización, que permite enviar al servidor más de una solicitud de HTTP simultáneamente, reduciendo así el tiempo de carga de las páginas web.
      Nota: la canalización sólo funciona con HTTP 1.1.

 

Preferencias avanzadas: instalación de software

En esta sección se explica el uso del panel preferencias de instalación de software. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. En la categoría Avanzadas, haga clic en Instalación de software. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)


El panel de preferencias de instalación de software sirve para activar la instalación de software y la notificación de las actualizaciones:

  • Habilitar instalación de software: seleccione esta opción si desea que Netscape le indique cuándo es necesario instalar y utilizar un software nuevo.
  • Buscar actualizaciones: seleccione esta opción para que se le envíe un aviso cuando salga una versión nueva de Netscape. Al comprobar su versión de Netscape, sus datos personales no se compartirán con Netscape.
    • semanalmente: elija esta opción si quiere que Netscape compruebe una vez a la semana si hay una versión nueva de Netscape.
    • mensualmente: elija esta opción si quiere que Netscape compruebe una vez al mes si hay una versión nueva de Netscape.

 

Preferencias avanzadas: rueda del ratón

En esta sección se explica el uso del panel preferencias de la rueda del ratón. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. En la categoría Avanzadas, haga clic en Rueda del ratón. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)


El panel de preferencias de la rueda del ratón permite controlar el funcionamiento en Netscape de la rueda de su ratón (que está entre los botones del ratón).

  • No hay tecla modificadora: en la lista desplegable, elija la tecla que desee usar con el ratón para realizar una de las siguientes funciones:
    • Desplazar el documento por: elija esta opción para que el documento se desplace el número de líneas indicadas en el campo.
      • Utilizar valores predeterminados del sistema: elija esta opción para cancelar el número de líneas indicadas y usar en su lugar el valor predeterminado del sistema.
    • Desplazarse una página atrás o adelante: elija esta opción para desplazarse una página adelante o atrás cada vez. Esta función permite desplazarse más rápidamente por la página con la rueda del ratón, pero se pierde precisión.
    • Trasladarse hacia adelante o hacia atrás en el historial de navegación: elija esta opción para desplazarse con la rueda del ratón hasta páginas que haya visitado previamente.
    • Aumentar o disminuir el texto: elija esta opción para aumentar o reducir el tamaño del texto de una página web con la rueda del ratón. Esta función sirve para mejorar la legibilidad de una página o para ajustar el texto en la pantalla.

    Nota: cada una de las teclas modificadoras puede asignarse a una función distinta.

 

Preferencias avanzadas: sistema

En esta sección se explica el uso del panel de preferencias de sistema. Si aún no ha llegado al panel, siga los pasos que se indican a continuación:

1. Abra el menú Editar y seleccione Preferencias.
2. En la categoría Avanzadas, haga clic en System. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)

Nota: el panel de preferencias de sistema sólo aparece en los sistemas basados en Windows.


El panel de preferencias de sistema permite indicar los archivos y los protocolos que se abren con Netscape.

  • Windows debe utilizar Netscape para abrir estos tipos de archivo: seleccione los tipos de archivo que desea abrir con Netscape de forma predeterminada.
  • Windows debe utilizar Netscape para manejar estos protocolos: seleccione los protocolos que desea abrir con Netscape de forma predeterminada.
  • Avisar si otras aplicaciones cambian estos valores: seleccione esta opción si desea que Netscape lo avise cuando otras aplicaciones modifiquen los valores predeterminados de los archivos y los protocolos de Netscape.

Sugerencia: Para establecer Netscape como navegador predeterminado de manera sencilla, consulte Preferencias de Navigator: Navigator. Si activa Avisar si otras aplicaciones cambian estos valores, también sabrá si otros programas se convierten en programas predeterminados en lugar de Netscape.
