La creación de páginas web con Netscape Composer

Con Netscape Composer, usted puede crear sus propias páginas y publicarlas en la Web. Para utilizar Composer no es necesario saber HTML; su manejo es tan sencillo como el de un procesador de textos.

Mediante los botones de la barra de herramientas, se pueden insertar en una página listas, tablas, imágenes, enlaces a otras páginas, colores, estilos de letra, etc. El aspecto que tendrá una página en la Web puede comprobarse a medida que se va creando, y los documentos pueden intercambiarse sin dificultad con otros usuarios, independientemente del tipo de navegador que posean o de si su programa de correo electrónico admite HTML.

Para comenzar a utilizar Netscape Composer:

  • Haga clic en el icono de Composer, incluido entre los que aparecen en la esquina inferior izquierda de cualquier ventana de Netscape.

Cómo se crea una página

Netscape Composer es un editor de HTML (Hypertext Markup Language) con el que es posible crear y editar páginas web. Como Composer es un editor WYSIWYG (lo que ve es lo que obtiene), es fácil comprobar, mientras se está creando, cómo se le mostrará la página a quien la visite. No es necesario que conozca el lenguaje HTML, ya que la mayoría de las funciones básicas de HTML están disponibles como comandos desde las barras de herramientas y los menús.

No obstante, Composer también le ofrece la posibilidad de editar directamente el código HTML, si lo prefiere. Para ver o modificar el código origen HTML, abra el menú Ver y seleccione Código origen de HTML, o bien haga clic en la pestaña <HTML> Origen, situada en la barra de herramientas de la parte inferior de la ventana de Composer.

Para crear una página web, sírvase de uno de los métodos descritos a continuación. Una vez que haya comenzado a crear una página, puede insertar y editar texto tal como lo haría en un procesador de textos.

Para crear una página desde el navegador de Navigator:

  • Abra el menú Archivo y elija Nuevo, y seguidamente Página de Composer. Se abrirá una ventana de Composer con una página blanco.

Para modificar una página por la que esté navegando en Navigator:

  • En la ventana de Navigator de la página que esté visitando, abra el menú Archivo y elija Editar página. Entonces se abre una ventana de Composer que contiene la página que estaba visualizando.

Para crear una página en Composer:

  • Haga clic en el botón Nuevo de la barra de herramientas de Composer.

Para empezar a trabajar a partir de un archivo HTML que tenga guardado en una unidad local:

  1. Abra el menú Ventana y elija Composer. Entonces aparece en pantalla la ventana de Composer.
  2. Abra el menú Archivo y elija Abrir archivo. Aparece el cuadro de diálogo Abrir archivo HTML.
  3. Localice en la unidad local el archivo que desee editar.
  4. Haga clic en Abrir para ver el archivo elegido en una ventana de Composer.

Para modificar una página web:

  1. Abra el menú Ventana y elija Navigator.
  2. Visite una página web escribiendo su URL (por ejemplo, http://home.netscape.com) en la barra Dirección y pulsando seguidamente la tecla Intro (Retorno, si trabaja con Macintosh).
  3. Abra el menú Archivo y seleccione Editar página.

Sugerencia: En la ventana de Composer, podrá abrir con rapidez el archivo más reciente con el que haya trabajado si elige Páginas recientes desde el menú Archivo, y selecciona a continuación el archivo que desee de la lista.

Cómo se guarda una página y se navega por ella

Puede guardar documentos de Composer en HTML o en formato de sólo texto. Al guardar un documento en formato HTML, se conserva el formato del documento, por ejemplo, los estilos de texto (si está en negrita o en cursiva), las tablas, los enlaces y las imágenes. Al guardar un documento en formato de sólo texto, se eliminan todas las etiquetas HTML, pero se conserva el texto del documento.

Para guardar un documento como archivo HTML:

  • Abra el menú Archivo y elija Guardar, o pulse el botón Guardar de la barra de herramientas de composición.

    (Si aún no le ha dado un título a la página, Composer le pedirá que lo haga.) Composer muestra el título de la página en la barra de título de la ventana del navegador cuando se visualiza la página en éste. El título de la página del documento también aparece en la lista de marcadores si se marca la página.

    A continuación, Composer le pide que introduzca un nombre de archivo o que especifique la ubicación en la que desea guardarlo. Asegúrese de que conserva la extensión .html (o .htm) del nombre del archivo.

Para modificar el nombre o la ubicación de un archivo HTML que ya existe:

  • Elija Guardar como, y seleccione un nombre de archivo o ubicación distintos.

Cuando se guarda una página en Composer, todos los elementos que la componen (el código HTML, las imágenes y demás archivos, como los de sonido o las hojas de estilo) quedan almacenados localmente en el disco duro del equipo. Si prefiere guardar la parte HTML de la página, debe modificar las preferencias de Composer para el almacenamiento de páginas. Consulte Preferencias de Composer: Composer si desea más información sobre cómo modificar las preferencias de Composer para guardar páginas.

Si la ubicación de una imagen es absoluta (es decir, comienza por "http://") y está conectado a Internet, seguirá viendo dicha imagen en el documento tanto en Composer como en Navigator. Sin embargo, si la ubicación de la imagen es relativa a la de la página (es decir, comienza por "file:///"), no la verá en la versión local del documento.

Para guardar un documento como archivo de sólo texto:

  1. En el menú Archivo, elija Exportar a texto.
  2. Escriba el nombre del archivo y especifique la ubicación en la que desee guardarlo.

Nota: Las imágenes no se ven en documentos guardados en formato de sólo texto.

Sugerencia: Puede elegir Volver a la última versión guardada desde el menú Archivo para recuperar la copia guardada más reciente del documento en el que esté trabajando. Tenga en cuenta que se perderán los cambios actuales.

Para visualizar su página en una ventana del navegador en la que podrá comprobar sus enlaces:

  • Abra el menú Archivo y seleccione Página Explorar (o haga clic en Examinar en la barra de herramientas de composición). (Si aún no ha guardado el documento, Composer le pedirá que escriba un título de página, un nombre de archivo y una ubicación.) La ventana de Composer quedará abierta detrás de la nueva ventana de Navigator.

Cómo se da formato a párrafos, títulos y listas

Para aplicar un formato a un párrafo, comience desde la ventana Composer.

  1. Sitúe el puntero donde quiera que dé comienzo el formato, o bien resalte el texto al que desee aplicárselo.
  2. Elija un formato de párrafo en la lista desplegable de la barra de herramientas de formato:
    • Texto del cuerpo: Aplica el tipo de fuente y el estilo de un texto normal, sin que afecte al espacio situado antes o después del texto.
    • Párrafo: Inserta una etiqueta de párrafo (utilícela para empezar un nuevo párrafo). El párrafo ya incluye un margen superior y otro inferior.
    • Encabezado 1  Encabezado 6: Le aplican al párrafo formato de encabezado. Encabezado 1 es el encabezado de nivel superior, mientras que Encabezado 6 es el del nivel más bajo.
    • Dirección: Puede utilizarse para una "firma" de página web, que indica el autor de la página y la persona con la que ponerse en contacto para obtener más información, por ejemplo: usuario@ejemplo.com

      Tal vez desee incluir también la fecha y un aviso de copyright. Este formato suele aparecer en la parte inferior de la página web debajo de una línea horizontal. Navigator muestra el formato de dirección en cursiva.

    • Con formato: Este formato es útil para elementos tales como ejemplos de código, tablas y mensajes de correo, si prefiere mostrarlos con una fuente de anchura fija. En el texto normal, la mayoría de los navegadores eliminan los espacios, las tabulaciones y los retornos de párrafo adicionales. Sin embargo, el texto que lleve asignado el estilo Con formato se visualizará con el espacio en blanco intacto.

Para dar formato al texto como encabezado:

  1. Haga clic para colocar el punto de inserción en cualquier parte del texto al que desee dar formato.
  2. En la lista desplegable de la barra de herramientas de formato, seleccione el nivel de encabezado que desee, desde 1 (el mayor) hasta 6 (el menor). Elija "Encabezado 1" para el encabezado principal, "Encabezado 2" para el siguiente nivel, y así sucesivamente.

Para aplicar a un texto formato de elemento de lista:

  1. Haga clic para colocar el punto de inserción en cualquier parte del texto al que desee aplicar este formato.
  2. Abra el menú Formato y seleccione Lista.
  3. Elija el estilo de la lista:
    • Viñetas: Cada elemento tiene una viñeta (un punto antepuesto), como en la lista.
    • Numerada: Los elementos están numerados.
    • Término y Definición: Estos dos estilos funcionan conjuntamente, y el resultado es una apariencia similar a la de un glosario. Utilice la etiqueta Término para la palabra definida y la etiqueta Definición para la explicación del término. El texto de Término aparece alineado a la izquierda, y el texto de Definición se muestra con sangría.

Sugerencia: Puede aplicar el estilo de la lista rápidamente a un bloque de texto seleccionándolo y haciendo clic en los botones Lista numerada o Lista de viñetas de la barra de herramientas de formato.

Para cambiar el estilo de viñetas o de números:

  1. Haga clic para colocar el punto de inserción dentro del texto del elemento de lista que desee modificar, o bien seleccione uno o más elementos en la lista si desea aplicarle un estilo nuevo a toda.
  2. Abra el menú Formato y elija Propiedades de la lista.
  3. Seleccione en la lista desplegable un estilo de viñeta o de número. Si es una lista numerada, puede especificar un número de inicio. Si es una lista de viñetas, puede modificar el estilo de la viñeta.

Sugerencia: También puede hacer doble clic en una viñeta o en un número de una lista para mostrar el cuadro de diálogo Propiedades de lista.

Para alinear un párrafo o texto en la página (por ejemplo, en el centro, a la izquierda o a la derecha):

  1. Haga clic para colocar el punto de inserción en el párrafo o en la línea del texto que desee alinear.
  2. Abra el menú Formato y elija Alinear; seleccione una opción de alineación.

Nota: Puede utilizar también la barra de herramientas de formato para alinear el texto.

Cómo se trabaja con listas

Para terminar una lista y continuar escribiendo texto del cuerpo de la página:

  • Haga clic para colocar el punto de inserción al final del último elemento de la lista, y pulse dos veces Intro (Retorno, si trabaja con Macinstosh) para dar por finalizada la lista.

Para cambiar uno o más elementos de una lista a texto del cuerpo:

  1. Haga clic para colocar el punto de inserción en el elemento de lista, o bien seleccione los elementos de lista.
  2. Si se trata de una lista numerada, haga clic en el botón de lista numerada (o en una lista de viñetas, haga clic en el botón de lista de viñetas) de la barra de herramientas de formato.

Para colocar texto con sangría bajo un elemento de una lista:

  1. Haga clic para colocar el punto de inserción en el elemento de lista.
  2. Pulse Mayúsculas-Intro para crear la sangría francesa.
  3. Escriba el texto al que desee aplicarle una sangría.
  4. Pulse Mayúsculas-Intro para crear otro párrafo con sangría, o bien pulse Intro para crear el siguiente elemento de la lista.

Sugerencia: El sangrado de una lista de elementos puede aumentarse o disminuirse colocando el punto de inserción en cualquier elemento de la lista y haciendo clic, respectivamente, en los botones Sangrar el texto o Sacar sangría del texto, situados en la barra de herramientas de formato. Si lo prefiere, también puede hacer clic en el elemento de la lista elegido y pulsar la tecla Tabulador tantas veces como niveles desee sangrarlo. Para reducir la sangría, mantenga pulsada la tecla Mayúsculas y pulse la tecla Tabulador tantas veces como niveles desee reducir el sangrado.

Para combinar dos listas adyacentes:

  1. Seleccione las dos listas que desee combinar. Asegúrese de que selecciona todos los elementos de las dos listas. Tenga en cuenta que cualquier texto que haya entre las dos listas también acabará formando parte de la lista combinada.
  2. A continuación, haga clic en el botón Lista numerada o en Lista con viñetas, situados en la barra de herramientas de formato, para combinar las listas.

Cómo se modifican el color, el estilo y la fuente del texto

Para cambiar el color, el estilo y la fuente del texto seleccionado:

  1. Seleccione el texto al que desee dar formato.
  2. Abra el menú Formato y elija una de las opciones siguientes:
    • Fuente: Utilice esta opción para elegir un tipo de letra. Si prefiere utilizar las fuentes especificadas por el navegador destinatario de la página, seleccione Anchura variable o Anchura fija.

      Nota: No todas las fuentes instaladas en el equipo aparecen recogidas en la lista. En lugar de especificar una fuente que pueda no estar disponible en todos los equipos en los que se visualice la página web, suele ser mejor seleccionar alguna de las fuentes incluidas en el menú, porque éstas sí funcionan en todos los equipos. Por ejemplo, las fuentes Helvética, Arial, Times y Courier normalmente tienen la misma apariencia cuando se visualizan en equipos diferentes. Si selecciona una fuente distinta, es posible que no presente la misma apariencia al visualizarla en otro equipo.

    • Tamaño: Elija un tamaño relativo de fuente o seleccione una opción para aumentar o disminuir el tamaño del texto (supeditado al texto que lo rodea).
    • Estilo del texto: Utilice esta opción para seleccionar el estilo (como cursiva, negrita o subrayado), o para aplicar un estilo estructurado (por ejemplo, Código).
    • Color del texto: Utilice esta opción para elegir un color de la paleta de colores. Si está familiarizado con los códigos de color hexadecimales de HTML, puede escribir el código del color elegido; si lo prefiere, basta con que escriba el nombre del color, pero deberá hacerlo en inglés: por ejemplo, "blue" (azul). Si pulsa aquí, podrá descargar un convertidor de colores muy útil.

Para cambiar el color de fondo de una página:

  1. Haga clic en cualquier lugar de la página.
  2. Haga clic en el pequeño recuadro opaco de los dos que conforman el botón para la elección de colores de la barra de herramientas de formato.
  3. Elija un color para el fondo de la página en el cuadro de diálogo Color del fondo de la página.
  4. Haga clic en Aceptar.

Sugerencia: Para sustituir rápidamente el color actual del texto por el que haya utilizado en último lugar, seleccione el texto, mantenga pulsada la tecla Mayúsculas y haga clic en el recuadro para seleccionar el color del texto, situado en la barra de herramientas de formato. Esto resulta útil cuando se desee utilizar un mismo color para líneas de texto sueltas.

También es posible utilizar una imagen como fondo de una página. Consulte Cómo se definen los colores y el fondo.

Cómo se elimina o se interrumpe un estilo de texto

Para eliminar todos los estilos (negrita, cursiva, etc.) del texto seleccionado:

  1. Seleccione el texto.
  2. Abra el menú Formato y seleccione Eliminar todos los estilos de texto.
  3. Continúe escribiendo.

Para continuar escribiendo texto con todos los estilos de texto eliminados:

  1. Coloque el punto de inserción en el lugar donde desee interrumpir los estilos de texto.
  2. Abra el menú Formato y seleccione Interrumpir los estilos de texto.
  3. Continúe escribiendo.

Cómo se busca y se sustituye texto

Para buscar texto dentro de la página en la que esté trabajando:

  1. Haga clic para colocar el punto de inserción donde desee comenzar la búsqueda.
  2. Abra el menú Editar y seleccione Buscar y reemplazar. Aparece el cuadro de diálogo Buscar y reemplazar.
  3. Escriba en el campo "Buscar" el texto que desee localizar. Para realizar una búsqueda avanzada, marque alguna de las siguientes opciones:


    • Coincidir mayúsculas y minúsculas: Utilice esta opción para especificar si desea que la búsqueda diferencie entre mayúsculas y minúsculas.
    • Ajustar: Utilice esta opción para buscar hasta el final de la página y comenzar de nuevo desde la parte superior o inferior (dependiendo de si está buscando hacia delante o hacia atrás).
    • Buscar hacia atrás: Utilice esta opción para buscar hacia atrás, desde el punto de inserción hasta el inicio de la página.
  4. Haga clic en Buscar siguiente para iniciar la búsqueda. Cuando Composer encuentre la primera coincidencia del texto, haga clic en Buscar siguiente para encontrar la siguiente coincidencia.
  5. Haga clic en Cerrar cuando haya terminado.

Para buscar y sustituir texto dentro de la página en la que esté trabajando:

  1. Haga clic para colocar el punto de inserción donde desee comenzar la búsqueda.
  2. Abra el menú Editar y seleccione Buscar y reemplazar. Aparece el cuadro de diálogo Buscar y reemplazar.
  3. Escriba el texto que desee localizar y, a continuación, aquél por el que quiera sustituirlo.
  4. Para realizar una búsqueda avanzada, marque alguna de las siguientes opciones:
    • Coincidir mayúsculas y minúsculas: Utilice esta opción para especificar si desea que la búsqueda diferencie entre mayúsculas y minúsculas. Si no selecciona esta opción, se mostrará el texto que coincida con el buscado independientemente de que también coincidan mayúsculas y minúsculas.
    • Ajustar: Busca hasta el final de la página y luego empieza otra vez desde el inicio.
    • Buscar hacia atrás: Busca desde el final hasta el inicio de la página.
  5. Haga clic en Buscar siguiente para encontrar la siguiente coincidencia. Composer selecciona la siguiente coincidencia en el texto.
  6. Haga clic en Reemplazar para sustituir el texto seleccionado por el texto especificado para reemplazarlo. Haga clic en Reemplazar y buscar para sustituir el texto seleccionado y buscar la siguiente coincidencia. Haga clic en Reemplazar todos para sustituir todas las coincidencias del documento por el texto especificado en el campo "Reemplazar con".
  7. Haga clic en Cerrar cuando haya terminado.

Cómo se insertan líneas horizontales

Las líneas horizontales se suelen utilizar para separar visualmente las diferentes secciones de un documento. Para insertar en la página una línea horizontal (también llamada filete), comience desde la ventana de Composer:

  1. Haga clic para colocar el punto de inserción donde desee que aparezca la línea.
  2. Abra el menú Insertar y elija Línea horizontal.

 

Cómo se configuran las propiedades de una línea horizontal

Puede especificar la altura, la longitud, la anchura, la alineación y el sombreado de la línea.

  1. Haga doble clic en la línea para que aparezca el cuadro de diálogo Propiedades de la línea horizontal.
  2. Modifique cualquiera de las siguientes propiedades:
    • Anchura: Introduzca la anchura y elija "% de ventana" o "píxeles". Si especifica la anchura porcentualmente, se modificará cada vez que cambie la anchura de la ventana de Composer o la ventana del navegador.
    • Altura: Escriba un número para la altura de la línea (en píxeles).
    • Sombreado 3-D: Seleccione esta opción para añadir profundidad a la línea mediante un sombreado.
    • Alineación: Especifique dónde desea situar la línea (izquierda, derecha o centrada).
  3. Haga clic en Usar como valor predeterminado para utilizar estos mismos valores cada vez que vuelva a insertar una línea horizontal.
  4. Para modificar manualmente las propiedades de una línea horizontal, haga clic en Edición avanzada. Consulte la sección Editor de propiedades avanzadas, para obtener información más detallada.

Sugerencia:Puede seleccionar "Mostrar todas las etiquetas" en el menú Ver para que aparezcan todos los elementos HTML en recuadros amarillos. Haga clic en cualquiera de los recuadros amarillos para seleccionar todo el contenido que abarca ese elemento o etiqueta HTML. Haga doble clic en cualquiera de los recuadros amarillos para mostrar el cuadro de diálogo Editor de propiedades avanzadas correspondiente al elemento o a la etiqueta HTML seleccionados.

Cómo se insertan caracteres especiales

Para insertar caracteres especiales, como acentos, o símbolos monetarios o de copyright:

  1. Haga clic para colocar el punto de inserción donde desee insertar el carácter especial.
  2. Abra el menú Insertar y elija Caracteres y símbolos. Aparece el cuadro de diálogo Insertar carácter.
  3. Seleccione una categoría de caracteres.
    • Si elige Acentuar mayúsculas o Acentuar minúsculas, abra la lista despegable Letra y seleccione la letra que desee acentuar. (Nota: No todas las letras tienen equivalente acentuado.) Seleccione Símbolos comunes para insertar caracteres especiales tales como símbolos de copyright o fracciones.
  4. En la lista desplegable Carácter, seleccione el carácter que desee insertar.
  5. Haga clic en Insertar.

    Puede continuar escribiendo en el documento (o en una ventana de composición de correo) mientras tenga abierto este cuadro de diálogo, por si desea volver a utilizarlo.

  6. Haga clic en Cerrar cuando haya terminado de insertar caracteres especiales.

Cómo se insertan elementos y atributos HTML

Si tiene conocimientos sobre código origen de HTML, puede insertar en una página web etiquetas adicionales, atributos de estilo y JavaScript. Si no sabe muy bien cómo debe manipularse el código origen de HTML, es aconsejable que no cambie nada. Utilice uno de los métodos descritos a continuación para trabajar con el código HTML:

  • Coloque el cursor donde quiera insertar el código HTML, o bien seleccione el texto que desee modificar, y seguidamente abra el menú Insertar y elija HTML. En el cuadro de diálogo Insertar HTML, introduzca las etiquetas HTML y el texto y, a continuación, haga clic en Insertar.
  • Seleccione un elemento, como, por ejemplo, una tabla, un anclaje con nombre, una imagen, un enlace o una línea horizontal. Haga doble clic en el elemento para abrir su cuadro de diálogo de propiedades correspondiente. Haga clic en Edición avanzada para abrir el Editor de propiedades avanzadas. Use el Editor de propiedades avanzadas para agregar a los objetos atributos HTML, JavaScript y CSS.
  • Abra el menú Ver y seleccione Código origen de HTML, o bien haga clic en la pestaña <HTML> Origen de la barra de herramientas de la modalidad de edición, situada en la parte inferior de la ventana de Composer. (Si no ve la barra de herramientas de la modalidad de edición, abra el menú Ver y seleccione Mostrar/ocultar; asegúrese de que esté seleccionada la barra de herramientas de la modalidad de edición).

 

Cómo se emplea el Editor de propiedades avanzadas

Puede utilizar el Editor de propiedades avanzadas para agregar atributos HTML y JavaScript a objetos tales como tablas, imágenes y líneas horizontales.

Nota: A menos que sepa muy bien cómo añadir, suprimir o modificar atributos HTML y los valores asociados a éstos, es preferible que no lo haga.

Si no está visualizando todavía el cuadro de diálogo Editor de propiedades avanzadas, siga los pasos que se indican a continuación:

  1. En el menú Ver (o la barra de herramientas de la modalidad de edición), seleccione Mostrar todas las etiquetas.
  2. Haga doble clic en el objeto que desee modificar para abrir el cuadro de diálogo Editor de propiedades avanzadas.
  3. Haga clic en Edición avanzada para abrir el Editor de propiedades avanzadas del objeto. Este cuadro de diálogo tiene tres pestañas, y en cada una de sus tres secciones se enumeran las propiedades actuales del objeto seleccionado:
    • Atributos HTML: Haga clic en esta pestaña para ver o escribir atributos adicionales HTML.
    • Estilo en línea: Haga clic en esta pestaña para ver o escribir propiedades adicionales CSS (hojas de estilo en cascada) mediante el atributo <style>. Si desea más información sobre el empleo de estilos CSS en Composer, consulte Preferencias de Composer: Composer.
    • Eventos de JavaScript: Haga clic en esta pestaña para ver o introducir eventos de JavaScript.
  4. Para editar una propiedad o atributo de cualquiera de las tres listas, seleccione el atributo que desee editar. A continuación, puede editar el nombre o el valor del atributo mediante los campos editables Atributo y Valor, situados en la parte inferior del cuadro de diálogo. Para agregar un atributo nuevo, escríbalo en el campo Atributo, situado en la parte inferior del cuadro de diálogo. El atributo nuevo se agrega automáticamente al hacer clic en el campo Valor. Para eliminar un atributo, selecciónelo en la lista y haga clic en Eliminar.

    Nota: Los atributos necesarios aparecen resaltados en amarillo en la lista Atributos.

  5. Haga clic en Aceptar para aplicar los cambios que haya realizado en el cuadro de diálogo Editor de propiedades avanzadas.
  6. Vuelva a hacer clic en Aceptar para salir del cuadro de diálogo Propiedades.

Composer entrecomilla automáticamente el texto de los atributos.

Cómo se revisa la ortografía

Composer siempre revisa la ortografía comenzando desde el principio del documento, independientemente del lugar en el esté situado el punto de inserción.

Para revisar la ortografía de una página:

  1. Haga clic en el botón Ortografía de la barra de herramientas, o abra el menú Editar y seleccione Revisar ortografía.
  2. Utilice el cuadro de diálogo Revisar ortografía para corregir las erratas y para editar palabras o incorporarlas al diccionario actual del usuario. Haga clic en Cerrar cuando haya terminado.
    • Volver a revisar página: Haga clic para volver a iniciar la revisión ortográfica. Esta función es útil, sobre todo, si cambia de idioma y después desea volver a revisar la ortografía del documento.
    • Revisar palabra: Haga clic para revisar la ortografía de cualquier palabra que escriba en el campo "Reemplazar con".
    • Reemplazar y Reemplazar todos: Haga clic para sustituir la palabra sólo esa vez o siempre que aparece en el documento.
    • Hacer caso omiso e Ignorar todo: Utilice estos dos botones para continuar la revisión sin cambiar la palabra sólo esa vez o ninguna de las veces que aparezca en el documento.
    • Agregar palabra: Haga clic para agregar la palabra a su diccionario personal, de modo que Composer la reconozca en lugar de resaltarla como si fuera un error ortográfico.
    • Edición: Haga clic para agregar, sustituir y eliminar palabras del diccionario personal.
    • Idioma: Seleccione el idioma que desee que utilice el revisor ortográfico.

Cómo se revisa el código HTML

Antes de publicar el documento en un servidor web para que lo vean otras personas, primero debe comprobar su formato HTML para asegurarse de que cumple todas las normas de la Web. Al validar un documento que contiene HTML, se reducen las probabilidades de que cause problemas cuando se visualice en navegadores diferentes. Aunque pueda comprobar visualmente sus páginas web en Navigator, ello no le garantiza que los documentos se vayan a mostrar correctamente en otros navegadores web.

Con Composer resulta muy cómodo y sencillo comprobar si los documentos cumplen los estándares HTML del W3C (World Wide Web Consortium). Composer utiliza el servicio de validación de HTML del W3C, que comprueba si la sintaxis de un documento HTML cumple las normas HTML 4.01. Este servicio también ofrece información sobre cómo subsanar posibles errores.

Nota: Debe estar conectado a Internet para usar esta función.

Para validar la sintaxis HTML de un documento:

  1. Abra el menú Herramientas y seleccione Validar HTML. Si tiene cambios sin guardar, Composer le preguntará si desea guardarlos antes de continuar.
  2. Cuando aparezca la página del servicio de validación de HTML del W3C, haga clic en Examinar e indique la ubicación en su equipo del archivo que desee validar.
  3. Haga clic en el botón "Validate this document".

Cómo se selecciona el modo de edición adecuado

Normalmente, no es necesario cambiar el modo de edición predeterminado (Normal). No obstante, si desea trabajar con el código origen de HTML del documento, quizás desee cambiar los modos de edición.

Composer permite cambiar rápidamente entre cuatro vistas, o modos de edición. Se puede pasar de un modo de edición a otro mientras se trabaja en el mismo documento; en cada uno de ellos las etiquetas HTML (y los iconos de éstas) se muestran con mayor o menor grado de detalle.

Antes de elegir un modo de edición:

  • Abra el menú Ver, seleccione Mostrar/Ocultar y, a continuación, asegúrese de que haya una marca de verificación junto a la barra de herramientas de la modalidad de edición.

La barra de la modalidad de edición consta de cuatro pestañas:

  • Normal: Elija este modo de edición para comprobar, mientras trabaja, qué aspecto tendrá el documento. Seleccione este modo para ver los bordes de las tablas y los iconos de anclajes con nombre. Los demás iconos de las etiquetas HTML quedan ocultos.
  • Mostrar todas las etiquetas: Elija este modo para mostrar los iconos de todas las etiquetas HTML.
  • <HTML> Origen: Seleccione este modo para visualizar y editar el documento como código origen de HTML sin formato. Si guarda el documento, regresará automáticamente al modo Normal.
  • Vista preliminar: Seleccione este modo para visualizar y editar el documento tal y como aparecería en una ventana del navegador, con la excepción de que los enlaces y las funciones JavaScript no estarán activos.

Nota: Las funciones, los marcos, los enlaces y los archivos GIF de JavaScript no están activos en ninguno de estos modos de edición. Para mostrar estos elementos en su estado activo, haga clic en el botón Examinar de la barra de herramientas de composición para cargar la página en una ventana del navegador.

Cómo se inserta una tabla

Las tablas son útiles para organizar texto, imágenes y datos en filas y columnas con formato. Para insertar una tabla:

  1. Haga clic para colocar el punto de inserción donde desee que aparezca la tabla.
  2. Haga clic en el botón Tabla de la barra de herramientas de composición. Aparece el cuadro de diálogo Insertar tabla.
  3. Escriba el número de filas y columnas que desee.
    • (Optativo) Introduzca la anchura de la tabla, expresándola en píxeles o como porcentaje de la ventana.
  4. Escriba un número para la anchura de la línea del borde (en píxeles); introduzca cero si no desea borde.

    Nota: Composer utiliza una línea roja discontinua para indicar que las tablas tienen borde cero; la línea discontinua desaparece cuando la página se visualiza en un navegador.

  5. Haga clic en Edición avanzada para mostrar el Editor de propiedades avanzadas , desde el que podrá aplicarle a la tabla otros atributos o funciones de JavaScript.
  6. Haga clic en Aceptar para confirmar los valores elegidos y mostrar la nueva tabla.

Para cambiar otras propiedades de la tabla recién creada, consulte Cómo se modifican las propiedades de una tabla.

Sugerencia: Para insertar una tabla dentro de otra, abra el menú Insertar y seleccione Tabla.

Cómo se modifican las propiedades de una tabla

En esta sección, se describe el procedimiento para modificar todas las propiedades de una tabla, o sólo las de algunas de las filas, columnas o celdas que la componen. Si no está visualizando todavía el cuadro de diálogo Propiedades de la tabla, siga los pasos que se indican a continuación:

  1. Seleccione la tabla, o haga clic en cualquier lugar dentro de ella.
  2. Haga clic en el botón Tabla de la barra de herramientas, o abra el menú Tabla y elija Propiedades de la tabla. El cuadro de diálogo Propiedades de la tabla tiene dos pestañas: Tabla y Celdas.
  3. Haga clic en la pestaña Tabla para editar las propiedades siguientes:
    • Tamaño: Especifique el número de filas y de columnas. Introduzca la anchura de la tabla y elija "% de la ventana" o "píxeles". Si especifica la anchura como porcentaje, ésta se modificará cada vez que cambie la anchura de la ventana de Composer o la ventana del navegador en la que se visualice la tabla.
    • Bordes y espaciado: Especifique (en píxeles) la anchura del borde, el espaciado entre las celdas y el relleno de éstas (es decir, el espacio entre el contenido y el borde).

      Nota: Composer utiliza una línea exterior discontinua para mostrar las tablas sin borde; esta línea desaparece cuando la página se visualiza en un navegador.

    • Alineación de la tabla: Alinea una tabla dentro de la página. Seleccione una opción en la lista desplegable.
    • Leyenda: Seleccione en la lista desplegable el sitio donde desea que aparezca la leyenda.
    • Color de fondo: Elija un color para el fondo de la tabla (o déjelo transparente).
  4. Haga clic en Edición avanzada para mostrar el Editor de propiedades avanzadas , desde donde podrá aplicarle a una tabla otros atributos o introducir eventos de JavaScript.
  5. Haga clic en Aplicar para mostrar una vista previa de los cambios sin cerrar el cuadro de diálogo, o haga clic en Aceptar para confirmarlos.

Para ver, cambiar o agregar propiedades a una o varias celdas:

  1. Seleccione la fila, la columna o la celda en cuestión, abra el menú Tabla y elija Propiedades de la tabla. Aparece el cuadro de diálogo Propiedades de la tabla.
  2. Haga clic en la pestaña Celdas para modificar las propiedades siguientes:
    • Selección: Elija Celda, Fila o Columna en la lista desplegable. Haga clic en Anterior o Siguiente para desplazar la selección entre las filas, columnas o celdas.
    • Tamaño: Escriba un número para Altura y Anchura y, a continuación, elija "% de tabla" o "píxeles".
    • Alineación contenido: Seleccione un tipo de alineación horizontal y otro vertical para el contenido de las celdas.
    • Estilo celda: Seleccione Cabecera en la lista desplegable para cabeceras de columna o fila (que hace que el texto de las celdas aparezca centrado y en negrita); si lo prefiere, seleccione Normal.
    • Ajustar texto: Seleccione "No ajustar" en la lista desplegable para que el texto no se ajuste hasta la siguiente línea, a menos que se inserte un salto de párrafo. En caso contrario, elija Ajustar.
    • Color de fondo: Seleccione un color para el fondo de las celdas o déjelo transparente.

    Nota: Haga clic en Edición avanzada para mostrar el Editor de propiedades avanzadas , desde el que podrá aplicarle a la tabla otros atributos o introducir eventos de JavaScript.

  3. Haga clic en Aplicar para mostrar una vista previa de los cambios sin cerrar el cuadro de diálogo, o haga clic en Aceptar para confirmarlos.

Sugerencia: Para cambiar rápidamente el color del texto o del fondo de una o varias celdas seleccionadas, o de toda la tabla, seleccione las celdas o haga clic en cualquier otra parte de la tabla y, a continuación, haga clic en el icono de color de texto o de fondo de la barra de herramientas de formato.

Sugerencia: Para sustituir rápidamente el color actual de una o varias celdas por el que haya utilizado en último lugar, selecciónelas, mantenga pulsada la tecla Mayúsculas y haga clic en el recuadro para seleccionar el color del fondo. Esto resulta útil cuando se desee utilizar un mismo color para celdas sueltas.

Cómo se añaden y se suprimen filas, columnas y celdas

Con Composer, se pueden agregar o suprimir con rapidez celdas de una tabla. Además, se pueden fijar opciones para mantener la estructura o el formato rectangular original de la tabla mientras se procede a su modificación.

Para agregar celdas, filas o columnas a una tabla:

  1. Haga clic dentro de la tabla a la que desee agregar celdas.
  2. Abra el menú Tabla y elija Insertar.
  3. Elija el tipo de celdas que prefiera. (También se puede insertar una tabla nueva dentro de una celda de otra tabla).

Para suprimir celdas, filas o columnas de una tabla:

  1. Haga clic en una celda, fila o columna para colocar el punto de inserción. O bien, seleccione las celdas contiguas para eliminar más de una fila al mismo tiempo. Para seleccionar celdas contiguas, arrastre el punto de inserción sobre ellas. Para seleccionar celdas no contiguas, mantenga pulsada la tecla Ctrl (en Windows y Linux) o la tecla Comando (en Macintosh) mientras va haciendo clic en las que desee seleccionar.
  2. Abra el menú Tabla y seleccione Suprimir.
  3. Elija el elemento que desee eliminar.

Para unir (o combinar) una celda con la situada inmediatamente a su derecha:

  • Haga clic dentro de la celda de la izquierda, abra el menú Tabla y elija Unir con la celda de la derecha.

Para unir (o combinar) celdas adyacentes:

  • Selecciónelas arrastrando el cursor.
  • Abra el menú Tabla y seleccione Unir las celdas seleccionadas.

Para volver a dividir celdas unidas en dos o más:

  • Haga clic dentro de la celda unida y elija Dividir celda en el menú Tabla. Composer coloca todo el contenido de la celda que estaba unida en la primera de las dos en las que aquélla queda dividida.

Consulte Cómo se seleccionan elementos de una tabla si desea información sobre cómo seleccionar celdas, filas y columnas no adyacentes.

 

Cómo se comporta Composer al insertar o eliminar celdas

De forma predeterminada, al suprimir una o varias celdas, Composer conserva la estructura de la tabla agregando celdas al final de una fila siempre que sea necesario. De esta forma, es posible suprimir una o varias celdas manteniendo el formato o la estructura rectangular original de la tabla. En caso contrario, es posible que la eliminación de celdas provoque que se creen espacios vacíos en la tabla o que sus bordes sean irregulares al quedar con un número impar de celdas.

Para determinar qué sucede al modificar una tabla, comience en la ventana de Composer:

  1. Abra el menú Editar, seleccione Preferencias y, a continuación, seleccione Composer.
  2. En la sección Editar tablas, proceda como se indica:
    • Asegúrese de que la opción "Mantener el diseño de la tabla al insertar o eliminar celdas" esté marcada para evitar que se muestren tablas con formas irregulares.
  3. Haga clic en Aceptar.

Consulte también Cómo se configuran las preferencias generales de Composer.

Cómo se seleccionan elementos de una tabla

Para seleccionar rápidamente una tabla, una celda o un grupo de celdas, puede optar por alguno de los dos procedimientos que se describen a continuación:

  • Haga clic en la tabla, abra el menú Tabla, elija Seleccione y seleccione un elemento del submenú. Por ejemplo, para seleccionar una tabla, haga clic en ella, abra el menú Tabla y, a continuación, elija Seleccione y seguidamente Tabla.
  • También puede utilizar el ratón como herramienta de selección:
    • Para seleccionar un grupo de celdas adyacentes: haga clic en una celda y arrastre el cursor para seleccionar las restantes. Arrastre el ratón hacia la izquierda o la derecha para seleccionar una fila y hacia arriba o hacia abajo para seleccionar una columna.
    • Para seleccionar celdas no adyacentes: pulse Ctrl (en Windows o Linux), o Comando (en Macintosh), y después haga clic dentro de una celda. Mantenga pulsada la tecla Ctrl (en Windows o Linux), o Comando (en Macintosh), mientras va haciendo clic en las demás celdas que desee seleccionar.
    • Para extender la selección e incluir celdas adyacentes: haga clic en una celda y seguidamente arrastre el cursor sobre aquéllas que quiera seleccionar.
    • Para seleccionar una o varias columnas o filas adyacentes: arrastre el cursor verticalmente para seleccionar la primera columna, y seguidamente desplácelo lateralmente para seleccionar columnas o filas adyacentes. Mantenga pulsada la tecla Mayúsculas y arrastre el cursor hacia la derecha para seleccionar una fila completa. Mantenga pulsada la tecla Mayúsculas y arrastre el cursor hacia abajo para seleccionar una columna completa.

  1. Haga clic dentro de la tabla.
  2. Abra el menú Tabla, elija Seleccione y, a continuación, Tabla.
  • Para copiar o desplazar la tabla: Utilice las opciones Cortar, Copiar y Pegar del menú Editar.

Cómo se convierte texto en una tabla

Para convertir texto en una tabla:

  1. Seleccione el texto que desee convertir en una tabla. Tenga en cuenta que Composer crea en la tabla una fila por cada párrafo que seleccione.
  2. En el menú Tabla, seleccione Crear tabla según la selección. Aparece en pantalla el cuadro de diálogo Convertir en tabla.
  3. Elija el carácter que deberá utilizar Composer para separar el texto seleccionado en columnas, o especifique otro que le parezca. Si elige Espacio como separador de columnas, seleccione si desea que Composer considere los espacios múltiples como uno solo.
  4. Deje marcada la opción "Suprimir carácter de separador" para que Composer elimine el separador cuando convierta el texto en tabla. Si no desea que Composer elimine el carácter separador, no marque esta opción.
  5. Haga clic en Aceptar.

Nota: Cuando se convierte en tabla, el texto seleccionado pierde el formato que tuviera asignado.

Cómo se inserta una imagen en una página web

En una página web se pueden insertar imágenes en formato GIF, JPEG y PNG (gráficos portátiles para red). También se pueden utilizar para crear enlaces. Al insertar una imagen, lo que Composer guarda en la página es una referencia a ella.

Nota: Si tiene previsto publicar las páginas en la Web, no es aconsejable que inserte en ellas imágenes en formato BMP.

Sugerencia: Es aconsejable que, antes de insertar una imagen, guarde o publique la página con la que esté trabajando. De esta forma, Composer utilizará automáticamente referencias relativas a las imágenes una vez que las inserte.

Para insertar una imagen en una página web:

  1. Haga clic para colocar el punto de inserción donde desee que aparezca la imagen.
  2. Haga clic en el botón Imagen de la barra de herramientas, o abra el menú Insertar y elija Imagen. Aparece el cuadro de diálogo Propiedades de la imagen.
  3. Escriba la ubicación y el nombre del archivo de la imagen, o haga clic en Seleccionar archivo para buscar un archivo de imagen en una unidad de disco de su equipo o de la red.
  4. Escriba una breve descripción de la imagen, que será el texto que aparezca en lugar de la imagen en navegadores que sólo muestren texto (y, a veces, también en otros) mientras ésta se esté cargando o si se ha desactivado la descarga de imágenes.

    Si lo prefiere, puede optar por no incluir ninguna descripción alternativa.

  5. Si es necesario, haga clic en otras pestañas para ajustar los demás valores (por ejemplo, la alineación) que figuran en el cuadro de diálogo Propiedades de la imagen.

Sugerencia: Para insertar rápidamente una imagen en una página web: Arrástrela y suéltela en la página.

Sugerencia: Para insertar un salto de línea después de todas las imágenes de un párrafo, elija Salto debajo de imagen en el menú Insertar.

Cómo se modifican las propiedades de una imagen

Una vez insertada la imagen en la página, puede modificar sus propiedades y personalizar el formato cambiando, por ejemplo, la altura, la anchura, los espacios y la alineación del texto. Si no está visualizando todavía el cuadro de diálogo Propiedades de la imagen, siga los pasos que se indican a continuación:

Para modificar las propiedades de la imagen seleccionada:

  1. Haga doble clic en la imagen, o bien selecciónela y haga clic en el botón Imagen de la barra de herramientas para que aparezca el cuadro de diálogo Propiedades.
  2. Haga clic en la pestaña Ubicación para modificar las propiedades siguientes:
    • Ubicación de imagen: Escriba el nombre y la ubicación del archivo que contiene la imagen. Haga clic en Seleccionar archivo para buscar un archivo de imagen en una unidad de disco de su equipo o de una red.
    • La dirección URL está relacionada con la ubicación de la página: Si está seleccionada esta opción, Composer convierte la URL para que sea relativa a la ubicación real de la página. Esto es especialmente útil si tiene previsto publicar las páginas en un servidor web para que otras personas las visiten. El uso de URL relativas permite mantener todos los archivos enlazados en el mismo lugar, independientemente de cuál sea su ubicación en el disco duro de su equipo o en un servidor web.

      Si no selecciona esta casilla, Composer convierte la URL en una URL completa (absoluta). Es aconsejable que utilice URL absolutas cuando inserte enlaces a imágenes que estén almacenados en otros servidores web (no en una unidad de disco de su equipo).

      Si nunca ha guardado o publicado la página, deberá almacenarla para que esta casilla aparezca activada. (Esta casilla no aparece activada si el cuadro de diálogo Propiedades de la imagen se abre desde una ventana de composición de mensajes.)

    • Texto alternativo: Escriba el texto que desee que aparezca en lugar de la imagen original; por ejemplo, una leyenda o una breve descripción de la imagen. Es muy recomendable incluir un texto alternativo destinado a aquellos usuarios que utilicen navegadores web que sólo muestren texto o que tengan desactivada la descarga de imágenes.
    • No utilizar texto alternativo: Marque esta opción si la imagen no precisa texto alternativo o si prefiere no incluirlo.
  3. Haga clic en la pestaña Dimensiones para modificar las propiedades siguientes:
    • Tamaño real: Seleccione esta opción para deshacer las modificaciones que haya realizado en las dimensiones de la imagen y devolverle su tamaño original.
    • Tamaño personalizado: Seleccione esta opción y especifique los nuevos valores de la altura y la anchura en píxeles o como porcentaje de la ventana. Este valor no afecta al archivo de imagen original, sino sólo a la imagen insertada en la página.
    • Confinar: Si cambia el tamaño de la imagen, no está de más marcar esta casilla para mantener proporcionalmente las dimensiones de la imagen con objeto de que no aparezca deformada. Si selecciona esta opción, luego bastará con que cambie la altura o la anchura, pero no ambas.
  4. Haga clic en la pestaña Apariencia para modificar las propiedades siguientes:
    • Espaciado: Especifique la cantidad de espacio que se insertará alrededor de la imagen para separarla del texto o de otros elementos que la circunden. También puede enmarcarla en un borde negro opaco y especificar el grosor de éste en píxeles. Introduzca el valor cero si prefiere no incluir ningún borde.
    • Alinear texto con imagen: Si ha situado la imagen al lado de un texto, seleccione uno de los iconos de alineación para indicar cómo quiere que aparezca éste en relación a la imagen.
    • Imagen parametrizable: Haga clic en Eliminar para suprimir los valores del mapa de la imagen.
  5. Haga clic en la pestaña Enlace para modificar las propiedades siguientes:
    • Introducir una ubicación de página web: Si desea enlazar la imagen, introduzca la URL de una página local o remota, o bien seleccione alguno de los anclajes o encabezados relacionados en la lista desplegable. Haga clic en Seleccionar archivo para buscar un archivo de imagen en una unidad de disco de su equipo o de una red.
    • La dirección URL está relacionada con la ubicación de la página: Si está seleccionada esta opción, Composer convierte la URL para que sea relativa a la ubicación real de la página. Esto es especialmente útil si tiene previsto publicar las páginas en un servidor web para que otras personas las visiten. El uso de URL relativas permite mantener todos los archivos enlazados en el mismo lugar, independientemente de cuál sea su ubicación en el disco duro de su equipo o en un servidor web.

      Si no selecciona esta casilla, Composer convierte la URL en una URL completa (absoluta). Es aconsejable que utilice URL absolutas cuando inserte enlaces a imágenes que estén almacenados en otros servidores web (no en una unidad de disco de su equipo).

      Si tiene cambios sin guardar, primero debe almacenar la página para que esta casilla de verificación aparezca activada. (Esta casilla no aparece activada si el cuadro de diálogo Propiedades de la imagen se abre desde una ventana de composición de mensajes.)

    • Mostrar borde de imagen enlazada: Si marca esta opción, aparecerá alrededor de la imagen el color seleccionado para destacar enlaces.
  6. Haga clic en Edición avanzada para mostrar el Editor de propiedades avanzadas, desde el que podrá aplicar otros atributos o introducir eventos de JavaScript.
  7. Haga clic en Aceptar para confirmar los cambios.

Cómo se crea un enlace dentro de la misma página

Para crear un enlace dentro de la misma página (por ejemplo, para que el usuario pueda saltar de una sección a otra), debe crear un anclaje (ubicación de destino) y, a continuación, un enlace que remita al anclaje. A los anclajes también se les denomina anclajes con nombre.

  1. Haga clic para colocar el punto de inserción al principio de la línea en la que desee crear un anclaje, o seleccione un fragmento de texto.
  2. Abra el menú Insertar y seleccione Anclaje con nombre. Aparece el cuadro de diálogo Propiedades del anclaje con nombre.
  3. Escriba un nombre único para el anclaje en el campo Nombre del anclaje (hasta 30 caracteres). Si incluye espacios, éstos se convertirán en subrayados ( _ ). Si ha seleccionado texto en el paso 1, este cuadro ya contendrá un nombre.
  4. Haga clic en Aceptar. Aparece un icono de anclaje en el documento para indicar la ubicación de éste:

Para crear un enlace de modo que el usuario pueda hacer clic en él para saltar hasta el anclaje:

  1. Seleccione el texto o la imagen que quiera enlazar con el anclaje.
  2. Haga clic en el botón Enlace o abra el menú Insertar y elija Enlace. Aparece el cuadro de diálogo Propiedades del enlace.
    • Si se trata de un enlace a un archivo HTML almacenado en su equipo, haga clic en Seleccionar archivo para localizarlo.
    • Si se trata de un enlace a un anclaje con nombre (destino), selecciónelo en la lista de anclajes la página.
    • Si se trata de un enlace a un encabezado con nivel (por ejemplo, Encabezado 1  Encabezado 6), selecciónelo en la lista de títulos de la página.
  3. Haga clic en Aceptar.

Nota: Para comprobar el enlace que ha creado, abra el menú Archivo, elija Página Explorar y haga clic en el enlace.

Sugerencia: Si no ha creado ningún anclaje con nombre previamente, puede utilizar el cuadro de diálogo Enlace para crear enlaces a los encabezamientos que ya existen en la página.

Cómo se crea un enlace a otras páginas

Puede crear enlaces desde su página a páginas locales de su propio sistema o de una red, o a otras remotas de cualquier lugar de Internet.

    • Enlazar dirección: Escriba la ruta de acceso local y el nombre del archivo, o la URL remota de la página a los que quiera que remita el enlace. Si no está seguro de la ruta de acceso y el nombre del archivo local, haga clic en Seleccionar archivo para localizarlo en el disco duro o una red. Para URL remotas, puede copiar la URL del campo Dirección del navegador. Si lo prefiere, puede seleccionar un anclaje con nombre o un encabezamiento de la página actual como destino del enlace.
    • La dirección URL está relacionada con la ubicación de la página: Si está seleccionada esta opción, Composer convierte la URL para que sea relativa a la ubicación real de la página. Esto es especialmente útil si tiene previsto publicar las páginas en un servidor web para que otras personas las visiten. El uso de URL relativas permite mantener todos los archivos enlazados en el mismo lugar, independientemente de cuál sea su ubicación en el disco duro de su equipo o en un servidor web.

      Si no selecciona esta casilla, Composer convierte la URL en una URL completa (absoluta). Es aconsejable que utilice URL absolutas cuando inserte enlaces a páginas que estén almacenadas en otros servidores web (no en una unidad de disco de su equipo).

      Si tiene cambios sin guardar, primero debe almacenar la página para que esta casilla de verificación aparezca activada.

  4. Haga clic en Edición avanzada para mostrar el Editor de propiedades avanzadas, desde el que podrá aplicar otros atributos o introducir eventos de JavaScript.
  5. Haga clic en Aceptar.
  6. Para comprobar el enlace que acaba de crear, haga clic en el botón Explorar y seguidamente en el propio enlace para asegurarse de que funciona como cabe esperar.

Sugerencia: Para copiar un enlace con rapidez, puede hacer clic en él y arrastrarlo directamente desde otra ventana hasta soltarlo en la página con la que esté trabajando. Por ejemplo, puede resaltar un enlace de una página web, un marcador o una ventana de correo y noticias, y arrastrarlo y soltarlo en su página. También puede hacer clic con el botón derecho en un enlace de en una página web (Control+clic en Macintosh) y elegir Copiar ubicación de enlace en el menú contextual. Seguidamente, puede pegar el enlace en el campo Enlazar dirección del cuadro de diálogo Propiedades del enlace.

Casi todos los ISP ofrecen espacio de sus servidores web para la publicación de páginas web. Para saber en qué servidor web puede publicar páginas, consúlteselo a su ISP, o pregúntele al administrador del sistema o al servicio de asistencia.

Cómo se publica un documento

Sugerencia: Es aconsejable que, antes de insertar enlaces o imágenes, guarde o publique la página con la que esté trabajando. De esta forma, Composer utilizará automáticamente referencias relativas a los enlaces e imágenes una vez que los inserte.

Para publicar un documento:

  1. Abra el documento HTML que desee publicar, o cree uno en Composer.
  2. Cuando esté listo para publicar el documento en un sistema remoto, haga clic en el botón Publicar.

    3. Si ya ha publicado este mismo documento con anterioridad, Composer le recordará los valores de configuración que en su momento estableció y procederá a publicarlo. Mientras transcurre el proceso de publicación, Composer le mantendrá al tanto de cómo evoluciona mediante un cuadro de diálogo.

    • Si nunca ha publicado un documento, Composer mostrará la pestaña Valores de configuración del cuadro de diálogo Publicar página para que introduzca los datos concernientes a la ubicación remota del documento. Consulte La pestaña Valores de configuración del cuadro Publicar página si desea más información. Cuando haya introducido la información, haga clic en Publicar.
    • Si aún no había guardado el documento, Composer muestra la pestaña Publicar del cuadro de diálogo Publicar página para que le asigne un nombre al archivo. Consulte La pestaña Publicar del cuadro Publicar página si desea más información. Cuando haya introducido el nombre del archivo, haga clic en Publicar.
  3. Para navegar por la página publicada, haga clic en el botón Examinar. Compruebe los enlaces de la página y que no falta ninguna imagen.
  4. Siga modificando la página si es necesario. Cuando esté listo para actualizar la página remota con los cambios realizados, haga clic en el botón Publicar.

Cuando publique un documento por primera vez, Composer sustituye la URL file:/// por una URL http:// para indicarle que está modificando la versión publicada. Si quiere guardar el documento en una unidad de disco de su equipo, haga clic en el botón Guardar. Se le pedirá que le asigne al archivo un nombre y una ubicación en la unidad de disco de su equipo.

Sugerencias para evitar que haya enlaces rotos o falten imágenes en una página


  • Compruebe que el nombre de los archivos de Composer acaba en la extensión .html o .htm. Compruebe que el nombre de los archivos de imagen acaba en las extensiones .JPG, .GIF o .PNG. No emplee espacios ni símbolos o caracteres especiales en los nombres de archivo. Procure emplear nombres cortos y utilice sólo letras mayúsculas o minúsculas, o números.
  • Si al explorar un documento en un servidor web, las imágenes aparecen como enlaces rotos, es posible que ello se deba a que ha olvidado incluir las imágenes entre los archivos publicados. En el menú Archivo, elija Publicar como para abrir el cuadro de diálogo Publicar página. Compruebe que en la pestaña Publicar esté seleccionada la la opción "Incluir imágenes y otros archivos", y seguidamente haga clic en Publicar.

Cómo se actualiza un documento ya publicado

Para publicar un documento ya publicado:

  1. Desde una ventana de Composer, abra el menú Archivo y elija Páginas recientes; a continuación, seleccione el documento en la lista.

    2. Si lo prefiere, introduzca en el campo Dirección de Navigator la dirección HTTP del documento (es decir, su dirección web) para su ubicación.

  2. Abra el menú Archivo y seleccione Editar página.
  3. Haga en el documento los cambios que considere oportunos.
  4. Cuando esté listo para actualizar la página remota con los cambios realizados, haga clic en Publicar en la barra de herramientas de Composer.

Sugerencia: Para suprimir una página o una imagen ya publicada en un servidor web, debe recurrir a un programa de FTP (Protocolo de transferencia de archivos). También deberá servirse de un programa de FTP si desea crear subdirectorios o cambiarles el nombre a los archivos almacenados en el servidor web. Consúltele a su proveedor de servicios qué programa de FTP aconsejan utilizar. Es habitual encontrar información sobre programas de FTP en las secciones de ayuda o de asistencia técnica del sitio web del proveedor. También es posible descargar programas compartidos de FTP de sitios como ZDNet (sección Downloads).

Cómo se crea un sitio para publicar páginas

Si tiene previsto publicar en varios sitios remotos, puede configurar Composer para que guarde la información relacionada con la publicación en cada uno de ellos, de tal manera que no tenga que volver a introducirla cada vez que quiera publicar alguna página.

Para crear un sitio de publicación, comience desde una ventana de Composer.

  1. Abra el menú Editar y seleccione Publicar configuración del sitio. Composer muestra el cuadro de diálogo Publicar configuración.
  2. Haga clic en Sitio nuevo.
  3. En el campo "Nombre del sitio", introduzca el sobrenombre con el que desee denominar el nuevo sitio.

    Por ejemplo, si va a utilizar el nuevo sitio para publicar documentos relacionados con el proyecto "Meteoro", podría asignarle ese nombre al sitio. El nombre de un sitio le sirve de recordatorio de los tipos de documentos que tiene publicados en él.

  4. En el campo "URL de publicación", introduzca la URL completa tal y como se la facilite su ISP, el administrador del sistema o el servicio de servidores web. La URL debe comenzar por ftp:// o bien por http://.

    La dirección de publicación indica el lugar en el que se publican los documentos, esto es, la ubicación del sitio a la que se transfieren. Si no está seguro de qué datos debe introducir, consúlteselos a su ISP o al administrador del sistema.

  5. En el campo "Dirección HTTP para examinar", introduzca la URL completa que deberá escribirse en Navigator para visitar las páginas de este sitio. No incluya el nombre de ningún archivo ni directorio como parte de la URL.

    La URL debe comenzar siempre por http://. En algunos casos, la URL coincide con la dirección de publicación. Si no está seguro de qué datos debe introducir, consúlteselos a su ISP o al administrador del sistema; si no los consigue, deje el campo en blanco.

  6. En el campo "Nombre de usuario", introduzca el nombre que utilizará para conectarse al ISP o al servicio de servidores web.
  7. En el campo "Contraseña", asígnele una contraseña al nombre de usuario introducido.
  8. Marque la casilla "Guardar contraseña" para que el Gestor de contraseñas la almacene y le ahorre tener que introducir los datos cada vez que acceda a este sitio.
  9. Haga clic en Aceptar.

Independientemente del número de sitios que haya configurado, siempre puede publicar un documento en cualquiera de ellos seleccionando Publicar como en el menú Archivo de Composer. Si desea más información, consulte Cómo se modifican el nombre del archivo o la ubicación.

    Si sólo tiene uno configurado, Composer lo utilizará automáticamente como predeterminado.

  3. Haga clic en el botón Definir como valor predeterminado.
  4. Haga clic en Aceptar para confirmar los cambios.

Cómo se resuelven problemas comunes de publicación

Si alguno de los archivos no consigue publicarse, aparecerá un mensaje de error el cuadro de diálogo de estado de la publicación para ayudarle a detectar el motivo y ponerle remedio. Desde el propio cuadro de diálogo de estado de la publicación, se le ofrece, mediante un botón, acceso a información para ayudarle a solucionar el problema.

Si, ni siquiera así, consigue publicar un archivo, guárdelo en una unidad de disco de su equipo seleccionando Guardar en el menú Archivo de Composer. Posteriormente puede abrir el archivo y volver a intentar publicarlo. Para agilizar la posterior localización del archivo, abra el menú Archivo de Composer y elija Páginas recientes.

Cómo se comprueba la configuración de publicación

Para comprobar la configuración de publicación:

  1. Si sigue abierto, cierre el cuadro de diálogo de estado de la publicación.
  2. Abra el menú Editar y seleccione Publicar configuración del sitio.
  3. En el cuadro de diálogo Publicar configuración, confirme si son correctos todos los valores del sitio en el que está intentando publicar sus documentos. Si tiene alguna duda, consúltesela a la ISP o al servicio de servidores web.
    • Cerciórese de haber introducido correctamente todos los valores de publicación. Es posible que haya cometido algún error al escribir alguno de los valores.
    • Compruebe asimismo que ha introducido correctamente la dirección de publicación. En ocasiones, los ISP y los servicios de sitios web emplean denominaciones como "nombre del servidor", "nombre del host", o "servidor/host", entre otras, para referirse a la dirección de publicación. Es frecuente también que reseñen la ubicación de publicación como ftp.miisp.com/usuario, donde usuario se refiere al nombre de usuario que usted tenga asignado.

      Para que la dirección de publicación sea correcta, debe anteponerle necesariamente ftp:// o bien http://. Por ejemplo, la dirección de publicación correcta que correspondería al sitio mencionado anteriormente es ftp://ftp.miisp.com/usuario.


Cómo se comprueban los nombres de archivo

Revise con atención los nombres de todos los archivos que no ha conseguido publicar. Compruebe que los nombres de los archivos:

  • Contienen sólo letras mayúsculas o minúsculas, o números. Aunque es perfectamente aceptable el empleo de mayúsculas en el nombre de un archivo, si las evita, prevendrá que se produzcan posibles errores a la hora de localizar el archivo publicado.

    Cuando se publican archivos en un servidor web, éste diferencia entre mayúsculas y minúsculas. Además, suele resultar más difícil recordar nombres de archivo que se compongan sólo de mayúsculas, o de una combinación de mayúsculas y minúsculas.

    Por ejemplo, cuando intente localizar un archivo publicado escribiendo su dirección web en el campo Dirección de Navigator, deberá teclearla tal y como cuando lo creó, con la misma combinación de mayúsculas y minúsculas.

  • No emplee ni signos de puntuación ni espacios. El subrayado ( _ ) y el guión ( - ) sí son admisibles.
  • Los nombres de los archivos de Composer deben acabar en la extensión .html o bien .htm.
  • No utilice nunca más de 32 caracteres.

También es posible descargar programas compartidos de FTP de sitios como ZDNet (sección Downloads).

Descripción del error: Este error puede deberse a múltiples causas. Por ejemplo:

  • La configuración del sitio de publicación puede no ser la correcta.
  • Se ha interrumpido la conexión a Internet.
  • El módem o el equipo con el que se conecta a Internet podría tener algún problema de funcionamiento.
  • Es posible que el servidor web en el que está intentando publicar no esté disponible a causa de un problema técnico o de alguna circunstancia desconocida.
  • Es posible que el ISP o el servicio de servidores web tenga algún problema técnico.

Soluciones posibles:

  • Compruebe que son correctos tanto la configuración para publicación como los valores introducidos. Consulte Cómo se comprueba la configuración de publicación si desea más información.
  • Visite alguna página web con Navigator para cerciorarse de que su conexión a Internet funciona correctamente. Por ejemplo, compruebe si puede acceder correctamente a la página http://home.netscape.com.
  • Si no consigue conectarse a Internet, compruebe si funcionan correctamente todo el equipo, las conexiones telefónicas, los módems y las conexiones de red.
  • Intente acceder con Navigator a alguna página que ya esté publicada en el sitio web al que está intentando acceder. Si puede ver correctamente otros sitios web pero no aquel en el que pretende publicar, es posible que el ISP o el servicio de servidores web tengan algún problema técnico.
  • Vuelva a intentar publicar la página en otro momento. Es posible que el ISP, el servicio de servidores web o el servidor al que intenta acceder tengan algún problema técnico.

Mensaje de error:

No tiene permiso para publicar páginas en el lugar especificado.

Descripción del error: Está intentando publicar en una ubicación, pero no cuenta con la autorización necesaria para hacerlo. Recuerde que sólo puede publicar páginas en sitios en los que un ISP o un servicio de servidores web le hayan autorizado a hacerlo.

  • Compruebe si son correctos el nombre de usuario y la contraseña introducidos en el cuadro de diálogo Publicar configuración del sitio, o en la pestaña Publicar del cuadro de diálogo Publicar.
  • Consúltele al ISP en qué lugar de su sitio web está autorizado a publicar páginas
  • Si su servicio de servidores web no le ofrece la posibilidad de publicar páginas, tendrá que cambiar de proveedor. En Navigator, realice una búsqueda de "alojamiento web", "ISP" u otras expresiones similares para localizar un proveedor que le ofrezca la posibilidad de publicar en la Web.
  • El servicio Netscape My Webpage http://mywebpage.netscape.com/ le ofrece de forma gratuita hasta 20 megabytes para alojar páginas web.

Descripción del error: Es posible que el disco del servidor web al que intenta acceder esté lleno, o que haya agotado la cuota de espacio en disco que le haya asignado el ISP o el servicio de servidores web.

Soluciones posibles:

  • Sírvase de un programa de FTP para suprimir los archivos innecesarios y dejar espacio libre en el sitio de publicación. Publicar página: Publicar

En la pestaña Publicar del cuadro Publicar página, puede indicarse dónde se desea publicar un documento. Estos valores de configuración se aplican al documento actual.

Si no está visualizando todavía la pestaña Publicar del cuadro de diálogo Publicar página, siga los pasos que se indican a continuación:

  1. Abra el menú Archivo y elija Publicar como. Aparecerá el cuadro de diálogo Publicar página.
  2. Pulse la pestaña Publicar.
  • Nombre del sitio: Es una relación de todos los sitios que haya creado en la que puede elegir el que desee publicar. Para crear un sitio, haga clic en el botón Sitio nuevo.
  • Título de la página: Especifique el título de la página tal y como aparecerá en la barra de título de la ventana del navegador cuando se visualice la página. El título de la página del documento también aparece en la lista de marcadores si marca la página.
  • Nombre de archivo: Especifique el nombre de archivo del documento. Asegúrese de incluir la extensión .html o .htm en el nombre del archivo.

  • Advertencia: Si en el sitio remoto ya hay algún archivo cuyo nombre coincida con el del que va a transferir, aquél quedará sobrescrito por este. No se le instará a que confirme la operación.

  • Subdirectorio del sitio para esta página: Si no rellena este campo, Composer publicará la página en el directorio de publicación principal, o raíz, del sitio. Si quiere publicar la página en algún subdirectorio remoto situado por debajo del directorio principal de publicación, deberá escribir su nombre o seleccionarlo en la lista. Composer "recuerda" las direcciones que introduzca en este campo para facilitarle posteriormente su selección en la lista. Tenga en cuenta que en los nombres de subdirectorios, se diferencia entre mayúsculas y minúsculas.

  • Nota: El subdirectorio elegido debe existir ya en el servidor remoto.

  • Incluir imágenes y otros archivos: Si se marca esta casilla, Composer publicará íntegramente todos los archivos e imágenes a los que se haga referencia de esta página. Puede optar por publicar estos archivos en el mismo lugar que la página o bien por hacerlo en algún subdirectorio del servidor remoto que ya exista dentro del directorio de publicación principal.

Sugerencia: Para crear un subdirectorio remoto, o para eliminar páginas o imágenes ya publicadas, deberá recurrir a un programa de FTP (Protocolo de transferencia de archivos). Consúltele a su proveedor de servicios qué programa de FTP aconsejan utilizar. Es habitual encontrar información sobre programas de FTP en las secciones de ayuda o de asistencia técnica del sitio web del proveedor. También es posible descargar programas compartidos de FTP de sitios como ZDNet (sección Downloads).

Publicar página: Valores de configuración

En la pestaña Valores de configuración del cuadro de diálogo Publicar página, ha de especificar la información necesaria para conectarse al servidor de publicación remoto, así como los valores de publicación de éste. Los valores especificados se aplicarán tanto al documento actual como a cualquier archivo que se publique en esta ubicación.

Si no está visualizando todavía la pestaña Valores de configuración del cuadro de diálogo Publicar página, siga los pasos que se indican a continuación:

  1. Abra el menú Archivo y elija Publicar como. Aparecerá el cuadro de diálogo Publicar página.
  2. Haga clic en la pestaña Valores de configuración.
  • Nombre del sitio: Escriba el sobrenombre que desee asignarle a este sitio de publicación. Procure emplear un nombre breve para que le resulte más fácil recordarlo.
  • URL de publicación: Especifique la URL completa, tal y como se la hayan facilitado el ISP o el administrador del sistema. La URL debe comenzar por ftp:// o bien por http://. En ocasiones, los ISP y los servicios de sitios web emplean denominaciones como "nombre del servidor", "nombre del host", o "servidor/host", entre otras, para referirse a la dirección de publicación.

  • La dirección de publicación indica el lugar en el que se publican los documentos, esto es, la ubicación del sitio a la que se transfieren. Si no está seguro de qué datos debe introducir, consúlteselos a su ISP o al administrador del sistema.

  • Dirección HTTP para examinar: Especifique la dirección completa del directorio raíz de publicación. Esta es la dirección web de la página inicial del sitio web publicado. No incluya el nombre de ningún archivo ni directorio como parte de la URL.

  • La URL debe comenzar siempre por http://. En algunos casos, la URL coincide con la dirección de publicación. Si no está seguro de qué datos debe introducir, consúlteselos a su ISP o al administrador del sistema; si no los consigue, deje el campo en blanco.

  • Nombre de usuario: Especifique el nombre de usuario con el que se conecta al ISP o a la red en cuestión.
  • Contraseña: Especifique la contraseña asignada a este nombre de usuario.
  • Guardar contraseña: Marque esta casilla para que el Gestor de contraseñas la cifre y la almacene, para ahorrarle tener que introducir los datos cada vez que publique páginas en este sitio.

Preferencias de Composer: Composer

Las preferencias de Composer sirven para especificar los valores de configuración relacionados con el almacenamiento de archivos y la modificación de tablas. Estos valores se aplican a todos los documentos que cree.

Si no está visualizando todavía la ventana Preferencias de Composer, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Haga clic en la categoría Composer.
  • Número máximo de páginas listadas: Especifique el número máximo de páginas recientes que aparecerán en la lista Páginas recientes del menú Archivo.
  • Retener formato de origen: Esta opción conserva los saltos de línea y el formato original de la página para el código origen de HTML. Seleccione esta opción si desea conservar el espacio en blanco (líneas adicionales, tabulaciones, etc.) que hace que el código origen de HTML sea legible. Esta preferencia no afecta a la apariencia de las páginas en la ventana del navegador.
  • Cambiar formato de código fuente HTML: Esta opción le vuelve a dar formato al código HTML para hacerlo más legible, insertando saltos de línea y sangrías. Esta preferencia no afecta a la apariencia de las páginas en la ventana del navegador.
  • Guardar imágenes y demás archivos asociados al guardar las páginas: Si se marca esta casilla, la totalidad de las imágenes, los elementos de JavaScript (JS) y las hojas de estilo en cascada (CSS), así como otros archivos asociados, quedan almacenados en la misma ubicación que el documento cuando éste se guarde por primera vez o cuando se almacene en una nueva ubicación. Si no se marca la casilla, sólo se guardará el archivo HTML.

  • Por ejemplo, al modificar una página remota, esta casilla garantiza que todos los archivos relacionados con ella se almacenarán localmente cuando guarde la página en una unidad de disco de su equipo.

  • Mostrar siempre el cuadro de diálogo Publicar al publicar las páginas: Si se marca esta casilla, Composer mostrará el cuadro de diálogo Publicar página siempre que haga clic en el botón Publicar o elija Publicar en el menú Archivo. Si no se marca, Composer sólo mostrará el cuadro de diálogo Publicar página cuando necesite solicitarle algún dato para publicar una página.
  • Mantener el diseño de la tabla al insertar o eliminar celdas: Seleccione esta opción si desea que Composer preserve siempre la forma de la tabla (es decir, que mantenga la forma rectangular) agregando celdas donde sea necesario. Si no marca esta opción, al eliminar una o más celdas, Composer elimina también el borde de la celda, lo que puede hacer que queden espacios vacíos en una tabla o que una línea exterior sea irregular a causa del número impar de celdas.
  • Utilizar estilos CSS en lugar de elementos y atributos HTML: Al activar esta opción, se habilita a Composer a que aplique formato a los documentos utilizando hojas de estilo en cascada (CSS). Si se activa esta opción, Composer genera formato HTML 4.01 aplicándoles a los elementos estilos CSS integrados.

  • Si se desactiva esta opción, Composer genera formato HTML 4.01, pero no emplea estilos CSS.

    El empleo de HTML con formato CSS, en comparación con el exclusivo de HTML, resulta más compatible, de mantenimiento más sencillo y facilita la visualización en navegadores diferentes. Si activa esta preferencia y modifica un documento creado sin estilos CSS, Composer sustituirá todos los elementos modificados por CSS.

    Si decide utilizar estilos CSS, se activará el botón de la barra de herramientas de formato que le permite resaltar con colores el texto seleccionado. Además, podrá asignar un color de fondo a cualquier elemento de la página. (Si la casilla de estilos CSS no está activada, no se puede disponer de las funciones descritas.)

Preferencias de Composer: Valores página nueva

En esta sección, se describe cómo configurar las preferencias de colores e imágenes de fondo que se aplican a todos los documentos que se creen.

Si no está visualizando todavía la ventana para la configuración de nuevas páginas, siga los pasos que se indican a continuación:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Abra la categoría Composer y haga clic en Valores página nueva.
  • Autor: Escriba su nombre. De esta forma, Composer lo incluirá automáticamente en el código origen HTML cada vez que cree una página.
  • Colores predeterminados del lector: Seleccione esta opción si desea que el texto y los elementos enlazados de las páginas utilicen los colores establecidos por el navegador de quien las visite.
  • Usar colores personalizados: Seleccione esta opción si desea especificar colores que se aplicarán al texto y a los elementos enlazados. Seguidamente, asígnele un color a cada elemento haciendo clic en el botón correspondiente situado junto a cada elemento.
  • Imagen de fondo: Escriba la ubicación y el nombre de un archivo de imagen, o haga clic en Seleccionar archivo para localizar un archivo de imagen en una unidad de disco de su equipo o de una red.

    Nota: Las imágenes de fondo aparecen en mosaico y prevalecen sobre el color de fondo.

Para cambiar el nombre del autor en una determinada página: Abra el menú Formato, y elija Título y propiedades de la página.

Para cambiar los colores y la imagen de fondo de una determinada página: Abra el menú Formato, y elija Colores y fondo de la página.
