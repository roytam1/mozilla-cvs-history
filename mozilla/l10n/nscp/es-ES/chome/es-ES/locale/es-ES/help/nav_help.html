La navegación por Internet

Bienvenido a Netscape. La navegación por Internet es uno de los usos que la gente da a Netscape con mayor frecuencia. Navigator, el componente de Netscape que permite visitar páginas web, ofrece numerosas formas de visitarlas y de hacer búsquedas en Internet.

En esta sección se presenta Navigator; léala para aprender a navegar y hacer búsquedas por Internet y a guardar páginas web.

Cómo se visualiza la página de inicio

Al iniciar Netscape, verá Navigator, su navegador. Al arrancar por primera vez Netscape, aparece automáticamente la página inicial en la ventana del navegador.

Cuando vuelva a abrir Netscape en ocasiones posteriores, normalmente verá su página de inicio. Suele verse la página de inicio elegida por el administrador de red o por el proveedor de servicios de Internet, o la página de inicio de Netscape, a no ser que elija usted mismo una página de inicio.

Para escoger su propia página de inicio, consulte Cómo se especifica el modo de inicio de Netscape.

Sugerencia:

  • Para aprovechar al máximo la interfaz de Netscape, aplique el modo Pantalla completa; de este modo las páginas web que vea ocuparán casi toda la pantalla. En Navigator, abra el menú Ver y elija Pantalla completa. También puede pulsar F11.
  • Para ir rápido a la página de inicio, pulse Alt+Inicio (Cmd+Inicio en Mac OS).

Cómo se cambia de página

Para ir a una página distinta, escriba la URL, es decir, su ubicación (dirección) en la web. Las direcciones URL normalmente empiezan por "http://", seguido por uno o más nombres que identifican la dirección. Por ejemplo "http://netscape.com".

  1. Haga clic en la barra Dirección para seleccionar la dirección URL que ya está escrita.
  2. Escriba la dirección URL de la página que desea visitar. La URL que escriba sustituirá cualquier texto existente en la barra Dirección.
  3. Pulse Entrar.

El icono con forma de candado que está en la esquina inferior derecha de la pantalla permite comprobar en cualquier momento el estado de seguridad de la página. Si desea más información, consulte Comprobación de la seguridad de una página web.

Sugerencia: para seleccionar rápidamente la URL de la barra Dirección, pulse Ctrl+L.

 

Barra Dirección

 

¿No conoce la dirección URL? Escriba una parte de una dirección, como "cnn" (para www.cnn.com); o escriba una palabra general, como "regalos" o "flores". Navigator intenta averiguar qué páginas quiere ver o muestra una página con varios enlaces relacionados con la palabra que ha escrito.

Si acaba de entrar en el mundo de Internet, consulte el manual en línea New to the Net Tutorial (Manual breve para novatos de Internet).

Cómo se hace clic en un enlace

La mayoría de las páginas web contienen enlaces donde puede hacer clic para ir a otras páginas.

  1. Mueva el puntero hasta que se transforme en una mano con un dedo que señala. El puntero adoptará esta forma cada vez que lo sitúe encima de un enlace. La mayoría de los enlaces están subrayados, pero los botones e imágenes también pueden actuar como enlaces.
  2. Haga clic en el enlace una vez. Mientras la red localiza la página que indica el enlace, en la parte inferior de la ventana aparecen mensajes de estado.

Cómo se regresa a una página ya visitada

Existen varios métodos para volver a visitar las páginas que ya ha visto:

  • Para volver una página atrás o ir una página adelante, haga clic sobre la flecha Anterior o Siguiente.
  • Para avanzar o retroceder más de una página, haga clic en los pequeños triángulos de los botones Anterior o Siguiente. Verá una lista de las páginas que ya ha visitado. Para volver a una página, escójala en la lista.

 

    Anterior  Siguiente

 

  • Para ver una lista de las URL escritas en la barra Dirección, haga clic en la flecha que está a la derecha de la barra. Para ver una página, elija una de la lista.

 

Barra Dirección

 

  • Para escoger entre las páginas que ha visitado durante la sesión en curso, abra el menú Ir y utilice la lista de la parte inferior del menú.
  • Para escoger entre las páginas que visitó durante sesiones pasadas, abra el menú Ir, y seleccione Historial. Aparecerá la lista del historial. La lista del historial contiene una lista de carpetas. Haga doble clic en las carpetas para ver subcarpetas o marcadores de las páginas web. Haga doble clic en la URL que está junto al icono Marcador para ver la página.

Sugerencia: la pestaña Historial de Mi Barra lateral también permite seleccionar las páginas visitadas durante las últimas sesiones. Si desea más información, consulte Cómo se agrega una pestaña a Mi Barra lateral.

 

Qué es el historial

El historial contiene enlaces a páginas visitadas recientemente. La lista de la barra Dirección contiene enlaces a páginas cuyas URL escribió en dicha barra para visitarlas.

Para acceder al historial desde Navigator, abra el menú Ir y seleccione Historial. Para acceder a la lista de la barra Dirección, haga clic en la flecha que está en el extremo derecho de la barra Dirección.

Sugerencia: para abrir rápidamente el historial, pulse Ctrl+H.

Si no quiere que la barra Dirección o el historial muestren las páginas que ha visitado, puede borrar totalmente o de forma selectiva la lista del historial y el historial de la barra Dirección.

Para borrar todas las páginas de la barra Dirección o del historial, comience desde la ventana Navigator:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Abra la categoría Navigator y seleccione Historial. (Si no aparece ninguna subcategoría, haga doble clic en Navigator para expandir la lista.)
  3. Haga clic en Borrar historial y Borrar barra Dirección para eliminar de la lista todas las páginas web visitadas.
  4. Si desea establecer un límite de páginas guardadas en el historial, haga clic en el campo junto a Número de páginas de historial de sesiones y escriba el número de páginas.

Para suprimir páginas de forma selectiva del historial, haga lo siguiente:

  • Para suprimir todas las páginas de un dominio, seleccione una página de ese dominio (carpeta) en la ventana del historial, abra el menú Editar y seleccione Suprimir todo el dominio [nombre de dominio]. Utilice este comando, por ejemplo, si desea suprimir todas las páginas que terminen en "netscape.com".
  • Para suprimir todas las páginas de un subdominio, seleccione una página de ese subdominio en la ventana del historial, abra el menú Editar y seleccione Suprimir todo de [subdominio]. Utilice este comando, por ejemplo, si desea suprimir todas las páginas de "health.netscape.com", pero no "netscape.com".
  • Para suprimir una sola página, seleccione una página de la lista del historial y pulse Supr.

Sugerencia: para ordenar el historial, haga clic en una de las categorías (Título, Ubicación o Última visitada). Haga clic de nuevo para invertir el orden.

Cómo se visita una página marcada

Las direcciones, o URL, de páginas web a veces son largas y difíciles de recordar. Afortunadamente, no es necesario aprenderse de memoria las URL para navegar por la web. Su navegador tiene una lista de marcadores que lo llevan a las páginas web más interesantes.

Para ir a una página marcada, empiece en la ventana Navigator:

  1. Abra el menú Marcadores. El menú contiene marcadores representados por un icono de marcador y carpetas que contienen más marcadores.
  2. Para visitar una página marcada, elija un marcador en el menú o abra una carpeta y elija un marcador.

Sugerencia: para volver atrás, haga clic en la flecha Anterior.

Guarde sus propios marcadores para ir a las páginas que visita con mayor frecuencia o a otros lugares interesantes de Internet. Si desea más información, consulte Cómo se crea un marcador.

Cómo se navega con pestañas

Cuando visite más de una página web al mismo tiempo, use la navegación con pestañas para navegar de un modo más rápido y sencillo.

La navegación con pestañas permite abrir pestañas, cada una con una página web, dentro de una sola ventana de Navigator. No hace falta abrir varias ventanas para visitar varias páginas web. De esta forma dispondrá de más espacio en el escritorio. Abra, cierre y recargue páginas web cómodamente en un solo lugar, sin necesidad de cambiar de una ventana a otra.

Gestione las pestañas de navegación fácilmente y controle la apertura automática de pestañas. Si desea más información sobre la navegación con pestañas, consulte Preferencias de Navigator: navegación con pestañas.

Para aprender a usar la navegación con pestañas, consulte Exploración con pestañas.

Cómo se emplea Mi Barra lateral

Además de navegar por la web con Navigator, también puede dejar que la web venga a visitarlo a usted con ayuda de Mi Barra lateral.

Mi Barra lateral es un marco personalizable del navegador en el que podrá mantener todos los elementos que necesite usar en cualquier momento, como las últimas noticias y el tiempo, el libro de direcciones o la Lista de contactos, los índices bursátiles y una agenda, entre muchas más opciones. Mi Barra lateral presenta estos elementos en pestañas que se actualizan continuamente.

Netscape incorpora algunas de la pestañas de Mi Barra lateral preestablecidas, pero puede personalizarla agregando, eliminando y reorganizando las pestañas. Si desea más información, consulte Mi Barra lateral.

Para ver un elemento de Mi Barra lateral, haga clic en la pestaña correspondiente.

Selector de Mi Barra lateral

Si aún no está abierta Mi Barra lateral, ábrala haciendo clic en el selector. Si no hay selector, abra el menú Ver de Navigator, elija Mostrar/ocultar, y luego seleccione Mi Barra lateral en el submenú.

Cómo se realiza una búsqueda rápida

Hay tres formas de realizar una búsqueda rápida: desde la barra Dirección, desde Mi Barra lateral y desde la página Buscar en la red.

 

Búsqueda desde la barra Dirección

Buscar páginas web sobre un tema determinado resulta tan fácil como escribir una palabra o dos en la barra Dirección del navegador, tal como se ilustra a continuación.

Por ejemplo, si desea encontrar información sobre muñecas:

  1. Haga doble clic en la barra Dirección para seleccionar el texto actual.
  2. Escriba la palabra "muñeca". Lo que introduzca sustituirá a lo que ya estuviera escrito.
  3. Realice una de las siguientes acciones:
  • Haga clic en el botón Buscar.
  • Haga clic en Buscar Netscape Search en Español para "muñeca" en la parte inferior de la lista desplegable en la barra Dirección. (Es posible que su buscador predeterminado sea distinto.)
    Se utilizará el buscador predeterminado que seleccione en Preferencias. Los resultados de la búsqueda de la palabra "muñeca" aparecerán en la ventana del navegador y en Mi Barra lateral. Haga clic en los enlaces para visitar las páginas web sobre muñecas.

Después de realizar una búsqueda, los resultados se guardarán en la pestaña Buscar de Mi Barra lateral hasta la nueva búsqueda. No es necesario que vuelva a hacer clic en el botón Anterior para recuperar los resultados de la búsqueda.

Nota: también puede usar palabras clave de Internet para realizar búsquedas desde la barra Dirección.

 

Búsqueda desde Mi Barra lateral

La pestaña Buscar de Mi Barra lateral le permite buscar rápidamente y marcar los resultados de la búsqueda. Por ejemplo, si quiere encontrar información sobre coches de juguete:

  1. Si aún no está abierta Mi Barra lateral, ábrala haciendo clic en el selector.
  2. Haga clic en la pestaña Buscar.
  3. Abra la lista desplegable Utilizando y elija un buscador.
  4. Escriba "coche de juguete" en el campo de búsqueda.
  5. Haga clic en Buscar. Los resultados de la búsqueda de "coche de juguete" aparecerán en la ventana del navegador y en Mi Barra lateral. Haga clic en los enlaces para visitar las páginas web sobre coches de juguete.

Nota: si la pestaña Buscar de Mi Barra lateral contiene una lista de buscadores entre los que elegir, es que está configurado en modo avanzado; de ser así, tendrá que elegir uno o varios buscadores antes de hacer clic en buscar. Puede dejarla así o ir a Preferencias y cambiarla a la configuración básica.

Sugerencia: Para mostrar rápidamente la página siguiente o anterior de resultados de la búsqueda, haga clic en los botones Siguiente y Anterior que están en la parte inferior de la pestaña Buscar de Mi Barra lateral.

 

Búsqueda desde la página Buscar en la web

La página Buscar en la web le permite escribir un término o frase de búsqueda o explorar varias categorías (como Artes, Recreación, Negocios y muchas otras) que le puedan interesar. Para visitar la página Buscar en la web, realice una de las siguientes acciones:

  • Haga clic en el botón Buscar de la barra de herramientas Personal.
  • Abra el menú Herramienta y elija Buscar en la Web.

 

Búsqueda de palabras seleccionadas en una página web

Netscape permite buscar palabras seleccionadas dentro de una página web:

  1. Seleccione (resalte) cualquier palabra de la página web.
  2. Haga clic con el botón derecho del ratón y seleccione Búsqueda en la web de [las palabras seleccionadas] en el menú emergente. (Para hacer lo mismo en Mac OS, pulse Ctrl y haga clic en el botón del ratón.)

Netscape abrirá una ventana nueva y usará el buscador predeterminado para buscar las palabras seleccionadas. Para aprender a cambiar el buscador que se usará para buscar las palabras seleccionadas, consulte Preferencias de Navigator: buscar en Internet.

Cómo se realiza una búsqueda avanzada

Puede establecer la pestaña Buscar de Mi Barra lateral en modo Avanzado; así, efectuará búsquedas más exactas y podrá escoger uno o más buscadores.

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Navigator, haga clic en Buscar en Internet. (Si no aparece ninguna subcategoría, haga doble clic en Navigator para expandir la lista.)
  3. En Preferencias de la pestaña Buscar de Mi Barra lateral, haga clic en Avanzadas y, luego, en Aceptar.
  4. Si aún no está abierta Mi Barra lateral, ábrala haciendo clic en el selector.
  5. Haga clic en la pestaña Buscar de Mi Barra lateral.
  6. Abra la lista desplegable Incluidos en y elija una categoría.
  7. Elija uno o más buscadores para la categoría seleccionada (si está disponible).
  8. En el campo de búsqueda, escriba la palabra sobre la que desea que se realice la búsqueda.
  9. Haga clic en Buscar.

 

 

Personalización de las categorías de búsqueda

Tal vez desee especificar qué buscadores deben utilizarse para las distintas categorías de búsqueda. Puede seleccionar, por ejemplo, un grupo de buscadores para buscar viajes y otro grupo de buscadores para software.

  1. Configuración de la búsqueda de Mi Barra lateral en Búsqueda avanzada. Si desea información más detallada, consulte Búsqueda avanzada.
  2. Si aún no está abierta Mi Barra lateral, ábrala haciendo clic en el selector.
  3. Haga clic en la pestaña Buscar de Mi Barra lateral.
  4. Abra la lista desplegable Incluidos en y haga clic en Editar categorías Aparecerá el cuadro de diálogo Editar categorías.
  5. En la lista desplegable Categorías, seleccione la categoría que desee personalizar.
    O
    Para crear una nueva categoría, haga clic en Nuevo y escriba el nombre de la categoría. En la lista desplegable se verá el nombre de la categoría seleccionada.
  6. Elija una de las opciones siguientes:
    • Para agregar un buscador a la categoría seleccionada, resalte el buscador en la lista de la izquierda y haga clic en Agregar.
    • Para eliminar un buscador de la categoría seleccionada, resalte el buscador en la lista de la derecha y haga clic en Eliminar.
    • Para cambiar el nombre de la categoría seleccionada, haga clic en Renombrar y escriba un nombre distinto.
    • Para suprimir la categoría seleccionada, haga clic en Suprimir.
  7. Cuando haya terminado de personalizar, haga clic en Aceptar.

Cómo se configuran las preferencias de búsqueda

De forma predeterminada, se usa el buscador Netscape Search. Si quiere, elija otro buscador como predeterminado. También puede especificar cómo quiere que se muestren los resultados de la búsqueda en Mi Barra lateral.

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Navigator, haga clic en Buscar en Internet. (Si no aparece ninguna subcategoría, haga doble clic en Navigator para expandir la lista.)
  3. En Mecanismo de búsqueda predeterminado, seleccione el buscador que desee utilizar para buscar en la web.
  4. En Resultados de la búsqueda, seleccione Abrir la pestaña Buscar de Mi Barra lateral cuando los resultados de la búsqueda estén disponibles, si desea que la pestaña Buscar de Mi Barra lateral se abra y muestre los resultados de la búsqueda.
  5. En Preferencias de la pestaña Buscar de Mi Barra lateral, seleccione Básicas si desea emplear un buscador único o Avanzadas si desea emplear varios buscadores.

Cómo se emplean las palabras clave de Internet

Las palabras clave de Internet se combinan con la barra Dirección para ayudarlo a encontrar rápidamente tipos determinados de información. Existen varias maneras de utilizar palabras clave de Internet:

  • Indique el nombre de un producto determinado, una marca o una empresa y haga clic en Entrar (Retorno en Mac OS) para ir directamente al sitio web de dicha empresa.
  • Escriba verbos, como "comprar" o "ayudar", seguidos por la palabra que está buscando, para obtener información específica. Por ejemplo, "comprar animales domésticos" lo lleva a páginas con información sobre compra de animales.
  • Escriba el nombre de una ciudad o pueblo, seguido por palabras como "películas" o "restaurantes", para obtener información local. Por ejemplo, escriba "restaurantes de san francisco".

Consejo: las palabras clave de Internet más utilizadas se relacionan en la lista desplegable que está a la izquierda de la barra Dirección. Haga clic en el triángulo y elija una palabra de la lista. Para ver una lista más larga de palabras clave, elija Lista de palabras clave.

Importante: para usar las palabras clave de Internet, pulse la tecla Entrar (Retorno en Mac OS) en lugar de hacer clic en el botón Buscar. Si pulsa Entrar, activa la función de palabras claves; si hace clic en Buscar inicia una búsqueda.

Para ver la diferencia, seleccione una opción:

  1. Escriba "apple imac" en la barra Dirección y pulse Entrar (Retorno). Aparece la página web del equipo de Apple iMac.
  2. Escriba "apple imac" en la barra Dirección y haga clic Buscar. Ahora verá una lista de vínculos relacionados con modelos iMac del equipo Apple.

Si escribe las palabras clave de Internet en la barra Dirección y no le funciona, compruebe las preferencias de navegación inteligente para asegurarse de que la función palabras clave de Internet (Internet keywords) está activada.

 

Consulta rápida de acciones en bolsa

Para ver las acciones actualizadas en bolsa de una compañía, escriba "acciones" y a continuación la denominación abreviada del valor de las acciones de la compañía y pulse Entrar.

Si no conoce dicho símbolo, escriba el nombre de la compañía. La página Acciones mostrará la lista de todas las compañías con nombres similares; elija aquella cuyo valor en bolsa desee conocer.

Cómo se realiza una búsqueda dentro de una página

Para buscar texto dentro de la página que está viendo con Navigator:

  1. Abra el menú Editar y seleccione Buscar en esta página. Si la página que está visitando contiene marcos, quizá tenga que hacer clic primero dentro de un marco. Aparecerá el cuadro de diálogo Buscar en esta página.
  2. Escriba el texto que desee buscar.
  3. Elija entre las siguientes opciones de búsqueda:
    • Coincidir mayúsculas y minúsculas: seleccione esta opción para encontrar una palabra o frase que coincida exactamente con la palabra o frase que usted busca (deben coincidir incluso las mayúsculas y minúsculas).
    • Ajustar: seleccione esta opción para que el navegador busque a partir del lugar en el que está el cursor hasta el final de la página, y que luego continúe buscando desde el principio de la página. Nota: normalmente, se empieza a buscar desde el último lugar en el que clic con el botón del ratón hasta el final de la página.
    • Buscar hacia atrás: seleccione esta opción para buscar desde el punto de inserción hacia atrás, hasta la parte superior de la página.
  4. Haga clic en Buscar para empezar la búsqueda.

Para buscar la misma palabra o frase otra vez:

  • Abra el Menú Editar y elija Volver a buscar.

Cómo se realiza una búsqueda en las listas de marcadores o en el historial

Para buscar en los marcadores, comience en la ventana Navigator:

  1. Abra el menú Marcadores y elija Gestionar marcadores. Se abrirá la ventana Marcadores.
  2. En la ventana Marcadores, abra el menú Herramientas y elija Buscar marcadores. Aparecerá el cuadro de diálogo Buscar marcadores.
  3. Emplee las listas desplegables para seleccionar las opciones que definan su búsqueda.
  4. Escriba una palabra de búsqueda.
  5. Haga clic en Buscar. Los marcadores que coincidan con los criterios de búsqueda se mostrarán en la ventana Resultados de la búsqueda - Marcadores.

Sugerencia: para abrir rápidamente el menú Marcadores, pulse Ctrl+B.

Para buscar en el historial:

  1. Abra el menú Ir y elija Historial. Aparecerá la lista del historial.
  2. En la ventana Historial, abra el menú Herramientas y elija Buscar historial. Aparecerá el cuadro de diálogo Buscar en historial.
  3. Emplee las listas desplegables para seleccionar las opciones que definan su búsqueda.
  4. Elija entre las siguientes opciones de búsqueda:
  • Seleccione título o ubicación en función del lugar en el que desee buscar. (La ventana de marcadores permite buscar nombre, descripción y palabra clave.)
  • Elija contiene, empieza por, o termina por si sólo conoce parte de la palabra o frase que desea buscar.
  • Elija Es si sabe exactamente el nombre de lo que busca.
  • Elija no es, o no contiene para excluir páginas.
  • Haga clic en el campo y escriba, total o parcialmente, el nombre o la URL (dirección Web) de la lista de marcadores o historiales que desea encontrar o excluir.
  1. Haga clic en Buscar. Las listas de historial que coincidan con los criterios de búsqueda se mostrarán en la ventana Resultados de la búsqueda.

Cómo se copia un fragmento de una página

Para copiar texto de una página, empiece en la ventana Navigator:

  1. Seleccione el texto.
  2. Abra el menú Editar y seleccione Copiar.

Puede pegar el texto en otros programas.

  2. Haga clic con el botón derecho del ratón en el enlace o la imagen para ver un menú emergente. (En Mac OS, pulse la tecla Control y haga clic en el botón del ratón.)
  3. Elija Copiar ubicación de enlace o Copiar ubicación imagen. Si una imagen actúa como enlace, se le ofrecerán ambas opciones.

Pegue el enlace en otros programas o en la barra Dirección de Navigator.

Cómo se guarda una página completa o parte de ella

Para guardar una página completa, comience en la ventana Navigator:

  1. Abra el menú Archivo y seleccione Guardar página como. Aparecerá el cuadro de diálogo Guardar como.
  2. Seleccione la carpeta en la que desee guardar el archivo.
  3. Elija un formato para la página que desee guardar:
    • Página web, completa: guarda la página web completa con imágenes. Esta opción permite ver la página en su forma original, como imágenes, pero tal vez no conserve la estructura de enlaces HTML de la página original. Netscape crea un directorio nuevo (donde se guarda la página) para guardar las imágenes y los demás archivos que hacen falta para mostrar la página web completa.
    • Página web, sólo HTML: guarda la página original sin imágenes. Esta opción conserva la estructura de enlaces HTML en un archivo.
    • Archivo de texto: guarda la página original como un archivo de texto. Esta opción no conserva la estructura de enlaces HTML original, pero permite abrir en un editor de texto la versión en formato de texto de la página.
  4. Escriba un nombre para el archivo la página y haga clic en Guardar.

Cuando se visita una página que contiene marcos y hay un marco seleccionado en ese momento, en la lista desplegable aparece la opción Guardar marco como, además de la opción Guardar página como. Esto le permite guardar sólo la página que se encuentra dentro del marco seleccionado.

Si guarda un archivo en el disco duro, podrá ver la página (o su código HTML) sin estar conectado a Internet.

Para guardar una imagen de una página:

  1. Coloque el puntero del ratón sobre la imagen:
  2. Haga clic con el botón derecho en la imagen (en Mac OS, pulse la tecla Control y haga clic en el botón del ratón) para ver un menú emergente.
  3. Seleccione Guardar imagen como. Aparecerá el cuadro de diálogo Guardar imagen.
  4. Seleccione la carpeta en la que desea guardar la imagen.
  5. Escriba un nombre de archivo para la imagen y haga clic en Guardar.

Para guardar una página sin verla (útil para recuperar una página sin formato, como un archivo de datos, que no está pensado para ser mostrado):

  1. Coloque el puntero del ratón sobre el enlace de la página.
  2. Haga clic con el botón derecho en el enlace de la página (en Mac OS, pulse la tecla Control y haga clic en el botón del ratón) para ver un menú emergente.
  3. Seleccione Guardar enlace de destino como. Aparecerá el cuadro de diálogo Guardar como.
  4. Seleccione la carpeta en la que desee guardar el archivo.
  5. Escriba un nombre para el archivo la página y haga clic en Guardar.

Importante: algunos enlaces descargan y guardan archivos en el disco automáticamente después de hacer clic sobre ellos. Los URL de estos enlaces a menudo empiezan por "ftp" o terminan con la extensión del tipo de archivo, como "au" o "mpeg". Estos enlaces pueden transmitir software, sonido o animaciones y pueden iniciar aplicaciones auxiliares que admitan los archivos.

Sugerencia: para establecer una imagen como tapiz en el escritorio de Windows, haga clic con el botón derecho sobre la imagen y elija Definir como papel tapiz en el menú emergente.

En la pantalla de vista preliminar, puede hacer lo siguiente con las páginas que desee imprimir:

  • Cambiar la escala (el tamaño) de las páginas: En el campo Escala, indique un porcentaje del tamaño original y pulse Entrar (Retorno en Mac OS). Por ejemplo escriba 50 y pulse Entrar para que el tamaño de las páginas se reduzca a la mitad. Haga clic en el botón Arriba o Abajo pata aumentar o reducir la escala, respectivamente.
  • Cambiar la orientación de la página: Elija Vertical para colocar la página en su posición normal, con el lado más corto hacia arriba. Elija Horizontal para colocar la página de lado, con el lado más largo hacia arriba.
  • Ir a Configuración de página: haga clic en Configuración de página para personalizar más las páginas que vaya a imprimir.
  • Ir a Imprimir: Haga clic en Imprimir para imprimir las páginas.

Configuración de la página

Nota: algunas funciones de vista configuración de página son diferentes (o inexistentes) para Mac OS y Linux.

Para personalizar la impresión de páginas en Netscape, vaya a Configuración de página:

  • Desde Navigator, abra el menú Archivo y seleccione Configurar página.

En la pantalla Configuración de página, puede hacer lo siguiente para las páginas que desee imprimir:

  • Formato & Opciones: elija la orientación, la escala y demás opciones:
    • Orientación:
      • Vertical: Elija esto para colocar la página en su posición normal, con el lado más corto hacia arriba.
      • Horizontal: Elija esto para colocar la página de lado, con el lado más largo hacia arriba.
    • Escala: escriba un porcentaje del tamaño original. Por ejemplo, escriba 50 para que el tamaño de las páginas se reduzca a la mitad.
      • Reducir hasta ajustar ancho de página: seleccione esta opción para ajustar automáticamente el tamaño de la página al ancho del papel.
    • Opciones:
      • Imprimir fondo (colores e imágenes): seleccione esta opción para imprimir las imágenes y los colores de fondo. Si no está seleccionada, sólo se imprimirán las imágenes y los colores en primer plano.
  • Márgenes & encabezado/pie: haga clic en esta pestaña para configurar los márgenes, los encabezados y los pies:
    • Márgenes:
      • Superior, inferior, izquierdo, derecho: indique la medida en pulgadas para los márgenes superior, inferior, izquierdo y derecho.
    • Encabezados y pies: cada una de las listas desplegables representa un encabezado o un pie. La fila superior de listas desplegables son para los encabezados izquierdo, central y derecho. La fila superior corresponde a los pies izquierdo, central y derecho. En cada lista desplegable, elija una de las siguientes opciones:
      • --en blanco--: en esta zona no se verá nada.
      • Título: se verá el título de la página web.
      • URL: se verá la URL de la página web (las URL suelen empezar por "http://").
      • Fecha/hora: Se verá la fecha y la hora de impresión de la página web.
      • Página n.º: mostrará el número de página de cada página. Por ejemplo,
      • Página # de #: mostrará el número de página junto al número total de páginas. Si imprime un sitio web con cinco páginas, por ejemplo, en la tercera página se verá "3 de 5".
      • Personalizar: escriba su propio texto. Indique uno de los siguientes códigos si desea introducir información específica:
        • &PT: número de página con el total (por ejemplo, "3 de 5")
        • &P: número de página
        • &D: fecha
        • &U: URL
        • &T: título de la página

Sugerencia: para obtener una vista preliminar de los cambios efectuados en Configuración de página, vaya a Imprimir Vista preliminar.

Cómo se elige una fuente o una codificación de caracteres

Si navega, redacta o envía y recibe mensajes electrónicos en más de un idioma, es posible que necesite seleccionar las fuentes y las codificaciones de caracteres apropiadas.

Un método de codificación de caracteres es la forma en que se ha convertido el documento o mensaje en datos que el ordenador puede utilizar. Todos los documentos web y los mensajes de correo o noticias utilizan un método de codificación de caracteres (también llamado codificación de caracteres o juego de caracteres).

El método de codificación de caracteres de un documento depende del idioma. Algunos idiomas, como la mayoría de las lenguas occidentales, comparten el mismo método de codificación. Otros, como el chino, el japonés y el ruso, utilizan métodos diferentes.

Su versión de Netscape tiene configurada de forma predeterminada la codificación de caracteres propia de su zona. Sin embargo, si utiliza más de un idioma, es posible que tenga que seleccionar los métodos adecuados de codificación de caracteres y designar las fuentes que desea utilizar para su método de codificación.

Para seleccionar una codificación de caracteres, comience en la ventana Navigator:

  1. Abra el menú Ver, seleccione Codificación de caracteres y, a continuación, seleccione Más.
  2. Elija una zona en la parte superior del submenú.
  3. Elija una codificación de caracteres dentro del submenú de la zona. Repita los pasos del uno al tres por cada método de codificación de caracteres que desee.

Los métodos de codificación de caracteres que ha seleccionado se agregan al menú Codificación de caracteres. Si ha seleccionado más de un método de codificación, el que esté activo estará marcado con un punto.

Netscape detecta la codificación de caracteres utiliza un documento y lo muestra correctamente en la pantalla. Para activar esta función, comience en la ventana Navigator:

  1. Abra el menú Ver, seleccione Codificación de caracteres y Detección automática.
  2. En el submenú, elija una de las opciones de Detección automática o Detección automática desactivada.

Para realizar cambios en la lista de los juegos de caracteres activos:

  1. Abra el menú Ver y elija Codificación de caracteres; seleccione Personalizar. Aparece el cuadro de diálogo para personalizar la codificación de caracteres.
  2. Elija entre los siguientes procedimientos:
    • Para ampliar la lista de juegos de caracteres activos, elija un juego de caracteres de la lista de la izquierda y haga clic en Agregar.
    • Para eliminar un juego de caracteres de la lista activa, elija un juego de caracteres en la lista situada a la derecha y haga clic en Eliminar.
    • Para cambiar el orden en que desea que su navegador verifique los juegos de caracteres, resalte el juego de caracteres de la lista de la derecha y utilice los botones con flechas para moverlos arriba o abajo en la lista.

Para cambiar las fuentes predeterminadas dentro de un grupo de idioma:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, elija Fuentes. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)
  3. En la lista desplegable Fuentes para, seleccione método de codificación de caracteres. Por ejemplo, para predeterminar un tipo de letra con el juego de caracteres occidentales, elija Occidental.
  4. Decida si el texto proporcional debe ser serif (como Times Roman) o sans-serif (como Arial). También puede especificar qué tipo de tamaño de fuente desea para el texto proporcional. El texto proporcional varía en anchura.
  5. (Si está disponible.) Seleccione una fuente para Serif, Sans-Serif, Itálica (cursiva) y Fantasía.
  6. Seleccione la fuente monospace (como Courier) que desea utilizar para las páginas web. El texto monospace es fijo en anchura; por tanto, cada carácter o letra ocupa el mismo espacio.

Muchos creadores de páginas web eligen la fuente y el tamaño. Para usar la configuración de fuentes del autor, seleccione Permitir que los documentos utilicen otras fuentes.

Para ajustar la legibilidad de las fuentes, seleccione los puntos por pulgada (dpi) para la visualización de páginas web en la lista desplegable. Seleccione Otra para abrir el cuadro de diálogo Calibrar resolución; la resolución se calibra estableciendo la longitud que aparentan las líneas en pantalla.

Cómo se configuran las preferencias de idiomas y contenidos

El idioma que utilice para Netscape afecta al texto de los botones, los cuadros de diálogo, los menús, las herramientas y demás elementos. Las funciones de los elementos no cambian.

Para establecer el idioma preferido:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, haga clic en Idiomas/Contenido. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)
  3. En Paquetes de contenido instalados, seleccione un paquete de contenido.
  4. Haga clic en Aceptar.

    Nota: después de cambiar el idioma preferido, debe reiniciar Netscape.

El paquete de contenido que utiliza afecta a la página de inicio, los marcadores, los contenidos de las barras de herramientas, Mi Barra lateral y demás elementos.

Para utilizar un paquete de contenido:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Apariencia, haga clic en Idiomas/Contenido. (Si no aparece ninguna subcategoría, haga doble clic en Apariencia para expandir la lista.)
  3. En Paquetes de contenido instalados, seleccione un paquete de contenido.
  4. Haga clic en Aceptar.

    Nota: tendrá que reiniciar Netscape después de cambiar a otro paquete de contenido.

Sugerencia: para descargar más paquetes de contenido para Netscape, haga clic en Descargar más.

A veces, algunas páginas web están disponibles en varios idiomas. Netscape presenta las páginas en el idioma que usted prefiera, si es posible. Puede especificar el idioma que desea ver por orden de preferencia.

Para fijar las prioridades de idioma:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Abra la categoría Navigator y seleccione Idiomas. (Si no aparece ninguna subcategoría, haga doble clic en Navigator para expandir la lista.)
  3. Haga clic en Agregar. Aparecerá el cuadro de diálogo Agregar idiomas.
  4. Seleccione un idioma y su código. Si desea agregar un código que no esté en la lista, escriba el idioma y su código en el campo Otros (véase la nota a continuación). Si desea reorganizar los idiomas de la lista, seleccione un idioma y haga clic en Trasladar arriba y Trasladar abajo.
  5. Haga clic en Aceptar.

Nota: entre paréntesis, al lado de cada idioma de la lista, se indican los códigos de dos letras. Se trata de abreviaciones normalizadas para los idiomas. Si desea ver una lista completa, consulte en Internet el documento Codes for the Representation of Names of Languages (Códigos de representación de nombres de idiomas).

Cómo se gestionan tipos de archivos diferentes

Navigator puede trabajar con muchos tipos de archivos. Sin embargo, para gestionar algunos archivos como los de películas o música, Navigator necesita conectores o aplicaciones auxiliares. Si Netscape no tiene las aplicaciones auxiliares o los conectores necesarios, puede guardar el archivo en el disco duro. Efectúe un seguimiento de los archivos que guarde con el Gestor de descargas.

 

Conectores

Los conectores son aplicaciones auxiliares que amplían las funciones de Navigator y que funcionan dentro de Netscape. Los conectores como Sun Java, Macromedia Flash y RealNetworks RealPlayer permiten a Netscape mostrar archivos multimedia y ejecutar pequeñas aplicaciones, como películas, animaciones y juegos.

Al instalar Netscape, se le brindó la oportunidad de agregar algunos de los conectores más conocidos. No hace falta que los instale por separado. No obstante, también se pueden instalar conectores después de instalar Netscape. Para ver los conectores de Netscape instalados, visite la página web Netscape Plug-in Manager (Gestor de conectores de Netscape).

Para consultar una lista completa de los conectores de Netscape que puede instalar, vea la página Browser Plug-ins (Conectores del navegador) de Netscape.

 

Aplicaciones auxiliares

Cuando los archivos no pueden abrirse dentro de Netscape, puede iniciar aplicaciones auxiliares que se abren fuera de Netscape. Por ejemplo, para reproducir archivos MP3, puede abrir programas como Winamp desde fuera de Netscape.

La configuración del panel de preferencias de aplicaciones auxiliares determina qué aplicaciones de este tipo se utilizarán en su equipo. Normalmente, no hace falta cambiar estas preferencias, ya que están preconfiguradas. Además, si Navigator no sabe qué aplicación auxiliar debe utilizar, le permite buscar una en el cuadro de diálogo de descarga.

El cuadro de diálogo de descarga tiene las siguientes opciones:

  • Abrir utilizando: Netscape abre el archivo utilizando el programa predeterminado en la configuración del sistema.
  • Elegir: Netscape le permite abrir el archivo mediante la aplicación que seleccione.
  • Guardar este archivo a disco: Netscape le permite seleccionar una ubicación para guardar el archivo.
  • Preguntar siempre antes de abrir este tipo de archivos: Si selecciona esta opción, Netscape abre el cuadro de diálogo de descarga al descargar un tipo concreto de archivos. Si desea restablecer estas opciones más tarde, vaya a las preferencias de las aplicaciones auxiliares.
  • Avanzadas: Netscape le permite definir el manejo de los archivos. Si desea más información, consulte Cómo se agrega una aplicación nueva a la lista.

Sugerencia: Se pueden eliminar las opciones guardadas para Preguntar siempre antes de abrir este tipo de archivos. Si desea más información, consulte Preferencias de Navigator: aplicaciones auxiliares.

Para los usuarios experimentados que deseen definir las preferencias de aplicaciones:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Navigator, haga clic en Aplicaciones auxiliares. (Si no aparece ninguna subcategoría, haga doble clic en Navigator para expandir la lista.)
  3. Lleve a cabo cualquiera de los procedimientos siguientes:

    Para agregar una nueva aplicación auxiliar a la lista:

    1. Haga clic en Nuevo tipo. Aparecerá el cuadro de diálogo Nuevo tipo.
    2. Escriba una descripción del tipo de archivos con el que trabaja la aplicación; por ejemplo, "películas" o "archivos de sonido".
    3. Escriba la extensión del archivo (sufijo) que utiliza este tipo de archivo. Si este tipo de archivo utiliza más de una extensión, escríbalas todos, separadas por espacios.
    4. Escriba el tipo MIME. Los tipos MIME constan de dos palabras separadas por una barra. Por ejemplo, un tipo MIME de un archivo de imagen TIFF es "image/tiff".
      Nota: al descargar un archivo, Netscape mostrará el tipo MIME en el cuadro de diálogo de descarga.
    5. Haga clic en Examinar y después seleccione una aplicación que gestione archivos de este tipo.
    6. Haga clic en Aceptar.

    Para especificar el modo en que Navigator gestionará los archivos descargados:

    1. Seleccione un tipo de archivo de la lista y haga clic en Edición.
    2. Escriba el tipo MIME que desea que use este archivo.
    3. Elija la opción Manipulado por para tener los archivos descargados guardados en su disco duro o abiertos en una aplicación que haya especificado. Haga clic en Examinar para especificar la aplicación.
    4. Haga clic en Aceptar.

    Para eliminar un elemento de la lista:

    1. Seleccione un tipo de archivo en la lista y haga clic en Eliminar.
    2. Haga clic en Aceptar.

Si desea más información sobre las preferencias de las aplicaciones auxiliares, consulte Preferencias de Navigator: aplicaciones auxiliares.

Cómo se configura Netscape como navegador predeterminado

Nota: en esta sección se comentan valores y funciones exclusivos de Windows.

Netscape es conocido, sobre todo, por mostrar páginas web, tanto en Internet como en su equipo. Para abrir las páginas web fácilmente, establezca Netscape como su navegador predeterminado.

 

Archivos y protocolos comunes de Internet

Al establecer Netscape como navegador predeterminado, le permitirá abrir automáticamente formatos de archivo y protocolos de uso común en Internet. Entre los formatos de archivo y los protocolos comunes de Internet, están los siguientes:

  • Archivos de imagen:

    • JPEG, GIF, PNG, y MNG

  • Archivos de documentos y lenguajes de Internet:

    • HTML, XML, XHTML y XUL

  • Protocolos de Internet:

    • HTTP, HTTPS, FTP, Chrome, Gopher

 

Cómo se cambia automáticamente la configuración del navegador predeterminado

Al terminar la instalación, Netscape comprueba si es el navegador predeterminado para algún formato de archivo o protocolo común de Internet. Si no es así, el programa le informará de que Netscape no es el navegador predeterminado y le preguntará si desea que lo sea a partir de ese momento.

Haga clic en Sí para que Netscape sea el navegador predeterminado. Si hace clic en No, se le hará la misma pregunta cada vez que se abra Netscape, a no ser que desactive la opción La próxima vez, realice la comprobación durante el inicio también.

Si desactiva dicha opción, aún puede establecer Netscape como navegador predeterminado; para ello, cambie la configuración en Preferencias. Si desea a aprender a establecer Netscape como su navegador predeterminado en las preferencias de Netscape, consulte Preferencias de Navigator: Navigator.

Cómo se personaliza la configuración del navegador predeterminado

Es posible personalizar el modo en que Netscape manipula los distintos formatos de archivos y los protocolos. Si desea más información, consulte Preferencias avanzadas: sistema.

Cómo se automatiza la carga de páginas

Cuando abre una página web en la pantalla, Netscape carga (inicia) automáticamente varios elementos que ayudan a interpretar las páginas web. Estos elementos, Java y JavaScript, hacen que las páginas web sean más atractivas, pero tardan bastante en cargarse.

Si desea aprender a desactivar Java, consulte Preferencias avanzadas: avanzadas.

Si desea aprender a desactivar JavaScript, consulte Preferencias avanzadas: secuencias de comandos y Windows.

Cómo se modifica la configuración del caché

El sistema almacena copias de las páginas a las que se accede con más frecuencia en el caché de memoria o en el caché de disco. De este modo, el sistema no tiene que recuperar la página de la red cada vez que quiere verla.

Para establecer el tamaño del caché de memoria o para vaciarlo (sólo Windows y Unix)

  1. Abra el menú Editar y haga clic en Preferencias.
  2. En la categoría Avanzadas, haga clic en Caché. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)
  3. Indique un número en el campo Caché de memoria para especificar el tamaño del caché de memoria. Entre 1.024 kB y 2.000 kB se considera un tamaño adecuado. Para vaciar el caché de memoria inmediatamente, haga clic en Vaciar caché de memoria.

Importante: un caché de memoria superior permite recuperar con rapidez una mayor cantidad de datos. Sin embargo, si no dispone de una gran cantidad de memoria de sobra en el sistema, 1.024 kB es un tamaño suficientemente grande.

Para establecer el tamaño del caché de disco o para vaciarlo:

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Avanzadas, haga clic en Caché. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)
  3. Introduzca un número en el campo Caché de disco para especificar el tamaño del caché de disco. 8.000 kB en Windows es suficiente (5.000 kB en Unix). Para vaciar el caché de disco inmediatamente, haga clic en Vaciar caché de disco.

Importante: un caché de disco superior permite recuperar con rapidez más páginas, pero se utiliza más espacio de disco duro.

Al salir de Netscape, éste realiza el mantenimiento del caché. Si el mantenimiento tarda más de lo deseado, pruebe a reducir el tamaño del caché de disco.

Para especificar la frecuencia con la que Navigator debe comprobar en la red si hay revisiones de las páginas (de modo que no conserve páginas obsoletas en el caché demasiado tiempo):

  1. Abra el menú Editar y seleccione Preferencias.
  2. En la categoría Avanzadas, haga clic en Caché. (Si no aparece ninguna subcategoría, haga doble clic en Avanzadas para expandir la lista.)
  3. Elija entre las siguientes opciones:
  • Cada vez que veo la página: seleccione esta opción si desea que Netscape compare una página web con la caché cada vez que la vea.
  • Cuando caduca la página: Seleccione esta opción si desea que Netscape compare una página web con la caché cuando el servidor determine que la página ha caducado.
  • Una vez por sesión: seleccione esta opción si desea que Netscape compare una página web con la caché una vez cada vez que inicie Netscape.
  • Nunca: Seleccione esta opción si no desea que Netscape compare la información de la caché con la red.

Si las páginas que deberían estar en el caché tardan demasiado tiempo en aparecer, asegúrese de que la preferencia escogida no es Cada vez que veo la página, porque la comprobación requiere una conexión de red que necesita tiempo.

Para actualizar una página en cualquier momento:

  • Haga clic en el botón Volver a cargar de la barra de navegación del Navigator. El sistema comprueba la red para asegurarse de que tiene la versión más reciente de la página.

Cómo se activa Quick Launch

Nota: Quick Launch sólo está disponible en las instalaciones de Netscape en sistemas Windows.

Quick Launch acorta el tiempo necesario para que inicie Netscape. Cuando Netscape utiliza Quick Launch, el icono del logotipo de Netscape está presente en la barra de tareas (junto al reloj).
Icono Quick Launch de Netscape

Sugerencia: Para iniciar rápidamente componentes (como Composer, correo y grupos de noticias y el Libro de direcciones), haga clic con el botón derecho en el icono del logotipo de Netscape y seleccione el componente del menú emergente.

Al instalar Netscape, se le ofreció la oportunidad de activar o desactivar Quick Launch. Si está activado, Quick Launch carga parte de Netscape en la memoria al iniciar Windows (y cada vez que se abre Netscape). Quick Launch permite que Netscape se quede en la memoria después de cerrar todas las ventanas de Netscape. De esta forma, Netscape se abre rápidamente cuando lo necesita, sin tener que cargar Netscape por completo. Si su equipo no dispone de mucha memoria, puede desactivar Quick Launch para ahorrar memoria.


Para activar o desactivar Quick Launch en Netscape:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Haga clic en la categoría Avanzadas.
  3. Seleccione Activar Habilitar Quick Launch para activarlo, o desactive la casilla para desactivarlo.
  4. Haga clic en Aceptar.

    Sugerencia: Para desactivar Quick Launch rápidamente, haga clic con el botón derecho en el logotipo de Netscape de la bandeja del sistema (junto al reloj) y seleccione Desactivar Quick Launch en el menú emergente.

Cómo se configuran los servidores proxy

Muchas empresas bloquean el acceso desde Internet a sus redes. Esto impide que las personas ajenas a la empresa obtengan acceso a información confidencial. Esta protección recibe el nombre de cortafuegos.

Si su empresa dispone de un cortafuegos, es posible que el navegador tenga que pasar por un servidor proxy antes de conectarse a Internet. El servidor proxy evita que los intrusos se introduzcan en la red privada de una empresa.

Antes de empezar:

  • Si existe un archivo de configuración para el servidor proxy en su lugar de trabajo, pida al administrador del sistema que le facilite el URL del archivo.
  • Si no existe ningún archivo de configuración para el servidor proxy, pida al administrador del sistema los nombres y números de puerto de los servidores que ejecutan software de proxy para cada servicio de la red.

Para configurar el navegador para que funcione con el servidor proxy:

  1. Abra el menú Editar y seleccione Preferencias.
  2. Abra la categoría Avanzadas y haga clic en Proxy. (Si no aparece ninguna subcategoría, haga doble clic en Navigator para expandir la lista.)
  3. Seleccione una de las opciones siguientes:
    • Conexión directa con Internet: seleccione esta opción si no desea utilizar un servidor proxy.
    • Configuración manual del servidor proxy: seleccione esta opción si no dispone de una ubicación de proxy (URL).
      • Proxy HTTP, Proxy SSL, Proxy FTP, Proxy de Gopher: escriba el nombre o la dirección IP numérica del servidor proxy. Indique el puerto en el campo Puerto.
      • Sistema SOCKS: escriba el nombre o la dirección IP numérica del servidor proxy. Indique el puerto en el campo Puerto.
        • SOCKS v4, SOCKS v5: cuando indique un sistema SOCKS, seleccione SOCKS v4 o SOCKS v5, según la versión de SOCKS que se use para el proxy.
      • No hay proxy para: escriba los dominios para los que no desea usar proxy. Separe cada dominio con una coma. (Ejemplo: suempresa.com, .suempresa.es.)
    • URL de configuración automática del servidor proxy: seleccione esta opción si dispone de una URL o un archivo de configuración de servidor proxy; a continuación escriba la URL de configuración.

Los nombres del dominio son la parte de un URL que contiene el nombre de una organización, empresa o institución docente, por ejemplo, netscape.com o washington.org. Si utiliza nombres de sistemas principales locales sin el nombre de dominio, especifíquelos de la misma manera. Utilice comas para separar varios nombres de sistemas principales. No se admite el carácter comodín [*].

Si desea más información sobre el panel de preferencias del proxy, consulte Preferencias avanzadas: proxies.
