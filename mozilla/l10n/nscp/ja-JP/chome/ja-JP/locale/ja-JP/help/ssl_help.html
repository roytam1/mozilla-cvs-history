SSL プロトコルのバージョン

SSL（Secure Socket Layers）プロトコルは、Web サイトとブラウザ ソフトウェアの間で、共通の認証設定を指定し、またその両者の間で伝送される情報の暗号化を定める規則です。TLS（Transport Layer Security）プロトコルは、SSL に基づく IETF 基準です。TLS 1.0 は SSL 3.1 とみなすことができます。

古い Web サーバでも新しい Web サーバでも機能するように、以下の 3 つのチェックボックスは通常オンにしておきます。

  • SSL バージョン 2 をオンにする： 古い Web サーバに対応できます。
  • SSL バージョン 3 をオンにする： 新しい Web サーバに対応できます。
  • TLS をオンにする： TLS をサポートしている Web サーバ用のオプションです。

個々の暗号法をオンまたはオフにするには、［暗号法を編集...］ボタンをクリックします。

  • 暗号法を編集： 暗号法の編集について正しい知識があり、システム管理者から編集の承認をもらっている場合以外は、暗号法を編集しないでください。 詳しくは、「暗号法を編集する」を参照してください。

TLS に関する重要事項： SSL を正しく実装していないサーバは、TLS をサポートしているクライアント ソフトウェア（ブラウザなど）と SSL ハンドシェークを行うことができません。このようなサーバを「TLS 不寛容」と言います。

［SSL］設定パネルで［TLS をオンにする］オプションを選択すると、ブラウザは、サーバと保護付き接続をしているときに、TLS プロトコルを使おうとします。サーバが TLS に対して不寛容なために保護付き接続を確立できない場合は、SSL 3.0 が使用されます。

 

SSL 警告

アクセスしている Web サイトへの接続が暗号化されているかどうかは一目でわかります。接続が暗号化されていると、［Navigator］ ウィンドウの右下にあるロック アイコンが閉じています。接続が暗号化されていない場合は、ロック アイコンが開いています。

さらに警告が必要な場合は、［SSL］設定パネルで、1つまたは複数の警告チェックボックスを選択できます。 ただし、ユーザによっては、表示される警告の数が多いと、嫌がる人もいます。

Navigator の警告をオンにするには、対応するチェックボックスを選択します。

  • 暗号化をサポートするページを読み込むとき：暗号化をサポートするページを読み込むたびに警告を表示したい場合は、このオプションを選択します。
  • 軽度の暗号化を使用しているページを読み込むとき：軽度の暗号化をサポートするページを読み込むたびに警告を表示したい場合は、このオプションを選択します（軽度の暗号化は、40 ビット キーを使った最も低いレベルの暗号化です）。
  • 暗号化をサポートしているページから離れるとき：暗号化をサポートするページから暗号化をサポートしないページを表示するたびに警告を表示したい場合は、このオプションを選択します。
  • 暗号化されていないページ間でフォーム データを送信するとき：暗号化されていない接続でデータを送信するたびに警告を表示したい場合は、このオプションを選択します。 暗号化されていない情報をインターネット上で送信すると、他の人に傍受される確率が高くなります。
  • 暗号化／非暗号化が混在したページを表示するとき：暗号化されていない情報を含むページにアクセスするたびに警告を表示したい場合は、このオプションを選択します。

簡単な定義については、認証設定、暗号化、または証明書を参照してください。

暗号法を編集する

このセクションでは、［SSL：暗号化の編集］ダイアログ ボックスの使い方について説明します。 ダイアログ ボックスを表示するには、以下の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。
  2. ［プライバシーとセキュリティ］カテゴリの［SSL］をクリックします（サブカテゴリが表示されていない場合は、［プライバシーとセキュリティ］カテゴリをダブルクリックすしてリストを展開表示します）。
  3. ［暗号法を編集］ボタンをクリックします。

［SSL：暗号化の編集］ダイアログ ボックスでは、暗号法（暗号化アルゴリズム）のオンとオフを切り替えることができます。 これらの設定を変更する前に、必ずシステム管理者に問い合わせください。

各チェックボックスは、それぞれ異なる暗号法（暗号化アルゴリズム）に対応しています。

暗号法および暗号化について詳しくは、以下のオンライン マニュアルを参照してください。

  • Introduction to Public-Key Cryptography（公開キー暗号法の概要）
  • Introduction to SSL（SSL の概要）
  • Encryption Technologies Available in NSS 3.4（NSS 3.4 で使用可能な暗号化技術）
