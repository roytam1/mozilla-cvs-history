Netscape のキーボード ショートカット

Netscape では、マウスの代わりにキーボードを使用してコマンドを実行できます。 キーボード ショートカットを使用してコマンドを実行すれば、時間を節約できます。さらに、この機能は、身体障害、視覚障害、および聴覚障害のある方が Netscape を使用する上で非常に役立ちます。

ショートカットを使用する

このドキュメントでは、以下の形式でショートカットを表記します。

例：

コマンド

Mac

コピー
Ctrl+C

コマンドを実行するには、示されているボタンを同時に押します。 たとえば、Windows でコピーを実行するには、Ctrl キーと C キーを同時に押します。

ヒント： プラス記号（+）が表示されている場合は、その前後のキーを同時に押す必要があります。 複数のプラス記号がある場合（例：Ctrl+Shift+C）も、その前後のキーをすべて同時に押す必要があります。

略語：

  • Ctrl = Control キー
  • Cmd = Macintosh キーボードの Command キー

注記： 一部のキーボード ショートカットは、カーソルの位置（フォーカス）に応じて実行される機能が異なります。 たとえば、Web ページを表示しているときに Windows マシンの Home キーを押すと、Netscape はそのページのトップに移動しますが、カーソルがテキスト フィールドにあるときに Windows マシンの Home キーを押すと、カーソルはテキスト フィールドの最初に移動します。

ヒント： Windows や Linux では、メニュー オプションへアクセスするのにもキーボードを使用できます。 各メニューを開くには、Alt キーを押したままメニュー名の末尾にある下線の付いた文字キーを押します。 メニューが開いたら、Alt キーを押さずにサブメニューを開いたり、コマンドを実行することができます。 たとえば、Alt+F キーを押すと［ファイル］メニューが開きます。
