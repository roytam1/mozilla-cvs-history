Netscape で ICQ を使用する

Netscape のインスタント メッセージ クライアントを使って、AIM および ICQ のメッセージを送受信できます。 ただし、ICQ メッセージを送受信するには、ICQ にログインする必要があります。AIM と ICQ を同時に使用することはできません。 また、ICQ アカウントを持っていること、および ICQ 連絡先リストを作成することが必要です。 AIM の友だちが ICQ アカウントを持っていれば、その人を ICQ 連絡先にすることは可能ですが、その設定が自動的に行われることはありません。

  1. 電子メール アドレスを入力して［Submit］（送信）をクリックします。

システム起動時に ICQ を起動する

Windows コンピュータの場合は、コンピュータの起動時に Netscape ブラウザを起動せずに ICQ だけが自動的に起動されるように設定できます。 ただし、ICQ 内でこのオプションを設定することはできません。AOL Instant Messenger for Netscape にログオンしている必要があります。 Sidebar ではなく、［ウィンドウ］メニューから選択できるメインの ICQ/AIM ウィンドウを開きます。

  1. ICQ にログインしている場合は、［ログオフ］ボタン をクリックします。
  2. ［サインオンするサービスを選択してください］ドロップダウン リストから［AIM］を選択します。
  3. AIM のスクリーン ネームとパスワードを入力し、［サインオン］ボタン をクリックします。
  4. ［編集］メニューから［設定］を選択します。
  5. ［Instant Messenger］をクリックします。
  6. ［システムの起動時に自動的に起動する］チェックボックスをオンにします。
  7. ［OK］をクリックします。
  8. ［AIM サインオフ］ボタン をクリックします。
  9. ［サインオンするサービスを選択してください］ドロップダウン リストから［ICQ］を選択します。
  10. ICQ 番号または表示名と、パスワードを入力して、［ログイン］ボタン をクリックします。

これで、コンピュータを起動するたびに ICQ が自動的に起動されるようになります。ただし、［ICQ］ウィンドウで［起動時に接続］チェックボックスをオンにしていないと自動ログオンは実行されません。

注記：このオプションは、Windows コンピュータでしか使用できません。また、スタンドアロンの ICQ クライアントまたは AOL Instant Messenger がシステムの起動時に起動するように設定されていると、このオプションは使用できません。

ICQ 連絡先を追加する

ICQ 連絡先を追加するには、以下の手順で操作します。

  1. ［連絡先リスト］タブをクリックして前面に表示します。
  2. ［連絡先を追加］ボタン をクリックします。［連絡先を追加］ダイアログ ボックスが表示されます。
  3. 連絡先の ICQ 番号および表示名（任意の名前を付けられます）を入力し、連絡先を割り当てるグループのチェックボックスをオンにして［OK］をクリックします。

1 つの連絡先は 1 つのグループにのみ追加できます。

連絡先であるユーザが許可要求メッセージを要求するように設定している場合は、許可要求メッセージが自動的に送信されます。 連絡先から許可要求メッセージへの応答が返ってくるまで、その表示名は［許可待ち］フォルダ内に入れられます。 許可要求が受け入れられると、その連絡先が連絡先リストに追加されます。 許可要求が拒否されると、その連絡先の表示名は［許可待ち］フォルダ内に残ります。 許可を得る必要がない場合は、メッセージは送信されず、表示名がすぐに連絡先リストに追加されます。

ICQ メッセージを送信する

相手が Netscape 内で ICQ を使用しているか、スタンドアロンの ICQ クライアントを使用しているかにかかわらず、すべての ICQ メンバーに ICQ メッセージを送信できます。

  1. ICQ が開いていない場合は、Sidebar から友だちリストを開くか、［ウィンドウ］メニューから［Instant Messenger］を選択します。
  2. 必要に応じて、［サインオンするサービスを選択してください］ドロップダウン リストから［ICQ］を選択します。
  3. 必要に応じて、［オンライン モード］タブをクリックして前面に表示します。
  4. 通信する ICQ 連絡先の番号または表示名をダブルクリックします。
  5. 表示されるメッセージ ウィンドウにメッセージを入力し、Enter キーまたは Return キーを押します。
  6. 返信を受け取ったら、メッセージ ウィンドウの下側のパネルに応答を入力し、Enter キーまたは Return キーを押します。
  7. 会話が終了するまで手順 5 と 6 を繰り返します。

連絡先リストに載っていない ICQ メンバーに対しては、上記の手順 1 ～ 3 を実行してから、［オンライン モード］の［送信］ボタン をクリックすることによって空白のメッセージ ウィンドウを開きます。 次に、相手の ICQ 表示名または番号を入力して、上記の手順 5 ～ 7 を実行します。

ICQ メッセージを受信する

ICQ メッセージは AIM メッセージと同じ方法で受信できますが、ICQ の場合はサービスにログオンしていなければなりません。 オフライン状態では、ICQ サービスを通じてメッセージを受信することはできません。 メッセージが着信すると、ウィンドウが開いてメッセージの内容が表示されます。応答するには、以下の手順で操作します。

  1. メッセージ ウィンドウの下側のパネルに応答を入力し、Enter キーまたは Return キーを押します。
  2. それに対する相手の応答を受け取ったら、手順 1 を繰り返します。
