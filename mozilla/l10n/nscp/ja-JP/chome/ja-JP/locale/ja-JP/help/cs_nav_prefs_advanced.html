詳細設定 - 詳細

このセクションでは、メインの［詳細］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリを選択します。

メインの［詳細］設定パネルでは、Java、FTP 用パスワード、簡易起動に関するオプションを設定できます。

  • Java を有効にする：Java を含むページを Netscape が解釈できるようにするには、このチェックボックスをオンにします。 Java は、Web ページ上でインタラクティブなマルチメディア コンテンツを作成するために使用されます。
  • XSLT を使用する： Netscape で XSLT を使用するには、これを選択します。 XSLT（eXtensible Stylesheet Language/Transform）は、XML ドキュメントを他の XML ドキュメントに変換するために使用する言語です。 XSLT について詳しくは、オンライン ドキュメント「Transforming XML: Netscape and XSLT（XML の変換：Netscape とXSLT）」を参照してください。
  • 匿名 FTP パスワードとしてこの電子メール アドレスを送信する： FTP サイトに匿名でログインする際にパスワードとして使用する電子メール アドレスを入力します。
  • 簡易起動を有効にする： このチェックボックスをオンにすると、Netscape の起動時間を短縮できます。 簡易起動について詳しくは、「簡易起動を使用する」を参照してください。

    注記： 簡易起動のオプションは、Windows ベースのプラットフォームでのみ表示されます。

 

詳細設定 - スクリプトとプラグイン

このセクションでは、［スクリプトとプラグイン］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［スクリプトとプラグイン］を選択します（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。


［スクリプトとプラグイン］設定パネルでは、Netcape における JavaScript とプラグインの使用について設定できます。

  • JavaScript を有効にする： JavaScript を有効にするには、これらのチェックボックスをオンにします。
    • Navigator： Navigator で開く Web ページに対して、JavaScript のオン・オフを切り替えます。
    • Mail & Newsgroups： Mail & Newsgroups で開く Web ページに対して、JavaScript のオン・オフを切り替えます。
  • Web ページを許可する： 以下のチェックボックスを選択して JavaScript の使用方法を設定します。
    • リンクを新しいウィンドウで開く： クリックしたリンクを新しいウィンドウを使って開きます。 リンクによっては、同じウィンドウでなく別の新しいウィンドウで開くように設定されているものもあります。 このオプションを有効にするには、Netscape を再起動する必要があります。
    • 既存のウィンドウを移動またはサイズ変更する： 開いているウィンドウのサイズ変更や移動を可能にします。
    • ウィンドウの位置を上げる／下げる： ウィンドウを他のウィンドウの前面または背面に動かすことができるようになります。
    • ステータス バーを隠す： ステータスバーを非表示にできます。
    • ステータス バーのテキストを変更する： ステータスバー内のスクロール テキストなど、ステータス バーのテキストを変更できます。
    • 画像を変更する： 画像のロールオーバー（マウス カーソルを上に置くと変化する画像）など、画像の変更やアニメ化が可能になります。
    • Cookie を作成または変更する： JavaScript を使って Cookie を変更または作成できます。
    • Cookie を読む： JavaScript を使って Cookie を読み取ることが可能になります。
  • プラグインを有効にする： 以下のチェックボックスを選択してプラグインの使用方法を設定します。
    • Mail & Newsgroups： Mail & Newsgroups でのプラグインの使用を許可します。

プラグインについて詳しくは、「さまざまなファイル タイプを管理する」を参照してください。

 

詳細設定 - キャッシュ

このセクションでは、［キャッシュ］設定パネルの使用方法について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［キャッシュ］を選択します（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。


［キャッシュ］設定パネルでは、Netscape のメモリとディスク キャッシュを調節できます。

  • メモリ キャッシュ： Netscape に割り当てるメモリ ベースのキャッシュ量を入力します。 メモリ キャッシュはコンピュータの電源を切ったり再起動すると消去されます。
  • ディスク キャッシュ： Netscape に割り当てるディスク キャッシュ量を入力します。 ディスク キャッシュはハードディスク（ドライブ）に保存され、コンピュータの電源を切った後でも再び使用できます。
  • ディスク キャッシュ フォルダ： 現在使用されているディスク キャッシュ フォルダの場所を表示します。
    • フォルダの選択： ディスク キャッシュを保存するフォルダの場所を選択できます。
  • メモリ キャッシュをクリア： メモリ キャッシュをクリアします。
  • ディスク キャッシュをクリア： ディスク キャッシュをクリアします。
  • キャッシュにあるページとネットワーク上のページの比較：
    • ページにアクセスするたび： Web ページにアクセスするたびに、そのページとキャッシュ内のページとを比較します。
    • ページが時間切れの場合： サーバにより Web ページの期限が切れていると判断された場合に、そのページとキャッシュ内のページとを比較択します。
    • セッション毎： Netscape を起動するたびに、Web ページとキャッシュ内のページとを比較します。
    • しない： キャッシュ内のページとネットワーク上のページは比較されません。

 

詳細設定 - プロキシ

このセクションでは、［プロキシ］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［プロキシ］を選択します。（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。


［プロキシ］設定パネルでは、プロキシを使用できるように Netscape を設定できます。

はじめに： ネットワーク管理者に問い合わせてプロキシ設定ファイルがあるかどうかを確認するか、またはプロキシ名とポート番号を尋ねてください。

  • インターネットに直接接続する： プロキシを使用しない場合に選択します。
  • 手動でプロキシを設定する： プロキシの場所（URL）がない場合に選択します。
    • HTTP プロキシ、SSL プロキシ、FTP プロキシ、Gopher プロキシ： 各プロキシ サーバのサーバ名または数値による IP アドレスを入力します。 ［ポート］フィールドにはポートを入力します。
    • SOCKS ホスト： 各プロキシ サーバのサーバ名または数値による IP アドレスを入力します。 ［ポート］フィールドにはポートを入力します。
      • SOCKS v4、SOCKS v5： SOCK ホストを入力する場合は、プロキシでどのバージョンのSOCKSを使用するかによって、［SOCKS v4］または［SOCKS v5］のいずれかを選択します。
    • プロキシなし： プロキシを使用しないドメインを入力します。 複数のドメインがある場合はカンマで区切ります（例： .yourcompany.com や .yourcompany.co.nz など）。
  • 自動プロキシ構成 URL： プロキシ設定ファイルや URL がある場合に選択し、その構成 URL を入力します。

 

詳細設定 - HTTP ネットワーク

このセクションでは、［HTTP ネットワーク］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［HTTP ネットワーク］をクリックします。（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。


［HTTP ネットワーク］設定パネルでは、HTTP ベースのネットワーク設定を行います。

  • 直接接続オプション、プロキシ接続オプション： HTTP のバージョンと直接またはプロキシ接続のオプションを選択します。
    • HTTP 1.0 を使用する： このオプションは、1996 年に標準化された HTTP の最初のバージョンを使用する場合に選択します。
    • HTTP 1.1 を使用する： このオプションは、HTTP 接続の効率化やクライアント側のキャッシュ サポート強化、複数の HTTP リクエスト処理（パイプライン機能）、キャッシュの有効期限や代替に関する方針の詳細管理など、パフォーマンスが強化された新しいバージョンの HTTP を使用する場合に選択します。
    • キープアライブ機能を有効にする： 追加の HTTP リクエストを送信できるように接続を開いたままにしておきます。これによって速度が上がります。
    • パイプライン機能を有効にする： パイプライン機能を使用する場合に選択します。これにより、一度に複数の HTTP リクエストを送信し、Web ページの読み込み遅延時間を短縮することができます。
      注記： パイプライン機能を使用できるのは HTTP 1.1 だけです。

 

詳細設定 - ソフトウェアのインストール

このセクションでは、［ソフトウェアのインストール］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［ソフトウェアのインストール］を選択します。（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。


［ソフトウェアのインストール］設定パネルでは、ソフトウェアのインストールと更新通知のオプションを指定できます。

  • ソフトウェアのインストール： このチェックボックスをオンにすると、新しいソフトウェアをインストールして Netscape と共に使用する必要がある場合に通知が出されます。
  • 更新を確認： このチェックボックスをオンにすると、Netscape の新しいバージョンが入手可能になったときに通知が出されます。 Netscape のバージョンを確認する際に、ユーザの個人情報が Netscape に送信されることはありません。
    • 週 1 回： Netscape の新しいバージョンが入手可能かどうかを週 1 回自動的にチェックします。
    • 月 1 回： Netscape の新しいバージョンが入手可能かどうかを月 1 回自動的にチェックします。

 

詳細設定 - マウス ホイール

このセクションでは、［マウス ホイール］設定パネルの使い方法について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［マウス ホイール］を選択します（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。


［マウス ホイール］設定パネルでは、お使いのマウスの（マウス ボタンの間にある）マウス ホイールを Netscape でどのように使用するかを設定できます。

  • 設定キー： ドロップダウン リストから、以下の操作を実行するためにマウス ホイールと組み合わせて使用するキーを選択します。
    • ドキュメントを ｎ 行ごとにスクロールする： フィールドに入力した行数ごとにドキュメントがスクロールします。
      • システム標準を使用する： 入力した行数を無視して、システムの標準設定を使用します。
    • 1 ページ単位で上下にスクロールする： 一度に 1 ページずつドキュメントを上下にスクロールします。 この設定を使用すると、マウス ホイールを使ってページ間をすばやく移動できますが、ページ内の正確な場所に移動することが難しくなります。
    • ブラウズした履歴を前後に移動する： マウス ホイールを使って以前にブラウズしたページを前後に移動できます。
    • 文字列のサイズを拡大／縮小する： マウス ホイールを使って Web ページ内のテキストのサイズを拡大または縮小できます。 この設定を使うと、ページを読みやすくしたり、あるいはテキストの表示を画面サイズに合わせることができます。

    注記： 設定キーによって異なる機能を割り当てることができます。

 

詳細設定 - システム

このセクションでは、［システム］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［詳細］カテゴリの［システム］をクリックします（サブカテゴリが表示されていない場合は、［詳細］カテゴリをダブルクリックしてリストを展開表示します）。

注記： ［システム］設定パネルは Windows ベースのプラットフォームでのみ表示されます。


［システム］設定パネルでは、Netscape を使って開くことのできるファイルやプロトコルを指定できます。

  • Netscape を使用して開くファイルの形式： 常に Netscape を使って開くファイルの形式を選択します。
  • Netscape を使用して処理するプロトコル： 常に Netscape を使って開くプロトコルを選択します。
  • これらの設定が他のアプリケーションによって変更される場合に警告を表示する： このチェックボックスをオンにすると、他のアプリケーションによって Netscape のファイルやプロトコルの標準設定が変更された場合に、Netscape から通知が出されます。

ヒント： Netscape を簡単に標準のブラウザにする方法については、「Navigator の設定 - Navigator」を参照してください。 ［これらの設定が他のアプリケーションによって変更される場合に警告を表示する］チェックボックスをオンにしておくと、他のプログラムが Netscape の代わりに標準ブラウザになった場合にも通知が出されます。
