表示設定 - 表示

このセクションでは、［表示］設定パネルの使い方について説明します。 このパネルを表示するには、以下の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。
  2. ［表示］カテゴリをクリックします。


［表示］設定パネルでは、Netscape の起動時に表示するコンポーネントを設定できます。

  • Netscape の起動時に開く：Netscape を起動したときに表示するコンポーネントを選択します。
  • ヒントを表示：Netscape のユーザ インターフェイス上にカーソルを置いたときにヒントを表示するには、このチェックボックスをオンにします。
  • Web サイト アイコンを表示： ブックマーク アイコン ではなく、サイト固有のアイコンを表示する場合（そのようなアイコンがある場合）は、このチェックマークをオンにします。 Web サイト アイコンは、場所ツールバーの左側、ブックマーク リスト内に表示されます。

 

表示設定 - フォント

このセクションでは、［フォント］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。
  2. ［表示］カテゴリの［フォント］をクリックします（サブカテゴリが表示されていない場合は、［表示］カテゴリをダブルクリックしてリストを展開表示します）。


［フォント］設定パネルでは、フォントの種類やサイズ、画面の解像度を設定できます。

注記： フォントによっては、選択した言語にそのスタイルがないために選択できないものもあります。

  • フォントの対象言語：言語の文字セットを選択します。たとえば、標準フォントを欧米の文字セットに設定する場合には［欧米］を選択します。
    • 可変ピッチ：可変テキストをセリフ（Times Roman など）またはサンセリフ（Arial など）のどちらで表示するかを選択し、フォント サイズを指定します。プロポーショナル テキストの幅は変更できるため、文字幅も変わります。
    • セリフ：Web ページに使用するセリフ フォントを選択します。
    • サンセリフ：Web ページに使用するサンセリフ フォントを選択します。
    • 筆記体：Web ページに使用する筆記体フォントを選択します。
    • ファンタジー： Web ページに使用するファンタジー フォントを選択します。
    • モノスペース： Web ページに使用するモノスペース フォント（例：Courier）とそのサイズを選択します。モノスペース テキストは幅が一定で、すべての文字が同じ幅で表示されます。
    • フォントの最小サイズ： Web ページで表示する最小のフォントサイズを選択します。
  • ドキュメントで他のフォントを使用できるようにする：このチェックボックスをオンにすると、ユーザの設定ではなく、Web ページのフォントとサイズ設定が使用されます。
  • 解像度を表示：Web ページを表示する解像度（dpi 単位）をドロップダウン リストから選択します。［その他］を選択すると、［解像度の較正］ダイアログ ボックスが開き、画面に表示される行の長さを計ることによって解像度を較正できます。 解像度を高くすると、画面によってはテキストが読みやすくなることがあります。

 

表示設定 - 配色

このセクションでは、［配色］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。
  2. ［表示］カテゴリの［配色］をクリックします。（サブカテゴリが表示されていない場合は、［表示］カテゴリをダブルクリックしてリストを展開表示します）。


［配色］設定パネルでは、Web ページの背景やテキストの色を設定できます。

  • 文字と背景：カラー ブロックをクリックして、Web ページで表示する文字と背景の色を選択します。
    • システムの配色を使用： システムの配色設定を使用する場合は、このチェックボックスをオンにします。
  • リンクの色：カラー ブロックをクリックして、Web ページでアクセスしていないリンクとアクセスしたリンクの色を選択します。
    • リンクに下線を引く： Web ページのリンクを下線付きで表示する場合は、このチェックボックスをオンにします。
  • Web ページに独自の色と背景が指定されている場合：
    • Web ページが指定する色と背景を常に使用： Web ページで指定されている配色と背景を使用します。
    • 自分が選択した色を使用し、指定されている色と背景画像は無視する： Web ページで指定されている配色や背景画像を無視して自分が選択した色を使用できます。
  • 構文ハイライト機能を使用： Web ページのソースを表示する際に、HTML 構文を色分けして表示する場合は、このチェックボックスをオンにします。

    ヒント： Web ページのソースを表示するには、Navigator から［表示］メニューを開いて［ページ ソース］を選択します。

 

表示設定 - テーマ

このセクションでは、［テーマ］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。
  2. ［表示］カテゴリの［テーマ］をクリックします（サブカテゴリが表示されていない場合は、［表示］カテゴリをダブルクリックしてリストを展開表示します）。


［テーマ］設定パネルでは、Netscape の表示テーマを選択して、その外観やスタイルを変更できます。

  • 使用するテーマ： インストールされているテーマのいずれかを選択して、Netscape の外観を変更します。
    • ［テーマ名］をアンインストール： 選択したテーマを削除します。
  • 新しいテーマを取得： このリンクをクリックすると、別のテーマをダウンロードするための Web ページへジャンプします。

    注記：

    • 新しいテーマで表示するには、そのテーマを選択した後に、Netscape を再起動する必要があります。
    • アンインストールしたテーマを元に戻すことはできません。

 

表示設定 - 言語／コンテンツ

このセクションでは、［言語パックとコンテンツ パック］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。

［言語パックとコンテンツ パック］設定パネルでは、Netscape で使用するコンテンツ パックと言語パックを選択できます。

  • インストールされているコンテンツ パック：Netscape で使用するコンテンツ パックを選択します。
  • インストールされている言語パック： Netscape で使用する言語パックを選択します。
  • さらにダウンロード： 追加のコンテンツ パックまたは言語パックをダウンロードするには、このボタンをクリックします。

    注記： 新しいコンテンツ パックや言語パックを有効にするには、Netscape を再起動する必要があります。
