このドキュメントは、情報提供のみを目的として Netscape により提供されたものです。 このドキュメントには、インターネット上で個人情報のプライバシーおよびセキュリティを保護するための手順が記載されています。 ただし、このドキュメントには、オンラインにおけるプライバシーおよびセキュリティに関するすべての問題が記載されているわけではなく、また、インターネット上における妥当なプライバシーおよびセキュリティ保護に関する Netscape からの勧告を表すものでもありません。

ページ情報を表示する

［ページ情報］ダイアログ ボックスには、現在表示しているページに関するさまざまな情報を表示したタブがあります。 このセクションでは、各タブの情報について簡単に説明します

ブラウザで現在表示しているページに関する情報を表示するには、［表示］メニューから［ページ情報］を選択します。 ［ページ情報］ダイアログ ボックスが表示されたら、目的の情報が含まれているタブをクリックします。

［ページ情報］ダイアログ ボックスの［一般］、［フォーム］、［リンク］、［メディア］タブに表示される情報の大部分は、そのページの HTML ソースから取り込まれたものです。 通常、これらは Web 開発者など専門職向けの情報です。

［ページ情報］ダイアログ ボックスに表示されている各タブを含め、HTML に関する詳細情報については、「HTML 4.01 Specification（HTML 4.01 仕様）」を参照してください。

［プライバシー］および［セキュリティ］タブには、アクセスしている Web サイトの P3P（Platform for Privacy Preferences）証書やそのページのセキュリティ ステータスに関する情報があります。

［一般］タブ（ページ情報）

［表示］メニューから［ページ情報］を選択すると、［一般］タブには、現在ブラウザに表示されているページに関する基本的な情報が表示されます。

上部にはページの名前（ある場合のみ）と以下の情報が表示されます。

  • URL：これは Web ページを表示したときにブラウザ ウィンドウの場所ツールバーに表示される標準のアドレスであり、Uniform Resource Locator の略語です。
  • タイプ：HTML、XML、オーディオ、アプリケーションなど、表示されているページのメディア タイプが表示されます。
  • レンダ モード：ページのレイアウトに旧バージョン互換モードまたは標準準拠モードのいずれを使用しているかが示されます。 旧バージョン互換モードとは、完全に標準準拠していない古いバージョンの Web ブラウザ用に設計された古い Web ページで使用されている、非標準的な機能を考慮したものです。 標準準拠モードとは標準仕様を厳守したものです。 レンダ モードは、Web ページに含まれている情報に従って自動的に選択されます。
  • ソース：このページのソース コードがキャッシュされたかどうかを示します。
  • エンコード：この HTML ドキュメントに使用されている文字セットです。
  • サイズ：ファイルのサイズです（サイズ情報がある場合のみ）。
  • 変更：ページの最終更新日です（ある場合のみ）。
  • 有効期限：ページに表示されている情報の有効期限です。

［一般］タブの下部には、ページに指定されているメタタグが表示されます。 メタタグには、ページに関する一般的な説明、検索エンジン用のキーワード、著作権に関する情報など、ページに表示されているコンテンツのタイプに関する情報が表示されます。

 

［フォーム］タブ（ページ情報）

［表示］メニューから［ページ情報］を選択し、［フォーム］タブをクリックすると、アクセスしている Web ページに表示されているすべてのフォームに関する情報が表示されます。

［フォーム］タブの上部には、HTML ソース内で指定されている、ページ内の各フォームに関する基本的な情報が表示されます。

  • 名前：フォームの名前です（ある場合のみ）。
  • 方法：フォームによって取り込まれた情報を Web サーバに送信するときに使用される HTML メソッドが表示されます。 GET が表示されている場合は、入力した値がフォームの送信先の Web サイト アドレスに追加されます。 POST が表示されている場合は、Web サイト上にあるプログラムが読み取ることのできるパラメータとして、値が Web サイトに送信されます。
  • フォーム操作：フォームの送信時に起動されるプログラムの URL です。

［フォーム］タブの上部に表示されているフォームを選択すると、タブの下部に、HTML ソース内で指定されている各フォームのエレメントの詳細情報が表示されます。

  • ラベル：エレメントのラベルです（HTML などでラベルがタグ処理されている場合）。
  • フィールド名：エレメントの名前です。
  • タイプ：エレメントの入力タイプが表示されます。TEXT（テキスト送信）、RADIO（ラジオ ボタン）、HIDDEN（送信されるが画面には表示されない情報を保存）などがあります。
  • 現在の値：エレメントの現在の値です。 たとえば、テキスト エレメントの現在の値は、テキスト エレメントに現在含まれているテキストです。

 

［リンク］タブ（ページ情報）

［表示］メニューから［ページ情報］を選択し、［リンク］タブをクリックすると、そのページに含まれているすべてのリンクが表示されます。 各リンクに対して以下の情報が表示されます。

  • 名前：リンクとしてブラウザに表示されているテキストです。
  • アドレス：リンク先のページの URL です。
  • タイプ：アンカー（HTML ドキュメント内の特定の場所へのリンク）やフォーム送信などのリンクのタイプです。

 

［メディア］タブ（ページ情報）

［表示］メニューから［ページ情報］を選択し、［メディア］タブをクリックすると、そのページに表示されているすべての画像やその他のメディア エレメントが表示されます。

［メディア］タブの上部には、アドレス（URL）やタイプなど、各エレメントの基本的な情報が表示されます。

［メディア］タブの上部に表示されているメディア エレメントを選択すると、タブの下部に、そのエレメントに関する情報（属性、サイズなど）が表示されます。

ダイアログ ボックスの下部にも選択したエレメントが表示されます。 画像を拡大するには、［ページ情報］ダイアログ ボックスの右下隅をクリックしてからドラッグします。

メディア エレメントをファイルとして保存するには、

  • 名前を付けて保存：保存するエレメントを選択して［名前を付けて保存］をクリックし、ファイルの保存先を指定します。

 

［セキュリティ］タブ（ページ情報）

［表示］メニューから［ページ情報］を選択し、［セキュリティ］タブをクリックすると、アクセスしている Web ページの認証設定と暗号化 に関する情報が表示されます。 ［セキュリティ］タブの上部には Web サイトの識別情報が確認されているかどうかが示され、下部にはブラウザがそのページを受信したときにページが暗号化されていたかどうかが示されます。

ブラウザ ウィンドウの右下のロック アイコンをクリックしても、［セキュリティ］タブが開きます。

［セキュリティ］タブの上部には以下のメッセージが表示されます。

  • Web サイトの識別情報を確認できませんでした：アクセスしている Web サイトに認証用の証明書が存在しなかったことを意味します。このため証明書マネージャは、この Web サイトの識別を確認できません。可能性が低いとは言え、この Web サイトが偽装サイトであることも考えられます。
  • Web サイトの識別情報が確認されました：証明書マネージャがこの Web サイトの識別情報を確認するために使用している証明書は、「信頼できる」と見なした認証局（CA）によって発行されたものです。この Web サイトは、おそらく偽装サイトではありません。

［セキュリティ］タブの下部には以下のメッセージが表示されます。

  • 接続が暗号化されていません： 可能性は低いですが、お使いのコンピュータからこの Web サイトに送信した情報、反対にこの Web サイトからお使いのコンピュータに送られた情報が、第三者によって参照されている可能性もあります。
  • 接続は暗号化されました： 一般に、暗号化した接続の強度は、暗号で使用するキーの長さ（ビット単位）に関係します。キーが長いほど暗号の難易度が上がります。 つまり、認証されていない第三者による暗号化情報のアンスクランブルは、非常に難しくなります。

    ［ページ情報］ウィンドウでは、次の 3 つのレベルを使って暗号化の難易度を表します。

    • 強度の高い暗号化：最低で 128 ビット キーを使用する、利用可能な暗号化の中で最も強度の高い暗号化。
    • 中間強度の暗号化： 56 ビットまたは 64 ビット キーを使用する、低レベルの暗号化よりいくらか強度の高い暗号化。
    • 強度の低い暗号化： 40 ビット キーを使用する、利用可能な暗号化の中で最も強度の低い暗号化。

    ほとんどの Web サイトでは強度の高い暗号化がサポートされています。 強度の低い暗号化しかサポートしていない古い Web サイトにアクセスしている場合は、可能性が低いとは言え、お使いのコンピュータからこの Web サイトに送信した情報、逆にこの Web サイトからお使いのコンピュータに送られた情報が、第三者によって傍受される可能性もあります。

暗号化機能がまったくサポートされいない場合、または強度の高い暗号化がサポートされていない場合は、クレジット カード番号などの機密情報を送信または表示する際、十分に注意する必要があります。

ブラウザ ウィンドウの右下にあるロック アイコンを見れば、Web ページの暗号化ステータスをすばやく確認することができます。 詳しくは、「Web ページのセキュリティを確認する」を参照してください。

 

［プライバシー］タブ（ページ情報）

［表示］メニューから［ページ情報］を選択し、［プライバシー］タブをクリックすると、他のページ、画像、またはスクリプトへのリンクなど、そのページにあるさまざまなリンクが表示されます。

［プライバシー］タブには、アクセスしている Web サイト全体またはそのページにリンクされているその他のサイトに対する P3P（Platform for Privacy Preferences ）証書が表示されます。 これらの証書には、そのサイトで収集される情報の種類、情報の提供先や用途が説明されています。

Web サイトは、人が判読可能な形式とブラウザが解釈できる XML ファイル形式で作成されたプライバシー証書を発行します。 プライバシー証書について詳しくは、「プライバシー証書を表示する」を参照してください。

ページ内の個々のリンクによって指定されている Web サイトのプライバシー情報を入手するには、以下の手順を実行します。

  1. プライバシー情報を表示したいリンク（ある場合）のカテゴリをダブルクリックします。
  2. 情報を表示するリンクをクリックします。
  3. 以下のボタンをクリックすると、選択したリンクに対するプライバシー証書の情報が表示されます。
    • 証書：このボタンをクリックすると、その Web サイトに関する判読可能なプライバシー証書が表示されます。
    • 要約：このボタンをクリックすると、サイトの証書が詳しく定義されている XML 証書ファイルを人が判読できるように解釈したものが表示されます。
    • 同意／拒否：このボタンをクリックすると、サイト提供の Web ページが表示され、ユーザは特定の情報が収集されることに対して同意または拒否することができます。

Web サイト全体のプライバシー情報を表示するには、リストの最上部にある URL をクリックしてから、いずれかのボタンをクリックします。
