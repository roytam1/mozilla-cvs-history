一般に Web サイトの P3P プライバシー証書には、そのサイトで収集される情報の種類、情報の提供先、および用途が説明されています。 Web サイトのプライバシー証書の内容について詳しくは、「プライバシー証書を表示する」を参照してください。

マスター パスワードを忘れた場合

画像を管理する

Web をブラウズするときに画像をダウンロードしないように選択することができます。 これによってオンラインで表示される内容がかなり制限されますが、インターネット接続速度が遅い場合や、ページを早く表示したい場合には便利です。

また、アニメーション画像の繰り返し回数を設定したり、アニメーション機能を完全にオフにすることもできます。

以下のセクションでは、これらの画像設定について説明します。 標準設定では、すべての画像がダウンロードされ、アニメーションが繰り返されるようになっています。

 

プライバシーとセキュリティの設定 - 画像

このセクションでは、画像の設定方法について説明します。画像の設定を表示するには、以下の手順で操作します。

  1. ［編集］メニューから［設定］を選択します。
  2. ［プライバシーとセキュリティ］カテゴリの［画像］をクリックします（サブカテゴリが表示されていない場合は、［プライバシーとセキュリティ］カテゴリをダブルクリックしてリストを展開表示します）。

 

画像受入証書

［画像受入証書］設定では、画像を表示するかどうか、またどこに表示するかを指定できます。

  • 画像を読み込まない：画像を表示したくない場合に選択します。
  • すべての画像を受け入れる：すべての画像を表示する場合に選択します （これは標準でオンになっています）。
  • Mail & Newsgroup メッセージにリモート画像を読み込まない：メッセージの添付ファイルとして送信された Web ページの埋め込み画像をダウンロードしたくない場合に選択します （これは標準でオンになっています）。

 

アニメーション画像のループ

この設定では、アニメーション画像の繰り返し回数を指定できます。

  • 画像が指定する回数：指定された回数だけアニメーションが繰返されます （これは標準でオンになっています）。
  • 1 回：アニメーションが 1 回だけ表示されます（指定した回数は無効になります）。
  • しない：アニメーションは表示されません。
