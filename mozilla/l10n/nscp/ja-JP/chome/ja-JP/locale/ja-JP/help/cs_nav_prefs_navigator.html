Navigator の設定 - Navigator

このセクションでは、メインの［Navigator］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリをクリックします。


［Navigator］設定パネルでは、ブラウザに関する 3 つの要素をカスタマイズできます。

  • Netscape の起動時に開くページ： Netscape の起動時に表示するページを指定します。
    • ブランク ページ： Web ページを自動的に読み込まずに Navigator を起動します。
    • ホーム ページ： Navigator を起動する度に（以下で指定する）ホームページを読み込みます。
    • 最後にアクセスしたページ： 前回 Netscape を終了したときに表示していたページを読み込んで Navigator を起動します。
  • 標準のブラウザ：Netscape を標準のブラウザとして指定できます。または、標準のブラウザであることが表示されます。
    • 標準のブラウザに設定： 淡色表示になっていなければ、これをクリックすると、Netscape を標準のブラウザとして使用できます。
  • ホーム ページ： ホーム ページにする Web ページを入力するか、［現在のページ］をクリックして、現在表示されている Web ページをホーム ページにします。［ファイルを選択］をクリックすると、ホーム ページとして読み込むディスク上のファイルを検索できます。
  • ツールバーに表示するボタンを選択してください： ツールバーに表示するボタンに対応するチェックボックスをオンにします。

    ［ジャンプ］、［検索］、［印刷］ボタンは、Navigator ウィンドウの右上のコマンド ツールバー内に表示されます。 他のボタンはすべて、ユーザ設定ツールバー内に表示されます。 ユーザ独自のブックマークをユーザ設定ツールバーに追加する方法については、「ユーザ設定ツールバー」を参照してください。

Navigator の設定 - 履歴

このセクションでは、［履歴］設定パネルの使い方について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［履歴］をクリックします（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［履歴］設定パネルでは、Navigator の履歴設定を指定できます。

  • ブラウズの履歴：
    • 前回アクセスしたページを記憶する：今までにアクセスした Web ページに関する情報が保管される期間を、日数で指定します。 たとえば「10 日」に設定すると、履歴には過去 10 日間にアクセスしたページが保管されます。
    • 履歴をクリア： アクセスしたサイトのリストを削除します。
  • 場所ツールバーの履歴：
    • 場所ツールバーをクリア： 場所ツールバー メニューに保管されているサイトの一覧を削除します。
  • セッション履歴：
    • セッション履歴内のページ数： 今までにアクセスした Web ページの中で、［戻る］／［次へ］ボタンを使って表示できるページ数を入力します。

Netscape の履歴について詳しくは、「履歴をたどる」を参照してください。

Navigator の設定 - 言語

このセクションでは、［言語］設定パネルの使い方法について説明します。 このパネルを表示するには、次の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［言語］を選択します（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［言語］設定パネルでは、Web ページを表示するための言語と文字コードを選択できます。

  • Web ページ用言語：
    • 上に移動／下に移動： リスト内の言語の優先順位を決めるために、選択した言語を上下に移動することができます。
    • 追加： Web ページの表示に使用する言語を追加できます。
    • 削除： 選択した言語を削除します。
  • 文字コード：
    • 標準文字コード: ドロップダウン リストから、Web ページの表示に使用する文字コードを選択します。

Navigator の設定 - サポート アプリケーション

このセクションでは、［サポート アプリケーション］設定パネルの使い方について説明します。 このパネルを表示するには、以下の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［サポート アプリケーション］を選択します（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［サポート アプリケーション］設定パネルでは、Netscape 内で様々なファイル タイプを開くための他のアプリケーションを設定できます。

  • 各ファイル タイプを開くアプリケーションを指定：
    • ファイル タイプ： Netscape が使用するファイルのタイプを表示します。 ファイル タイプのいずれかを選択すると、以下の情報が表示されます。
      • 拡張子：選択したファイル タイプの拡張子が表示されます。
      • MIME タイプ：選択したファイル タイプの MIME タイプが表示されます。
      • 動作： 選択したタイプのファイルを開くために使用するプログラムが表示されます。
    • 新規タイプ： 新しいファイル タイプを追加するには、このボタンをクリックします。 タイプの説明、ファイル拡張子、MIME タイプ、使用アプリケーションを入力または選択します。
    • 編集： 選択したファイル タイプのファイル処理情報を変更できます。
    • 削除： 選択したファイル タイプを削除します。
  • Plug-in Finder Service: 新しいプラグイン サービスが必要な場合に必ず PFS（Plugin Finder Service）を使用するには、このオプションを選択します。選択しなかった場合は、プラグインを必要とする Web ページでプラグインの場所が指定されていない場合にのみ、PFS が使用されます。
  • ファイルを開く：
    • リセット： ［ダウンロード］ダイアログ ボックスの［この形式のファイルを開く前にメッセージを表示する］チェックボックスをオフにすることにより設定した内容をクリアします。

様々なファイル タイプの処理方法について詳しくは、「さまざまなファイル タイプを管理する」を参照してください。

Navigator の設定 - スマート ブラウジング

このセクションでは、［スマート ブラウジング］設定パネルの使い方について説明します。 このパネルを表示するには、以下の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［スマートブラウジング］を選択します（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［スマート ブラウジング］設定パネルでは、インターネット キーワードや場所ツールバーの自動補助を使って Webページ をより快適にブラウズするための設定を行います。

  • インターネット キーワード：
    • インターネット キーワードを有効にする： 場所ツールバーから株式情報や検索、その他の情報にすばやくアクセスできるようになります。
    • 詳細情報： インターネット キーワードに関する詳細情報を表示します。
  • 場所ツールバー自動補助：
    • 場所ツールバーに入力したテキストを自動的に補助する： 場所ツールバーに以前入力したテキストが、自動的に補完されるようになります。
    • 詳細： 次の中から 1 つまたは複数のオプションを選択できます。
      • 入力時に第一候補を使用して自動補助する： 場所ツールバーに入力し始めると、以前にアクセスした Web サイトの中から最も近い Web アドレスを使用して自動的にアドレス入力が完了します。
      • 照合結果のリストを表示する：場所ツールバーに入力し始めると、以前にアクセスした Web アドレスの中で一致するアドレスがドロップダウン リストに表示されます。
      • インターネット検索エンジンを表示する：入力した語句を検索するための標準の検索エンジンがドロップダウン リストに表示されます。
      • 以前に入力した Web サイトだけを照合する： 場所ツールバーに入力したことがある Web サイトだけが表示され、他の Web ページからリンクをクリックしてアクセスした場合など、それ以外のサイトは表示されません。

Navigator の設定 - インターネット検索

このセクションでは、［インターネット検索］設定パネルの使い方について説明します。 このパネルを表示するには、以下の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［インターネット検索］を選択します（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［インターネット検索］設定パネルでは、Netscape での検索方法を指定できます。

  • 標準の検索エンジン：
    • 使用するエンジン：ドロップダウン リストから Web ページの検索に使用する検索エンジンを選択します。
  • 検索結果：
    • 検索結果がある場合は、Sidebar の［検索］タブを開く： このチェックボックスをオンにすると、Sidebar に検索結果が表示されるようになります。
  • Sidebar の検索タブの設定：
    • ベーシック： Netscape で検索するときに検索エンジンを 1 つだけ使用します。
    • 詳細： Netscape で検索を実行するときにリストから １ つまたは複数の検索エンジンを選択できます。

Navigator の設定 - タブ式ブラウジング

このセクションでは、［タブ式ブラウジング］設定パネルの使い方について説明します。 このパネルを表示するには、以下の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［タブ式ブラウジング］を選択します（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［タブ式ブラウジング］設定パネルでは、タブ式ブラウジングを設定できます。

注記： Mac OS をお使いの方へ： Ctrl キーを押すように指示がある場合、 Command キーを押してください。

  • タブの表示：
    • 開いているタブが 1 つだけの場合はタブ バーを隠す： 複数の Navigator タブが開いている場合にのみタブ式ブラウジング バーを表示します。
    • リンクを背景に読み込む： ［新しいタブでリンクを開く］を使ってリンクを開く場合に、新しいタブに自動的に切り替わらないようにします。
  • 次の操作で、ウィンドウの代わりにタブを開く：
    • Web ページでリンクを中クリックまたは Control-クリック： マウスの中ボタンをクリックするか Ctrl キーを押したままリンクをクリックすると、Web ページのリンクが新しいタブ内に開くようになります。
    • 場所ツールバーで Control+Enter： 場所ツールバーに URL を入力して Ctrl+Enter キーを押すと、Web ページのリンクが新しいタブ内に開くようになります。

Navigator の設定 - ダウンロード

このセクションでは、［ダウンロード］設定パネルの使い方法について説明します。 このパネルを表示するには、以下の手順で操作します。

1. ［編集］メニューから［設定］を選択します。
2. ［Navigator］カテゴリの［ダウンロード］をクリックします（サブカテゴリが表示されていない場合は、［Navigator］カテゴリをダブルクリックしてリストを展開表示します）。


［ダウンロード］設定パネルでは、Web ページからダウンロードしたファイルの処理方法を設定できます。

  • ダウンロードを開始するとき：
    • ダウンロード マネージャを開く： 現在および以前のダウンロードに関するステータス情報を表示したダウンロード マネージャを開きます。 すべてのダウンロードに関するステータス情報が 1 つのウィンドウにまとめて表示されます。
    • 進捗状況ダイアログを開く： 進行中のダウンロードに対するステータス情報を示す進捗状況ダイアログ ボックスを表示します。 各ダウンロードのステータス情報は別のウィンドウに保管されます。
    • 何も開かない： ダウンロード用のダイアログボックスを開かないでダウンロードを実行します。 どのダウンロードのステータスも表示されません。

ファイルのダウンロードについて詳しくは、「さまざまなファイル タイプを管理する」を参照してください。
