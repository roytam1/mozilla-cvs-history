プロファイルを管理する

インターネットを自宅と職場の両方で使用する場合には、アクセスする場所に応じて、ブックマーク、設定、アドレス帳、電子メール アカウント、Sidebar などの設定を変えたいことがあります。 また、家族で同じブラウザ ソフトウェアを共有している場合には、各自のインターネット識別情報を個別にしておきたいこともあります。

プロファイル マネージャを使用すると、複数のプロファイルを作成し、各プロファイルに個別のブックマーク、環境設定、電子メール設定を指定することができます。ブラウザ ソフトウェアを最初にインストールする際には、自動的に標準のプロファイルが作成されます。 1 つ以上のプロファイルを作成すると、ブラウザの起動時にどのプロファイルを使用するかを尋ねるメッセージが表示されます。

新しいプロファイルを作成する

プロファイルを作成するには：

  1. ブラウザを終了します。Navigator、Mail & Newsgroups、またはブラウザ ソフトウェアの他のコンポーネントを実行している間は、新しいプロファイルを作成できません。
  2. プロファイル マネージャを開きます。
    • Windows の場合は、［スタート］メニューの［プログラム］から［Netscape 7.0］を選択し、［プロファイル マネージャ］を選択します。
    • Mac OS の場合は、［Netscape］フォルダを開き、［プロファイル マネージャ］をダブルクリックします。
    • Linux の場合は、コマンド ラインに次のように入力します。 ./netscape -profilemanager
  3. ［プロファイルの作成］をクリックし、プロファイルの説明を読んだ後、［次へ］をクリックします。
  4. プロファイル名を入力します。プロファイル名は自由に付けることができます。たとえば、実際の名前や、プロファイルを使用する場所に関連する名前（「学校」など）を入力します。
  5. 新しいプロファイルの保存先として、標準のディレクトリを受け入れるか、または［フォルダ］ボタンをクリックして目的の保存先を選択します。
  6. 言語パックや地域コンテンツ パックをお持ちの場合は、［エリアの選択...］をクリックし、使用する言語／コンテンツを選択します。
  7. ［完了］をクリックします。

開いていたプロファイル マネージャの画面が再び表示され、新しいプロファイルがハイライト表示されます。

新しいプロファイルに対するブラウザを設定するには、［Netscape 7.0 を起動］をクリックします。 （すでにスクリーン ネームを持っている場合は）Netscape Webmail、Netscape Instant Messenger、およびカスタマイズした My Netscape で使用するスクリーン ネームを登録するよう求められます。 スクリーン ネームを持っていない場合は、［今すぐ登録］をクリックしてスクリーン ネームを取得してください。

この時点でスクリーン ネームを登録したくない場合は、［キャンセル］をクリックすると、プログラムが開きます。 Netscape の起動中にスクリーン ネームを登録するには、［ヘルプ］メニューから［Netscape を登録］を選択します。

新しいプロファイルを使い始めると、電子メールの確認などのタスクを実行するたびに、さらに必要な情報を求められます。

 

プロファイルを削除または名前を変更する

普段使用しないプロファイルを削除したい場合があります。既存のプロファイルを削除またはその名前を変更するには：

  1. ブラウザを終了します。Navigator、Mail & Newsgroups、またはブラウザ ソフトウェアの他のコンポーネントを実行している間は、新しいプロファイルを作成または削除したり、名前を変更することはできません。
  2. プロファイル マネージャを開きます。
    • Windows の場合は、［スタート］メニューの［プログラム］から［Netscape 7.0］を選択し、［プロファイル マネージャ］を選択します。
    • Mac OS の場合は、［Netscape］フォルダを開き、［プロファイル マネージャ］をダブルクリックします。
    • Linux の場合は、コマンド ラインに次のように入力します。./netscape -profile
  3. プロファイルを削除するには、削除するプロファイルを選択し、［プロファイルの削除］をクリックします。
  4. プロファイルの名前を変更するには、プロファイルの名前を選択し、［プロファイルの名前を変更］をクリックした後、画面の指示に従います。
