在 Netscape 中使用 ICQ

Netscape 快信传递客户程序既可用于发送和接收 AIM 邮件，又可用于发送和接收 ICQ 邮件。在发送和接收 ICQ 邮件时，您必须先登录到 ICQ 客户程序；另外，您不能同时使用 AIM 和 ICQ。您必须拥有一个 ICQ 帐户，而且需建立一个 ICQ 联系人列表。一些 AIM 好友可能有 ICQ 帐户，因而可以是 ICQ 联系人，但这不是自动的。

设立您的第一个 ICQ 帐户

要在 Netscape 中设立一个新的 ICQ 帐户，请按下列步骤操作：

  1. 从“我的快捷栏”中，打开“好友”列表，或从“窗口”菜单中选择“快信信使”。
  2. 从“选择需登录使用的服务”下拉列表中选择 ICQ。
  3. 单击“注册新用户”

ICQ 互联网注册页随即在浏览器中打开。

  1. 输入您的电子邮件地址并单击“提交”。

此时将出现第二个互联网注册页。

  1. 单击“单击此处注册并获取新的 ICQ 号码”。
  2. 阅读互联网注册许可协议并单击“不同意”或“同意”。

如果单击“同意”，出现最后一个互联网注册页，其上有一个要求输入确认码的文本框。

  1. 当收到带有确认码的电子邮件（通常在几分钟之内）后，把确认码复制并粘贴到确认码文本框中并单击“提交”。另一种选择是单击邮件中提供的链接使确认码自动输入。
  2. 在出现的表单中输入您选择的别名，然后是您的名和姓，您的性别和年龄以及四到八个字符的密码。
  3. 单击“提交”。

您的 ICQ 号码此时将出现在新的屏幕上。您现在可以准备发送 ICQ 邮件和接收来自其他 ICQ 会员的 ICQ 邮件了。

使用现有号码和密码设立 ICQ 帐户

如果您已有一个 ICQ 号码和密码，那么在 Netscape 中设置 ICQ 帐户的方法如下：

  1. 从“我的快捷栏” 打开“好友”列表，或从“窗口”菜单中选择“快信信使”。
  2. 从“选择需登录使用的服务”下拉列表中选择 ICQ。
  3. 单击“注册新用户”。

此时“ICQ 互联网注册”页在浏览器中打开。

  1. 输入您的电子邮件地址并单击“提交”。
  2. 在显示的第二个互联网注册页中输入您的 ICQ 号码和密码并单击“提交”。
  3. 阅读互联网许可协议并单击“不同意”或“同意”。

如果单击“同意”，屏幕上将出现最后的“互联网注册”页，其中有一个要求您输入确认码的文本框。

  1. 当收到带有确认码的电子邮件（通常在几分钟之内）后，把确认码复制、粘贴到确认码文本框中并单击“提交”按钮。另一种选择是单击邮件中提供的链接使确认码自动输入。
  2. 在出现的表单中随意输入您选择的别名，然后是您的名和姓，您的性别和年龄，然后单击“提交”按钮。

您现在可以准备发送 ICQ 邮件和接收来自其他 ICQ 会员的 ICQ 邮件了。

登录过程

每次要使用 ICQ 服务时可手动登录到 ICQ，或设置在启动 Netscape 或启动计算机时启动 ICQ。

本节包括：

登录到 ICQ

启动系统时启动 ICQ

 

登录到 ICQ

使用下列过程登录到 ICQ ：

  1. 通过选择“窗口”菜单中的“快信信使”或单击“我的快捷栏”中的“好友列表”选项卡，都可打开 ICQ。
  2. 如果还未选择 ICQ，从“选择需登录使用的服务”下拉列表中选择。
  3. 输入您的 ICQ 号码或屏幕用名以及您的 ICQ 密码。

  4. 如果愿意，可以选中“保存密码”和“启动时连接”这两个复选框中的任意一个或两个。

如果选中“启动时连接” 复选框，以后在启动 Netscape 时会自动登录到 ICQ。这一次，您需要手动连接。

  1. 单击“登录”按钮。

备注：登录到 ICQ 后，登录按钮变为“断开连接”按钮。单击此按钮即断开连接。

窗口改而显示“在线方式”选项卡，其上有您的所有当前在线的 ICQ 联系人的列表。不在线的联系人不会出现在列表中。如果某个联系人在线，您可以发送 ICQ 邮件给他（她）。

启动系统时启动 ICQ

在 Windows 操作系统的计算机上，您可以设置在启动计算机时启动 ICQ， 而不启动 Netscape 浏览器本身。但在 ICQ 里无法设置这个选项，您必须先登录到 Netscape 的 AOL 快信信使。您应该在主 ICQ/AIM 窗口，在“窗口”菜单（而不是“我的快捷栏”）中选择即可进入该窗口。

  1. 如果您已登录到 ICQ ，单击“注销”按钮。。
  2. 在“选择需登录使用的服务”的下拉表单中选择 AIM。
  3. 输入您的 AIM 名和密码，并单击“登录”按钮。
  4. 打开“编辑”菜单并选择“个性设置”。
  5. 单击主要的“快信信使”个性设置。
  6. 选择“启动系统时自动启动”
  7. 单击“确定”。
  8. 单击“AIM 注销”按钮 。
  9. 从“选择需登录使用的服务”的下拉列表中选择 ICQ。
  10. 输入您的 ICQ 号码或屏幕用名和密码，并单击“登录”按钮。

此后在每当启动计算机时 ICQ 将打开。尽管这个选项打开了 ICQ 窗口，但并不自动登录，除非您在 ICQ 窗口中也选中了“启动时连接”这一选项。

备注：在系统启动时启动 ICQ 的功能只适用于 Windows 系统的计算机。另外，如果此前已将独立的 AOL 快信信使客户程序或独立的 ICQ 客户程序设置成在系统启动时启动，则此选项不可用。

添加 ICQ 联系人

使用下列过程添加 ICQ 联系人：

  1. 单击“联系人列表”选项卡显示联系人列表。
  2. 单击“添加联系人”按钮。此时可以看到“添加联系人”对话框。
  3. 输入联系人的 ICQ 号码和屏幕用名（这可以是您选择的任何名字），单击想为这个联系人分配的组的复选框，然后单击“确定”。

一个联系人只能被添加到一个组中。

如果用户需要这样的安全措施，系统会自动发送一封认证请求邮件。在被请求的联系人对此邮件作出应答之前，他（或她）的屏幕用名出现在一个叫“Awaiting Authorization”的文件夹中。如果联系人的应答是肯定的，他（或她）即被添加到联系人列表中。如果联系人的应答是否定的，他（或她）的名字则保留在“Awaiting Authorization”文件夹中。如果被请求的联系人不需要授权，则不会有任何邮件送出。他的（或她的）屏幕用名会立即显示在您的联系人列表中。

移除 ICQ 联系人

要移除一个 ICQ 联系人：

  1. 在“联系人列表”中选择一个联系人的名字。
  2. 单击“移除联系人”按钮。
  3. 在出现的确认对话框中单击“确定”。

该联系人即被移除。

添加 ICQ 组

ICQ 客户程序自带四个标准组：即一般、家庭、朋友和同事。可以通过下列过程添加更多的组：

  1. 单击“添加组”按钮。即可看到“添加组”对话框。
  2. 输入组的名字并单击“确定”。

此时新组显示在组列表中。

发送 ICQ 邮件

可以给任何 ICQ 会员发送邮件，无论他们使用的是 Netscape 中的 ICQ 或是独立的 ICQ 客户程序。

  1. 如果 ICQ 没有打开，从“我的快捷栏”打开“好友列表”，或从“窗口”菜单选择“快信信使”。
  2. 如有必要，从“选择需要登录使用的服务”的下拉列表中选择 ICQ。
  3. 如有必要，单击“在线方式”选项卡使其在前端显示。
  4. 双击要与之通信的 ICQ 联系人的号码或屏幕用名。
  5. 在打开的邮件窗口中， 输入您的邮件，然后按 Enter 或 Return 键。
  6. 当收到回复时，在邮件窗口底部的窗格中输入一个应答，然后按 Enter 或 Return 键。
  7. 重复步骤 5 和 6 直到对话结束。

如果 ICQ 会员没在您的联系人列表中，完成上面的步骤 1 到 3，然后从“在线方式”下单击“发送”按钮打开一个空白的编写窗口。输入要与之通信的人的 ICQ 屏幕用名或号码，然后完成上面的步骤 5 到 7。

接收 ICQ 邮件

除了必须登录到服务器之外，接收 ICQ 邮件与接收 AIM 邮件的操作过程相同。在离线时 ICQ 服务不能接收邮件。当有入站邮件时，系统将打开一个邮件窗口，并显示收到的内容。应答时时，请使用下列操作步骤：

  1. 在邮件窗口底部的窗格中输入您的应答，然后按 Enter 或 Return 键。
  2. 当收到回复时，重复步骤 1。

将状态设置为“不在线”

如果您要离开计算机一段时间并想让您的联系人知道此事，则单击“不在线”按钮，或从“文件”菜单选择“不在线”。想与您联系的用户将知道您离开了。

常规 ICQ 个性设置

本节描述如何使用“常规 ICQ 个性设置”面板。如果尚未见到该面板，请遵循以下步骤：

  1. 打开“编辑”菜单并选择“个性设置”。
  2. 单击 ICQ。

从下列选项中选择：

  • 按 Enter 键插入 Return：如果想在按 Enter 键时插入一个换行符则选中此项。如果想在按 Enter 键时发送您的邮件则取消其选中状态。
  • 按 Tab 键盘插入 Tab:如果想在按 Tab 键时插入一个 Tab 则选中此项。如果想在按 Tab 键时在 ICQ 窗口的各个字段间切换则取消其选中状态。
  • 在新窗口中显示邮件的时间戳：如果想让新的 ICQ 会话中的邮件标记上发送时间，则选中此项。

ICQ 个性设置 - 隐私与授权

本节说明如何使用“ICQ 的隐私与授权”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

  1. 登录到 ICQ。必须登录才能改变隐私个性设置。
  2. 打开“编辑”菜单并选择“个性设置”。
  3. 在 ICQ 类别下，单击“隐私与授权”。（如果看不到子类别，双击 ICQ 即可展开该列表。）

从下列选项中选择：

  • 如果您希望只有联系人列表中的用户才能发邮件给您，则在“谁能与我联系”下，单击“只允许我的联系人列表中的用户”。
  • 如果您想阻止一个或多个联系人，为第一个您想阻止的用户单击“添加 ICQ #”，然后在出现的对话框中输入联系人的 ICQ 号码。重复上述操作直到所有您想阻止的用户都被阻止。
  • 如果您想解除对一个或多个用户的阻止，在阻止用户列表中选择他们的号码并单击“移除”。

ICQ 个性设置 - 通知与声音

本节说明如何使用“ICQ 通知与声音”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

  1. 打开“编辑”菜单并选择“个性设置”。
  2. 在 ICQ 类别下，单击“通知与声音”。（如果看不见子类别，双击 ICQ 即可展开该列表。）

选择下面列出的选项来设置您的个性设置：

  • 在下列情况下播放声音：如果您选择了这个选项，则缺省情况下所有列出的事件发生时都将听到声音。对于不想听到声音的事件，取消选择。可通过浏览找到一个声音，方法是单击“浏览”按钮，然后定位到“控制面板/声音和多媒体”。一旦选择了一种声音,可以通过单击“预览”按钮试听。如果您选择播放声音，但未指定具体选择，则使用缺省的蜂鸣声。

ICQ 个性设置 - 连接

本节说明如何使用“ICQ 连接”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

  1. 打开“编辑”菜单并选择“个性设置”。
  2. 在 ICQ 类别下，单击“连接”。（如果看不见子类别，单击 ICQ 即可展开该列表。）

如果是通过互联网服务供应商上网访问，几乎不需要改动“连接”个性设置对话框的内容。如果是通过局域网防火墙上网访问（例如，在您的工作单位），则网络管理员也许需要向您提供一些信息，使您能够配置“连接”个性设置面板的代理服务器部分。

服务器主机

这个字段包含 Netscape 集成快信传递服务器的缺省互联网地址，此地址既支持 Netscape 的 AOL 快信信使又支持 Netscape 的 ICQ。一般情况下，并不需要改变这个字段的内容。如果您的代理服务器需要解析主机名，也许需要在这个字段输入 IP 地址而不是代替服务器的名字。可以从网络管理员那里获得 IP 地址。

服务器端口

这个字段包含用于 Netscape 集成快信传递服务器的缺省服务器端口号。一般情况下，并不需要改变这个字段的内容。

使用代理连接

如果单击这个按钮，快信信使软件将启动代理服务器控制功能。 从网络管理员那里获得信息后，输入适当的代理服务器、认证及协议信息。
