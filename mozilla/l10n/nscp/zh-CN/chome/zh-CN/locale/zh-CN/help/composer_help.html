用 Netscape 编写器创建网页

您可通过 Netscape 编写器创建您自己的网页，并在网上发布。您无需掌握 HTML 就可以使用这个编写器；它用起来就象一个文字处软件一样简单易用。

“工具栏”按钮用于添加列表、表格、图像、到其他页面的链接、颜色和字体样式。文档在网上的样式与您创建时的样式是一致的，并且可以很容易的与其他用户共享，不论他们使用什么类型的浏览器或哪一种具 HTML 功能的电子邮件程序。

要开始使用 Nescape 编写器：

  • 单击在任意 Netscape 窗口左下角的“编写器”图标。

创建新页面

Netscape 编写器是 HTML（超文本标记语言）编辑器，可以创建和编辑网页。编写器是一个 WYSIWYG（所见即所得）编辑器，也就是说在创建页面时看到的页面与读者将会看到的页面是一样的。由于大多数基本 HTML 功能可从工具栏和菜单作为命令使用，所以您无需了解 HTML。

编写器也用于在需要时编辑 HTML 源代码。要查看或编辑 HTML 源代码，打开“查看”菜单并选择“HTML源代码”，或者在“编写器”窗口底部的“编辑模式”工具栏中单击“<HTML>源代码”选项卡。

要创建新网页，请使用下述方法之一。新建页面后，可以象在文字处理器中那样添加和编辑文本。

从浏览器创建新页面：

  • 打开“文件”菜单，选择“新建”，然后再选择“编写器页面”。此时打开一个包含空白页面的“编写器”窗口。

要编辑当前正在浏览器中浏览的页面：

  • 在当前查看的页面的浏览器窗口中，打开“文件”菜单，并选择“编辑页面”。此时会出现一个“编写器”窗口，其中包含正在查看的页面。

在编写器中创建新页面：

  • 单击编写器工具栏中的“新建”按钮。

要从存储在本地驱动器上的 HTML 文件开始：

  1. 打开“窗口”菜单并选择“编写器”。即可看到“编写器”窗口。
  2. 打开“文件”菜单，选择“打开文件”选项。即可看到“打开 HTML 文件”对话框
  3. 在本地驱动器上找到要编辑的文件。
  4. 单击“打开”按钮，以在“编写器”窗口显示指定文件。

要编辑网页：

  1. 打开“窗口”菜单，选择“浏览器”。
  2. 在地址栏中键入网页的 URL(如 http：//home.netscape.com)并按下 Enter 键(在 Macintosh OS 中是 Return 键。
  3. 打开“文件”菜单，并选择“编辑页面”。

提示：在编写器窗口中，可通过打开“文件”菜单，选择“最近访问的页面”，然后从列表中选择您要的文件来快速打开最近用过的文件。

保存和浏览新页面

编写器文档可保存为 HTML 或纯文本格式。以 HTML 格式保存文档可保留文档的格式信息，如文本样式（例如，黑体或斜体）、表格、链接和图像。以纯文本格式保存文档会删除所有 HTML 标记，仅保留文档文本。

要将文档保存为 HTML 文件：

  • 打开“文件”菜单并选择“保存”，或单击编写工具栏中的“保存”按钮。

    如果尚未给页面添加标题，编写器会提示您。在浏览器中查看该页时，编写器会在浏览器窗口的标题栏中显示该页的标题。如果将文档的页标题加到了书签中，它还会出现在您的书签列表中。

    然后，编写器会提示您输入文件名，并指定文件的保存位置。确保在文件名中保持扩展名 .html。

若需改变现有 HTML 文件的名称或位置，请下列步骤操作：

  • 选择“另存为”并选择一个不同的文件名或位置。

在编写器中保存页面时，页面中的所有部分（HTML、图像和其他的文件如声音文件、样式表等）都保存到本地硬盘中。如果仅想保存页面中的 HTML 部分，必须改变有关保存页面的编写器个性设置。有关改变编写器保存页面设置方面的详细信息。请参见编写器个性设置 - 编写器一节。

如果图像地址是绝对的（以“http://”开头），并且您已连接到互联网，您仍会在编写器和浏览器中的文档中看到该图像。然而，如果图像的地址是相对于页面地址的，（以“file：///”开始的）, 则不能在本地版文档中看到该图像。

要将文档保存为纯文本文件：

  1. 打开“文件”菜单，并选择“导出到文本”。
  2. 输入文件名并指定文件的保存位置。

备注：图像不在以纯文本格式保存的文档中显示。

提示：可从“文件”菜单中选择“恢复到上次保存的文件”，使当前文档恢复到最近一次存盘的结果。请记住，当前所做的更改将会丢失。

要在浏览器窗口中查看您的页面以便测试链接：

  • 打开“文件”菜单，并选择“浏览页面”（或在“编写”工具栏中单击“浏览”）。如果尚未保存文档，编写器会提示您输入页面标题、文件名和位置。“编写器”窗口将在新的浏览器窗口后面保持打开状态。

格式化段落、标题和列表

要把一种格式应用到一个段落中，请从“编写器”窗口开始：

  1. 单击想要格式化部分的开始处以放置插入点，或者选中想要格式化的文本。
  2. 使用“格式”工具栏中的下拉式列表选择段落格式：
    • 正文：把应用程序的缺省字体和样式应用于正常文本，同时不影响文本前后的间隔。
    • 段落：插入一个段落标记（从而开始一个新段落）。段落包括顶部和底部的页边距。
    • 标题 1 - 标题 6：把段落格式化为标题。标题 1 是最高级的标题，标题 6 是最低级的标题。
    • 地址：可用作网页的“签名”，以注明网页的作者以及要获得的更多信息而须与之联系的人，例如：user@example.com

      还可以包括日期和版权通知。此格式通常出现在网页底部的水平线下面。浏览器以斜体显示地址格式。

    • 预设格式：这对于希望以固定宽度字体显示的元素，如代码例子、列数据、邮件信息等，十分有用。对于普通文本，大多数浏览器都会移除多余的空格、制表符和段落回车符。然而，使用“预定格式”样式的文本显示时将保留原有的空格，从而可保持原来文本的布局。

要将文本设置为标题格式：

  1. 单击文本内任何想要格式化的地方以放置插入点。
  2. 使用“格式”工具栏中的下拉式列表，选择想要的标题级别，可以为 1（最大）到 6（最小）。选择“标题 1”为主标题，选择“标题 2”为下级标题，以此类推。

要应用列表项格式：

  1. 单击希望格式化的文本行内，以放置插入点。
  2. 打开“格式”菜单，并选择“列表”。
  3. 选择列表样式：
    • 项目符号：每一个项目旁边都有一个项目符号（圆点）(如此列表中)。
    • 编号：项目的编号。
    • 术语和定义：这两种样式需一起使用，用于创建词汇表样式的外观。被定义的词使用“术语”标记，定义使用“定义”标记。“术语”文本左对齐显示，“定义”文本缩进显示。

提示：要快速将一列表样式应用于一个文本块，可以通过选中文本并单击“格式”工具栏中的“编号列表” 或“项目符号”列表 。

要更改项目符号或编号的样式：

  1. 在要更改的列表项目的文本中单击放置插入点，或者如果想将新样式应用于整个列表，可在列表中选择一个或多个项目。
  2. 打开“格式”菜单，并选择“列表属性”。
  3. 从下拉式列表中选择项目符或编号样式。对编号列表，可以指定开始编号。对项目符列表，可以更改项目符样式。

提示：也可以在列表中双击项目符或编号以显示“列表属性”对话框。

若需对齐页面中的段落或文本，如居中、左对齐或右对齐等，请按下列步骤操作：

  1. 在要对齐的文本的段落或行中单击放置插入点。
  2. 打开“格式”菜单，然后选择“对齐”选项。

备注：还可以使用“格式”工具栏来对齐文本。

处理列表格式

要结束列表并继续键入正文文本：

  • 单击最后一个列表项的尾部以放置插入点，然后按 Enter 键（在 Macintosh OS 中是 Return 键）两次，便可结束列表。

要将一个或多个列表项目改为正文文本：

  1. 在列表项中单击放置插入点，或选中这些列表项。
  2. 在一个编号列表中，单击格式工具栏中的“编号列表”按钮（对于项目符号列表，单击“项目符号列表”按钮）。

要在列表项下面放置缩进文本：

  1. 单击以在列表项中放置插入点。
  2. 按 Shift-Enter 创建悬挂式缩进。
  3. 键入要缩进的文本。
  4. 按 Shift-Enter 创建另一缩进段落，或按 Return 创建下一个列表项。

提示：可以通过单击列表项的任意位置然后单击在“格式”工具栏中的“缩进”或者“凸出”按钮来增加或者减少列表项的缩进。还可以单击一个列表项的任意位置，按 Tab 键缩进一级。按 Shift+Tab 凸出一级。

要合并两个相邻的列表：

  1. 选择要合并的两个列表。确保选中两个表中的所有元素。注意，这两个列表之间的任何文本也将成为合并后的列表的一部分。
  2. 单击格式工具栏中的项目符号列表或编号列表，以合并列表。

更改文本颜色、样式和字体

要更改所选文本的样式、颜色或字体：

  1. 选择要格式化的文本。
  2. 打开“格式”菜单并选择下列选项之一：
    • 字体：用于选择字体。如果更愿使用读者浏览器指定的字体，可选择“可变宽度”或“固定宽度”。

      备注：计算机中安装的字体不一定都在此显示。观看您的网页的人可能没有您指定的字体，故而通常最好是选择一种菜单中提供的字体，因为这些字体在每台计算机上都有效。举例来说，在不同的计算机上，字体 Helvetica、Arial、 Times 和 Courier 通常看起来是一样的。如果选择别的字体，当用不同的计算机查看时，可能就不一样。

    • 大小：用于选择一个相对的字体大小或选择一个可增加或者是减小文本大小（相对于周围的文本）的选项。
    • 文本样式：用于选择一种样式，如：斜体、粗体或者是下划线，或者选择一种结构化样式，如：Code。
    • 文本颜色：用于在颜色采集器中选择颜色。如果您熟悉 HTML 十六进位颜色代码，可以键入特定的代码或只键入颜色名（例如 blue）。若需使用便捷的颜色代码转换器，请单击此处。

改变页面的背景颜色：

  1. 单击页面的任何位置。
  2. 单击“格式”工具栏中的背景色块。
  3. 从“背景色块”对话框中选择背景颜色。
  4. 单击“确定”。

提示：要快速将文本的颜色改变到上一次使用的颜色，先选择文本，然后在按下 Shift 键同时单击格式工具栏中的文本色块。当您要对文本中的不同行使用同一种颜色时，这很有用。

也可以使用图像作为背景。参见设置页面颜色和背景。

移除或停用文本样式

要从选中的文本中删除所有文本样式（黑体、斜体等）：

  1. 选择该文本。
  2. 打开“格式”菜单，并选择“移除所有文本样式”。
  3. 继续键入。

移除了所有文本样式后，若需继续键入文本，请按下列方式操作：

  1. 在要中断文本样式的地方放置插入点。
  2. 打开“格式”菜单，并选择“中断文本样式”。
  3. 继续键入。

查找并替换文本

要在当前工作页面中查找文本：

  1. 单击想要搜索部分的开始处以放置插入点。
  2. 打开“编辑”菜单，并选择“查找并替换”。即可看到“查找和替换”对话框。
  3. 在“查找内容”字段中键入您想要查找的文本。若需缩小搜索范围， 请在下列选项中选中一个或多个：
    • 匹配大／小写：用于指定搜索时是否需区分大小写。
    • 绕回查找：用于搜索到页面的端头后再从顶部或底部（取决于是向前还是项后搜索）开始搜索。
    • 向后搜索：用于从插入点开始向页面的开始部分搜索。
  4. 单击“查找下一个”开始搜索。当编写器找到第一处的文本时，单击“查找下一个”便可搜索下一处。
  5. 完成后，单击“关闭”。

要在当前工作页面中查找并替换文本：

  1. 单击想要搜索部分的开始处以放置插入点。
  2. 打开“编辑”菜单，并选择“查找并替换”。即可看到“查找和替换”对话框。
  3. 键入您要查找的文本，然后键入替换文本。
  4. 要缩小搜索范围， 请在下列选项中选中一个或多个：
    • 匹配大／小写：用于指定搜索是否需区分大小写。如果不选择此选项，搜索所匹配文本时将不区分大小写。
    • 绕回查找：用于搜索到页面的底部后再从顶部开始搜索。
    • 向后搜索：用于从尾部到开始部分进行搜索。
  5. 单击“查找下一个”以搜索下一处。编写器会选中下次出现的文本。
  6. 单击“替换”用替换文本来取代选中的文本。单击“查找和替换”以替换选中的文本并寻找该文本的下一处。单击“全部替换”用替换文本来取代文档中出现的每一处文本。
  7. 完成后，单击“关闭”。

插入水平线

水平线通常用于将文档的不同部分分开。若需在页面中插入一水平线（也称为嵌线) ，请从“编写器”窗口开始：

  1. 在要显示水平线的地方单击放置插入点。
  2. 打开“插入”菜单，选择“水平线”。

 

设置“水平线属性”

可以定制线的高度、长度、宽度、对齐方式和阴影。

  1. 双击该线显示“水平线属性”对话框。
  2. 然后便可编辑以下任一属性：
    • 宽度：输入宽度，然后选择“窗口的百分比”或是像素。如果以百分比形式指定宽度，只要“编写器”窗口或浏览器窗口的宽度发生改变，线条宽度也会随之变化。
    • 高度：键入线条的（以像素为单位的）高度。
    • 三维阴影：用于通过增添加倾斜阴影来增加线条的深度。
    • 对齐方式：指定何处放置线条（左对齐，居中，右对齐）。
  3. 单击“用作缺省值”将这些设置作为下次插入水平线时的缺省值。
  4. 若需以手动方式编辑水平线属性，可单击“高级编辑”。详细信息，请参见高级属性编辑器一节。

提示：可以从“查看”菜单中选择“显示所有标记”，在一个个的黄框中显示所有 HTML 元素。单击任意一个黄框以选择该 HTML 标记（或元素）中的全部内容。双击任意一个黄框为该 HTML 标记（元素）显示高级属性编辑器对话框。

插入特殊字符

要插入特殊字符（如重音标记、版权或货币符号）：

  1. 单击想要特殊字符出现的地方以放置插入点。
  2. 打开“插入”菜单，并选择“字符和符号”。即可看见“插入字符”对话框。
  3. 选择字符种类。
    • 如果选择“重音大写字母”或“重音小写字母”，那么打开“字母”下拉式列表，并选择要应用重音的字母。（备注：并非所有字母都有重音形式。）选择“通用符号”可插入类似版权符号或分数这样的特殊字符。
  4. 从“字符”下拉式列表中，选择要插入的字符。
  5. 单击“插入”。

    此对话框的打开状态不会影响您在文档（或邮件书写窗口）中继续输入，如果想要再次使用这个对话框，即可方便地转换过去。

  6. 完成插入特殊字符后，请单击“关闭”。

插入 HTML 元素和属性

如果知道如何使用 HTML 源代码，则可以在您的页面中插入附加标记、样式属性和 JavaScript。如果对使用 HTML 源代码并无把握，最好不要对其进行更改。要运用 HTML 代码，请使用以下方法之一：

  • 在希望插入 HTML 代码的地方放置插入点，或选中要编辑的文本，然后打开“插入”菜单并选择 “HTML”。在“插入 HTML”对话框内输入 HTML 标记和文本，然后单击“插入”。
  • 选取一个诸如表格、命名锚点、图像、链接或水平线之类的元素。双击该元素便可打开与该项相关的属性对话框。单击“高级编辑”，打开“高级属性编辑器”。用高级属性编辑器可以向对象中添加 HTML 属性、JavaScript 和 CSS。
  • 打开“查看”菜单，选择“HTML 源代码”，或单击编写器窗口底部“编辑模式”工具栏内的“<HTML> 源代码”选项卡。（如果没有“编辑模式”工具栏，可以打开“查看”菜单，选择显示／隐藏；然后检查“编辑模式工具栏”的复选框已被选取。）

 

使用“高级属性编辑器”

备注：除非您十分了解如何添加、删除或修改 HTML 属性及其相关的值，否则最好不要这样做。

如果您目前不在“高级属性编辑器”对话框内，请执行以下步骤：

  1. 在“查看”菜单（或“编辑模式”工具栏）中，选择“显示所有标记”。
  2. 双击想要修改的对象，以打开其“属性”对话框。
  3. 单击“高级编辑”，以打开该对象的“高级属性编辑器”。“高级属性编辑器”有三个选项卡，每个选项列出被选对象当前属性：
    • HTML 属性：单击此选项卡可查看或输入其它 HTML 属性。
    • 内嵌样式：单击此选项卡可查看或通过（样式）属性输入其他 CSS（级联样式表）属性。有关在编写器中使用 CSS 样式的详细说明，请参见编写器个性设置 - 编写器。
    • JavaScript 事件：单击此选项卡可查看或者输入 JavaScript 事件。
  4. 若需编辑某一属性或三个列表中的任何一个属性，请选择要编辑的属性。该对话框底部有可编辑的“属性”和“值”字段，可用来编辑属性的名称和值。若需添加新属性，请在对话框底部的“属性”字段中键入。单击“值”字段时，新属性便会自动添加在字段中。若需删除某一属性，请在列表中将其选中，然后单击“移除属性”。

    备注：系统将在“属性”列表中醒目显示必须输入的属性。

  5. 单击“确定”，以应用对“高级属性编辑器”对话框所做的修改。
  6. 再次单击“确定”，此时便可退出“属性”对话框。

编写器会自动在任何属性文字上放置引号。

检查 HTML

在互联网服务器上放置文档之前， 应首先检查文档的 HTML 格式，以确保其符合互联网标准。使用不同浏览器查看时，如文档中包含已验证的 HTML，则产生问题的机率较小。只在浏览器内凭视觉检查网页并不能确保文档在其它互联网浏览器上能正确显示。

编写器提供了一种方便的方法来检查文档是否符合 W3C（万维网协会）HTML 标准。编写器使用 W3C HTML 验证服务来检查文档的 HTML 语法是否符合 HTML 4.01 标准。这种服务也提供了如何改正错误的信息。

备注：这种功能必须在联机时才能使用。

要验证文档的 HTML 语法：

  1. 打开“工具”菜单，选择“验证 HTML”。如果尚未保存更改，在进行下一步之前，编写器会询问是否将其保存。
  2. 当“W3C 验证服务”页面出现时，单击“浏览”并在硬盘中查找要验证的文件。
  3. 然后单击“验证此文档”。

选择正确的编辑模式

一般情况下，不需要更改编辑缺省模式（普通）。然而，如果要以文档的 HTML 源代码方式工作，可能要更改编辑模式。

编写器允许在四种编辑模式或视图之间进行快速切换。每种编辑模式都允许持续操作文档，但会显示不同的 HTML 标记等级（和标记图标）。

选择某一种编辑模式之前：

  • 打开“查看”菜单，选择“显示/隐藏”，并确保在“编辑模式工具栏”旁边有一个复选标记。

“编辑模式”工具栏有四个选项卡：

  • 普通：选择这种编辑模式可以在创建时看到文档在线时的样子。选择此模式可显示表格的边框和命名锚点的图标。隐藏所有其它的 HTML 标记图标。
  • 显示所有标记：选择这种模式以显示所有 HTML 标记图标。
  • <HTML> 源代码：选择这种模式将文档作为未格式化的 HTML 源代码来查看和编辑。保存文档时，重新出现“普通”模式。
  • 预览：选择此模式能以与浏览器窗口相同的显示效果来显示和编辑文档，不同的只是链接和 JavaScript 功能是非活动的。

备注：JavaScript 功能、框架、链接和动画 GIF 文件在任何编辑模式中都是非活动的。要以激活态显示这些项目，请单击“编写”工具栏上的“浏览”按钮，将其加载到一个浏览器窗口。

插入表格

在将文本、图像和数据编排到格式化的行和列中时，表格是非常有用的。要插入一个表格：

  4. 输入边框宽度的数值（以像素计）；若无边框，则输入零。

    备注：编写器用红色虚线来表示表格无边框；该虚线在浏览器中是看不见的。

  5. 要应用其它表格属性或 JavaScript，请单击“高级编辑”，以显示高级属性编辑器。
  6. 单击“确定”，确认设置并查看新的表格。

若需更改新表格的其它属性，请参阅更改表格属性。

提示：若需在一个表内插入另一个表格，请打开“插入”菜单，并选取“表格”。

更改表格属性

本节介绍如何修改可应用于整个表格以及表格内的行、列或单个单元格的属性。如果尚未进入“表格属性”对话框，遵循以下步骤：

  1. 选择某表格，或单击其内部的任何部位。
  2. 单击工具栏中的“表格”按钮 ，或打开“表格”菜单，并选择“表格属性”。“表格属性”对话框包含两个选项卡：“表格”和“单元格”。
  3. 单击“表格”选项卡可编辑以下属性：
    • 大小：用来指定行数和列数。输入表的宽度，然后选择“窗口大小百分比”或“像素”。如果以百分比形式指定宽度，只要“编写器”窗口或浏览器窗口的宽度发生改变，表格宽度也会随之改变。
    • 边框和间隔：用于以像素为单位指定边框的宽度，单元格的间距，以及单元格的填充（单元格内容与其边框间的距离）。

      备注：编写器用虚线勾画出以零边框的表格； 当页面在浏览器中查看时虚线就消失了。

    • 表格对齐：用于在页面内对齐表格。从下拉式列表中选择一个选项。
    • 标题：从下拉列表中选择标题布局。
    • 背景颜色：用来为表格选择背景颜色，或者让表格背景为透明。
  4. 要应用其它属性或 JavaScript 事件，请单击“高级编辑”以显示高级属性编辑器。
  5. 单击“应用”，可在不关闭对话框的情况下预览更改，或单击“确定”予以确认。

要对一个或多个单元格进行查看、更改或添加属性：

  1. 选择行、列或单元格，然后打开“表格”菜单，并选择“表格属性”。“表格属性”对话框出现。
  2. 单击“单元格”选项卡，以编辑如下属性：
    • 选择：从下拉列表中选择单元格、行或列。单击“上一个”或“下一个”，在行、列和单元格之间移动。
    • 大小：输入一个高度和宽度，然后选择“表格大小百分比”或“像素”。
    • 内容对齐：为单元格中的文本或数据选择垂直对齐和水平对齐的方式。
    • 单元格样式：从下拉列表中为行或列选择标题样式（其文本在单元格中心并用粗体字体）；另外也可选择“普通”。
    • 文本折行：从下拉列表中选择“不要折行”使文本不折行，除非插入一个段落终止符。否则，请选择“折行”。
    • 背景颜色：为单元格的背景选择一种颜色，或使单元格保持透明。

    备注：要应用其它属性或 JavaScript 事件，请单击“高级编辑”以显示高级属性编辑器。

  3. 单击“应用”，可在不关闭对话框的情况下预览更改，或单击“确定”予以确认。

提示：要改变一个或者多个单元格或者整个表格的文本颜色或背景颜色，首先要选择单元格或单击表格中的任意位置，然后在“格式”工具栏中单击文本颜色或背景颜色的图标。

提示：要将单元格的颜色改为上一次使用的颜色，选择单元格，按下 Shift 并且单击背景颜色采集器。当想要为单个单元格选用一个颜色时，这项功能很有用处。

添加和删除“行”、“列”和“单元格”

编写器允许在表格内迅速添加或删除一个或者多个单元格、列或行。此外，还可以通过设置选项，允许在执行编辑任务时保留表的原始矩形结构或布局。

在表格中添加单元格、行、列：

  1. 单击表格内要添加一个（或多个）单元格的位置。
  2. 打开“表格”菜单，选择“插入”。
  3. 选择单元格组中的一个单元格。（也可以在一个表格单元格内插入一个新表格。）

要删除一个单元格、行或列：

  1. 单击一个行、列或单元格以放置插入点。或者选择相邻的单元格以同时删除多个行。要选择相邻的若干单元格，拖拉越过想要选择的单元格。要选择表格中的一些孤立的单元格，按住 Ctrl 键（对于 Windows 和 Linux） 或 Command 键（对于 Macintosh OS）并分别单击要选择的单元格。
  2. 打开“表格”菜单，选择“删除”。
  3. 选择想要删除的条目。

要在单元格右侧联接（或合并）一个单元格：

  • 单击单元格内的左侧，打开“表格”菜单，选择“合并右侧的单元格”。

要联接或合并相邻单元格：

  • 通过拖动选择相邻的若干单元格。
  • 打开“表格”菜单，并选择“合并选定的单元格”。

要将一个合并的单元格再分割成两个或多个单元格：

  • 单击该单元格内部，打开“表格”菜单，然后选择“分割单元格”。编写器把合并的单元格的完整内容放到分割的两个单元格中的第一个单元格中。

参考选择表格元素以获得关于如何选择不相邻的单元格、行和列方面的信息。

 

更改缺省表格编辑性能

缺省状态下，删除一个或多个单元格时，编写器可根据需要通过在一行的结尾添加单元格来保留表格的结构。这使得在删除一个或多个单元格时，仍可保持表格的原始矩形布局或结构。否则，删除单元格将导致表格内出现空格，或由于单元格的数量不均匀而显示不规则的轮廓。

要改变缺省的表格编辑行为，请从“编写器”窗口开始：

  1. 打开“编辑”菜单，选择“个性设置”，然后选择“编写器”。
  2. 在“表格编辑”下，进行下列个性设置：
    • 确保选中“插入或删除单元格时保持表格布局”以保证不会产生不规则形状的表格。
  3. 单击“确定”。

另请参见设置一般编写器个性设置。

页面插图

在网页中可以插入 GIF、JPEG、BMP 和 PNG（可移植网络图形）等类型的图像。还可用之创建链接。当插入图像时，编写器会在页面中保存图像的参照位置。

    也可以选择不包含备用文本。

  5. 如果必要，单击其他选项卡，这样就可以调整图像属性对话框中的设置（例如，对齐方式）。

编辑图像属性

一旦已将图像插入页面，就可以编辑其属性并定制其在页面内的布局，例如，高度、宽度、间距及文本对齐。 如果当前尚未看到“图像属性”对话框，遵循以下步骤：

要编辑选定图像的属性：

  1. 双击图像，或者先选取图像然后单击工具栏上的“图像”按钮，以显示“图像属性”对话框。
  2. 单击“位置”选项卡编辑下列属性：
    • 图像位置：输入图像的文件名和位置。单击“选择文件”以在硬盘驱动器或网络上查找图像文件。
    • URL 与页面位置相对应：如果选中，编写器将把 URL 转换为相对于页面位置。如果计划将页面发布到互联网服务器以便其他人查看，此项功能特别有用。您可通过相关的 URL 在相对于彼此的相同的位置保留所有链接文件，而与它们在硬盘或互联网服务器上的位置无关。

      取消选中此复选框，编写器会将 URL 转换为完整（绝对）URL。在链接到其它互联网服务器中的图像时（即未存储在本地硬盘上的），通常使用绝对 URL。

      如果从未保存或者发布过某个网页，就必须首先保存该网页才可使这个复选框可用。（如果在邮件编写窗口打开“图像属性”对话框，则不可使用这个复选框。）

    • 备用文本：输入替代原图像而显示的文本；例如，图像的标题或者对图像的简短说明。建议为使用纯文本浏览器或关闭图像载入功能的读者指定备用文本。
    • 不使用备用文本：如果图像不要求备用文本或不想使用它，可选择此选项。
  3. 单击“尺寸”选项卡编辑以下这些属性：
    • 实际大小：选择此项可以撤销所有的关于图像大小的改变，使图像回到原始尺寸。
    • 自定义大小：选择此项可指定以像素为单位或以百分比表示的新的高度和宽度。此设置不会影响原始图像文件，只对插入页面中的图像有效。
    • 限定大小：若要改变图像的大小，最好选用此项以保持图像的纵横比（这样图像就不会失真）。如果选择此选项，则只需更改高度或宽度中的一项。
  4. 单击“外观”选项以编辑下列属性：
    • 间隔：指定图像周围（图像与相邻文本之间）的间隔距离。也可在图像四周放置实心黑色边框并指定其宽度（以像素计）。如果无边框，则指定零。
    • 将文本与图像对齐：如果把图像放在文本的旁边，选择一个对齐方式图标来指明希望的文本相对于图像的位置。
    • 图像映射：单击“移除”可移除所有图像映射设置。
  5. 单击“链接”选项卡可编辑下列属性：
    • 输入网页地址：如果想要为该图像确定一个链接，则须输入一个远程页面的或本地页面的 URL，或者从下拉菜单中选择一个命名锚点或标题。单击“选择文件”以在硬盘驱动器或网络上查找图像文件。
    • URL 与页面位置相对应：如果选中，编写器将把 URL 转换为相对于页面的位置。如果计划将页面发布到互联网服务器以便其他人查看，此项功能特别有用。您可通过相关 URL 允许在相对于彼此的相同的位置保留所有链接文件，而与它们在硬盘或互联网服务器上的位置无关。

      取消选中此框，编写器会将 URL 转换为完整（绝对）URL。在链接到其它互联网服务器中的图像时（未存储在本地硬盘上），通常使用绝对 URL。

      如果未保存更改，则必须首先保存页面，才能启用此复选框。(如果在邮件编写窗口中打开“图像属性”对话框，这个复选框不可用。)

    • 显示链接图像的边框：如果选中，则在该图像周围以醒目颜色显示链接。
  6. 要应用其它属性或 JavaScript 事件，请单击“高级编辑”以显示高级属性编辑器。
  7. 单击“确定”以确认您的更改。

验证发布设置

要验证您的发布设置：

  1. 如果“发布状态”对话框是打开的，将之关闭。
  2. 打开“编辑”菜单，选择“发布网站设置”。
  3. 在“发布设置”对话框内，检查准备作为发布目标的网站的设置是否正确。如果不能确定，请咨询您的 ISP 或者托管网站。
    • 检查输入的发布设置是否正确。 有可能在某个设置中有意外的输入错误。
    • 检查输入的发布地址是否正确。 托管主机或 ISP 可能把发布地址称为“服务器名”，“主机名”或者“服务器/主机”。他们经常把发布位置指定为 ftp.myisp.com/username， 这里的 username 是用户名。

检查文件名

检查任何发布失败的文件的名称。确认文件名：

  • 只使用数字或大小写字母。尽管允许使用大写字母创建文件，但如果在文件名仅使用小写字母，可避免潜在的、在以后定位发布文件时的错误。

    在向互联网服务器发布文件时，互联网服务器中的文件名是区分大小写的。如果使用大写字母或混合应用大小写字母，可能会更加难记住文件名。

    例如，当您在浏览器的“地址”字段中输入文件名的网址以查找一个已发布的文件时，您必须输入与创建时完全相同的文件名，使用同样的大写字母和小写字母的组合。

  • 请勿使用标点符号或空格。下划线（_）和连字符（-）可以使用。
  • 以 .html 或 .htm 结尾（对于编写器文件名而言）。
  • 所用字符不得多于 32 个。

  • 在“此页面的网站子目录”处，可能输入了一个在发布位置不存在子目录名。
  • 选中了“包括图像和其它文件”，但输入的子目录名在发布位置不存在。

  • 使用浏览器在试图对之进行发布的网站中设法查看某个网页。如果能成功地查看其他网站，但却不能查看发布网站中网页，则可能是 ISP 或托管网站遇到技术问题。
  • 请过一会儿再尝试发布。ISP、托管网站或互联网服务器可能面临着暂时的技术问题。

  • 保存密码：选择此项以用“密码管理器”安全地保存您的密码，这样就不用每次发布时重复输入密码了。

编写器个性设置 - 编写器

您可通过编写器个性设置为文件保存和表格编辑指定设置。这些设置将应用于您创建的每个文档。

如果尚未进入编写器个性设置，请按以下步骤操作：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 单击“编写器”类别
  • 列出的最大页面数：指定在“文件”菜单中的“最近访问的页面”下列出的页面的最大数目。
  • 保留源代码格式：此选项保留 HTML 源代码的换行符和页面原始格式。如果要保留空白（附加行、制表符等）以使 HTML 源代码更具可读性，请选择此选项。此个性设置不会影响页面在浏览器窗口中的显示方式。
  • 重新格式化 HTML 源代码：此选项通过插入换行符和缩进重新格式化 HTML 源代码，使其更易读。此个性设置不会影响页面在浏览器窗口中的显示方式。
  • 保存页面时保存图像和其他相关文件：如果选中，所有的图像、JavaScript（JS）、级联样式表（CSS）和其他的相关文件都在文档被首次保存或保存在一个新位置时与文档保存在同一位置。如果取消选中，则只有 HTML 文件被保存。

  • 例如，在编辑一个远程网页时，如将这个页面保存到本地硬盘中，则该设置保证所有与这个远程网页相关的文件都也将在本地保存。

  • 在发布页面时始终显示发布对话框：如果选中，在单击“发布”按钮或从“文件”菜单中选择“发布”时，始终显示“发布网页”对话框。如果取消选中，编写器只在为了发布页面而需要更多信息时才显示“发布网页”对话框。
  • 当插入或删除单元格时维护表格布局：如果希望编写器通过在需要的地方添加单元格来保持表格的布局（也即，保持其矩形形状），则选择此项。如果取消选中此选项，在删除一个或多个单元格时，编写器会同时删除单元格边框，这样会在表格中形成空白区域，或由于单元格数量不一致而显示为不规则轮廓。
  • 使用 CSS 样式而不是 HTML 元素及属性：启用编写器文档中的级联样式表（CSS）格式化功能。启用此个性设置后，编写器将用 CSS 针对各元素的内嵌样式生成 HTML4.01 格式。

  • 如果此个性设置没有启用，编写器仍可生成 HTML 格式但不使用 CSS 样式。

    与 HTML 相比，使用 CSS 格式化功能的 HTML 文档更简洁，更便于维护，更能与不同的浏览器兼容。如果启用了该个性设置而随后又编辑一个创建时没有使用 CSS 的文档，编写器将用 CSS 样式替换被编辑的元素。

    如果启用了 CSS 样式，则可用“格式”工具栏上的文本高亮度颜色按钮为选取的文本选择一个高亮度颜色。还可以为页面的任何元素选择背景颜色。(如果此个性设置未启用，这些特征不能使用。)

编写器个性设置 - 新页面设置

新建页面个性设置用于为每一个创建的文档指定颜色和背景图像的设置。

如果尚未进入“新页面设置”，请按以下步骤操作：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 双击“编写器”类型并且单击“新建页面设置”。
  • 作者：输入您的姓名。这样就将把您的名字加入到所创建的每个新页面的 HTML 源代码中。
  • 用户的缺省颜色：如果希望网页使用查看者的浏览器中为文本和链接所设置的颜色，选择此项。
  • 使用自定义颜色：如果每次都需为文本和链接元素指定颜色，在可选择此选项。然后通过单击在每个元素旁边的颜色按钮为每一个元素选择颜色。
  • 背景图像：输入图像文件名和地址，或者单击“选择文件”在本地硬盘或网络中查找图像文件。

    备注：背景图像被平铺并覆盖背景颜色。

若需在单独的页面中改变作者名：打开“格式”菜单，然后选择“页面标题和属性”。

要改变一个单独的页面的颜色和背景图像：打开“格式”菜单，然后选择“页面颜色和背景”。
