高级个性设置 - 高级

本节说明如何使用主要的“高级”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 单击“高级”类别。

主要的“高级”个性设置面板用于自定义 Java、 FTP 密码和“快速启动”：

  • 启用 Java：选择此项允许 Netscape 解释包含 Java 程序的网页。有些网页使用 Java 加入交互式和多媒体内容。
  • 启用 XSLT：选择此项允许 Netscape 使用 XSLT。XSLT （eXtensible Stylesheet Language/Transform）是一种用于将 XML 文档转换为其他 XML 文档的语言。有关 XSLT 的详细信息，请参阅在线文档，转换 XML：Netscape 和 XSLT。
  • 将此电子邮件地址作为匿名 FTP 密码发送：在匿名登录一个 FTP 站点时须键入一个希望作为密码的电子邮件地址。
  • 启用“快速启动”：如希望缩短启动 Netscape 的时间，可选择此项。有关使用快速启动的详细信息，请参阅“使用快速启动”。

    备注：“快速启动”只在 Windows 平台上才会出现。

 

高级个性设置 - 脚本和插件

本节说明如何使用脚本和插件个性设置面板。如果尚未见到该面板，请遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下，单击“脚本和插件”。（若未见子类别，双击“高级”即可展开子类别列表。）


“脚本和插件”个性设置用于控制在 Netcape 中使用 JavaScript 及其插件的方式：

  • 启用下列项的 JavaScript 功能：选择这些复选框使 JavaScript 针对下列目标开启：
    • 浏览器：将 JavaScript 切换为针对浏览器中打开的网页。
    • 邮件和新闻组：将 JavaScript 切换为针对在邮件和新闻组中打开的网页。
  • 允许从网页：选择这些复选框以控制 JavaScript 的使用：
    • 在新窗口中打开链接：使被单击的链接在新窗口中打开。有些链接配置为在新窗口而不是原有窗口打开。对此控制的更改在重新启动 Netscape 后生效。
    • 移动现有窗口或调整现有窗口大小：允许移动打开的窗口或调整其大小。
    • 升高或降低窗口：允许窗口位于其他窗口的上面或下面。
    • 隐藏状态栏：允许状态栏隐藏。
    • 改变状态栏文本：允许更改状态栏文本，比如状态栏中的滚动文本。
    • 改变图像：允许图像变样或动起来，如滚动图像（当鼠标的光标置于图像上时，图像就会改变）。
    • 建立或更改 cookie：允许用 JavaScript 建立或更改 cookie。
    • 读取 cookie：允许用 JavaScript 读取 cookie。
  • 启用插件：选取此复选框以控制插件的使用：
    • 邮件和新闻组：允许插件在“邮件和新闻组”中使用。

有关插件的更多信息，请参阅管理不同的文件类型。

 

高级个性设置 - 高速缓存

本节说明如何使用“高速缓存”个性设置面板。如果尚未见到该面板，遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下，单击“高速缓存”。（若未见子类别，双击“高级”即可展开子类别列表。）


“高速缓存”个性设置面板用于调整 Netscape 内存和磁盘高速缓存：

  • 内存高速缓存：键入希望分配给 Netscape 的高速缓存的内存量。内存高速缓存在关闭或重新启动计算机时被擦除。
  • 磁盘高速缓存：键入希望分配给 Netscape 的磁盘高速缓存量。磁盘高速缓存保存在硬盘（驱动器）上，即使关机仍可再次使用。
  • 磁盘高速缓存文件夹：显示当前磁盘高速缓存文件夹的位置。
    • 选择文件夹：单击此选项以选择放置磁盘高速缓存的一个文件夹位置。
  • 清除内存高速缓存：单击此选项以清除内存高速缓存。
  • 清除磁盘高速缓存：单击此选项以清除磁盘高速缓存。
  • 比较高速缓存中的页面和网络上的页面：
    • 每次查看页面时：如希望 Netscape 每次都将看到的网页和高速缓存进行比较，可选择此选项。
    • 页面过期时：若希望 Netscape 将服务器认定已过期的网页与高速缓存进行比较，可选择此项。
    • 每个会话比较一次：若希望每次启动 Netscape 时都将网页与高速缓存进行比较，可选择此项。
    • 不比较：不比较若不想让 Netscape 对高速缓存中的信息和网络信息进行比较，可选择此项。

 

高级个性设置 - 代理

本节说明如何使用“代理”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下单击“代理”选项。（若未见子类别，双击“高级”即可展开子类别列表。）


“代理”个性设置面板用于通过代理服务器设置 Netscape：

启动前：须询问您的网络管理员您是否有一个代理配置文件具，或询问代理的名称和端口号。

  • 直接连接到互联网：如不想使用代理，可选择此项。
  • 手工代理配置：如没有代理服务器地址（URL），可选择此项。
    • HTTP 代理、SSL 代理、FTP 代理、Gopher 代理： 键入代理服务器的名称或数字 IP 地址。在“端口”字段键入端口号。
    • SOCKS 主机：键入代理服务器的名称或数字 IP 地址。在“端口”字段中键入端口号。
      • SOCKS v4，SOCKS v5：进入到一个 SOCK 主机后，选择“SOCKS v4”或“SOCKS v5”取决于代理所用的 SOCKS 是哪个版本。
    • 对以下域名不使用代理：键入不想使用代理的域。域与域之间用逗号隔开。（例如：yourcompany.com, yourcompany.co.nz）
  • 自动代理配置 URL：如果有代理配置文件或 URL，可选择此项，然后键入配置 URL。

 

高级个性设置 - HTTP 网络连接

本节说明如何使用“HTTP 网络连接”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下，单击“HTTP 网络连接”。（若未见子类别，双击“高级”即可展开子类别列表。）


“HTTP 网络连接”个性设置面板用于配置基于 HTTP 的网络连接：

  • 直接连接选项、代理连接选项：为直接连接和代理连接选择 HTTP 版本和选项。
    • 使用 HTTP 1.0：选择此项以使用 HTTP 的原始版本，1996 年标准版。
    • 使用 HTTP 1.1：选择此项以使用新版 HTTP，新版提供了改进了的性能，包括更有效地使用 HTTP 连接，更好地支持客户机端的高速缓存，多个 HTTP 请求（流水线作业）和更精确的控制高速缓存过期和置换的策略。
    • 启用保持连接功能：选择此项以保持连接畅通用于发送额外的 HTTP 请求并提高速度。
    • 启动流水线作业： 选择此项以启用流水线，这样可一次向服务器发送多个 HTTP 请求，缩短载入网页时间。
      备注：流水线功能仅在 HTTP 1.1版本下可用。

 

高级个性设置 - 软件安装

本节说明如何使用“软件安装”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下单击“软件安装”。（若未见子类别，双击“高级”即可展开子类别列表。）


“软件安装”个性设置用于启用软件安装和更新通知：

  • 启用软件安装：若希望在需要安装和 Netscape 一起使用的新软件时得到 Netscape 的提示，可选择此项。
  • 检查更新：选择此项以便在 Netscape 提供新版本时得到通知。当验证您的 Netscape 版本时，Netscape 不会共享您的个人信息。
    • 每周：若想每周检查一次是否有 Netscape 最新版本，可选择此项。
    • 每月：若想每月检查一次是否有 Netscape 最新版本，可选择此项。

 

高级个性设置 - 鼠标滚轮

本节说明如何使用“鼠标滚轮”个性设置面板。如果尚未见到该面板，遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下单击“鼠标滚轮”。（若未见子类别，双击“高级”即可展开子类别列表。）


“鼠标滚轮”个性设置面板用于控制鼠标上的鼠标滚轮（在鼠标左右键之间）在 Netscape 下的使用：

  • 修改键：使用下拉列表选择希望与鼠标滚轮一同使用以执行下列功能之一的键：
    • 文档滚动行数：选择此项以键入字段的行数滚动文档。
      • 使用：选择此项以系统缺省设置覆盖输入的行数。
    • 向上或向下滚动页面：选择此项以便每次向上或向下滚动一个页面。此设置可使鼠标滚论能更快地滚动一个页面，但准确性略差一些。
    • 在浏览历史中前后移动：选择此项使鼠标滚轮用于前后浏览访问过的页面。
    • 放大或缩小文本：选择此项使鼠标滚轮用于放大或缩小网页上文本。此设置有助于更好地阅读页面或使文本适应屏幕宽度。

    备注：每个修改键可被赋予一个不同的功能。

 

高级个性设置 - 系统

本节说明如何使用“系统”个性设置面板。如果尚未见到该面板，请遵循以下步骤：

1. 打开“编辑”菜单，选择“个性设置”。
2. 在“高级”类别下单击“系统”。（若未见子类别，双击“高级”即可展开子类别列表。）

备注：“系统”个性设置面板只在 Windows 平台上出现。


“系统”个性设置面板用于指定使用 Netscape 时打开哪些文件和协议：

  • Windows 应使用 Netscape 打开这些文件类型：选择希望在使用 Netscape 时打开文件的缺省类型。
  • Windows 应使用 Netscape 处理这些协议：选择希望在使用 Netscape 时打开的缺省协议。
  • 其他应用程序更改这些设置时发出警告：如果希望在其他应用程序更改了缺省的 Netscape 文件及协议设置时发出警告，可选择此项。

提示：要很简单的把 Netscape 设置为缺省浏览器，请参阅浏览器个性设置 - 浏览器。同样选择“其他应用程序更改这些设置时发出警告”也可获知其他程序何时代替 Netscape 成为缺省程序。
