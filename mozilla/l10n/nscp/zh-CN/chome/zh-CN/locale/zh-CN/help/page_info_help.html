查看页面信息

“页面信息”对话框包含多个选项卡，显示正在查看的页面的各种类型的信息。本节对每个选项卡上的可用信息提供一个简短的概述。

要查看浏览器当前显示页面的页面信息， 请打开“查看”菜单并选择“页面信息”。然后单击对应于需要查看的那类信息的选项卡。

“页面信息”对话框的“常规”、“表单”、“链接”和“媒体”选项卡中显示的大多数信息是从页面的 HTML 源代码中获得的。通常只有互联网开发者和其他专业人员对这些信息感兴趣。

关于 HTML 的详细信息，其中包括“页面信息”显示的标记，请参阅 HTML 4.01 说明书。

“隐私”和“安全”选项卡包含有关正在查看页面的网站的隐私个性设置平台（P3P）策略的信息以及该页面的安全状态的信息。

本节包括：

“常规”选项卡

“表单”选项卡

“链接”选项卡

“媒体”选项卡

“隐私”选项卡

“安全”选项卡

 

常规（页面信息选项卡）

在“查看”菜单中选择“页面信息”，“常规”选项卡显示正在浏览器中查看的页面的基本信息。

在顶部显示该页面的名称（如果存在）和下列信息：

  • URL：   页面的统一资源定位符，即出现在浏览器窗口顶部附近的地址栏中的标准化地址。
  • 类型：显示出的页面的媒体类型，例如 text/HTML、text/XML、音频或应用程序。
  • 绘制方式： 说明浏览器是使用 quirks 模式 还是与标准相容模式展示页面。Quirks 模式顾及到一些旧的互联网页面使用的非标准行为，这些页面是为不完全与标准相容的旧版浏览器而设计的。与标准相容模式严格坚持标准规范。浏览器会根据网页自身包含的信息来自动选择绘制模式。
  • 源代码：说明此页面的源代码是否储存在高速缓存中。
  • 编码：此 HTML 文档使用的字符集编码。
  • 大小：文件的大小，如果有的话。
  • 修改日期：页面最后一次被修改的日期，如果有的话。
  • 过期：页面显示的信息何时过期。

在底部显示页面指定的元标记（metatag）。元标记提供关于页面所显示内容的类型方面信息，例如页面的一般描述，搜索引擎关键字，版权信息等等。

 

表单（页面信息选项卡）

在“查看”菜单中选择“页面信息”并单击“表单”选项卡，可以看到当前在浏览器中查看的页面所显示的所有表单的信息。

在顶部列出关于页面中每个表单在 HTML 源代码中被指定的方式的基本信息：

  • 名称：表单的名称，如果存在。
  • 方法：用于把被表单捕获的信息发送回互联网服务器的 HTML 方法。GET 将输入值添加至表单向之提交的网站地址。POST 将值作为能被网站的程序读取的参数发送到网站。
  • 表单操作：当提交表单时被启动的程序的 URL。

如果选择了“表单”选项卡顶部列出的某个表单，底部会显示关于该表单的每个元素在 HTML 源代码中被指定的方式的详细信息：

  • 标签：元素的标签（如果是被标记的，如 HTML 中那样）。
  • 字段名：元素的名称。
  • 类型：元素的输入类型，例如 TEXT（对于要提交的文本），RADIO（对于音频按钮），或者 HIDDEN（对于被提交但不显示在屏幕上的存储信息）。
  • 当前值：元素的当前值。例如，一文本元素的当前值就是它当前包含的文本。

 

链接（页面信息选项卡）

在“查看”菜单中选择“页面信息”并单击“链接”选项卡，可以看到该页面上所有链接的列表。对于每个链接显示如下信息：

  • 名称：在浏览器中作为链接所显示的文本。
  • 地址。链接所指向页面的 URL。
  • 类型：链接的类型，如锚点（到 HTML 文档中一指定位置的链接）或表单提交。

 

媒体（页面信息选项卡）

在“查看”菜单中选择“页面信息”并单击“媒体”选项卡，可以看到页面所显示的所有图像和其它媒体元素的列表。

顶部列出每个元素的基本信息，包括其地址（URL）和类型。

如果选择在“媒体”选项卡顶部列出的一个媒体元素，在底部会显示这个元素的已有信息，如描述、大小、尺寸。

也能够在此对话框的底部看到选定的元素。要看更大尺寸的图像，请单击“页面信息”对话框的右下角并拖动。

要将媒体元素存储为单独的文件：

  • 另存为：选择需要保存的元素，单击“另存为”，然后定位要存放的位置。

 

安全（页面信息选项卡）

在“查看”菜单中选择“页面信息”并单击“安全”选项卡，可以看到关于正在查看页面的认证和加密方面的信息。在顶部指出网站的标识是否已被验证，在底部说明浏览器所接收的页面是否已加密。

也可以通过单击任何浏览器窗口右下角的锁图标来直接打开“安全”选项卡。

“安全”选项卡的顶部能够显示下面的任何信息：

  • 未验证网站标识：正在查看的网站没有出示证书来认证其自身。因此，证书管理器不能验证其标识。这可能说明，虽然可能性不大，该网站并不是其所声称的站点。
  • 已经验证网站标识：证书管理器用于验证此网站标识的证书是由标记为可信的认证中心（CA）颁发的。有理由相信此网站是其所声称的站点。

“安全”选项卡的底部可显示下面的任何信息：

  • 连接未加密： 其他人可能能够查看从您的计算机发送到网站的信息或从网站发送到您的计算机的信息，但实际有人这样做的可能性不大。
  • 连接已加密：通常，加密连接的强度依赖于加密所使用的密钥的长度（用位度量）。密钥越长，加密强度越大。也就是说，未被授权的人破译出加密信息的难度越大。

    “页面信息”用以下三种方式之一描述加密强度：

    • 高级加密： 可获得的最强加密，至少使用 128 位密钥。
    • 中级加密： 比低级加密要强，使用 56 或 64 位密钥。
    • 低级加密： 可获得的最弱加密，使用 40 位密钥。

    大多数网站支持高级加密。如果正在查看的是支持弱加密的旧网站，其他人可能能够查看从您的计算机发送到网站的信息或从网站发送到您的计算机的信息，但实际有人这样做的可能性不大。

只有当发送保密信息时，例如信用卡号，才需要关注是否缺少加密或缺少强化加密。

通过注意浏览器窗口右下角锁图标的状态，可以快速检查网页的加密状态。详细信息请参阅检查网页安全性。

 

隐私（页面信息选项卡）

在“查看”菜单中选择“页面信息”并单击“隐私”选项卡，可以看到页面中各种链接（例如链接到其它页面、图像或脚本）的列表。

可使用此选项卡查看有关网站整体的，或者页面所链接的或为页面提供元素的其他网站的隐私个性设置平台（P3P）策略。这样的策略描述站点收集的信息的种类，这些信息给谁，以及它如何使用这些信息。

网站发布隐私权策略既采用可读的形式，同时也以能够被浏览器解释的 XML 文件提供。关于隐私权策略的更多信息，请参阅查看隐私权策略。

要获得页面上某个链接指定的任何网站的隐私信息，遵循以下步骤：

  1. 双击要查看其隐私信息的链接（如果存在）的类别。
  2. 单击要查看信息的链接。
  3. 单击以下按钮之一，查看选定链接的隐私权策略信息：
    • 策略：单击此按钮以查看站点在互联网上的可读的隐私权策略。
    • 摘要：单击此按钮以查看站点的 XML 策略文件的可读解释，它详细定义了网站策略。
    • 选项：单击此按钮以查看由某些站点提供的网页，在这些站点可以选择撤销或承认对特定种类信息收集的许可。

要查看网站整体的隐私信息，单击列表顶部的 URL 后再单击一个按钮。
