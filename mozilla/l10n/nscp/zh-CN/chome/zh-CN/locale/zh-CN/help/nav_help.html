查看主页

启动 Netscape 时，即可看到“浏览器”，这就是您的浏览器。首次启动 Netscape 时，浏览器窗口会自动出现“最新动态”页面。

首次启动后，通常会在启动 Netscape 时看到您的主页。除非您自己选择主页，否则您的网络或互联网服务提供商会为您选择，或者即可看到 Netscape 的主页。

要选择您自己的主页，请参阅指定 Netscape 如何启动。

提示：

  • 若需简化 Netscape 界面，可以使用“全屏”方式来显示网页，这几乎会占用您的全部屏幕。在浏览器中打开“查看”菜单并选择“全屏”。也可以按 F11 按钮。
  • 如果想快速回到主页，请按 Alt +“Home（在 Mac OS 上是 Cmd+Home 按钮）。

转到另一页面

通过键入 URL - 网页位置（地址）可以转到新页面。URL 通常由 http：// 开始，跟着是一个或多个标识地址的名称。例如，http://netscape.com

  1. 单击地址栏以选择已经存在的 URL。
  2. 键入要访问页面的 URL。您所键入的 URL 将替换掉地址栏内的任何文字。
  3. 按下“Enter”键。

使用窗口右下角的锁定图标，可以在任何时间查看网页的安全情况。详情请参阅查看网页安全性。

提示：想快速选择“地址栏” 里的 URL，按下 Ctrl+L 键。

 

地址栏

 

不了解 URL? 您可以键入 URL 的一部分，比如 cnn（代表 www.cnn.com）；或者键入普通词，如 gifts 或 flowers。浏览器可推断您要浏览的页面，或显示带有与您键入的词有关的链接选择的页面。

如果您对国际互联网还很陌生，请参阅网络初学者指南。

单击链接

大多数的网页都包含单击就可以转到其它页面的链接。

  1. 移动光标直到它变成手指形的光标。当光标位于链接之上的时候会变成手指形。大多数链接为带有下划线的文字，但链接也可以是按钮和图片。
  2. 单击链接。当网络查找链接所指向的页面时，窗口底部显示状态信息。

返回各步骤

有几种方式可重新访问页面：

  • 要前进或后退一页，单击“前进”或“后退”箭头。
  • 要前进或后退多页，单击“前进”或“后退”按钮上的小三角。您会看到曾经访问过页面的列表。要返回到一个页面，在列表中选择它。

  • 要查看已键入到地址栏的所有 URL 地址的列表，单击地址栏右端的箭头。要查看一个页面，在列表中选择它。

 

地址栏

 

  • 要选择当前会话期间曾访问过的页面，打开“转到”菜单并使用菜单底部的列表。
  • 要选择前几次会话期间曾访问过的页面，打开“转到”菜单并选择“历史”。即可看到历史列表。历史列表显示一个文件夹列表。双击文件夹显示子文件夹或网页的书签。您可以双击书签图标旁边的 URL 地址来浏览该页面。

提示：“我的快捷栏历史” 选项卡也可以用来选择在过去几次会话期间访问过的页面。详情请参阅添加我的快捷栏选项卡。

 

关于历史列表

历史列表包含最近访问过的页面链接。“地址栏”列表包含您键入“地址栏”并访问了的页面链接。

要从 浏览器 访问历史列表，打开“转到”菜单，然后选择“历史”。要访问“地址栏”列表，单击在“地址栏”右下方的箭头。

提示：要想快速打开历史列表，按下 Ctrl+H 键。

如果不愿意“地址栏”或历史列表显示访问过的页面，可以彻底地或有选择行地清除历史列表和“地址栏”历史。

要删除“地址栏”或历史列表中的所有页面，请从“浏览器”窗口开始：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“浏览器”类别下，单击“历史”。（如果未见到子类别，双击“浏览器”即可展开子类别列表。）
  3. 单击“清除历史”和“清除地址栏”可从列表中清除所有曾经访问过的网页。
  4. 如果想对历史列表中可存储多少页面进行设置，单击“期间浏览历史中的页面数”字段，然后键入页面数。

如果想从历史列表中选择性地删除页面，请按下列步骤：

  • 要删除“历史”列表中一个域（文件夹）下的所有页面，打开“编辑”菜单，选择“删除整个域[域名]”。例如，如果您要删除以 netscape.com 结尾的所有页面则使用此命令。
  • 要删掉“历史”列表中子域下的页面，打开“编辑”菜单，选择“删除全部[子域]”。例如，如果您要从 health.netscape.com 而不是 netscape.com 中删除所有页面则使用此命令。
  • 要删除单个页面，从历史列表中选择一页面然后按“删除”。

提示：如果想给历史列表排序，单击一类别（“标题”、“位置”或“上次访问时间”）。如果想逆向排序请再次单击标题。

访问带有书签的页面

网页的地址或 URL 可能会相当长，难以记忆。好在浏览网页不需要必须记住 URL。浏览器有一个书签列表，可用来指向您感兴趣的网页。

要进入已插入了书签的页面，请从“浏览器”窗口开始：

  1. 打开“书签”菜单。该菜单由书签图标表示的书签和包含更多书签的文件夹组成。
  2. 要访问已加入书签的页面，从该菜单中选择书签或打开一个文件夹并选择一个书签。

提示：如果想撤销步骤，单击“后退”箭头。

您可以保存自己的书签，以快速进入到经常访问或感兴趣的网站。详情请参阅创建新书签。

使用选项卡浏览

当同时访问一个或多个网页时，可以使用“通过选项卡浏览”来更快更方便地浏览互联网。

“通过选项卡浏览” 可以在一个单一的“浏览器”窗口内打开若干选项卡，而每个选项卡显示一个网页。这样就不必为了浏览若干不同的网页而打开几个不同的窗口。这可节省桌面空间。可以在一个窗口内方便地打开、关闭、重新装载网页，不需切换到另外一个窗口。

当选项卡自动打开后，就可以轻松地管理导航选项卡。关于设置“通过选项卡浏览”个性设置的详情，请参阅浏览器个性设置 - 通过选项卡浏览。

有关使用“通过选项卡浏览”的更多信息，请参阅通过选项卡浏览。

使用我的快捷栏

除了用浏览器在网上冲浪以外，您也可以通过使用“我的快捷栏”访问网页。

“我的快捷栏”是浏览器中的一个自定义框架，再那里可保存频繁使用的项目，如最近的新闻和天气预报、您的通讯录或“在线好友”、股票报价、日历，以及其他很多项目。这些项目显示在“我的快捷栏”选项卡中，并且不断更新。

Netscape 显示一些已经设置的“我的快捷栏”选项卡，但您可以通过添加、删除和重新排列选项卡来自定义“我的快捷栏”。详情请参阅我的快捷栏。

要查看“我的快捷栏”中的一项，单击其选项卡。

我的快捷栏缩放柄

如果尚未打开，单击“我的快捷栏”的缩放柄将其打开。如果没有缩放柄，打开浏览器中的“查看”菜单，选择“显示／隐藏”，然后再单击子目录里的“我的快捷栏”。

快速搜索

一共有三种快速搜索的方法：从“地址栏”、“我的快捷栏”和“网络搜索”页面。

 

从“地址栏”搜索

搜索特定主题的网页很容易，只需在浏览器的地址栏内键入一个问题或只是一两个词，如下图所示。

例如，您要查找有关儿童玩具的信息：

  1. 双击地址栏以选择当前文字。
  2. 键入单词“儿童玩具”。您键入的词将替换掉当前文字。
  3. 按照以下步骤之一操作：
  • 单击“搜索”按钮。
  • 在地址栏下拉列表的底部单击“搜索 Netscape”以搜索有关“儿童玩具”的信息。（您的缺省搜索引擎可能会不同。这里用的是在个性设置中设置的缺省搜索引擎。关于“儿童玩具”的搜索结果将同时显示在浏览器窗口和我的快捷栏内。单击链接以访问有关儿童玩具的页面。

执行搜索之后，结果保存在我的快捷栏的搜索选项卡内，直到进行一项新的搜索为止。您不必单击“后退”按钮来获得搜索结果。

 

从“我的快捷栏”搜索

“我的快捷栏”选项卡可使您快速搜索并把搜索结果加入书签。例如，如果您需要查找有关玩具汽车的信息：

  1. 如果尚未打开，单击“我的快捷栏”的缩放柄将其打开。
  2. 单击“搜索”选项卡。
  3. 打开“使用”下拉列表，选择搜索引擎。
  4. 在搜索区域中键入“玩具汽车”。
  5. 单击“搜索”。“玩具汽车”的搜索结果同时显示在浏览器窗口和我的快捷栏内。单击链接可访问有关玩具汽车的页面。

备注：如果“我的快捷栏搜索”选项卡包含一个搜索引擎的列表以供选择，然后将搜索设置成了“高级”模式，而在单击“搜索”前需要选择一个或多个搜索引擎。可以以这种方式保留设置或改变“个性设置”中的基本设置。

提示：要快速显示搜索结果的下一页或当前页，单击在“我的快捷栏搜索”选项卡中的“前进”和“后退”按钮。

 

从“网络搜索”页搜索

“网络搜索”页可以使您键入一个搜索项或词组，或打开几个感兴趣的类别（如艺术和娱乐、商业和其它类别）。要访问网络搜索页，按照下列步骤之一操作：

  • 单击“个人工具栏”中的“搜索”按钮 。
  • 打开“工具”菜单选择“搜索互联网”。

 

在一网页中搜索选定的词

Netscape 可在一个网页中搜索您选定的词。

  1. 选择（突出显示）网页中的文字。
  2. 右击选择菜单上的“在互联网上搜索[您选定的词]”。（在 Mac OS 中右击，按下 Ctrl 键再单击鼠标。）

Netscape 打开一个新窗口，然后用缺省搜索引擎来搜索选定的词。有关如何改变用来搜索您选定的词的搜索引擎，请参阅 浏览器个性设置 - 互联网搜索。

高级搜索模式

可以把“我的快捷栏搜索”选项卡设置为高级模式，它可以缩小搜索范围，或者选择一个或多个搜索引擎。

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“浏览器”类别下，单击“互联网搜索”。（如果未见到子类别，双击“浏览器”即可展开子类别列表。）
  3. 在“快捷栏搜索选项卡个性设置”中，单击“高级”，然后单击“确定”。
  4. 如果尚未打开，单击“我的快捷栏”的缩放柄将其打开。
  5. 单击“我的快捷栏”上的“搜索”选项卡。
  6. 打开“位于”下拉列表，选择一个类别。
  7. 给选中的类别选择一个或多个搜索引擎（如果有可能的话）。
  8. 在搜索字段中键入要搜索的词。
  9. 单击“搜索”。

 

 

自定义搜索类别

您可以为不同的搜索类别指定使用的搜索引擎。例如，您可以选择一套搜索引擎搜索旅游信息，另一套搜索引擎搜索软件信息。

  1. 将“我的快捷栏搜索”设置为“高级搜索模式”。有关详细信息，请参见高级搜索模式。
  2. 如果尚未打开，单击“我的快捷栏”的缩放柄将其打开。
  3. 单击“我的快捷栏”上的“搜索”选项卡。
  4. 打开“位于”下拉列表，单击“编辑目录”。将看到“编辑类别”对话框。
  5. 从类别下拉列表中选择想要自定义的类别。
    或者
    要创建一个新的类别的话，就单击“新建”然后在对话框中键入类别的名称。下拉菜单列中即显示出选中类别的名称。
  6. 按照下列步骤之一操作：
    • 要为选中的类别添加搜索引擎，突出显示列表左边的搜索引擎，单击“添加”。
    • 要为选中的类别删除搜索引擎，突出显示列表右边的搜索引擎，单击“删除”。
    • 要重命名选中的类别，单击“重命名”然后键入一个新名称。
    • 要删除选中的类别，单击“删除”。
  7. 自定义完成后单击“确定”。

设置搜索个性设置

在缺省状态下，单击“搜索”时所使用的是 Netscape 搜索引擎。可以选择别的搜索引擎作为缺省引擎。您也可以指定在“我的快捷栏”上怎样显示搜索结果。

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“浏览器”类别下，单击“互联网搜索”。（如果未见到子类别，双击“浏览器”即可展开子类别列表。）
  3. 在缺省搜索引擎下，选择一个用于网络搜索的搜索引擎。
  4. 如果想要打开“我的快捷栏”搜索选项卡并显示搜索结果，在搜索结果下，选择“当搜索结果可用时打开快捷栏内的搜索”选项卡。
  5. 在“我的快捷栏搜索选项卡”个性设置下，如果想使用单个搜索引擎搜索，选择“基本”，或者选择“高级”以使用多个搜索引擎。

在页面内搜索

若需在当前浏览器浏览的页面中查找文本：

  1. 打开“编辑”菜单，然后选择“在本页面中查找”。如果浏览的页面包含框架，可能需要在搜索之前在一个框架内点击一下。您将看到“在本页中查找”对话框。
  2. 键入要查找的文字。
  3. 在下列搜索选项中选择：
    • 大小写匹配：选中它以搜索一个与您键入的词完全匹配的词或词组（包括大小写字母）。
    • 绕回查找：选择此项使浏览器从光标所在的位置搜索到页面底部，然后继续从页面顶部开始搜索。备注：通常，搜索页面时从最后点击鼠标处开始搜索到页面底部。
    • 向后搜索选择此项以便从插入点反向搜索到页面顶部。
  4. 单击“查找”以开始搜索。

要再次找到相同的词或词组：

  • 打开“编辑”菜单，选择“再次查找”。

搜索书签或历史列表

要搜索书签列表，从“浏览器”窗口开始。

  1. 打开“书签”菜单并选择“管理书签”。 即可看到“书签”窗口。
  2. 在“书签”窗口打开“工具”菜单，然后选择“搜索书签”。您将看到查找书签对话框。
  3. 使用下拉列表来选择选项以定义搜索。
  4. 键入搜索词。
  5. 单击“查找”。符合搜索标准的书签就会显示在“搜索结果 - 书签”窗口。

提示：要想快速打开“书签”菜单，请按 Ctrl＋B 键。

要搜索历史列表：

  1. 打开“转到”菜单，然后选择“历史”。即可看到历史列表。
  2. 在“历史”列表中，打开“工具”菜单，然后选择“搜索历史”。将看到“在历史中查找”对话框。
  3. 使用下拉列表来选择选项以定义搜索。
  4. 在下列搜索选项中选择：
  • 选择“标题”或“位置”以便选择在何处搜索。（书签窗口可用于搜索“名称”、“描述”和“关键字”。）
  • 如果您只知道要搜索的词或短语的一部分，选择“包含”、“起始字符”或“结束字符”。
  • 如果您确认正在搜索什么，选择“是”。
  • 选择“不是”或“不包括”来排除不需要的页面。
  • 对于需要查找或排除的书签或历史列表，在该字段中单击并键入全部或部分字母或 URL（网站地址）。
  1. 单击“查找”。符合搜索条件的历史列表显示在搜索结果窗口中。

要使用搜索结果：

  • 双击搜索结果窗口中的一个书签便可转到那个网页。

提示：

要保存整个页面，从“浏览器”窗口开始：

  1. 打开“文件”菜单，然后选择“页面另存为”。即可看到“另存为”对话框。

  3. 为要保存的页面选择一个格式：

    • 网页，仅 HTML：保存原页面，不带图片。这个选项在一个文件中保存 HTML 链接结构。
    • 文本文件：把原页面另存为一个文本文件。此选项不保存原网页的 HTML 链接结构， 但可在任何文本编辑器中看到网页的文本格式版本。

  5. 为图像键入一个文件名，然后单击“保存”。

若需在不显示页面的情况下进行保存（这在检索一个非格式化的不用于查看的页面（如数据文件）时非常有用）：

  1. 把光标定位到该页面的链接上。

  3. 选择“另存链接目标为”。将看到“保存文件”对话框。
  4. 选择一个文件夹以保存该文件。
  5. 为页面键入一个文件名，然后单击“保存”。

要点：一些链接在单击之后，可自动下载并保存文件到您的硬盘。这些链接的 URL 通常以“ftp”开始或以“au”或“mpeg”等文件类型扩展名结尾。这些链接可以传送软件、声音或电影文件，并可启动支持该文件的帮助器应用程序。

  • 转到页面设置：单击“页面设置”来进一步自定义要打印的页面。
  • 转到打印：单击打印即可打印页面。

在“页面设置”中，可以改变要打印页面的下列设置：

      • 缩小到页面宽度：选择此项可使页面自动缩小到纸张的宽度。

      • URL：显示网页的 URL （URL 通常由 http：// 开始）。
      • 日期／时间：显示打印网页时的日期和时间。
      • 第#页：显示每页的页码。例如
      • 第#页，共#页：显示页码和总页数。例如，当打印一个五页的网页时，在第三页上就会出现“第 3 页，共 5 页”。

        • &PT：页码和总页数（例如：“第 3 页，共 5 页”）
        • &P：页码
        • &D：日期

选择字符编码和字体

如果用一种以上的语言浏览、编写或收发电子邮件，您需要选择适当的字符编码和字体。

字符编码是一种转换方法，它可将文档或邮件转换为能被您的计算机使用的数据。所有的互联网文档和邮件以及新闻邮件都使用某一种字符编码方法（也叫字符集）。

文档的字符编码方法取决于其所用语言。有些语言（比如大多数西方语言）有共同的编码方法。其它如中文、日文和俄文，使用不同的方法。

您的 Netscape 版本把缺省字符编码设置为适合您所在地区的编码。然而，如果使用一种以上的语言，用户可能需要选择适当的字符编码方法，并指定用于编码方法的字体。

要选择字符编码，从“浏览器”窗口开始：

  1. 打开“查看”菜单，选择“字符编码”，然后选择“详细信息”。
  2. 从子菜单顶部选择一个地区。
  3. 在地区子菜单中选择一个字符编码。重复步骤 1-3 来设置每个想使用的字符编码方法。

选中的字符编码方法将添加到“字符编码”菜单。如果选中了一种以上的编码方法，正在使用的编码旁边有一个项目符号（点）。

Netscape 可检测出文档使用了哪种字符编码，并可把字正确的显示在屏幕上。要充分利用这个能力，从“浏览器”窗口开始：

  1. 打开“查看”菜单，选择“字符编码”，然后选择“自动检测”。
  2. 选择“自动探测”选项中的一个，或从子目录中选择“关闭”。

若需改变在用的字符集列表：

  1. 打开“查看”菜单，选择“字符编码”，然后选择“自定义”。将看到“自定义字符编码”对话框。
  2. 从下列选择：
    • 要添加到在用的字符集列表，选择列表左边的字符集并单击“添加”。
    • 要从在用的列表中删除某字符集，选择列表右边的字符集并单击“删除”。
    • 要改变浏览器检查字符集的顺序，突出显示列表右边的字符集，并使用箭头按钮在列表中上下移动字符集。

要改变语言组中的缺省字体：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“外观”类别下单击“字体”。（如果未见到子类别，双击“外观”即可展开子类别列表。）
  3. 从“字体用于”下拉列表中选择字符编码方法。例如要为西方字符集设置缺省字体，选择“西欧语族”。
  4. 选择比例文本应为 serif（如 Times Roman）还是 sans serif （如 Arial）。也可选择比例文本的字体大小。比例文本的宽度可以变化。
  5. （如果可用）可选择 Serif、Sans-Serif、Cursive 和 Fantasy（衬线、无衬线、草体和花体）等字体。
  6. 选择用于网页的等宽字体（如 Courier）。等宽字体的宽度不变，所以每个字符或字母占用相同的空间。

许多网页作者选择自己的字体和字体大小。您可通过设置“允许文档使用其它字体”来使用作者的字体设置。

要调整字体的可读性，从“点每英寸”（dpi）下拉列表中选择适合的分辨率显示网页。选择“其它”打开“校准分辨率”对话框，它可以通过测量屏幕上行的长度来校准分辨率。

管理不同的文件类型

浏览器可处理多种类型的文件。但是对有些文件来说，比如电影或音乐， Netscape 则需要能够处理这些文件的插件或者“帮助器”应用程序。如果 Netscape 没有所需的帮助器应用程序或者插件，它仍可以将这些文件保存到硬盘中。保存文件时，可以用“下载管理器”记录其踪迹。

 

插件

插件是扩大浏览器功能、在 Netscape 内部运行的帮助器应用程序。Netscape 程序可通过 Sun Java、Macromedia Flash 和 RealNetworks RealPlayer 等插件运行多媒体文件和小应用程序，比如电影、动画和游戏。

在安装 Netscape 时，就可以选择添加这些最流行的插件中的一部分。因此不用分别对它们进行安装。尽管如此，在安装了 Netscape 之后，仍然可以安装插件。若想查看 Netscape 当前安装了那些插件，请参阅线上网页 Netscape 插件管理器。

关于可以安装的插件完整清单，请参阅 Netscape 的浏览器插件页面。

 

帮助器应用程序

当文件不能在 Netscape 中使用时，可以选择启动帮助器应用程序在 Netscape 之外打开这些文件。例如，要运行 MP3 文件，Winamp 这样的程序就可以在 Netscape 外面打开。

浏览器可根据帮助器应用程序个性设置面板中的设置，确定使用哪种帮助器应用程序。这些个性设置已经预先配置妥当，通常无需改变。另外，如果浏览器不知道使用哪种帮助器应用程序，它会允许您在“下载”对话框内查找帮助器应用程序。

“下载”对话框有下列选项：

  • 使用 xxx 打开：Netscape 用系统设置中的缺省程序打开文件。
  • 选择：您可通过该选项在 Netscape 中用您选择的应用程序打开文件。
  • 将此文件保存到磁盘：您可通过该选项在 Netscape 中选择一个保存文件的位置。
  • 在打开此类文件之前始终询问：选定此项后， Netscape 在下载一个特定类型文件时打开“下载”对话框。可以随后在帮助器应用程序个性设置中重新设置这些选项。
  • 高级：您可通过该选项在 Netscape 中微调文件处理方法。请参见“添加新帮助器应用程序到列表”中的详细说明。

提示：可以清除个性设置中保存的“打开此类文件之前始终询问”选项。更多信息请参阅浏览器个性设置 - 帮助器应用程序。

对于要精细调整帮助器应用程序个性设置的有经验用户：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“浏览器”类别下，单击“帮助器应用程序”。（如果未见到子类别，双击“浏览器”即可展开子类别列表。）
  3. 执行任一下列操作：

    在列表中添加新帮助器应用程序：

    1. 单击“新建类型” 即可看到“新建类型”对话框。
    2. 键入应用程序处理的文件类型的描述；例如，“电影”或“声音文件”。
    3. 键入该文件类型使用的文件扩展名（后缀）。如果该文件类型使用多个扩展名，则全部键入，用空格分隔。
    4. 键入 MIME 类型。MIME 类型由斜杠分隔的两个词组成。例如，TIFF 图像文件的 MIME 类型为 image/tiff。
      备注：在下载文件时，Netscape 会把 MIME 类型显示在对话框中。
    5. 单击“选择”，然后选择处理该类文件的应用程序。
    6. 单击“确定”。

    要指定浏览器应如何处理下载的文件：

    1. 从列表中选择文件类型，单击“编辑”。
    2. 键入该文件使用的 MIME 类型。
    3. 选择“处理程序”选项把下载文件保存到硬盘或用指定的应用程序打开。单击“选择”指定应用程序。
    4. 单击“确定”。

    要从列表中删除一项：

    1. 从列表中选择文件类型，然后单击“删除”。

把 Netscape 设置成缺省浏览器

备注：本节讨论仅针对 Windows 的设置和功能。

Netscape 以其网页显示闻名，包括在互联网上的和在您的电脑上。要便捷地打开网页，应把 Netscape 设置成缺省浏览器。

 

一般互联网文件和协议

把 Netscape 设置成缺省浏览器使得它能够自动打开互联网上使用的一般文件格式和协议。一般互联网文件和协议包括：

  • 图像文件：

    • JPEG、GIF、PNG 和 MNG

  • 互联网文档和语言文件：

    • HTML、XML、XHTML 和 XUL

  • 互联网协议：

    • HTTP、HTTPS、FTP、 Chrome、Gopher

 

自动改变缺省浏览器设置

安装完成后，Netscape 检查其是否是一般互联网文件或协议的缺省浏览器。如果不是，它就会询问：“Netscape 当前未被设为缺省浏览器， 是否将其设为缺省浏览器？”

单击“是”即可把 Netscpape 设为缺省浏览器。如果单击“否”，每次 Netscape 启动时都会被问到这个问题，除非取消对“下次启动时还进行检查”复选框的选取。

如果取消对“下次启动时还进行检查”复选框的选取，仍可以通过改变个性设置中的设置把 Netscape 设为缺省浏览器。有关如何通过 Netscape 个性设置把 Netscape 设为缺省浏览器，请参阅浏览器个性设置 - 浏览器。

自定义缺省浏览器设置

可以手工自定义 Netscape 如何处理不同格式和协议。详情请参阅高级个性设置 - 系统。

改变高速缓存的设置

计算机把经常访问页面的副本保存到内存高速缓存或磁盘高速缓存中。这样计算机就可以不必在每次浏览时都从网络上检索页面了。

要设置内存高速缓存的大小或清除之（仅适用于 Windows 和 Unix 系统）：

  1. 打开“编辑”菜单并选择“个性设置”。
  2. 在“高级”类别下，选择“高速缓存”。（如果未见到子类别，双击“高级”即可展开子类别列表。）
  3. 在“内存高速缓存”字段键入一个数字，以指定内存高速缓存的大小。1024K 到 2000K 比较合适。要立即清除内存高速缓存，单击“清除内存高速缓存”。

要点：较大的高速缓存可以快速地检索更多的数据。但是，这要求您的计算机上有很多的额外内存。1024K 足矣。

要设置磁盘高速缓存的大小或清除之：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“高级”类别下，选择“高速缓存”。（如果未见到子类别，双击“高级”即可展开子类别列表。）
  3. 在“磁盘高速缓存”字段键入一个数字以指定磁盘高速缓存的大小。对于 Windows 而言，8000K 足矣（Unix，5000K）。若需立即清除磁盘高速缓存，单击“清除磁盘高速缓存”。

要点：大一些的磁盘高速缓存可以快速检索更多的页面,但会占用更多的硬盘空间。

当退出 Netscape 时，它会进行高速缓存维护。如果维护时间太长，试着减小磁盘高速缓存的大小。

要指定浏览器多长时间检查网络上的页面更新（不会使高速缓存中的页面过期）：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“高级”类别下，选择“高速缓存”。（如果未见到子类别，双击“高级”即可展开子类别列表。）
  3. 从下列选项中选择：
  • 每次查看页面时：如果想每次查看这个网页时都让让 Netscape 比较高速缓存和网页，就选择此选项。
  • 当网页过时时：如果希望网页被服务器认定为过期时让 Netscape 比较网页和高速缓存，请选择此项。
  • 每个会话比较一次：如果希望每次启动 Netscape 时都比较网页和高速缓存，请选择此项。
  • 不比较：如果不希望 Netscape 比较高速缓存信息和网络，请选择此项。

如果应位于高速缓存中的页面显示慢于正常速度，确保个性设置没有设置为“每次查看页面时”，因为验证需要花费时间连接网络。

要随时刷新页面：

  • 单击浏览器浏览工具栏上的“重新载入”按钮。计算机检查网络以确保您看到的是最新版本的页面。

使用快速启动

备注：“快速启动”只在 Netscape 的 Windows 版本中可用。

快速启动可缩短 Netscape 启动的时间。当 Netscape 使用快速启动时，Netscape 标志显示在任务栏内（靠近时间）。
Netscape 快速启动图标

提示：要快速启动部件（比如“编写器”窗口、“邮件和新闻组”窗口和“通讯录”窗口），右击 Netscape 徽标图标，然后在弹出菜单中选择部件。

安装 Netscape 时，您可以选择启用或禁用快速启动功能。如果启用了快速启动，当 Windows 首先启动（以及每次 Netscape 首先启动）时，“快速启动”功能将把 Netscape 的一部分载入内存。“快速启动”使得 Netscape 能够在关闭了所有的 Netscape 窗口后仍然驻留在内存中。这让 Netscape 能在您需要时快速启动，而不需载入全部的 Netscape。如果您的计算机内存不大，可以禁用“快速启动”以节省内存。


要在 Netscape 中启用或禁用快速启动：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 单击“高级”类别。
  3. 选择“启用快速启动”启用之，或取消选择以禁用之。
  4. 单击“确定”。

    提示：一种便捷地禁用“快速启动”的方法是：右击 Netscape 在系统通知区域（靠近时间显示）的徽标图标然后选择弹出菜单中的“禁用快速启动”。

代理服务器

 

本节内容包括：

设置代理服务器的值

本节解释如何使用代理服务器。

设置代理服务器的值

许多机构阻止在互联网上对其网络进行访问。这可防止了外部人员访问敏感信息。这种保护叫做防火墙。

如果您所在的机构设有防火墙，浏览器在连接到互联网之前，可能需要通过一个代理服务器。代理服务器可防止外部人员闯入您所在机构的专用网络。

准备工作：

  • 如果在您的工作地点有代理服务器配置文件，向系统管理员询问其 URL 地址。
  • 如果没有代理服务器配置文件，向系统管理员询问为每个网络服务运行代理软件的服务器的名称和端口号。

若需设置浏览器，以使其能使用代理服务器，请按下列步骤操作：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“高级”类别下，选择“代理服务器”。（如果未见到子类别，双击“浏览器”即可展开子类别列表。）
  3. 选择下列一项：
    • 直接连接到互联网：如果不想使用代理就选择此项。
    • 手工配置代理服务器：如果没有代理服务器位置（URL）就选择此项。
      • HTTP 代理、SSL 代理、FTP 代理、Gopher 代理： 键入代理服务器的名称或者数字 IP 地址。在“端口”字段键入端口号。
      • SOCKS 主机：键入代理服务器的名称或者数字 IP 地址。在“端口”字段键入端口号。
        • SOCKS v4, SOCKS v5：进入一 SOCK 主机时，选择“SOCKS v4”或是“SOCK v4H”，取决于代理服务器使用的 SOCKS 版本。
      • 对以下域名不使用代理：键入不愿意使用代理的域名。在每个域名之间用逗号隔开。（例如：.yourcompany.com, .yourcompany.co.nz)
    • 自动配置代理服务器 URL：如果有代理配置文件或 URL，选择此项然后键入配置 URL。

域名是 URL 的一部分，其中包含了机构、企业或学校的名称，例如 netscape.com 或 washington.org。如果使用不带域名的本地主机名，请以同样的方式列出。使用逗号分隔多个主机名。不能使用通配符。

有关使用代理服务器个性设置面板方面的详细说明，请参阅高级个性设置 - 代理服务器。
