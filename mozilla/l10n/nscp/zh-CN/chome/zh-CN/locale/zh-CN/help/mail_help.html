使用邮件帐户设置向导

要设置一个邮件帐户或新闻组帐户，首先打开“窗口”菜单并选择“邮件和新闻组”。如果您还没设置一个帐户，系统会自动显示“帐户向导”，使您能够设置一个帐户。

帐户向导会引导您完成创建新帐户的过程。如果您不了解某个设置，单击“取消”并询问您的互联网服务供应商（ISP）或帮助台。

如果已有一个帐户，则帐户向导不会在“邮件”窗口打开时自动出现。而在打开“邮件”窗口后，打开“文件”菜单并选择“新建”，然后选择“帐户”。详细说明，请参见设置另外的邮件和新闻邮件帐户。

 

设置 AOL 帐户或 Netscape 网际邮件帐户

美国在线（AOL）是许多最受欢迎的互联网服务供应商之一。Netscape 网际邮件是一种免费的邮件服务，它使用容易并在任何地方都能使用。

如果您已经有 AOL 或 Netscape 网际邮件帐户，则可用“帐户向导”设置 Netscape 的邮件和新闻组，以从该帐户存取邮件。从“编辑”菜单，选择“邮件和新闻组帐户设置”，然后单击“添加帐户”。

如果您还没有设立 Netscape 网际邮件帐户，则可通过单击文件夹窗格中的“免费网际邮件”图标注册一个免费帐户。

备注：在设置和使用 AOL 或 Netscape 网际邮件帐户之前，您必须安装 AOL 快信信使。（在缺省情况下，AOL 快信信使将在安装 Netscape 7.0 时一并安装）。

 

在 ISP 或电子邮件供应商处开立邮件帐户

在您设置邮件帐户之前，您的 ISP 或电子邮件供应商应该提供下列信息：

  • 您的用户名
  • 您的电子邮件地址
  • 接收服务器和发送邮件服务器的名称
  • 接收服务器的类型（IMAP 或 POP）

在开立新闻组帐户之前，您的 ISP 或电子邮件供应商应该提供下列信息：

  • 您的电子邮件地址
  • 新闻组服务器名
  • 帐户名称

要设置新的邮件帐户或新闻组帐户，请从“邮件”窗口开始：

  1. 打开“编辑”菜单并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。
  2. 单击“添加帐户”以启动“帐户向导”程序。

    帐户向导需要的信息取决于在其第一个窗口中所指定的新帐户的类型。在设置 ISP 或电子邮件供应商帐户时，您将对应于不同的窗口看到下列黑体字的标题。

  3. 新帐户设置：选择您要设置的帐户的类型，然后单击“下一步”。
  4. 标识：输入适合帐户的名字和电子邮件地址，然后单击“下一步”。
  5. 服务器信息：表明您想要 POP 帐户还是 IMAP 帐户。并非所有的服务供应商都同时支持这两个选项。更多的信息，请参见邮件和新闻组帐户设置 - 服务器设置。

    在这里还必须指定接收邮件服务器和发送邮件服务器（SMTP）的名字。这是发送邮件的服务器的名字（也称为 SMTP 主机）。单击“下一步”继续。

    备注：即使您有几个邮件帐户，您也只需要指定一个发送邮件服务器（SMTP）。在提供给您的帐户设置信息中，您的 SMTP 主机的名字可能没有明确的列出。例如，您的 SMTP 主机可能与 POP 或 IMAP 主机的名字相同。如有任何疑问，请与您的 ISP 或系统管理员联系。

  6. 用户名：输入 ISP 或电子邮件供应商提供给您的用户名，然后单击“下一步”。
  7. 帐户名：输入任意一个用于访问此帐户时您想要的名字，然后单击“下一步”。
  8. 恭喜！请检查键入的信息是否正确。必要时，请让您的 ISP 或系统管理员检查您键入的信息。当您确定输入的信息正确时，单击“完成”即可结束帐户的设置操作。
  9. 您的新帐户将在“邮件和新闻组帐户设置”对话框的左边列出。单击“确定”即可开始使用新帐户。

现在可以从您的帐户接收邮件了。有关详细说明，请参见获取新邮件。

设置另外的邮件和新闻组帐户

您可用“帐户设置”对话框添加新帐户，或更改现有帐户的信息，其中包括：

  • 邮件和新闻组服务器的设置（例如，删除邮件和下载个性设置）
  • 邮件副本和文件夹存储的设置
  • 您的回复地址、组织名称和签名

要添加新帐户或更改现有帐户的设置，请从“邮件”窗口开始：

  1. 打开“编辑”菜单并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。您可以执行以下任务：
    • 添加帐户：单击这个按钮设置一个新的邮件或新闻邮件帐户。确保正确无误地键入帐户信息。单击“下一步”或“后退”切换显示的窗口中的内容，或单击“取消”停止帐户创建。
    • 设置为缺省帐户：选择一个帐户，然后单击这个按钮使得选择的帐户出现在邮件窗口上的帐户列表的顶部。所做的改动在下一次打开“邮件和新闻组”时生效。

      缺省帐户是一启动“邮件和新闻组”时您想登录并（仅对于 IMAP 帐户）自动检查新邮件的帐户。（对于 POP 帐户，每次您必须单击“获取邮件”按钮来获取新邮件。）

    • 移除帐户：选择一个帐户，然后单击这个按钮以从“邮件”窗口完全移除该帐户。
    • 发送服务器（SMTP）：单击这个按钮（位于帐户列表的底部）后，您便可编辑有关发送邮件服务器的信息。参见邮件和新闻组帐户设置 - 发送服务器（SMTP）以获得更多信息。
  2. 单击任何帐户名字下面的标题并编辑相应的在面板右边设置。
  3. 单击“确定”以保存所做的更改。

更改帐户设置

要查看或更改一个现有邮件帐户或新闻组帐户的信息，请从“邮件”窗口开始：

  1. 打开“编辑”菜单并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。
  2. 单击“帐户设置”对话框左侧的帐户名。在对话框右侧可看见帐户信息，如您的电子邮件地址和签名文件。
  3. 单击一个帐户名下面的任何信息项来查看相应的设置：
    • 服务器设置：可用的设置取决于服务器的类型（IMAP、POP 或新闻组服务器）。更多的信息，请参见邮件和新闻组帐户设置 - 服务器设置。

      重要事项：如果您需要改变服务器的类型（例如，从 POP 改编为 IMAP），必须首先删除已有的帐户。接着，必须退出 Netscape 并重新启动。然后重新打开“邮件和新闻组帐户设置”对话框并通过单击“添加帐户”以重新创建一个具有新服务器类型的帐户。

    • 副本和文件夹：这些设置用于确定是否发送自动邮件（密件抄送）以及把出站邮件的副本、草稿和邮件模板存储于何处。更多的信息，请参见邮件和新闻组帐户设置 - 副本和文件夹。
    • 寻址：这些设置可覆盖在“个性设置”对话框中为所有通讯录指定的全局目录服务器的设置。更多的信息，请参见邮件与帐户设置 - 寻址。
    • 离线和磁盘空间（仅限 IMAP 和新闻邮件帐户）：这些设置适用于在离线状态（与互联网断开连接）下工作或需要节省下载时间和保留磁盘空间。更多的信息，请参见离线和磁盘空间设置（IMAP）或离线和磁盘空间设置（新闻邮件）。
    • 磁盘空间（仅限 POP 帐户）：这个设置决定了将下载到硬盘中的最大邮件的大小。更多的信息，请参见磁盘空间设置（POP）。
    • 安全：这些设置确定用哪些证书对发送邮件进行数字签名和加密。数字签名用于在您发送的邮件中向他人证明自己是可靠的。加密有助于保证您的邮件在互联网上传输时可保持其私密性。更多的信息，请参见邮件和新闻组帐户设置 - 安全。
  4. 单击“确定”以保存所做的更改。

获取新邮件

对于 IMAP 帐户，可以通过打开“邮件和新闻组”并为 IMAP 帐户选择“收件箱”而自动检索新邮件并在收件箱中显示。

对于 POP 帐户，必须选择收件箱并单击“获取邮件”来检索邮件。缺省情况下，来自 POP 帐户的邮件在检索后将从 POP 服务器上删除。可以改变 POP 服务器设置使得在将邮件下载到您的计算机中后，服务器上仍保存一个副本。

也可以把邮件和新闻组设置为启动时获取新邮件并以一定的时间间隔检查是否有新邮件。

  邮件和新闻组图标

状态栏上的“邮件和新闻组”图标显示为绿色箭头时，就是通知您有新邮件。

  新邮件通知

设置自动检查新邮件的邮件帐户时，请从“邮件”窗口开始：

  1. 打开“编辑”菜单并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。
  2. 如果拥有多个帐户，选择一个帐户并单击该帐户的“服务器设置”类别。
  3. 在“服务器设置”部分选择下面选项中的一个或两个都选：
    • 启动时检查新邮件：如果无论什么时候启动“邮件和新闻组”都自动检查这个帐户有无新邮件，则选中这个复选框。对于 POP 帐户，“邮件和新闻组”可检查新邮件但并不下载邮件。只有单击“获取邮件”或选择了“自动下载所有新邮件”才会下载邮件。
    • 每 ___ 分钟检查一次新邮件：如果您想指定检查新邮件所间隔的分钟数，则选择这个复选框。您仍然可以随时单击“邮件”窗口中的“获取邮件”来检查新邮件。
  4. 单击“确定”。
    所做的设置将在下一次启动 Netscape 邮件和新闻组时生效。

要设置 Netscape 邮件和新闻组以便在有新邮件时播放声音或显示提醒信息，请参见邮件和新闻组个性设置 - 邮件和新闻组。

可以随时以手动方式获取邮件。要获取选中帐户或新闻组的新邮件，执行下列其中一项：

  • 单击邮件工具栏上的“获取邮件”。
  • 打开“邮件”窗口中的“文件”菜单并选择“获取新邮件”。

要获取所有邮件帐户的新邮件，从“邮件”窗口开始：

  1. 单击邮件工具栏上的“获取邮件”按钮上的小三角形。
  2. 选择“获取所有新邮件”。Netscape 邮件和新闻组即为所有帐户检索新邮件。

    如果当前还未登录到您的某个邮件帐户，“邮件和新闻组”在获取该帐户的新邮件之前首先提示键入您的用户名和密码。（如果已经使用密码管理器保存了用户名和密码，“邮件和新闻组”将不再提示该信息。

备注：也可以打开“文件”菜单（在“邮件”窗口）并选择“获取用户新邮件”。

要获取指定邮件帐户的新邮件，从“邮件”窗口开始：

  1. 单击邮件工具栏中“获取邮件”按钮上的三角形。
  2. 选择要获取其邮件的帐户。

备注：第一次为某个帐户检索邮件时，“邮件和新闻组”会提示您输入密码。这时可以选择让“邮件和新闻组”在密码管理器中保存您的密码。

密码管理器可以将所有的用户名和密码保存在您的计算机中并在使用时自动输入。更多信息，请参见使用密码管理器。

保存和打印邮件

要将一邮件保存为纯文本、HTML 或 Outlook Express 文件：

  1. 在“邮件”窗口中选择该邮件。
  2. 打开“文件”菜单并选择“保存为”，然后选择“文件”。
  3. 对于“保存类型”，选择一个文件类型（“HTML”、“文本”或“邮件文件”）。如果想使保存的邮件能用 Microsoft Outlook 或 Outlook Express 打开，则选择“邮件文件”。
  4. 选择文件的扩展名使之以“.html”、“.txt”或“.eml”结尾，具体扩展名需视您在第三步中所选文件类型而定。
  5. 为文件选择一个目的路径并单击“保存”。

要打印选中的邮件：

  • 单击“打印”。

控制图像、脚本和插件

在缺省情况下，可以在收到的邮件中查看远程的图像。要避免下载嵌入网页中作为邮件附件发送的图像：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“隐私与安全”类别下，单击“图像”。（如果未见到任何子类别，双击“隐私与安全”即可展开子类别列表。）
  3. 单击“在邮件和新闻组邮件中不载入远程图像”。
  4. 单击“确定”使改变生效。

在缺省情况下，在收到的邮件中不启用 JavaScript 但启用插件。要改变这些设置：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“高级”类别下，单击“脚本和插件”。（如果未见到任何子类别，双击“高级”即可展开子类别列表）。
  3. 在“启用下列项的 JavaScript 功能”下，选中“邮件和新闻组”为在邮件中查看的网页启用 JavaScript。
  4. 在“启用插件”下，取消对“邮件和新闻组”的选中以禁用插件。
  5. 单击“确定”使改变生效。

编写邮件和新闻组邮件

通过进行下列操作之一，可以定位、编写、回复或发送一个新邮件：

  • 在任何 Netscape 窗口中，打开“文件”菜单并选择“新建”，然后选择“邮件”。
  • 单击邮件工具栏上的“编写”。
  • 显示邮件时，在邮件工具栏内单击“回复”、“转发”或“回复全部”。
  • 从“通讯录”窗口，选择一个地址并单击“在通讯录中编写”。

提示：使用“邮件和新闻组帐户设置”对话框来指定一种 HTML 文本编辑器，以便编写从此帐户发送的邮件（可以为每一个帐户指定不同的编辑器）。先选择帐户，然后选中“以 HTML 格式编写邮件”以便使用 HTML 文本编辑器来编写所有的邮件。请参见改变帐户设置以获得更多的信息。

用 HTML 格式编写邮件允许在邮件中使用不同的字体、文本样式（如粗体或斜体）和文本颜色、表格、编号或项目符号列表以及图片。然而，一些收件人只能阅读以纯文本格式编写的邮件。如果只是偶尔需要使用纯文本编辑器，可以在按住 Shift 键的同时，单击“编写”或“回复”按钮，即可在需要的时候使用纯文本编辑器。

使用邮件编写窗口

使用“编写”窗口来寻址、编写、和发送电子邮件和新闻组邮件信。首先在帐户设置个性设置面板（打开“编辑”菜单并选择“邮件和新闻组帐户设置”）内指定是以纯文本格式还是以 HTML 格式编写邮件。

要查看“编写”窗口，单击邮件工具栏上的“编写”按钮。

“编写”窗口包含以下各项：

  • 邮件工具栏

  • 可以单击以下按钮：

    • 发送：发送一封编写完的邮件。
    • 寻址：在通讯录中搜索名字。
    • 附加：在邮件上附加文件。请参见使用附件以获得更多的信息。
    • 拼写：检查邮件文本的拼写。
    • 安全性：显示有关信息，表明邮件是否将加密或数字签名（或两者同时）后发送。
    • 保存：将一邮件作为草稿保存。
  • 地址区：输入收件人地址的地方。
  • 附件区：当附加文件到一个邮件中（通过单击这个区域或通过单击“附加”按钮），相关的文件名将在“地址区”右边的“附件区”列出。
  • 邮件正文区：输入邮件内容的地方。

如果选择使用 HTML 编辑器编写邮件，则可见到另一个工具栏，其上带有与 Netscape 编写器中的按钮相似的一套文本格式按钮。

有关使用 HTML 编辑器的帮助信息，请参见格式化网页。

邮件寻址

要寻址邮件：

  1. 在地址区输入名字。

    2. 如果已经启用（缺省即为启用）自动填写地址，只需键入收件人名称的前几个字母，便可等待“邮件和新闻组”补充完成该地址的输入。（或在键入部分名称后立即按下 Enter 键，以让“邮件和新闻组”去尝试完成该地址的输入。

  2. 如果显示出多个地址，则选择一个地址并单击 Enter。

    3. 备注：在同一行使用逗号来分隔多个地址。不要用逗号来分隔姓和名。例如，多个地址可能是：

    user1@netscape.net,user2@netscape.net

  3. 如果想从另一个帐户发送此邮件，单击“发件人”字段来选择一个需要的帐户。请参见改变邮件的发送帐户以获得更多的信息。
  4. 必要时，可单击“收件人”来选择别的收件人类型：
    • 收件人：用于邮件的主要收件人。
    • 抄送：用于次要收件人（抄送）。
    • 密件抄送：用于不向其他收件人（包括“抄送”列表中的收件人）标识其身份的次要收件人（密件抄送）。
    • 回复地址：若需让收件人用另一个电子邮件地址回复，可选用此选项。
    • 新闻组：针对新闻组的邮递。
    • 跟踪地址：用于重定向新闻组邮递，这样随后的回复邮件直接转到到重定向的新闻组，而不是原新闻组。

提示：若需快速寻找一邮件的地址，可单击所读邮件中包含的电子邮件地址，然后从弹出的菜单中选择“将邮件发送至”。

改变邮件的发送帐户

如果有多个邮件帐户，在“发件人”字段列出的帐户是基于您在创建邮件时所选的帐户（或服务器）。然而，Netscape 邮件和新闻组也允许编写邮件时改变发送邮件的帐户。单击“发件人”字段查看帐户列表，然后选择所需帐户。在发送邮件所用帐户的“已发送邮件”文件夹中将保存邮件的一个副本。

关于自动填写地址

自动填写地址功能可使在“编写”窗口中很容易地为电子邮件添加地址而不必搜索名称或键入完整的地址。“邮件和新闻组”可自动检查您的通讯录和 LDAP 目录服务器（如果可用），而且如果找到匹配的信息，则自动完成地址的填写。如果找到了多个匹配项，则可显示带有附加信息的所有可能的选项，从而防止错误的发生。自动填写地址功能是缺省启用项。

如果不想使用“邮件和新闻组”提供的地址，可通过 Backspace 或 Delete 键删除填入的内容，然后另输入一个地址。

要禁用自动填写地址功能：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“邮件和新闻组”类别下，单击“寻址”。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。
  3. 在“自动填写地址”部分，取消对“本地通讯录”和“目录服务器”的选中。
  4. 单击“确定”。

选择邮件发送选项

编写邮件时，可在“选项”菜单中选择这些附加的邮件发送选项：

  • 选择地址：“选择地址”选项用于在通讯录或远程目录中选择收件人的电子邮件地址。要在通讯录或目录中查找一个地址，输入收件人的姓或名的前几个字母以开始搜索。选择一个地址，然后单击“收件人”、“抄送”或“密件抄送”：以确定邮件的发送地址。
  • 拼写检查：用于在发送邮件前检查邮件文本的拼写。也可以单击“拼写”。
  • 自动折行：如果正在使用纯文本编辑器编写一个邮件，可以使用“自动折行”命令使引用文本中的很长的行能够自动折行以适应“编写”窗口的大小。这个命令将选中的引用文本自动换行为指定字符数的行，这个字符数可在编写个性设置中指定。当原始邮件被引用在回复邮件中，且包含很长的行时，这个命令很有用。

    若需指定使用纯文本编辑器编写邮件，则须用“编辑”菜单上的“邮件和新闻组帐户设置”命令进行指定。在“帐户设置”对话框内，选择该帐户，然后取消对“以 HTML 格式编写邮件”的选中以使用纯文本编辑器编写所有邮件。如果只是偶尔需要使用纯文本编辑器，可以在按住 Shift 键的同时，单击“编写”或“回复”按钮，即可在需要的时候使用纯文本编辑器。

  • 回执：选中这个选项就是要求在收件人显示（打开）邮件时，返回一个确认信息。记住，收件人可以选择不发送回执。这个选项用于启用或禁用针对每个邮件的回执请求。要自动为发送的邮件要求回执，可使用该回执个性设置。请参见邮件和新闻组个性设置-回执以获得更多的信息。
  • 格式：将邮件作为纯文本或 HTML（已格式化）发送，或两种都用。如果选择“自动检测”，则在不知道收件人的邮件程序是否能够显示 HTML 邮件时，邮件和新闻组会询问要使用的格式。这里选择的格式将覆盖您在“编辑”菜单中使用“个性设置”命令所指定的发送格式。
  • 优先级：可选择的邮件优先级有：最低、低、普通、高或最高。
  • 将副本发送至：如果想把发送的邮件的副本归档在其他文件夹中而不是缺省的“已发送邮件”文件夹中，则选择此项。然后选择需要的文件夹。
  • 安全性：您可通过此项来改变邮件的缺省安全性选项。

回复邮件

要回复一个邮件：

  • 选择该邮件。
  • 若只回复发件人，请单击“回复”。
  • 若需回复邮件中的所有地址，请单击“回复全部”。

若需每次回复邮件时包含原邮件并指定如何在回复中放置原邮件：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“邮件和新闻组”类别下单击 编写。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。
  3. 选择“回复时自动引用原邮件”。
  4. 指定在邮件何处放置回复。缺省设置为“在引用文本的上面开始编写回复邮件”。
  5. 单击“确定”。

转发邮件

当转发一邮件时，可以相对于原文本指定新文本放置在何处：直接引用 （在邮件的正文中，这是缺省设置）或附件。

转发邮件：

  1. 选择邮件并单击“转发”。
  2. 键入收件人的名称或电子邮件地址。
  3. 单击“发送”。

设置转发邮件的缺省值：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“邮件和新闻组”类别中，单击编写。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。
  3. 对于转发邮件，可选择直接引用（于邮件正文中）或附件。
  4. 单击“确定”。

提示：若需越过缺省的转发邮件设置，可选择一个邮件，打开“邮件”菜单，然后选择“另转发为”，然后单击“直接引用”或“附件”。

确认邮件已打开

您可通过回执功能在收件人阅读（打开）邮件时通知您。收件人必须使用支持“邮件去向通知”（MDN）标准的邮件程序。请记住，收件人可以选择不给您发送回执，尽管您有这样的要求。发送到新闻组地址的邮件不会包含回执请求，因为新闻组服务器不支持这个功能。

若需为所有发送的邮件请求回执，可以启用全局的“回执”个性设置。也可以使用“回执”个性设置指定如何管理收到的回执请求。对于单独的帐户，您可越过这些全局的个性设置。

若需单独针对每封邮件请求回执：

  • 从“邮件编写”窗口内打开“选项”菜单，然后选择“回执”。

若需在您的每个帐户发送邮件时自动请求回执：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“邮件和新闻组”类别中，单击回执。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。
  3. 选择“发送邮件时，总要求提供回执”。
  4. 单击“确定”。

有关回执的个性设置方面的详细说明，请参见邮件和新闻组个性设置 - 回执。

保存和编辑邮件草稿

如果需将邮件作为草稿保存，以便以后再用：

  • 在编写窗口中，单击“保存”，或打开“文件”菜单并选择“保存为草稿”。缺省情况下，该邮件保存在当前帐户的“草稿”文件夹内。

    备注：在将邮件保存为草稿后，邮件仍然是打开的。

若需编辑或发送邮件草稿，可从“邮件”窗口开始：

  1. 单击创建草稿时所用帐户中的“草稿”文件夹。
  2. 单击要编辑的邮件。
  3. 在邮件的右上角，单击“编辑草稿”按钮。
  4. 按需要编辑邮件。
  5. 单击“发送”可发送该邮件，或单击“保存”以保存该邮件，以便以后再继续编写。

  6. 备注：邮件一经发送即被从“草稿”文件夹中移除。

提示：也可以双击邮件以打开进行编辑。当邮件窗格关闭时，这一功能尤其有用。

若需删除一个或多个不想要的邮件草稿，可从“邮件”窗口开始：

  1. 单击创建草稿时所用帐户内的“草稿”文件夹。
  2. 选择想删除的邮件草稿。
  3. 单击邮件工具栏上的“删除”。

创建和使用模板

模板是用来为日常发送的邮件（如每周的形势报告）设置缺省格式的。可以在任何显示邮件的窗口中把邮件保存为一个模板，在邮件编写窗口中也可如此。

要保存作为模板使用的邮件：

  1. 在“邮件”窗口中，单击“编写”创建一个新邮件，然后设置缺省的字体、字号、文本颜色、背景颜色和其他一些需要指定的缺省设置。

    2. 也可打开一个现有的并且有您想要的格式的邮件。

  2. 邮件开启后，打开“文件”菜单，选择“另存为”，然后选择“模板”。在当前邮件帐户的“模板”文件夹中，该邮件将被存为模板。

使用模板编写邮件：

  1. 在“邮件”窗口中，在创建邮件模板时所用的帐户中选择“模板”文件夹。
  2. 双击该邮件模板将其打开。
  3. 编辑邮件，然后便可保存（存入“草稿”文件夹）或发送。

  4. 备注：发送邮件并不会从“模板”文件夹中删除模板。模板将保留，供以后使用。

若需删除一个或多个不想要的邮件模板，请从“邮件”窗口开始：

  1. 单击在创建邮件模板时所用帐户中的“模板”文件夹。
  2. 选择想要删除的模板。
  3. 在邮件工具栏单击“删除”。

在邮件中使用 HTML

HTML 邮件可包括带有格式的文本、链接、图像和表格，就像网页一样。然而，某些收件人可能无法接收 HTML 邮件。您可通过 Netscape 邮件和新闻组程序，在您的每个邮件帐户中，用 HTML 格式（多功能文本）编辑器或纯文本编辑器编写邮件和新闻组邮件。另外，您还可选择您的收件人在缺省情况下是接收 HTML 格式的邮还是接收纯文本格式的邮件，并在不知道收件人是否可以接收 HTML 格式邮件的情况下，邮件和新闻组程序应该如果处理邮件。

若需指定是否在缺省情况下使用 HTML 编辑器编写邮件，请从“邮件”窗口开始：

  1. 打开“编辑”菜单，并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。
  2. 选择要使用的邮件帐户或新闻组帐户。
  3. 选择“以 HTML 格式编写邮件”。此时，您将在“编写”窗口中看到格式工具栏。如果该帐户使用纯文本编辑器，则不要选中该复选框。

编辑和插入 HTML 元素

如果您知道任何编写 HTML 源代码，则可在邮件中编辑或插入附加的 HTML 标记、样式属性和 JavaScript。如果对使用 HTML 源代码并无把握，最好不要对其进行更改。若需使用 HTML 代码，请用以下其中一种方法：

  • 将插入点置于预期位置处，然后打开“插入”菜单并选择 HTML。在“插入 HTML”对话框中，输入 HTML 标记和文本，然后单击“插入”，以插入改变的内容。
  • 选择要编辑的 HTML 源代码，然后打开“插入”菜单并选择 HTML。在“插入 HTML” 对话框中，编辑 HTML 标记和文本，然后单击“插入”，以插入改变的内容。
  • 选取一个诸如表格、命名锚点、图像、链接或水平线之类的元素。双击该元素以打开相关的属性对话框。单击“高级编辑”，打开“高级属性编辑器”。可以使用“高级属性编辑器”为对象添加 HTML 属性和 JavaScript。

选择 HTML 邮件发送选项

按照缺省设置，“邮件和新闻组”在不知道收件人的邮件程序是否能显示 HTML 格式的邮件时，它会在发送邮件之前给出提示。

要选择邮件的发送格式选项，请从“邮件”窗口开始：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“邮件和新闻组”类别下，单击“发送格式”。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。

    备注：此个性设置只能用于邮件，不能用于新闻组邮件。

  3. 选择需要的选项，并单击“确定”。

如果在编写邮件时意识到一个或更多的收件人可能无法接收 HTML 格式的邮件，可以在单击“发送”时很容易的把该邮件转换为别的格式：

  1. 在编写窗口内，打开“选项”菜单并选择“格式”。
  2. 从子菜单中选择发送邮件想用的格式：
    • 自动检测：“邮件和新闻组”功能可为邮件选择适当的格式。如果不能确定邮件的格式，它会提出询问让您选择一种格式。
    • 仅纯文本：尽管邮件可能不能显示如粗体这样的格式，但所有邮件程序都能显示这种邮件。
    • 仅多功能文本（HTML）：一些邮件程序在显示 HTML 格式的邮件时可能会有问题。只有在可以确定收件人的邮件程序能显示 HTML 格式的邮件时才可选择此项。
    • 纯文本或多功能（HTML）文本：这种文本需使用更多的磁盘空间，但在不能确定收件人的邮件程序是否能显示 HTML 格式的邮件时，这是最好的选择。
  3. 编写邮件结束后，单击“发送”。

指定 HTML 邮件的收件人

为了节省时间，可以标明通讯录中的人是愿意接收 HTML 邮件还是纯文本邮件。

关于通讯录

通讯录可保存接收您邮件的人（如同事、朋友和家庭）的电子邮件地址和联系信息。Netscape 邮件和新闻组提供了两个通讯录：个人通讯录和收集通讯录 - 当然还可以自己创建另外的通讯录。也可以从其他的邮件程序和以前版本的 Netscape 导入通讯录。这些通讯录的内容都保存在本地硬盘中。

通讯录也可以列出位于 LDAP 目录服务器上的 LDAP 目录中的电子邮件地址。目录服务器可存储不在本地通讯录中存储的那些人的电子邮件地址。LDAP（轻型目录访问协议）是一种用于访问互联网或内部网的目录服务（如公司通讯录服务）的工业标准方法。

个人通讯录

您可通过“个人通讯录”选择添加具体人的名字。可创建邮递列表并编辑各个地址条目。如果设有 Netscape 网际邮件或 AOL 帐户，还可使个人通讯录中的条目与网际邮件或 AOL 通讯录中的条目同步。

收集地址

“收集通讯录”可自动收集入站邮件和出站邮件中包含的电子邮件地址。对于入站邮件，当打开每封邮件时，“收集通讯录”都将保存其中发件人的地址。对于出站邮件，邮件程序会在单击“发送”时存储其地址。也可以通过更改电子邮件地址收集个性设置来选择要收集哪些地址（仅限邮件中的地址和邮件和新闻组邮件中的地址）。

LDAP 目录（如果可用）

LDAP 目录（也称地址查找服务）可存储不在本地通讯录中存储的那些收件人的电子邮件地址。LDAP 目录可向您提供访问中心维护的大型电子邮件地址数据库之便利，这对自动填写地址功能特别有用。

缺省设置为启用“自动地址收集”。若需改变自动地址收集的设置，请从“邮件”窗口开始：

  1. 打开“编辑”菜单，然后选择“个性设置”。
  2. 在“邮件和新闻组”目录下，单击“添加地址”。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。
  3. 在“电子邮件地址收集”下，对下列选项做出是或否的选择：


    • 将该功能用于入站邮件或出站邮件，或两者都使用。
    • 将该功能用于新闻组邮件。
    • 限定收集通讯录的大小。缺省大小为 700 张卡片。如果添加新卡片时超出了限值，最久的卡片被删除，添加新卡片，从而保持卡片总数不变。
  4. 单击“确定”。

搜索通讯录和目录

Netscape 邮件和新闻组使您可以通过名字或电子邮件地址快速搜索通讯录或目录，或使用组合条件在通讯录或目录中执行更有针对性的搜索。

要在通讯录或目录中快速搜索一个名字或电子邮件地址，请从“通讯录”窗口开始：

  1. 在“通讯录”窗口，从通讯录列表中选择要搜索的通讯录或目录。
  2. 在“名字或电子邮件包含”字段，键入想查找的名字或电子邮件地址。可以只键入名字或电子邮件地址的一部分，也可键入要查找的精确文本。

    一旦键入完成， Netscape 邮件和新闻组会变为只显示那些包含输入的搜索文本的名字或电子邮件地址的条目。

  3. 单击“清除”以清除搜索文本并恢复显示所有的条目。

 

搜索指定的条目

可以在通讯录或目录中搜索指定的条目。如果还没有看到“高级通讯录搜索”对话框，从“通讯录”窗口开始：

  1. 打开“工具”菜单并选择“搜索地址”。即可看到“高级通讯录搜索”对话框。
  2. 在“在以下通讯录中搜索”后面，选择要搜索的通讯录或目录。
  3. 选择“邮件和新闻组”用来搜索条目的匹配选项，即“匹配全部条件（标准）”或“匹配选择条件（标准）中的至少一个”。

添加和删除 LDAP 目录

添加 LDAP 目录到通讯录允许您在目录中搜索电子邮件地址和其他联系人信息。 LDAP 目录还可用于在邮件寻址时自动填写地址。

通常应遵照系统管理员提供的指示来添加或删除 LDAP 目录。请与您的系统管理员核对添加新目录到通讯录中所需要的信息。

要添加一个新目录，从“通讯录”窗口开始：

  1. 打开“文件”菜单，然后选择“新建”，然后选择 “LDAP 目录”。即可看到“目录服务器属性”对话框。
  2. 在“目录服务器属性”对话框的“常规”选项卡中键入下列信息：


    • 名称：输入目录服务的名称（例如，InfoSpace Directory）。
    • 主机名：输入主机名服务器的名称，例如 ldap.infospace.com。
    • 基本 DN：这个设置用于设置基本判别名。输入代码以将搜索限定在指定的国家或组织。例如，c=JP 限定只在日本进行搜索。基础 DN 还可在目录中指定要在其中进行搜索的组织（例如，o=Netscape Communications Corporation，c=US）。
    • 端口号：输入 LDAP 服务器的端口号。缺省值为 389。
    • Bind DN：用于 LDAP 服务器认证（登录）时使用的判别名。如果空着不填，则 LDAP 服务器将以匿名绑定。
    • 使用安全连接（SSL）：如果您的 LDAP 服务器支持安全（加密）连接则选择此设置。 如果不能确定，请与您的系统管理员联系。
  3. 单击“高级”选项卡，配置 LDAP 目录服务器设置。
  4. 键入下列信息：


    • 返回数量不超过：这个设置用于限制目录服务器返回的用于自动填写地址的匹配结果的数量。输入为自动填写地址而显示的电子邮件地址匹配项的最大数量。
    • 范围：用于定义搜索界限：选择以下选项之一：


      • 一级：通过搜索基本 DN 和基本 DN 的下一级来检索匹配条目。
      • 子树：通过搜索基本 DN 和基本 DN 下的所有级来检索匹配条目。这是最小限制性搜索。
    • 搜索过滤器：输入应用于指定搜索范围内匹配结果的搜索过滤器。
  5. 单击“确定”，关闭“目录服务器属性”对话框。
  6. 单击“确定”，关闭“LDAP 目录服务器”对话框。
  7. 单击“确定”，关闭“个性设置”对话框。

您添加的目录出现在“通讯录”窗口中的通讯录列表中。

要删除一个目录：

  1. 打开“编辑”菜单，选择“个性设置”。即可看到“个性设置”对话框。
  2. 在“邮件和新闻组”类别下，选择“寻址”。（如果未见到任何子类别，双击“邮件和新闻组”即可展开子类别列表）。
  3. 在对话框右侧的“自动填写地址”下，单击“编辑目录”。
  4. 在“LDAP 目录服务器”对话框中，选择要删除的目录，然后单击“删除”。
  5. 单击“确定”，然后再单击“确定”，关闭“个性设置”对话框。

有关下载或同步可离线使用的目录（以便在离线工作状态时仍可搜索和使用其条目进行通讯录的自动填写地址）方面的说明，请参见下载目录条目以便离线使用。

与其他用户共享文件夹（仅限 IMAP）

使用 IMAP 邮件帐户的用户可以与在同一网络中的其他用户共享邮件夹。共享文件夹允许一些用户阅读和操作同一批邮件，类似于新闻组。要使用共享文件夹，您的 IMAP 邮件服务器必须支持“访问控制列表”（ACL）管理。如果不能确定您的 IMAP 邮件服务器是否支持共享文件夹，请咨询系统管理员或帮助工作台。

要与网络中的其他用户共享邮件夹，或要查看文件夹的共享信息，从“邮件”窗口开始：

  1. 在一个 IMAP 帐户中，选择想共享的文件夹，或选择要查看共享权限的文件夹。

    列在“本地文件夹”或 POP 邮件帐户下文件夹不能共享。

  2. 打开“编辑”菜单，然后选择“文件夹属性”。
  3. 单击“共享”选项卡。
  4. 单击“特权”。系统会提示您输入网络用户名和密码。

    “特权”按钮只有 IMAP 邮件服务器允许设置共享权限时才可用。如果这个按钮不可用，仍可以查看这个文件夹的共享权限但不能修改。

  5. 按照屏幕上的说明可以添加用户及设置他们的文件夹访问权限。


    • “读”权限：用户可以阅读邮件和复制其内容，但不能修改或删除邮件，或从其他地方复制邮件到这个文件夹。用户可将邮件标记为已读或未读。请参见为邮件加标记或标志 中有关标记邮件的说明。
    • “读写”权限：除了“读”权限之外，用户可以修改和删除邮件。用户也可以复制和移动邮件到这个文件夹。
    • “管理“权限：除了“读写”权限之外，用户还可以添加和删除用户及他们对文件夹的权限。
  6. 单击“确定”以确认更改。
  7. 单击“确定”以关闭“文件夹属性”对话框。

在您的邮件帐户的文件夹列表中，共享文件夹将显示为一个与众不同的图标以说明它是共享的。

    只有共享同一个网络的邮件收件人能订阅您的共享文件夹。

  6. 单击“发送”。

创建邮件过滤器

邮件过滤器使您能够管理和整理邮件。您可创建邮件过滤器，以使 Netscape 邮件和新闻组可依据指定的标准对入站邮件自动执行特定的操作。例如，您可创建一个可自动将入站邮件移动到一个指定文件夹的邮件过滤器。邮件过滤器的操作是区分帐户的。

如果尚未打开“邮件过滤器”对话框，请从“邮件”窗口开始：

  1. 打开“工具”菜单并选择“邮件过滤器”。即可看到“邮件过滤器”对话框。
  2. 如果有多个邮件帐户，请选择要应用该过滤器的帐户。
  3. 单击“新建”。用“过滤器规则”对话框指定需过滤的邮件类型，以及想要过滤器执行的操作。
  4. 输入过滤器的名称。
  5. 选择您想让邮件程序使用的匹配选项：“所有下列条件”（标准），或“至少一个”条件。
  6. 用下拉列表选择搜索标准（例如“主题”、“发件人”、“包含”、“不包含”），然后输入要匹配的文本或短语。

    提示：要搜索一个其标题没有在第一个下拉菜单中列出的邮件（例如，您想搜索一个包含“Resent-From”标题的邮件），请选择“自定义”并键入您想搜索的标题。Netscape “邮件新闻组”将自定义的标题添加到下拉列表中，于是以后就可以选择它来搜索匹配条目。一定保证您键入的自定义标题的正确性，因为邮件将只搜索严格与键入的标题相匹配的条目。

  7. 单击“更多”添加标准，“更少”删除标准。
  8. 用下拉列表选择想要过滤器对邮件执行的操作（例如，“移动到文件夹”）。

    提示：要自动给入站邮件加上标签，从下拉列表中选择“为邮件加标签”。

  9. 选择要在其中存储邮件的目标文件夹，或创建新文件夹。
  10. 单击“确定”以确认您的设置。
  11. 在“邮件过滤器”对话框中单击“确定”。您单击了“确定”后过滤器立即开始过滤入站邮件。

要管理过滤器，从“邮件”窗口开始。

  1. 打开“工具”菜单并选择“邮件过滤器”。即可看到“邮件过滤器”对话框。
  2. 如果有多个邮件帐户，请选择要应用该过滤器的帐户。
  3. 从以下项中选择：
    • 打开或关闭过滤器：单击过滤器名字右边的复选框以启用之，或再单击一次关闭之。
    • 编辑过滤器：选择过滤器名字并单击“编辑”（或双击过滤器名字）。使用“过滤器规则”对话框来修改过滤器。
    • 删除过滤器：选择一个过滤器名字并单击“删除”。
    • 改变过滤器的应用顺序：在过滤器列表中，单击过滤器名字，然后单击“上移”或“下移”来移动。

      • 备注：过滤器按选择的顺序依次应用于每一个入站邮件，直到发生这样的情况为止：某过滤器操作的结果使得邮件被删除或移动到了收件箱文件夹。

  4. 完成过滤器管理后，请单击“确定”。如果创建了新的过滤器，则单击“确定”后，就开始过滤入站邮件。

备注：如果您删除一个用来存储过滤邮件的文件夹，过滤器将不再起作用。匹配过滤器标准的来件将出现在收件箱中。如果重命名或移动该文件夹，则过滤器将自动更新以使用该重命名或移动的文件夹。

提示：过滤器不能用于已存在的邮件。如果想将现有邮件移动到另一个文件夹，可搜索这些邮件，然后将它们归档到其他文件夹。

过滤指定发件人的邮件

可以快速创建一个针对指定发件人的邮件的过滤器。例如，如果您想自动地把从孩子的老师那里收到的所有邮件移动到一个叫“学校”的文件夹中，就可以快速设置一个过滤器来做这件事情。

要创建一个针对指定发件人的邮件的过滤器，从“邮件”窗口开始：

  1. 选择指定发件人的一个邮件。
  2. 打开“邮件”菜单，然后选择“从邮件创建过滤器”。即可看到“过滤器规则”对话框。使用发件人的邮件地址， Netscape 预填一个名字，过滤器匹配条件，以及过滤器动作（移动到文件夹）。
  3. 选择一个存储收到指定发件人邮件的目标文件夹，或新建一个文件夹。
  4. 单击“确定”以确认您的设置。即可看到邮件过滤器对话框，在那里可以创建、删除或编辑邮件过滤器。
  5. 单击“确定”。
    单击“确定”后，过滤器就开始过滤从指定发件人处入站的邮件。

搜索邮件文本

Netscape 邮件和新闻组可在单个邮件中快速查找文本信息，通过主题或发件人搜索邮件，或使用一个联合的条件执行从指定的邮件夹、新闻组或帐户彻底把所有邮件搜索一遍的操作。

要在一单个邮件中查找文本，从“邮件”窗口开始：

  1. 选择邮件，打开“编辑”菜单，然后选择“在此邮件中查找”。
  2. 在对话框中键入想查找的文本。
  3. 单击“查找”以定位到文本的第一次出现。
  4. 继续单击“查找”找到其他匹配，或在完成后单击“取消”。
  5. 从“编辑”菜单中选择“再次查找”，以继续在该邮件的其余部分中搜索该文本。

要在一个选定的文件夹中通过主题或发件人快速查找一个邮件，从“邮件”窗口开始：

  1. 在“主题或发件人包含”的右边，键入想查找主题的文本或发件人的名字。您可以只键入主题文本或发件人名字的一部分，也可以键入想查找的准确文字或名字。

    键入一结束，Netscape 邮件和新闻组只显示选定文件夹中的其主题或发件人包含输入文本的邮件。

  2. 单击“清除”以擦除搜索文本并在文件夹中恢复显示所有邮件。

 

搜索指定的邮件

可以在邮件夹或新闻组中搜索指定的邮件。如果尚未看到“搜索邮件”对话框，从“邮件”窗口开始：

  1. 打开“工具”菜单并选择“搜索邮件”。即可看到“搜索邮件”对话框。
  2. 在“在以下目标中搜索邮件：”中，选择想在其中查找邮件的帐户、新闻组或文件夹。
  3. 选择“搜索子文件夹”以便在搜索时包含所有的子文件夹。
  4. 选择“邮件和新闻组”用来搜索条目的匹配选项，即“匹配全部条件（标准）”或“匹配选择条件（标准）中的至少一个”。
  5. 使用下拉列表说明搜索条件（例如“主题”或“包含”）然后键入您想匹配的文本或短语。

    提示：要搜索一个其标题没有在第一个下拉菜单中列出的邮件（例如，您想搜索一个包含“Resent-From”标题的邮件），请选择“自定义”并键入您想搜索的标题。 Netscape “邮件新闻组”将自定义的标题添加到下拉列表中，于是以后就可以选择它来搜索匹配条目。一定保证您键入的自定义标题的正确性，因为邮件将只搜索严格与键入的标题相匹配的条目。

  6. 单击“更多”添加标准，“更少”删除标准。
  7. 单击“搜索”开始，或单击“清除”重新设置输入。搜索结果将出现在“搜索邮件”对话框的下边。
    • 要打开邮件阅读，可选择邮件并单击“打开”，或双击该邮件。
    • 要以不同顺序排序邮件，可单击要按其排序的列。
    • 要将“结果”区域中的邮件移动或复制到另一文件夹，可选择该邮件，然后从“文件”下拉列表中选择目标文件夹。如果目标文件夹位于同一帐户中，则邮件将移动到该文件夹。如果目标文件夹位于不同帐户，则邮件将复制到该文件夹。
    • 要从“结果”区域删除一个邮件，选择邮件并单击“删除”。
    • 要打开存储邮件的文件夹，选择邮件并单击“打开邮件夹”。

订阅新闻组

如果已在新闻组服务器上建立了帐户，就可以加入（订阅）新闻组（也称为讨论组）。

要订阅新闻组， 从“邮件”窗口开始：

  1. 打开“文件”菜单，选择“订阅”。将出现“订阅”对话框。
  2. 如果需要，单击“帐户”下拉列表，选择其他的新闻组帐户。
  3. 选择新闻组。要选择多个新闻组， Ctrl+单击（Windows 和 Linux）或 Command+单击（Macintosh OS）每个要添加的新闻组。
  4. 单击“订阅”，或在新闻组旁边的“订阅”列中单击。在所订阅的新闻组旁边会出现一个复选标记。单击“撤销订阅”，取消对某个订阅选择。
  5. 单击“确定”。
    订阅的新闻组列表显示在“邮件”窗口中。

如果是 IMAP 邮件用户，还可订阅 IMAP 服务器上的邮件夹。（“收件箱”是一种邮件夹。） 按照上面介绍的方法订阅，但从“帐户”下拉列表中选择一个帐户。有关共享文件夹或订阅文件夹的更多信息，请参见与其他用户共享文件夹（仅限 IMAP）。

监控论题

要监控感兴趣论题中的未阅读邮件：

  1. 选择论题中的一个邮件。
  2. 打开“邮件”菜单，然后选择“关注论题”。
  3. 如果要监控其他论题，对其他论题中的邮件重复步骤 1 和步骤 2。
  4. 当准备好监控这些论题中的邮件时，打开“查看”菜单，选择“邮件”，然后选择“有未读邮件的关注论题”。Netscape 邮件和新闻组只显示包含未读邮件的被关注的论题。
  5. 打开“查看”菜单，选择“邮件”，然后选择“全部”以查看该新闻组中的全部邮件。

要忽略一个邮件论题：

  1. 选择该论题中的一个邮件。
  2. 打开“邮件”菜单，然后选择“忽略论题”。Netscape 邮件和新闻组 将该论题中的所有邮件标记为已读，发布到该论题的新的回复将显示为已读。
  3. 要查看忽略的论题，在“查看”菜单中，选择“邮件”，然后选择“忽略论题”。

设置 Netscape 邮件和新闻组以便在离线状态下工作

Netscape 邮件和新闻组的离线功能可用来下载邮件并离线阅读（与互联网断开连接的状态下阅读）。如果使用拨号（调制解调器）连接访问电子邮件并想减少连接持续的时间，或者因外出或改变位置而需要暂时断开与公司网络的连接，则可以下载电子邮件以便在离线状态下阅读。离线功能可在重新连接上之后自动下载入站邮件并随后发送要发出的邮件。

如果偶尔想在离线状态下工作，Netscape 邮件和新闻组将使您能方便地：

  • 下载“收件箱”供离线使用。
  • 下载单个文件夹供离线使用。
  • 下载选中的或有标志的邮件供离线使用。
  • 在通讯录中下载目录条目供离线使用。

如果经常离线工作，Netscape 邮件和新闻组也可让您：

  • 将一个或多个帐户设置为离线使用。
  • 为每个帐户设置离线和磁盘空间个性设置。
  • 选择要离线查看的文件夹和新闻组。

下载收件箱供离线使用

可以让 Netscape 邮件和新闻组自动下载“收件箱”邮件以供离线使用。随后，当返回到在线状态时，Netscape 邮件和新闻组会自动使“收件箱”邮件和服务器同步。

要自动下载“收件箱”供离线使用，从“邮件”窗口开始：

  1. 打开“编辑”菜单并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。
  2. 在对话框的左边，在要离线使用的帐户的名称下面，选择“离线和磁盘空间”。（此类别不适用于 POP 帐户。）
  3. 选中“让我在离线工作时仍可访问收件箱中的邮件”复选框。
  4. 单击“确定”。
  5. 在文件夹窗格中，在想离线使用的帐户名字下面，单击除了“收件箱”之外的任何文件夹。例如，单击“发件人”或“草稿”文件夹。然后单击“收件箱”以开始下载其中的邮件。

关于互联网邮件访问协议（IMAP）

优点：邮件及对它们的任何改变均保存在服务器上，从而可节省本地磁盘空间。另外，您总是有权访问更新的邮箱，还可以从多个不同的位置获取邮件。调制解调器的表现会比较快一些，因为一开始只下载邮件的标题。

缺点：并非所有的 ISP 都支持 IMAP。

[ 返回到本节开始处 ]

 

关于邮局协议（POP）

优点：邮件立即被下载到本地计算机，但也可以指定是否在服务器上保存邮件的副本，以及当本地邮件被删除时是否同时也在服务器上删除之。目前大多数 ISP 都支持 POP。

缺点：如果使用多台计算机，邮件将保存在其中一台而不是所有的计算机内。在速度慢的连接上，POP 效能不如 IMAP。另外，不能从多个不同地点访问所有的邮件夹。

IMAP 服务器设置

如果还没看过 IMAP 服务器设置，从“邮件”窗口开始：

  1. 打开“编辑”菜单并选择“邮件和新闻组帐户设置”。即可看到“邮件和新闻组帐户设置”对话框。

  • 服务器类型：创建此帐户时指定的服务器类型（IMAP 邮件服务器）。要改变与帐户的关联的服务器类型，必须删除帐户，然后重新建立。
  • 服务器名：在创建此帐户时指定的服务器名。如果从一个帐户上获取邮件时碰到问题，请向服务供应商或系统管理员验证所输入的服务器名是否正确。
  • 用户名：创建此帐户时指定的用户名。
  • 端口：除非您的服务供应商或系统管理员有特别的指示，否则不要改变这个设置。

关于证书

“邮件和新闻组帐户设置”中的“安全性”窗格的主要用途是选择两个证书：

  • 用于签署发送给其他人的邮件的电子邮件证书。
  • 其他人发送加密邮件给您时使用的电子邮件证书

根据颁发证书的认证中心（CA）的策略，您可以为两种用途使用一个证书或使用两种不同的证书。即使您只使用一个证书，也必须指定两次，一次用于数字签名，一次用于加密。

这里选择的证书将包含在您发送的每个签名邮件中。这些证书使您的收件人可以验证您的数字签名以及对发送给您的邮件进行加密。

 

数字签名

使用在安全性窗格上的“数字签名”区域来指定如何签署电子邮件。

  • 数字签名邮件：如果想对发送的所有邮件都进行数字签名则选择这个复选框。（在选择这个复选框之前 必须指定一个个人证书。）
  • 用此证书对您发送的邮件进行数字签名：如果这个字段为空或显示错误的证书，则须单击“选择”以便从您文件中的证书中挑选一个。

要改变“编写”窗口中正在编写的邮件的数字签名设置，单击靠近窗口顶部的“安全性”图标下面的箭头，然后选中或取消选中“此邮件需数字签名”。有关细节，请参见签署和加密一个新邮件。

 

加密

使用“安全性”窗格中 的“安全性”区域来指定在发送邮件时通常需要如何时使用加密：

  • 从不：如果从不使用加密或只是偶尔使用加密，则选择此项。
  • 需要加密：如果总是使用加密则选择此项。如果没有所有必需的证书，则不能发送邮件，除非特地针对这样的邮件关闭加密。
  • 使用这个证书加密和解密发送给您的邮件：如果这个字段为空或显示为错误的证书，须单击“选择”以便从文件中的证书中挑选一个。

邮件和新闻组个性设置 - 发送格式

“发送格式”个性设置用来指定出站邮件使用的格式。如果还没有看到“发送格式”的设置，按下列步骤做：

  1. 打开“编辑”菜单，然后选择“个性设置”。将出现“个性设置”对话框。
  2. 在“邮件和新闻组”类别下，单击“发送格式”。（如果尚未见到子类别，双击“邮件和新闻组”即可展开子类别列表。）
  • 询问我该如何操作：这个选项要求“邮件和新闻组”提示您在发送邮件之前选择一种格式。
  • 把邮件转换为纯文本：这个设置可能导致邮件丢失诸如粗体文本这样的格式。
  • 无论如何都要以 HTML 格式发送：如果选择此项，切记：有些邮件程序可能在显示邮件时会碰到问题。
  • 以纯文本和 HTML 两种格式发送：这个选项会使用更多的磁盘空间。

  • 可以随时使用“邮件编写”窗口中的“选项”菜单来为个别邮件覆盖这些个性设置。

  • 接受 HTML 和纯文本的网域：如果知道哪些网域可以显示 HTML 格式的邮件以及哪些网域只能显示纯文本格式的邮件，则可使用“添加”按钮来添加作为发送邮件典型目标的域名。

例如，如果经常将邮件发送到具有相同域名的多个收件人（例如，您同事的电子邮件地址都以 “netscape.net” 结尾），而且您知道该域名可以显式 HTML 邮件，那么，就可以将 netscape.net 域添加到“HTML 域”列表中，这样，邮件程序将会自动以 HTML 格式将邮件发送给这些收件人。

同样，如果经常发送邮件到只能接收“纯文本”邮件的域的收件人，可以将该域名添加到“纯文本”域列表中。这样，邮件程序将会自动以纯文本格式将邮件发送到该域。

备注：如果经常编写 HTML（格式化的）邮件，请记住，有时并非所有的收件人都使用可正确显示 HTML 格式的邮件程序。“发送格式”个性设置可以用来针对发往那些不能显示 HTML 格式邮件的收件人的邮件指定如何对之格式化。可以将邮件转换为纯文本，只格式化为 HTML ，或既格式化为 HTML又格式化为纯文本。这些个性设置适用于所有邮件帐户，但只适用于电子邮件，而不适用于新闻组邮件。

向通讯录添加一个收件人或地址卡片时，可指定该收件人是否可以接收 HTML 格式的邮件。无论如何，当无法获得这样的信息时，可以通过“发送格式”个性设置来指定“邮件和新闻组”为这些邮件使用什么格式。

邮件和新闻组个性设置 - 寻址

寻址个性用于控制 Netscape 邮件和新闻组的通讯录的设置。（例如，邮件地址收集和自动填写地址）。如果尚未看到“寻址”的设置，按下列步骤操作：

    备注：除“个人通讯录”外，Netscape 邮件和新闻组还提供了“收集通讯录”，它可自动存储入站邮件和出站邮件的电子邮件地址，或两种邮件的地址。对于入站邮件，“邮件和新闻组”会在每个邮件一打开时存储它们的地址。对于出站邮件，邮件程序会在单击“发送”后立刻存储其地址。您的改变要在下一次启动 Netscape 时生效。

    备注：在“个性设置”对话框输入的目录服务器设置将应用于所有帐户。通过使用“帐户设置”对话框中一个帐户的“寻址”设置，指定不同的 LDAP 目录服务器或服务器设置，可令单个帐户忽略这些设置。要为指定的帐户设置不同的地址选项，打开“编辑”菜单并选择“邮件和新闻组”帐户设置。
