互联网上的隐私权

浏览器向网站提供了哪些信息？

每当您的浏览器显示一个网页时（例如，您每次单击一个链接或输入一个 URL 时）或每当一个网页在电子邮件中显示时，浏览器会向网页的网站提供某种信息。此类信息可能包括（但不限于）您的操作环境、互联网地址和您来自的网页。

 

操作环境

网站被告之有关您的操作环境的信息，如浏览器类型和操作系统等。这有助于该网站以最佳方式在您的屏幕上呈现页面。例如，网站可能得知您所使用的是 Windows 2000的计算机和法语版本 Netscape 6.2.3。

 

互联网地址

您的浏览器必须向网站告知您的互联网地址（也叫互联网协议，或 IP 地址），这样该网站就知道将您所请求的页面发送到何处。除非网站知道您的 IP 地址，否则无法发送您希望看到的页面。

您的 IP 地址既可是临时的，也可为固定的（静态的）。

如果是通过与电话线相连的标准调制解调器连接到互联网，那么互联网服务供应商 (ISP) 可在每次登录上网时分配给您一个临时的 IP 地址。您在互联网会话期间使用这个临时的 IP 地址，直到注销或挂断拨号连接，或中断计算机与互联网的在线连接时为止。每个 ISP 都有很多 IP 地址并将这些地址随机分配给各个用户。

如果拥有 DSL，电缆调制解调器，或光纤连接，您在每次连接上网时就可拥有一个固定的 IP 地址。

您的 IP 地址不同于您的电子邮件地址。

 

转荐页面

当单击一个链接查看网站的一个页面时，该网站也会得知您正在阅读哪个页面。这样该网站就可得知您是由哪个网站转荐过来的。或者当您浏览网站时可得知您来自该网站的哪些页面。

 

Cookie 是什么，如何工作？

cookie 是某些网站使用的一小块信息。当访问一个使用 cookie 的网站时，该网站可能会要求在浏览器在硬盘上放置一个或多个 cookie。

此后，当您返回到同一网站时，您的浏览器会将属于该网站的 cookie 发送回去。

若使用默认的 cookie 设置，上述活动是看不见的，您不会知道网站何时设置一个 cookie 或何时将一个网站的 cookie 发送回去。然而，您可以通过个性设置，要求在放置 cookie 前询问。有关具体操作，请参阅设置隐私级别。

 

网站是如何使用 Cookie 信息的？

Cookie 可使一网站得知您先前访问的信息。例如，如果您在一个网站只搜索有关当地天气和购买图书的信息，该网站可使用 cookie 记住您所居住的城市或喜欢的作家，这样就可使您的再次访问变得更简单、更实用。

很多网站都发布隐私权政策，说明其如何使用接收到的信息。

您可以查看网站的隐私权政策，了解该网站收集的信息及其使用情况，以及就该信息的使用您拥有哪些选择权。有关具体操作，参见查看隐私权政策。

有关如何在已发布的隐私权政策的协助下管理 cookie 的信息，请参阅设置隐私级别。

 

什么是“第三方 Cookie”？

如果浏览器存储了某个网站的 cookie，那么它只能将该 cookie 返回到特定的同一网站。浏览器不会向一个网站提供另一网站放置的 cookie。由于一个网站只能接收其本身的 cookie，因此只能获悉您在该网站活动时的信息，而不能得知您在上网时的全部信息。

但有时一网站也可显示驻留在另一网站的内容。 这些内容可涉及从图像到文本或是广告的任何信息。驻留这些内容其他网站也能够在您的浏览器上存储 cookie，即使您不直接访问那些网站。

这些您未访问过的网站所储存的 Cookie 称作第三方 cookie 或外来 cookie。网站有时会通过透明 GIF 使用第三方 cookie，“透明 GIF”是一种能协助网站计算用户数量、追踪电子邮件应答和详细了解访问者如何使用网站的特殊图像。（“透明 GIF”也称作网络信标或网络窃听器。）

如需要，可调整您的 cookie 个性设置，使网站可储存普通 cookie，而不储存第三方 cookie。

 

如何控制电子邮件中的网页？

可以完全禁用作为电子邮件的一部分而接收的网页中的 cookie、图像、JavaScript 和插件。

尽管在浏览网页时启动这些功能的部分或全部是十分便利，但没有必要以单一网页的形式作为邮件的附件发送。

有关启动或禁用电子邮件中的 cookie、图像、JavaScript 和插件的信息，请参阅下列各节：

  • 隐私与安全个性设置 - Cookie
  • 隐私与安全个性设置 - 图像
  • 高级个性设置 - 脚本和插件

 

如何确认未经授权的人无法使用有关我的信息？

保护您的信息隐私权的最佳途径是阅读所访问网站和所使用的互联网服务的隐私权政策，并在上网时保持谨慎，防止泄露个人信息。

互联网是一个公共网络。当您通过网络（通过网页、电子邮件或其他方式）发送您的姓名、电话号码和其他个人信息时，有可能被其他人拦截。

以下是您可能问到的有关网站隐私权政策的一些问题：

  • 此网站收集的是哪类个人信息？
  • 网站是如何使用这些信息的？
  • 网站是否会与其他方共享信息，对任何共享信息的使用我是否有选择权？
  • 我能否访问一个网站收集到的有关我的部分或全部信息以对其进行检查或更新？
  • 网站如何保护这些信息？
  • 如果有疑问或问题如何与网站联络？

有关网站隐私策略的信息，请参阅查看隐私权政策。
