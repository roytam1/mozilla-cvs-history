Netscape 键盘快捷方式

Netscape 允许使用键盘代替鼠标执行许多命令。使用键盘执行命令通常可以节省时间，并且对于那些身体，视觉和听觉有问题的人们，快捷方式可以帮助他们方便的使用 Netscape。

执行命令时，同时按下列在一起的键即可。例如，在 Windows 系统中执行复制命令，同时按下 Ctrl 键和 C 键。

提示：加号（+）表示您必须同时按下加号两边的两个键。如果多于一个的加号（例如 Ctrl+Shift+C），表示同时按下加号周围的三个键。

缩写：

  • Ctrl = Control 键
  • Cmd = Macintosh 系统键盘上的命令键

注意：一些键盘快捷方式会因光标位置（焦点）的不同而执行不同的功能。例如，在 Windows 系统上浏览网页时按下 Home 键，Netscape 将把浏览器中的内容将移动到网页的顶部。可是，如果 Windows 系统中光标在某文本区域时按下 Home 键，光标将跳到文本的开始处。

提示：在 Windows 系统和 Linux 系统中，可以使用键盘访问菜单选项。要打开顶层菜单，按下 Alt 键和菜单名中带下划线的字母。打开一个菜单后，可以不按 Alt 键打开下面的子菜单和命令。例如，可以按 Alt+F 打开“文件”菜单。
