什么是“我的快捷栏”？

“我的快捷栏”是浏览器中的一个可自定义区域，在那里可以保存总会用到的项目 - 最新的新闻和天气预报、您的地址簿或“好友”列表、股票报价、日历 - 以及很多其他的可选取项目。“我的快捷栏”在选项卡中列出并不断更新这些项目。

Netscape 已设置了一些“我的快捷栏”选项卡，但您可以通过添加、移出和重新安排选项卡来自定义“我的快捷栏”。

使用“我的快捷栏”

除非您已经关闭了“我的快捷栏”，否则它会一直在浏览器的左侧打开。要查看选项卡：

  • 须单击一个选项卡的标题；例如单击“搜索”一词。“搜索”选项卡即被打开并可用于搜索网页。

我的快捷栏
缩放柄

若没看到“我的快捷栏”，可能已关闭。要打开“我的快捷栏”：

  1. 沿 Netscape 窗口的左手边框上下移动鼠标指针。鼠标指针在接接触到“我的快捷栏”的“缩放柄”时会变成一个手的形状。
  2. 单击该缩放柄打开“我的快捷栏”。

    备注：若未从浏览器的“查看”菜单（在“显示／隐藏”子菜单下）选择“我的快捷栏”，将不会看到该快捷栏的边框和缩放柄。

提示：要重新载入“我的快捷栏”选项卡，须用鼠标右键单击选项卡标题，并从弹出菜单中选择“重新载入”。

添加“我的快捷栏”选项卡

若需添加新选项卡，请按下列步骤操作：

  1. 单击“我的快捷栏”顶部的“选项卡”，从菜单中选择“自定义我的快捷栏”。
  2. 在“自定义我的快捷栏”对话框中，从左侧列表中选择一个选项。单击文件夹将其打开或关闭。
  3. 单击“添加”。
  4. 如有必要，继续添加其他选项卡。
  5. 单击“确定”完成添加。

备注：若在“我的快捷栏”中添加了八个以上的选项卡， Netscape 会隐藏多余的选项卡以防混乱。要滚动查看隐藏的选项卡，可单击“我的快捷栏”底部的下箭头按钮直到看到所需选项卡。单击上箭头按钮可再次向上滚动。

提示：

  • 要在添加前预览“我的快捷栏”选项卡，可从“自定义我的快捷栏”对话框左侧的列表中选择一个选项卡，单击“预览”即可。几秒钟后，选项卡就会显示在弹出的“选项卡预览”窗口中。
  • 要查看“我的快捷栏”提供的全面、分类的选项卡列表，可单击“我的快捷栏”顶部的“选项卡”，选择“我的快捷栏目录”。

也可将“我的快捷栏”选项卡在开启与关闭之间切换。

  1. 单击“我的快捷栏”顶部的“选项卡”。当前选项卡在菜单下部列出。
  2. 选择希望在“我的快捷栏”中显示的选项卡。去掉对钩（不选中）以关闭一个选项卡（但仍能从菜单中使用）。

提示：要快速关闭一个“我的快捷栏”选项卡，可用鼠标右键单击其名称然后选择“隐藏选项卡”。

自定义单个“我的快捷栏”选项卡

备注：不是所有选项卡都能自定义。

  1. 单击“我的快捷栏”顶部的“选项卡”，从菜单选择“自定义我的快捷栏”。
  2. 从右侧列表中选择一个可用的选项卡。
  3. 如果“自定义选项卡”已启用，单击该选项。会出现一个窗口，显示有关自定义选项卡的信息和选项。

    指令会根据选项卡的来源不同有所改变，包括 Netscape，选项卡供应商可以是任何公司、组织或使用互联网的个人。

  4. 执行选项卡供应商的指令后，应关闭自定义窗口（或按供应商的指令关闭。）
  5. 单击“确定”完成操作。

什么是“通过选项卡浏览”？

“通过选项卡浏览”可使您在单一窗口打开一个以上的网页。每个网页在单一的浏览器窗口上方都有一排自己的选项卡。每个选项卡都出现在“选项卡栏”上。例如，可在一个窗口内访问 netscape.com、icq.com 和 cnn.com，而无须打开三个窗口。

在访问几个网页时无须打开多个的窗口，因此就可以释放更多空间到您的桌面。相反，您无须转换到另一窗口就可以很方便地在同一位置打开、关闭和重新载入网页。

设置“通过选项卡浏览”

有很多方法可自定义“通过选项卡浏览”。例如，可以改变您的个性设置，使得能够从“地址栏”打开新的浏览器选项卡。也可以用其他方式设置“通过选项卡浏览”，例如在后台载入新的浏览器选项卡，这样在载入第二个页面时第一个页面保持在最前面。要了解更多有关“通过选项卡浏览”的信息，请参阅浏览器个性设置 - 通过选项卡浏览。

打开选项卡

可用下列方法打开一个浏览器选项卡：

打开一个新的空浏览器选项卡：

  • 从“文件”菜单中打开：打开“文件”菜单，选择“新建”，再选择“新建浏览器选项卡”。
  • 从“选项卡栏”中打开：如该栏可见，单击“选项卡栏”左侧的“新选项卡”图标。
  • 从弹出菜单中打开：若“选项卡栏”可见，可用鼠标右键单击该栏，从弹出菜单中选择“新选项卡”。

在浏览器选项卡中打开一个网页链接：

  • 从弹出菜单中打开：用鼠标右键单击一个网页连接，选择“在新选项卡中打开链接”。（用鼠标右键单击 Mac OS，按下 Ctrl 键并单击鼠标键。）
  • 从“地址栏”打开：在“地址栏”输入一个网页地址，并按下 Ctrl+Enter。（在 Mac 中须按 Cmd+Enter）
    备注：必须设置“通过选项卡浏览”个性设置，使得可从“地址栏”打开一个浏览器选项卡。更多信息请参阅浏览器个性设置 - 通过选项卡浏览。

提示：

  • 要快速打开一个新的空浏览器选项卡，按下 Ctrl+T。
  • 要重新载入一个或所有的浏览器选项卡，可用鼠标右键单击“选项卡栏”任何位置，分别选择“重新载入选项卡”或“重新载入所有选项卡”。

为选项卡创建书签

要为当前窗口中的那一组浏览器选项卡创建书签（即添加一个组书签）：

  1. 打开“书签”菜单，选择“将此组选项卡存入书签”。
  2. 在“名称”字段中输入一个组书签名。
  3. 选择下列任何选项：
    • 在以下文件夹中创建：选择一个用于创建您的“组书签”的文件夹。
    • 新文件夹：单击此按钮创建一个用于创建您的“组书签”的新文件夹。
    • 使用缺省文件夹：单击此按钮将“组书签”存入指定的“新书签”文件夹。有关指定您的“新书签”文件夹的详细信息，请参阅整理您的书签。
  4. 单击“确定”添加“组书签”。

改变缺省字体

通常，网页以浏览器设置的缺省字体或以网页作者选择的字体显示。

要改变缺省字体：

  1. 打开“编辑”菜单，选择“个性设置”。
  2. 在“外观”类别下，选择“字体”。（若在该类别中未显示任何选项，双击“外观”即可展开选项列表。）
  3. 从“字体用于”下拉菜单选择一个字符集。例如，要将缺省字体设置为“西欧语族”（roman）字符集，可选择“西欧语族”。
  4. 选择比例文本是否应为 serif（如 Times Roman）或 sans-serif（如 Arial）。然后指定希望用于比例文本的字体大小。
  5. 如果您的语言字符集中有合适的字体，为 Serif 体、Sans-Serif 体、草体、浮雕体和等宽字体选择字体。还可指定希望用于等宽字体的字体大小。
  6. 指定缺省字体应为 serif 还是 sans serif。
  7. 选择一种固定宽度的字体和大小。某种类型的文本，比如方程式和公式，是以固定宽度的字体显示的。

很多网页作者选择自己的字体和字符大小。要使用非个性设置中指定的字体，可选定“允许文档使用其他字体”。

要调整字体可读性，可从下拉列表中为网页显示选择每英寸的点数（dpi）。选择“其他”，打开“校准分辨率”对话框，即可通过测量显示在屏幕上的一条线的长度来校准分辨率。增加屏幕分辨率会提高文本某些屏幕的可读性。

改变缺省颜色

通常，网页背景和文本颜色由浏览器或网页作者设置的缺省颜色决定。

要改变缺省颜色：

  1. 打开“编辑”菜单，选择“个性设置”。
  2. 在“外观”类别下，选择“颜色”。（若在该类别中未显示任何选项，双击以展开选项列表。）
  3. 单击“文本”、“背景”、“未访问链接”和“已访问链接”边上的色块。从彩色框中为每个选项选择一种颜色。也可指定链接应加下划线。

大多数网页作者选择自己的颜色。可通过选择“使用我所选的颜色，忽略指定颜色”来替换作者的创意。

在查看网页源代码时，可通过选择“启用语法突出显示功能”看到以特定颜色突出显示的网页源代码的 HTML 语法。

改变主题

可通过改变主题更改 Netscape 的外观和感觉。

  1. 打开“编辑”菜单，选择“个性设置”。
  2. 在“外观”类别下，选择“主题”。
  3. 单击以选择希望使用的主题。
  4. 单击“确定”确认您的选择。

    备注：在应用了新主题后，要使此改变生效，须退出并重新启动 Netscape。

快捷方式：也可打开“查看”菜单，选择“应用主题”，然后选择希望使用的主题，即可改变主题。

个人工具栏

“个人工具栏”可以全部自定义，您可以决定希望保留在此栏的全部内容。“个人工具栏”的一些按钮已经启用，包括“我的 Netscape 页面”、“您的主页”、“搜索”和“邮件”。

个人工具栏

您可以轻易在“个人工具栏”上添加、删除和重新排列工具栏项目。

开启和关闭按钮

  1. 打开“编辑”菜单，选择“个性设置”。
  2. 单击“浏览器”。
  3. 在“选择需在工具栏中显示的按钮”下选择希望在工具栏中看到的按钮。
  4. 单击“确定”。


添加个人工具栏书签

您可以为最喜欢的书签或包含组书签的文件夹添加按钮。要创建一个新书签添加到“个人工具栏”中：

  1. 打开希望添加书签的网页。
  2. 将书签图标 （位于“地址栏”中 URL 的左侧）拖动到“个人工具栏”中的希望位置。可将该图标直接拖到“个人工具栏”或“个人工具栏”上的一个文件夹中。有关在“个人工具栏”中添加书签文件夹的详细信息，请参阅添加个人工具栏书签文件夹。

备注：如果在个性设置中选择了“显示网站图标”的话，书签图标 则可能以另一种特定页面图标的形式出现。有关更改此个性设置的详细信息，请参阅外观个性设置 - 外观。

“个人工具栏”文件夹的每个项目均作为工具栏按钮出现。要看到全部项目，可能需要扩大浏览器窗口。

添加个人工具栏书签文件夹

可以在“个人工具栏”中添加书签文件夹，用于分类整理您最喜爱的书签。例如，可以在“个人工具栏”中拥有一个与爱好相关的书签文件夹和另一个与工作有关的书签文件夹。要在“个人工具栏”中添加一个新书签：

  1. 打开“书签”菜单，选择“管理书签”。
  2. 选择您指定的“个人工具栏文件夹”。
  3. 单击工具栏上的“新建文件夹”。
  4. 输入新书签文件夹的名称。缺省名为“新建文件夹”。
  5. 单击“确定”确认新书签文件夹名。

新书签文件夹会出现在“个人工具栏”尾端。


指定一个“书签文件夹”作为“您的个人工具栏文件夹”。


  1. 打开“书签”菜单，选择“管理书签”。
  2. 选择希望其中的项目出现在工具栏的书签文件夹。
  3. 从“查看”菜单选择“设置为个人工具栏文件夹”。

现在您的“个人工具栏”上的按钮就与指定的文件夹中的书签相符合了。


从“个人工具栏”中移除书签


  1. 打开“书签”菜单，选择“管理书签”。
  2. 单击“个人工具栏文件夹”。
  3. 选择希望删除的书签或文件夹。
  4. 按下键盘上的 Delete 键。
  5. 关闭“管理书签”窗口。

提示：要快速移除位于“个人工具栏”（而不是文件夹）中的书签，用鼠标右键单击书签，然后选择“删除”。


重新安排“个人工具栏”


  1. 打开“书签”菜单，选择“管理书签”。
  2. 在“书签”窗口，单击“个人工具栏文件夹”。
  3. 选择一个书签或文件夹，并将其拖到一新位置。
  4. 结束重新安排项目后，关闭“书签”窗口。

备注：“个人工具栏”上的标准按钮，如“搜索”和“主页”按钮，不能被重新安排，但可在开启与关闭之间切换。。

提示：要快速移动“个人工具栏”上的书签，可单击并拖动该书签到“个人工具栏”上的另一位置或拖至一个文件夹中。

状态栏

“状态栏”位于任何 Netscape 窗口底部。它包括下列各部分：

  • 组件栏 用于在各组件之间转换。更多信息请参阅状态栏。
  • 状态信息：象网页 URL 的那样显示信息，并载入状态信息。
  • Cookie 通知图标 ：当一个网站以一种使您得到通知的方式使用 cookie 时，此图标出现。更多信息请参阅 Cookie 通知。
  • “离线工作” 或“在线工作” 图标：单击此图标在离线工作和在线工作间切换。离线工作可防止 Netscape 连接到互联网，去进行诸如载入网页图像或自动查看邮件之类的工作。
  • 锁定图标（例如：）：可显示页面的全部信息是否在被您的计算机接收时加密。更多信息请参阅查看网页安全性。

什么是书签？

书签是访问您最爱和最常访问网站的快捷方式。您只须创建书签而无须输入很长的 URL（网址）就可直接访问您希望看到的页面。

可通过“书签”菜单，“我的快捷栏”上的“书签”选项卡和“管理书签”窗口找到您的书签。可为您最喜欢的网页创建书签，也可采用任何您希望的方式整理书签列表，从而控制书签列表。

 

使用书签

Netscape 本身自带一些可用的书签。要使用一个书签：

  1. 打开“书签”菜单。
  2. 从列表或列表中的文件夹中选择一个书签。

创建新书签

可以为您的最喜欢的网页创建书签，以便今后能很容易地再找到这些网页。

要为当前网页创建书签，须执行下列步骤之一：

  • 要在“书签”菜单中添加一个书签，须打开“书签”菜单，选择“将此页面加入书签”。
  • 要将书签添加到“书签”菜单的一个特定文件夹中，或要为您的书签提供一个特定名称或 URL：
    1. 打开“书签”菜单，选择“保存书签”。选择下列任何选项之一：
      • 名称：若希望有不同名称，输入一个书签名。
      • 位置：若希望有不同的 URL，输入一个书签 URL。
      • 在以下文件夹中创建：选择用于创建书签的文件夹。
      • 新文件夹：单击此项创建一个用于放置书签的新文件夹。
      • 缺省文件夹：单击此项将书签存入指定的“新书签文件夹”中。
    2. 单击“确定”添加书签。

    提示：如果在一个窗口打开了多个浏览器选项卡，可选择“将此选项卡组存入书签”，就会添加一个可以打开所有当前窗口选项卡的书签。

  • 要在“个人工具栏”中添加一个书签，须将“地址栏”旁边的书签图标 拖动到“个人工具栏”中的某个位置上。可将书签拖到下列位置：
    • “个人工具栏”的“书签”文件夹中。
    • 您在“个人工具栏”创建的书签文件夹中。
    • “个人工具栏”中所有书签文件夹的右侧。

    有关在“个人工具栏”中添加书签的详细信息，请参阅“添加“个人工具栏”书签”。

个人工具栏
  • 要在“我的快捷栏”的“书签”选项卡中添加一个书签，可打开“我的快捷栏”，选择“书签”选项卡，并将“地址栏”旁边的书签图标 拖到“书签”选项卡的书签列表中的某个位置上。

备注：

  • 如果您在个性设置中选择了“显示网站图标”的话，书签图标 可能以另一种特定页面图标的形式出现。有关更改此个性设置的详细信息，请参阅外观个性设置 - 外观。
  • 在使用上述任何方法添加了一个书签后，就可通过“我的快捷栏”中的“书签”选项卡、“管理书签”窗口和“书签”菜单使用书签了。

整理书签

要整理您的书签，须打开“书签”菜单，选择“管理书签”。在“管理书签”窗口中执行下列任何任务。

提示：可从“我的快捷栏”中的“书签”选项卡打开“管理书签”窗口。单击“书签”选项卡顶部的“管理”选项。

查看文件夹内的书签：

  • 双击文件夹查看其内容。

  • 将希望移动的书签或文件夹拖到新位置。要将书签放置到一个文件夹中，只须将其拖到该文件夹。

创建新文件夹或分隔符：

  • 单击“书签”窗口顶部的”新建文件夹”或“新建分隔符”。新文件夹或新分隔符会出现在当前选择之下。

从列表移除书签或文件夹：

    提示：要添加更多的列，可打开“查看”菜单，再打开“显示列”，在列表中选择一个列标题。


指定一个新书签文件夹

  1. 打开“书签”菜单，选择“管理书签”。
  2. 在您的“书签”窗口，选择一个用来放置新书签的文件夹。
  3. 打开“查看”菜单，选择“设置为新书签文件夹”。

改变个人书签

您可以改变任何个人书签信息。

  1. 打开“书签”菜单，选择“管理书签”。
  2. 在“书签”窗口，单击一个书签。
  3. 单击“属性”。
  4. 在书签“属性”对话框，单击“信息”选项卡。

可以对书签进行重命名（名称出现在书签列表中）、添加描述信息或设置一个密码。（您可以在位置字段输入书签密码，访问书签链接的站点。）

也可以设置 Netscape，使其查看书签链接的网站的更新情况。

  1. 打开“书签”菜单，选择“管理书签”。
  2. 在“书签”窗口，单击一个书签。
  3. 单击“属性”。
  4. 在书签“属性”对话框，单击“时间表”选项卡。
  5. 使用下拉菜单，指定 Netscape 多久查看一次书签链接页面的更新情况。
  6. 若希望在书签链接页面有变动时得到通知，可单击“通知”选项卡，选择一个通知选项。

搜索书签

要搜索书签列表：

  1. 打开“书签”菜单，选择“管理书签”。
  2. 在“书签”窗口，打开“工具”菜单，选择“搜索书签”。即可看到“查找书签”对话框。
  3. 在下拉列表中，选取选项以定义搜索，然后单击“查找”。与搜索标准匹配的书签就会显示出来。从下列搜索选项中选择：
    • 若只知道要搜索项的部分单词或词组，可选择“包含”、“开头是”或“结尾是”进行搜索。
    • 若知道要搜索项的确切名称，可选择“是”。
    • 选择“不是”或“不包含”排除不需要的页面。
    • 单击要填写的字段，输入希望找到或排除的书签或历史列表的名称（全部或部分）或URL（网址）。
    • 选择“在书签中保存查询”保存此搜索，供今后之用。
  4. 双击列表中的书签并转至该页面。

提示：若很难读列表，可尝试扩大搜索结果窗口。

导出或导入“书签列表”

您的书签存储在命名为 bookmarks.html 的文件中。可导出此文件的一个副本，并将其保存在您所选择的一个文件夹中。然后您就可编辑它，像对任何 HTML 文件一样地进行处理。

  1. 打开“书签”菜单，选择“管理书签”。
  2. 在“书签”窗口，打开“工具”菜单，然后选择“导出”。
  3. 在“导出书签文件”对话框，选择一个文件夹。 bookmarks.html 文件即被保存在这个指定的文件夹中。
  4. 单击“保存”。

您的 Netscape 书签不会因此操作程序而被更改。

也可以从其他来源导入书签文件。例如，可以从早期的 Netscape 版本，其他浏览器或从朋友发来的书签文件中导入。

在开始前，应确认希望导入的书签文件是一个 HTML 文件。

  1. 打开“书签”菜单，选择“管理书签”。
  2. 在您的“书签”窗口，打开“工具”菜单，然后选择“导入”。
  3. 在“导入书签文件”对话框，找到并选择希望导入的书签文件。
  4. 单击“保存”。

导入的书签文件会被作为一组新书签对待，并被添加到书签列表的底部。如果已指定了一个新书签文件夹，导入的书签则会被添加到该文件夹中。

备注：导入一个书签文件就是从该文件导入书签和文件夹。不会创建两个书签文件。

指定起始页

您可以指定浏览器启动时希望载入的页面：

  1. 打开“编辑”菜单，选择“个性设置”。
  2. 在“浏览器启动时显示”下，从空白页、您的主页还是最后访问的网页中选择一种在浏览器启动时希望自动启动页面。

    备注：如果选择了“主页”，须在“位置”字段输入它的 URL。

[ 返回到本节开始处 ]

 

更改您的主页

您的主页是指您单击“个人工具栏”上的“主页”按钮时打开的页面。依照不同的个性设置，主页也可以是您在启动 Netscape 时自动打开的页面。

指定您的主页：

  1. 打开“编辑”菜单，选择“个性设置”。
  2. 单击“浏览器”类别。
  3. 在“主页”部分，执行下列步骤之一：
    • 在“位置”字段输入您的主页 的 URL（网址）。
    • 单击“使用当前页”使当前在浏览器窗口中显示的页面成为您的主页。
    • 单击“选择文件”从您计算机的硬盘上选择一个文件。

提示：要快速指定您的主页，可将书签图标 从“地址栏”拖至“个人工具栏”上的“主页”按钮即可。

  3. 选择希望每次启动 Netscape 自动打开的组件。
