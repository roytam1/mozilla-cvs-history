Hiding the Introduction Screens

If you don't want the opening screen of Account Setup to display the Internet Preview, the Communicator Preview, or the Set Up Communicator screens, you can hide these screens so that they don't appear.

To remove the Introduction screens, follow these steps:

  1. Using the Account Setup navigation pane, move back to the first screen of Account Setup.
  2. Click Screen Options in the Toolbar.
  3. Uncheck the Show Account Setup Introduction Screens checkbox.
