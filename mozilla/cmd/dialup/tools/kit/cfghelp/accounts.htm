Start Account Setup

This is the first screen of the main part of Account Setup. It lets users choose whether they want to:

  • Create a new account--Account Setup runs in Internet Account Server (IAS) mode-- or
  • Set up access to an account they already have--Account Setup runs in Network Configuration (NCI) mode.

Account Setup doesn't present this screen to users if you selected either IAS mode or NCI mode in Account Setup's global options. For more information about setting global options, click [More Help], below.
