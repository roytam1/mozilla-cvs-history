Offer Your Registration Server (IAS Mode) and Offline Account Creation (NCI Mode)

This option lets users choose whether they want to use your registration server to create a new Internet account (IAS mode), or set up access to an existing account (NCI mode).

In this mode, Account Setup automatically presents the "Start Account Setup" screen, which gives users a choice of IAS or NCI mode.

Note that if you choose this option, you must have a dedicated Internet account server for users who choose the IAS mode. If you don't have such a server, choose the Network Configuration (NCI) Mode option or the Offer Netscape Registration Server and Offline Account Creation (NCI Mode) option instead.
