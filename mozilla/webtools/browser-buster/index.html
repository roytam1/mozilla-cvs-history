Browser Buster: an eyeball hour testing project

This is a sanity checking and stress testing system for the browser. The system loads
various lists of pages for as long as the browser will continue to run.

The tests can be started from any browser by visiting one of the links below.
 

test_url_25.html

  • top 100 site list - new page loaded every 25 seconds
Sorter cycles are good for flipping through lots of pages and covering lots of tests quickly.
Longer cycles are for using on dialup/slower speed connections, running under debug or
purify builds,  browsing around looking at sites and reading content, etc...
