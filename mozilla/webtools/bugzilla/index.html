This is Bugzilla: the Mozilla bug system. For more information about what Bugzilla is and what it can do, see mozilla.org's bug pages.
This is our database for tracking bugs with Mozilla and other mozilla.org projects.

Use the Netscape bug reporting form to report bugs with Netscape Navigator. If your browser uses the blue N logo, your bug report should go there. Navigator bugs reported here will be marked invalid.

For help with Netscape products, visit Netscape's help site. If the answer to your question isn't on their web site, try posting to one of their newsgroups.
