tinderbox graphs

query.cgi Wrapper script for graph.cgi, this adds UI toggles, html links, etc.
graph.cgi Core gnuplot driver script.
rawdata.cgi Subset of graph.cgi, just blasts out the data that graph.cgi is plotting. Use this for a closer look, or plot the data yourself, etc.
collect.cgi Tests report to the graph server using this script.
