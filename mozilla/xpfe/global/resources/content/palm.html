Mozilla Address Book Sync for Palm-Compatible Devices


Mozilla Address Book Sync for Palm devices lets you transfer data between Mozilla address books and your Palm-compatible device.

Note: For information on HotSync setup, consult your Palm device's documentation.

Installing Mozilla Address Book Palm Sync

Before installing Mozilla Address Book Palm Sync, make sure your Palm device contains the latest address book data. If you have updated or entered new address book data in your Palm Desktop software, you should first perform a HotSync to update your Palm device's address books with the latest data. This will insure that your latest Palm address book data will be synchronized with your Mozilla address books.

Note: After you synchronize, unfiled records in your Palm address book will get filed in the Personal category and will also get added to your Mozilla Personal Address Book. If you don't want these Palm address book records to get filed in the Personal category then you should manually file them in another category before synchronizing. (not true?)

The Mozilla Address Book Palm Sync installer utility lets your Palm device synchronize directly with your Mozilla address books by replacing the Palm Desktop Address Book Sync with Mozilla Address Book Sync.

To install Mozilla Address Book Palm Sync:
  1. Start the Mozilla Address Book Palm Sync installer:
    1. Open the Windows Start menu.
    2. Choose Programs.
    3. From the Programs menu, choose Mozilla 7.1.
    4. From the Mozilla 7.1 menu, choose Palm Tools.
    5. From the Palm Tools menu, choose Address Book Palm Sync Install.
  2. If the Palm HotSync Manager is already open, the Mozilla Address Book Palm Sync utility will prompt you to restart the HotSync Manager software.

Synchronizing With Your Palm Device

After you synchronize with your Palm device, your Palm address book will contain all the records from your Mozilla address books. Each Mozilla address book is listed as a category in your Palm address book. Your Mozilla address book will contain a new address book for each category in your Palm address book. For more information, see Field Mapping.

After synchronizing, records that exist in a Palm category and in a Mozilla address book with the same name will get duplicated. This most commonly occurs between your Palm's "Personal" category and your Mozilla Personal Address Book. You can avoid this by either deleting or re-filing records before synchronizing, or else deleting duplicate records after synchronizing.

Synchronizing With Another Mozilla User Profile
To synchronize your Palm device with a different Mozilla user profile, follow the steps under Installing Mozilla Address Book Palm Sync. If you have existing records in your Mozilla address book, syncing may result in the creation of duplicate records which you will have to manually delete.

Synchronization Guidelines

Maximum number of address books
A maximum of 15 address books can be synchronized.
  • Mozilla Address Book Palm Sync can synchronize only the first 15 address books due to the Palm device's limit of 15 categories. Before you synchronize, make sure that the total number of categories plus the total number of Mozilla address books does not exceed 15. Palm categories that have the same name as a Mozilla address book will be merged when you synchronize, so they should only be counted once.
  • If your Palm address book already contains 15 categories, you can either remove some categories before you synchronize or make sure that your Mozilla address books are named the same as the Palm categories, so that they will get merged when you synchronize.
Mozilla address book fields
Mozilla address book fields, with the exception of the Notes and Address fields, cannot contain multiple lines. When you synchronize, field values in your Palm address book will be converted to single line entries in your Mozilla address book, with each line separated by a comma.

Palm category names

Uninstalling Mozilla Address Book Palm Sync

Note: Before uninstalling Mozilla Address Book Palm Sync, make sure your Palm device contains the latest address book data. If you have updated or entered new address book data in your Mozilla address book, you should perform another HotSync to update your Palm device's address books with the latest data. This will insure that your Palm address book contains the latest data from your Mozilla address books before you uninstall Mozilla Address Book Palm Sync.

To uninstall Mozilla Address Book Palm Sync:
  1. Start the Mozilla Address Book Palm Sync uninstaller:
    1. Open the Windows Start menu.
    2. Choose Programs.
    3. From the Programs menu, choose Mozilla 7.1.
    4. From the Mozilla 7.1 menu, choose Palm Tools.
    5. From the Palm Tools menu, choose Address Book Palm Sync Uninstall.
  2. If the Palm HotSync Manager is already open, the Mozilla Address Book Palm Sync uninstall program will prompt you to restart the HotSync Manager software.
You have successfully uninstalled Mozilla Address Book Palm Sync. The Palm Desktop Address Book Sync is now enabled. You can synchronize your Palm device with your Mozilla address book again at any time by following the instructions under Installing Mozilla Address Book Palm Sync.
