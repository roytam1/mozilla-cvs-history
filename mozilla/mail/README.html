mozilla/mail

The goal of this project is to produce a stand alone mail client.

The initial work will be to take the current mozilla/mailnews FE code and collapse all the overlays, and remove all the integration with browser (links will launch the default browser), editor (except for mail compose), chat, etc.

There won't be any sidebar.

See http://bugzilla.mozilla.org/show_bug.cgi?id=90293 for some details
