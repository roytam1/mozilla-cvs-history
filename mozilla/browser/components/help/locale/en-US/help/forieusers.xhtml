About Your IE Favorites

Your IE Favorites are imported automatically. To access them, open the Bookmarks menu and choose Imported IE Favorites.

&brandShortName;'s Features

  • Tabbed Browsing: Instead of opening a separate browser window for each site you want to visit, you can open multiple sites within the same window and tab between them. You can also set a group of tabs as your home page.
  • Pop-up Window Controls: Lets you allow or suppress both popup and popunder windows.
  • Profile Manager: Create different profiles, each with its own bookmarks, preferences, and so on. This is useful if you must share &brandShortName; on the same computer with other people, or if you want to keep your work and personal settings separate.
  • Cookie Manager: Lets you change what &brandShortName; will do when accepting cookies.
  • Image Manager: Enables you to disable images from certain Web sites or disable them altogether. This is useful if you wish to decrease the amount of time it takes for Web sites to load.

Keyboard Shortcuts
