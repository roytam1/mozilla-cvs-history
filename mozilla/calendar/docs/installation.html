Mozilla Calendar Installation

Installation Instructions

  • Download and install a clean version of Mozilla 0.9.5. Presently, the calendar only works with this version.
  • Download and install libical (version 0.23 ). We've provided an RPM that installs it to /usr here.
  • Download and unzip calendar.tar.gz
  • Follow the instructions in the INSTALL file, which are:
    • Ensure that calendar and setupcalendar variables MOZ_PATH point to your install directory of Mozilla 0.9.5.
    • Edit content/penglobal/penroot.js and change "/home/mikep" to your home directory.
    • Run "./setupcalendar" from the calendar directory.
    • Launch the calendar by running "./calendar"
