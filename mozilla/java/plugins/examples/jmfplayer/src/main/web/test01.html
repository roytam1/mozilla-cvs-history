Media Player Using Java Media Framework

This small example shows how you can leverage the power of the Java Media Framework to simply and easily play media content in your browser.


Code Artifacts

The "Simple Pluglet" example, which you can access from the index page, gives more of a tutorial view of how to build a pluglet. In this example, I'll just include the source code and you can take it from there.
