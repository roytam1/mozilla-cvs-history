Localization notes for LICENSE

You must include the Netscape End User License Agreement included with this kit in your localized product.  The license file may be found at: products\client\windows\platform\en\license\export\eula\license.txt. The Localization Kit gives no information on the nature of this license agreement; it only tells you how to put a localized license into the software. Whether or not you localize this license (this may depend on your desires or upon the laws of your country), you must include this in your localized binaries by following this procedure. (This example uses Microsoft® MSDEV; if you prefer a different resource editor you may need to execute slightly different steps):
  • Open license.txt in a text editor window.
  • Choose Edit | Select All.
  • Choose Edit | Copy.
  • Open resdll.dll (make sure you have the right one!).
  • Expand the DATA section, open the LICENSE entry.
  • Choose Edit | Select All.
  • Choose Edit | Paste to paste the new text over the old:
  • Terminate the entry with a null character: type "0"

    • Save and exit.
