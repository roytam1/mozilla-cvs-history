Localization notes for SETUP.INI

General Notes:
  • Only translate the strings which are shown in bold type face below.
  • The "StartCopy Dialog Message" should be less than 256 bytes long.
  • For the "Multiple Components" section, both the key and the value may be localized (optional).
  • If the key "Multimedia Support" is localized then next section name should be identical to the localized key.
  • Change Netscape trademarks to your own product name(s).
  • Update the version number as appropriate.
  • Modify the locale code according to the ISO 639 and ISO 3166 standards. (See the Reference section for details).
