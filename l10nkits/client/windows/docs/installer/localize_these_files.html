Localize These Files

This section tells you exactly what files need to be localized. The following tables contain two columns:
  • The file to be localized column gives the specific file name. The path name temp refers to wherever you have copied the uncompressed installer files in Step 2 of the Localization Checklist. If the file or resource name is a hypertext link, further information can be found by following that link.
  • The purpose column describes the basic purpose of this file.
Table 1. Binary files to be localized
File to be localized  Purpose
rsutil.dll Installer STRINGTABLE and ICON resources. 
_isuser.dll Installer DIALOG resources.
