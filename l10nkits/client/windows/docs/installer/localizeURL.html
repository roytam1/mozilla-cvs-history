Localizing URLs

Many resources of all types in the Communicator contain Internet addresses in the form of URLs (Universal Resource Locators). Most of these point to locations within the Netscape domain; as part of the Unlimited Localization Program, you can leave these as they are, or you may change them as you desire.

Internal URLs

There are a number of internal URLs that should not be changed.  These are the strings that look like about:[some value].  These point to internally contained resources, such as the main About Communicator page.  If you localize these strings you will not be able to load these pages.  The main examples you might find are:
  • about:
  • about:plugins
  • about:fonts
For more examples and information on these internal URLs go here.
