Localization notes for CD.INI

General Notes:
  • Only translate the strings which are shown in bold type face below.
  • Update the product version number as appropriate.
  • Modify the locale code according to the ISO 639 and ISO 3166 standards. (See the Reference section for details).
  • You may need to move the placement of "%s" variables in your translation.
  • Do not exceed the length of US string for Deinstall Dialog Wait Message.
  • Change Netscape trademarks to your own product name(s).
